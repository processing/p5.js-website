# Translation Documentation

This folder contains language-specific documentation for translation contributors working on the p5.js website.

Each language has its own file in this folder. Each file contains:

- **Glossary** - agreed translations for technical and p5.js-specific terms
- **Style points** - writing and formatting conventions specific to that language
- **Technical exceptions** - code level changes required to support that language correctly

- [Chinese](./zh.md)
