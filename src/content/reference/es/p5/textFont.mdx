description: >
  <p>Establece la fuente utilizada por la función <a href="/reference/p5/text">text()</a>.</p>
  <p>El primer parámetro, <code>font</code>, establece la fuente. <code>textFont()</code> reconoce tanto un objeto <a href="/reference/p5/p5.Font">p5.Font</a> como una cadena con el nombre de una fuente del sistema. Por ejemplo, <code>'Courier New'</code>.</p>
  <p>El segundo parámetro, <code>size</code>, es opcional. Establece el tamaño de la fuente en píxeles. Esto tiene el mismo efecto que llamar a <a href="/reference/p5/textSize">textSize()</a>.</p>
  <p>Nota: El modo <code>WEBGL</code> solo admite fuentes cargadas con <a href="/reference/p5/loadFont">loadFont()</a>.</p>
