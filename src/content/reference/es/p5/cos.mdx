  <p>Calcula el coseno de un ángulo.</p>


  <p><code>cos()</code> es útil para muchas tareas geométricas en programación creativa.
  Los valores devueltos oscilan entre -1 y 1 a medida que aumenta el ángulo de entrada.
  <código>cos()</código>
  tiene en cuenta el valor actual de <a
  href="/reference/p5/angleMode">angleMode()</a>.</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      describe('A white ball on a string oscillates left and right.');
    }

    function draw() {
      background(200);

      // Calculate the coordinates.
      let x = 30 * cos(frameCount * 0.05) + 50;
      let y = 50;

      // Draw the oscillator.
      line(50, y, x, y);
      circle(x, y, 20);
    }
    </code>

    }
