description: >
  <p>Crea una luz que brilla desde un punto en una dirección.</p>

  <p>Las luces de foco son como linternas que brillan en una dirección creando un cono de luz. La forma del cono se puede controlar utilizando los parámetros de ángulo y concentración. Se pueden activar como máximo 5 luces de foco a la vez.</p>

  <p>Hay ocho formas de llamar a <code>spotLight()</code> con parámetros para establecer el color, la posición y la dirección de la luz. Por ejemplo, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0)</code> crea una luz roja <code>(255, 0, 0)</code> en el origen <code>(0, 0, 0)</code> que apunta hacia la derecha <code>(1, 0, 0)</code>.</p>

  <p>El parámetro <code>angle</code> es opcional. Establece el radio del cono de luz. Por ejemplo, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0, PI / 16)</code> crea una luz roja <code>(255, 0, 0)</code> en el origen <code>(0, 0, 0)</code> que apunta hacia la derecha <code>(1, 0, 0)</code> con un ángulo de <code>PI / 16</code> radianes. Por defecto, <code>angle</code> es <code>PI / 3</code> radianes.</p>

  <p>El parámetro <code>concentration</code> también es opcional. Enfoca la luz hacia el centro del cono de luz. Por ejemplo, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0, PI / 16, 50)</code> crea una luz roja <code>(255, 0, 0)</code> en el origen <code>(0, 0, 0)</code> que apunta hacia la derecha <code>(1, 0, 0)</code> con un ángulo de <code>PI / 16</code> radianes y una concentración de 50. Por defecto, <code>concentration</code> es 100.</p>
