description: >
  <p>Combina dos colores para encontrar un tercer color entre ellos.</p>

  <p>El parámetro <code>amt</code> especifica la cantidad de interpolación entre
  los dos

  valores. 0 es igual al primer color, 0.1 está muy cerca del primer color,

  0.5 está a medio camino entre los dos colores, y así sucesivamente. Los números negativos se establecen

  en 0. Los números mayores que 1 se establecen en 1. Esto difiere del comportamiento de

  <a href="/reference/lerp">lerp</a>. Es necesario porque los números fuera de
  del

  intervalo [0, 1] producirán colores extraños e inesperados.</p>

  <p>La forma en que los colores se interpolan depende del

  <a href="/reference/colorMode">colorMode()</a> actual.</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Create p5.Color objects to interpolate between.
      let from = color(218, 165, 32);
      let to = color(72, 61, 139);

      // Create intermediate colors.
      let interA = lerpColor(from, to, 0.33);
      let interB = lerpColor(from, to, 0.66);

      // Draw the left rectangle.
      noStroke();
      fill(from);
      rect(10, 20, 20, 60);

      // Draw the left-center rectangle.
      fill(interA);
      rect(30, 20, 20, 60);

      // Draw the right-center rectangle.
      fill(interB);
      rect(50, 20, 20, 60);

      // Draw the right rectangle.
      fill(to);
      rect(70, 20, 20, 60);

      describe(
        'Four rectangles. From left to right, the rectangles are tan, brown, brownish purple, and purple.'
      );
    }
    </code>
