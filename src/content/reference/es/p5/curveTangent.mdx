description: >
<p>Calcula coordenadas a lo largo de una línea que es tangente a una curva de tipo spline.</p>

<p>Las líneas tangentes rozan la superficie de una curva. La pendiente de una línea tangente es igual a la pendiente de la curva en el punto donde se intersectan.</p>

<p><code>curveTangent()</code> calcula coordenadas a lo largo de una línea tangente usando los puntos de anclaje y de control de la curva spline. Requiere los puntos en el mismo orden que la función <a href="/reference/p5/curve">curve()</a>. <code>curveTangent()</code> trabaja un eje a la vez. Pasar las coordenadas en x de los puntos de anclaje y control calculará la coordenada en x de un punto en la línea tangente. Pasar las coordenadas en y de los puntos de anclaje y control calculará la coordenada en y de un punto en la línea tangente.</p>

<p>El primer parámetro, <code>a</code>, es la coordenada del primer punto de control.</p>

<p>El segundo y tercer parámetros, <code>b</code> y <code>c</code>, son las coordenadas de los puntos de anclaje.</p>

<p>El cuarto parámetro, <code>d</code>, es la coordenada del último punto de control.</p>

<p>El quinto parámetro, <code>t</code>, es la cantidad para interpolar a lo largo de la curva. 0 es el primer punto de anclaje, 1 es el segundo punto de anclaje, y 0.5 es a la mitad entre ellos.</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Set the coordinates for the curve's anchor and control points.
      let x1 = 5;
      let y1 = 26;
      let x2 = 73;
      let y2 = 24;
      let x3 = 73;
      let y3 = 61;
      let x4 = 15;
      let y4 = 65;

      // Draw the curve.
      noFill();
      curve(x1, y1, x2, y2, x3, y3, x4, y4);

      // Draw tangents along the curve's path.
      fill(255);

      // Top circle.
      stroke(0);
      let x = curvePoint(x1, x2, x3, x4, 0);
      let y = curvePoint(y1, y2, y3, y4, 0);
      circle(x, y, 5);

      // Top tangent line.
      // Scale the tangent point to draw a shorter line.
      stroke(255, 0, 0);
      let tx = 0.2 * curveTangent(x1, x2, x3, x4, 0);
      let ty = 0.2 * curveTangent(y1, y2, y3, y4, 0);
      line(x + tx, y + ty, x - tx, y - ty);

      // Center circle.
      stroke(0);
      x = curvePoint(x1, x2, x3, x4, 0.5);
      y = curvePoint(y1, y2, y3, y4, 0.5);
      circle(x, y, 5);

      // Center tangent line.
      // Scale the tangent point to draw a shorter line.
      stroke(255, 0, 0);
      tx = 0.2 * curveTangent(x1, x2, x3, x4, 0.5);
      ty = 0.2 * curveTangent(y1, y2, y3, y4, 0.5);
      line(x + tx, y + ty, x - tx, y - ty);

      // Bottom circle.
      stroke(0);
      x = curvePoint(x1, x2, x3, x4, 1);
      y = curvePoint(y1, y2, y3, y4, 1);
      circle(x, y, 5);

      // Bottom tangent line.
      // Scale the tangent point to draw a shorter line.
      stroke(255, 0, 0);
      tx = 0.2 * curveTangent(x1, x2, x3, x4, 1);
      ty = 0.2 * curveTangent(y1, y2, y3, y4, 1);
      line(x + tx, y + ty, x - tx, y - ty);

      describe(
        'A black curve on a gray square. A white circle moves back and forth along the curve.'
      );
    }
