description: >
  <p>Una función que se ejecuta cuando el usuario mueve el ratón.</p>

  <p>Declarar la función <code>mouseMoved()</code> establece un bloque de código que se ejecutará

  automáticamente cuando el usuario mueva el ratón sin tener presionado ningún botón:</p>

  <code>

  function mouseMoved() {
    // Code to run.
  }

  </code>


  <p>Las variables del sistema del ratón, como <a
  href="/reference/p5/mouseX">mouseX</a> y

  <a href="/reference/p5/mouseY">mouseY</a>, se actualizarán con su valor más
  reciente

  cuando <code>mouseMoved()</code> sea ejecutada por p5.js:</p>

  <code>

  function mouseMoved() {
    if (mouseX < 50) {
      // Code to run if the mouse is on the left.
    }

  <p>  if (mouseY > 50) {
      // Code to run if the mouse is near the bottom.
    }
  }

  </p></code>

  <p>El parámetro, <code>event</code>, es opcional. <code>mouseMoved()</code> siempre se le pasa un

  objeto <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent"
  target="_blank">MouseEvent</a> con propiedades que describen el evento de movimiento del ratón:</p>

  <code>

  function mouseMoved(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>Los navegadores pueden tener comportamientos predeterminados asociados a ciertos eventos del ratón. Por ejemplo, algunos navegadores resaltan texto cuando el usuario mueve el ratón mientras presiona un botón. Para evitar cualquier comportamiento predeterminado para este evento,

  agrega <code>return false;</code> al final de la función.</p>
