Cuando se utiliza <code>random()</code> para generar números, todos los números en el rango de salida tienen la misma probabilidad de ser producidos. Cuando\n<code>random()</code> se utiliza para seleccionar elementos de una matriz, todos los elementos tienen la misma probabilidad de ser elegidos.</p>\n<p>De forma predeterminada, <code>random()</code> produce resultados diferentes cada vez que se ejecuta un boceto.\nLa función <a href=\"/reference/p5/randomSeed\">randomSeed()</a> se puede utilizar para\ngenerar la misma secuencia de números o elecciones cada vez que se ejecuta un boceto.</p>\n<p>La versión de <code>random()</code> sin parámetros devuelve un número aleatorio desde 0\nhasta 1 (sin incluirlo).</p>\n<p>La versión de <code>random()</code> con un parámetro funciona de dos maneras. Si el argumento es un número, <code>random()</code> devuelve un número aleatorio desde 0\nhasta el número (sin incluirlo). Por ejemplo, llamar <code>random(5)</code> devuelve\nvalores entre 0 y 5. Si el argumento es una matriz, <code>random()</code>\ndevuelve un elemento aleatorio de esa matriz. Por ejemplo, llamando\n<code>random(['\U0001F981', '\U0001F42F', '\U0001F43B'])</code> devuelve un emoji de león, tigre u oso.</p>\n<p>La versión de <code>random()</code> con dos parámetros devuelve un número aleatorio dentro de\nun rango dado. Los argumentos pasados ​​establecen los límites superior e inferior del rango.\nPor ejemplo, llamar a <code>random(-5, 10.2)</code> devuelve valores desde -5 hasta 10.2 (sin incluirlo).</p>"
