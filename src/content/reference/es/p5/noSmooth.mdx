description: |2-

  <p>Dibuja ciertas características con bordes irregulares (aliased).</p>


  <p><a href="/reference/p5/smooth">smooth()</a> está activo de forma predeterminada. En 2D
  mode,


  <code>noSmooth()</code> es útil para ampliar imágenes sin que se vean borrosas. Las


  funciones no afectan las formas ni las fuentes.</p>


  <p>En modo WebGL, <code>noSmooth()</code> hace que todas las formas se dibujen con bordes irregulares

  (aliased). Las funciones no afectan a las imágenes ni a las fuentes.</p>
