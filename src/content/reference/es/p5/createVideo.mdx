description: >
  <p>Crea un elemento <code><video></code> para reproducción de audio/video simple.</p>

  <p><code>createVideo()</code> devuelve un nuevo

  objeto <a href="/reference/p5/p5.MediaElement">p5.MediaElement</a>. Los vídeos se 
  muestran por

  defecto. Se pueden ocultar llamando a <code>video.hide()</code> y dibujar en 
  el

  lienzo usando <a href="/reference/p5/image">image()</a>.</p>

  <p>El primer parámetro, <code>src</code>, es la ruta del vídeo. Si se pasa una sola
  cadena, 
  
  como en <code>'/assets/topsecret.mp4'</code>, se carga un único vídeo.
  Un arreglo

  de cadenas o <em>strings</em> se puede usar para cargar el mismo vídeo en diferentes formatos. Por

  ejemplo, <code>['/assets/topsecret.mp4', '/assets/topsecret.ogv',
  '/assets/topsecret.webm']</code>.

  Esto es útil para asegurar que el vídeo se pueda reproducir en diferentes navegadores 
  con

  diferentes capacidades. Consulta

  <a
  href="https://developer.mozilla.org/en-US/docs/Web/HTML/Supported_media_formats">MDN</a>

  para obtener más información sobre los formatos admitidos.</p>

  <p>El segundo parámetro, <code>callback</code>, es opcional. Es una función
  que se llamará una vez

  que el vídeo esté listo para reproducirse.</p>

    <code>
    let video;

    function setup() {
      noCanvas();

      // Load a video and add it to the page.
      // Provide an array options for different file formats.
      // Call mute() once the video loads.
      video = createVideo(
        ['/assets/small.mp4', '/assets/small.ogv', '/assets/small.webm'],
        muteVideo
      );

      // Show the default video controls.
      video.showControls();

      describe('A video of a toy robot with playback controls beneath it.');
    }

    // Mute the video once it loads.
    function muteVideo() {
      video.volume(0);
    }
    </code>
