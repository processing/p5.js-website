description: >
  <p>Rota el sistema de coordenadas alrededor del eje x en modo WebGL.</p>

  <p>El parámetro, <code>angle</code>, es la cantidad a rotar. Por ejemplo,
  llamar a

  <code>rotateX(1)</code> rota el sistema de coordenadas alrededor del eje x por 1
  radián.

  <code>rotateX()</code> interpreta los valores de ángulo usando el

  <a href="/reference/p5/angleMode">angleMode()</a>.</p>

  <p>De forma predeterminada, las transformaciones se acumulan. Por ejemplo, llamar
  a <code>rotateX(1)</code>

  dos veces tiene el mismo efecto que llamar a <code>rotateX(2)</code> una vez. Las

  funciones <a href="/reference/p5/push">push()</a> y <a
  href="/reference/p5/pop">pop()</a>

  se pueden usar para aislar transformaciones dentro de grupos de dibujo distintos.</p>

  <p>Nota: Las transformaciones se restablecen al principio del bucle de dibujo. Llamar

  a <code>rotateX(1)</code> dentro de la función <a href="/reference/p5/draw">draw()</a>
  no hará que

  las formas giren.</p>
