description: >
  <p>Inclina el eje x- para que las formas parezcan sesgadas.</p>

  <p>De forma predeterminada, los ejes x- e y- son perpendiculares. La función <code>shearX()</code>
  transforma 
  
  el sistema de coordenadas de modo que las coordenadas x- se traducen mientras

  las coordenadas y- permanecen fijas.</p>

  <p>El primer parámetro, <code>angle</code>, es la cantidad de inclinación. Por 
  ejemplo, llamar a

  <code>shearX(1)</code> transforma todas las coordenadas x- usando la fórmula

  <code>x = x + y * tan(angle)</code>. <code>shearX()</code> interpreta los 
  valores de ángulo utilizando el

  <a href="/reference/p5/angleMode">angleMode()</a> actual.</p>

  <p>De forma predeterminada, las transformaciones se acumulan. Por ejemplo, llamar a

  <code>shearX(1)</code> dos veces tiene el mismo efecto que llamar a 
  <code>shearX(2)</code> una vez. Las

  <a href="/reference/p5/push">push()</a> y

  <a href="/reference/p5/pop">pop()</a> funciones se pueden utilizar para aislar

  transformaciones dentro de grupos de dibujo distintos.</p>

  <p>Nota: Las transformaciones se restablecen al principio del bucle de dibujo. Llamar

  a <code>shearX(1)</code> dentro de la función <a href="/reference/p5/draw">draw()</a>
  no hará

  que las formas se inclinen continuamente.</p>
