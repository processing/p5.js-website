description: >
  <p>Una función que se ejecuta una vez cuando se libera cualquier tecla.</p>

  <p>Declarar la función <code>keyReleased()</code> establece un bloque de código
  para ejecutarse

  automáticamente cuando el usuario libera cualquier tecla:</p>

  <code>

  function keyReleased() {
    // Code to run.
  }

  </code>


  <p>Las variables <a href="/reference/p5/key">key</a> y <a
  href="/reference/p5/keyCode">keyCode</a>

  se actualizarán con el valor liberado más recientemente cuando

  <code>keyReleased()</code> sea llamado por p5.js:</p>

  <code>

  function keyReleased() {
    if (key === 'c') {
      // Code to run.
    }

  <p>  if (keyCode === ENTER) {
      // Code to run.
    }
  }

  </p></code>

  <p>El parámetro, <code>event</code>, es opcional. <code>keyReleased()</code> siempre
  se le pasa un

  <a href="https://developer.mozilla.org/es/docs/Web/API/KeyboardEvent"
  target="_blank">objeto KeyboardEvent</a>

  con propiedades que describen el evento de tecla:</p>

  <code>

  function keyReleased(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>Los navegadores pueden tener comportamientos predeterminados asociados a

  ciertos eventos de tecla. Para evitar cualquier comportamiento predeterminado
  para este evento, agregue <code>return false;</code> al final

  de la función.</p>
