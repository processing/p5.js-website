description: >
  <p>Crea una luz que brilla en una dirección.</p>

  <p>Las luces direccionales no brillan desde un punto específico. Son como un sol

  que brilla desde algún lugar fuera de la pantalla. La dirección de la luz se establece

  utilizando tres valores <code>(x, y, z)</code> entre -1 y 1. Por ejemplo, establecer la dirección de una luz como <code>(1, 0, 0)</code> iluminará los objetos de <a href="/reference/p5/p5.Geometry">p5.Geometry</a>

  desde la izquierda, ya que la luz apunta directamente hacia la derecha.</p>

  <p>Hay cuatro formas de llamar a <code>directionalLight()</code> con parámetros para establecer el color y la dirección de la luz.</p>

  <p>La primera forma de llamar a <code>directionalLight()</code> tiene seis parámetros. Los primeros tres parámetros, <code>v1</code>, <code>v2</code> y <code>v3</code>, establecen el color de la luz utilizando el <a href="/reference/p5/colorMode">colorMode()</a> actual. Los últimos tres parámetros, <code>x</code>, <code>y</code> y <code>z</code>, establecen la dirección de la luz. Por ejemplo, <code>directionalLight(255, 0, 0, 1, 0, 0)</code> crea una luz roja <code>(255, 0, 0)</code> que brilla hacia la derecha <code>(1, 0, 0)</code>.</p>

  <p>La segunda forma de llamar a <code>directionalLight()</code> tiene cuatro parámetros. Los primeros tres parámetros, <code>v1</code>, <code>v2</code> y <code>v3</code>, establecen el color de la luz utilizando el <a href="/reference/p5/colorMode">colorMode()</a> actual. El último parámetro, <code>direction</code>, establece la dirección de la luz utilizando un objeto <a href="/reference/p5/p5.Geometry">p5.Geometry</a>. Por ejemplo, <code>directionalLight(255, 0, 0, lightDir)</code> crea una luz roja <code>(255, 0, 0)</code> que brilla en la dirección en la que apunta el vector <code>lightDir</code>.</p>

  <p>La tercera forma de llamar a <code>directionalLight()</code> tiene cuatro parámetros. El primer parámetro, <code>color</code>, establece el color de la luz utilizando un objeto <a href="/reference/p5/p5.Color">p5.Color</a> o un arreglo de valores de color. Los últimos tres parámetros, <code>x</code>, <code>y</code> y <code>z</code>, establecen la dirección de la luz. Por ejemplo, <code>directionalLight(myColor, 1, 0, 0)</code> crea una luz que brilla hacia la derecha <code>(1, 0, 0)</code> con el valor de color de <code>myColor</code>.</p>

  <p>La cuarta forma de llamar a <code>directionalLight()</code> tiene dos parámetros. El primer parámetro, <code>color</code>, establece el color de la luz utilizando un objeto <a href="/reference/p5/p5.Color">p5.Color</a> o un arreglo de valores de color. El segundo parámetro, <code>direction</code>, establece la dirección de la luz utilizando un objeto <a href="/reference/p5/p5.Color">p5.Color</a>. Por ejemplo, <code>directionalLight(myColor, lightDir)</code> crea una luz que brilla en la dirección en la que apunta el vector <code>lightDir</code> con el valor de color de <code>myColor</code>.</p>

    <code>
    // Click and drag the mouse to view the scene from different angles.
    // Double-click to turn on the directional light.

    let isLit = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A sphere drawn on a gray background. A red light starts shining from above when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Control the light.
      if (isLit === true) {
        // Add a red directional light from above.
        // Use RGB values and XYZ directions.
        directionalLight(255, 0, 0, 0, 1, 0);
      }

      // Style the sphere.
      noStroke();

      // Draw the sphere.
      sphere(30);
    }
    </code>
