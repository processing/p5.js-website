description: >
  <p>Establece el color utilizado para dibujar puntos, líneas, y trazos o contornos de las formas.</p>
  <p>Llamar a <code>stroke(255, 165, 0)</code> o <code>stroke('orange')</code> significa que el borde de todas las formas dibujadas después de llamar a <code>stroke()</code> será de color naranja. La manera en que se interpretan estos parámetros puede cambiarse con la función <a href="/reference/p5/colorMode">colorMode()</a>.</p>
  <p>La versión de <code>stroke()</code> con un parámetro interpreta el valor de tres maneras. Si el parámetro es un <code>Number</code> (número), se interpreta como un valor de escala de grises. Si el parámetro es un <code>String</code> (cadena de caracteres), se interpreta como una cadena de caracteres de color de CSS (colores predefinidos de CSS). También se puede proporcionar un objeto <a href="/reference/p5/p5.Color">p5.Color</a> para establecer el color de trazo.</p>
  <p>La versión de <code>stroke()</code>  con dos parámetros interpreta el primero como un valor de escala de grises. El segundo parámetro establece el valor alfa (transparencia).</p>
  <p>La versión de <code>stroke()</code>  con tres parámetros los interpreta como colores RGB, HSB o HSL, dependiendo del modo de color actual (<code>colorMode()</code>).</p>
  <p>La versión de <code>stroke()</code>  con cuatro parámetros los interpreta como colores RGBA, HSBA o HSLA, dependiendo del modo de color actual (<code>colorMode()</code>). El último parámetro establece el valor alfa (transparencia).</p>
