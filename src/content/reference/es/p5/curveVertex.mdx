description: >
  <p>Agrega un segmento de curva spline a una forma personalizada.</p>

  <p><code>curveVertex()</code> agrega un segmento curvado a formas personalizadas. Las curvas spline que crea están definidas como las que se hacen con la función <a href="/reference/p5/curve">curve()</a>. <code>curveVertex()</code> debe llamarse entre las funciones <a href="/reference/p5/beginShape">beginShape()</a> y <a href="/reference/p5/endShape">endShape()</a>.</p>

  <p>Las curvas spline pueden crear formas y curvas con pendientes suaves. Son como cables que están unidos a un conjunto de puntos. Los splines se definen por dos puntos de anclaje y dos puntos de control. <code>curveVertex()</code> debe llamarse al menos cuatro veces entre <a href="/reference/p5/beginShape">beginShape()</a> y <a href="/reference/p5/endShape">endShape()</a> para dibujar una curva:</p>

  <code>

  beginShape();


  <p>// Add the first control point.

  curveVertex(84, 91);</p>

  <p>// Add the anchor points to draw between.

  curveVertex(68, 19);

  curveVertex(21, 17);</p>

  <p>// Add the second control point.

  curveVertex(32, 91);</p>

  <p>endShape();

  </p></code>
  <p>El fragmento de código anterior solo dibujaría la curva entre los puntos de anclaje, similar a la función <a href="/reference/p5/curve">curve()</a>. Los segmentos entre los puntos de control y de anclaje pueden dibujarse llamando a <code>curveVertex()</code> con las coordenadas de los puntos de control:</p>

  <code>

  beginShape();


  <p>// Add the first control point and draw a segment to it.

  curveVertex(84, 91);

  curveVertex(84, 91);</p>

  <p>// Add the anchor points to draw between.

  curveVertex(68, 19);

  curveVertex(21, 17);</p>

  <p>// Add the second control point.

  curveVertex(32, 91);</p>

  <p>// Uncomment the next line to draw the segment to the second control point.

  // curveVertex(32, 91);</p>

  <p>endShape();

  </p></code>  
  <p>Los dos primeros parámetros, <code>x</code> y <code>y</code>, establecen la ubicación del vértice. Por ejemplo, llamar a <code>curveVertex(10, 10)</code> agrega un punto a la curva en <code>(10, 10)</code>.</p>

  <p>Las curvas spline también pueden dibujarse en 3D utilizando el modo WebGL. La versión 3D de <code>curveVertex()</code> tiene tres argumentos porque cada punto tiene coordenadas x, y y z. Por defecto, la coordenada z del vértice se establece en 0.</p>

  <p>Nota: <code>curveVertex()</code> no funcionará cuando se pasa un argumento a <a href="/reference/p5/beginShape">beginShape()</a>.</p>

    // Click the mouse near the red dot in the bottom-left corner
    // and drag to change the curve's shape.

    let x1 = 32;
    let y1 = 91;
    let isChanging = false;

    function setup() {
      createCanvas(100, 100);

      describe(
        'A black U curve drawn upside down on a gray background. The curve passes from one red dot through two black dots and ends at another red dot.'
      );
    }

    function draw() {
      background(200);

      // Style the shape.
      noFill();
      stroke(0);
      strokeWeight(1);

      // Start drawing the shape.
      beginShape();

      // Add the first control point and draw a segment to it.
      curveVertex(x1, y1);
      curveVertex(x1, y1);

      // Add the anchor points.
      curveVertex(21, 17);
      curveVertex(68, 19);

      // Add the second control point and draw a segment to it.
      curveVertex(84, 91);
      curveVertex(84, 91);

      // Stop drawing the shape.
      endShape();

      // Style the anchor and control points.
      strokeWeight(5);

      // Draw the anchor points in black.
      stroke(0);
      point(21, 17);
      point(68, 19);

      // Draw the control points in red.
      stroke(255, 0, 0);
      point(x1, y1);
      point(84, 91);
    }

    // Start changing the first control point if the user clicks near it.
    function mousePressed() {
      if (dist(mouseX, mouseY, x1, y1) < 20) {
        isChanging = true;
      }
    }

    // Stop changing the first control point when the user releases the mouse.
    function mouseReleased() {
      isChanging = false;
    }

    // Update the first control point while the user drags the mouse.
    function mouseDragged() {
      if (isChanging === true) {
        x1 = mouseX;
        y1 = mouseY;
      }
    }
    </code>
