description: >
  <p>Dibuja un solo punto en el espacio.</p>

  <p>El ancho predeterminado de un punto es de un píxel. Para colorear un punto, utiliza la función <a href="/reference/p5/stroke">stroke()</a>. Para cambiar su ancho, utiliza la función <a href="/reference/p5/strokeWeight">strokeWeight()</a>. Un punto no puede ser rellenado, por lo que la función <a href="/reference/p5/fill">fill()</a> no afectará el color del punto.</p>

  <p>La versión de <code>point()</code> con un parámetro permite establecer la ubicación del punto con sus coordenadas x, y, como en <code>point(10, 20)</code>.</p>

  <p>La versión de <code>point()</code> con tres parámetros permite dibujar el punto en el espacio 3D con coordenadas x, y, z como en <code>point(10, 20, 30)</code>. Para hacer esto, es necesario agregar el argumento <code>WEBGL</code> a la función <a href="/reference/p5/createCanvas">createCanvas()</a>.</p>

  <p>La versión de <code>point()</code> con un parámetro permite establecer la ubicación del punto con un objeto <a href="/reference/p5/p5.Vector">p5.Vector</a>.</p>
