description: >
  <p>Comienza a utilizar formas para borrar partes del lienzo.</p>
  <p>Todo dibujo que siga después de <code>erase()</code> borrará del lienzo, revelando la página web que se encuentra debajo. Las áreas borradas se volverán transparentes, permitiendo que el contenido detrás del lienzo se muestre. <a href="/reference/p5/fill">fill()</a>, <a href="/reference/p5/stroke">stroke()</a> y <a href="/reference/p5/blendMode">blendMode()</a> no tienen efecto una vez que se llama a <code>erase()</code>.</p>
  <p>La función <code>erase()</code> tiene dos parámetros opcionales. El primer parámetro establece la intensidad del borrado dado por el interior de la forma. Un valor de 0 significa que no ocurrirá ningún borrado. Un valor de 255 significa que el interior de la forma borrará completamente el contenido que se encuentra debajo. El valor predeterminado es 255 (máxima intensidad).</p>
  <p>El segundo parámetro establece la intensidad del borrado dado por el borde de la forma. Un valor de 0 significa que no ocurrirá ningún borrado. Un valor de 255 significa que el borde de la forma borrará completamente el contenido que se encuentra debajo. El valor predeterminado es 255 (máxima intensidad).</p>
  <p>Para cancelar el efecto de borrado, utiliza la función <a href="/reference/p5/noErase">noErase()</a>. </p>
  <p><code>erase()</code> no tiene efecto en los dibujos realizados con las funciones <a href="/reference/p5/image">image()</a> y <a href="/reference/p5/background">background()</a>.</p>
