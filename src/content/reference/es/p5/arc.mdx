description: >
  <p>Dibuja un arco.</p>

  <p>Un arco es una sección de una elipse definida por los parametros <code>x</code>,
  <code>y</code>, <code>w</code>,y

  <code>h</code>. <code>x</code> y <code>y</code> establece la ubicación
  del centro del arco. <code>w</code> y

  <code>h</code> establecen el ancho y la altura del arco. Ver

  <a href="/reference/p5/ellipse">ellipse()</a> y

  <a href="/reference/p5/ellipseMode">ellipseMode()</a> para más detalles.</p>

  <p>Los quinto y sexto parámetros, <code>start</code> y <code>stop</code>,
  establecen los ángulos 

  entre los cuales dibujar el arco. Los arcos siempre se dibujan en sentido horario desde

  <code>start</code> a <code>stop</code>. Los ángulos siempre se dan en
  radianes.</p>

  <p>El séptimo parámetro, <code>mode</code>, es opcional, este determina el estilo
  de relleno 

  del arco. Los modos de relleno son un semicírculo (<code>OPEN</code>), un semicírculo
  cerrado

  (<code>CHORD</code>), o un segmento de pastel cerrado (<code>PIE</code>).</p>

  <p>El octavo parámetro, <code>detail</code>, también es opcional. Este determina
  cuántos

  vértices se utilizan para dibujar el arco en el modo WebGL. El valor predeterminado es 25.</p>

      background(0);

      // Style the arc.
      noStroke();
      fill(255, 255, 0);

      // Update start and stop angles.
      let biteSize = PI / 16;
      let startAngle = biteSize * sin(frameCount * 0.1) + biteSize;
      let endAngle = TWO_PI - startAngle;

      // Draw the arc.
      arc(50, 50, 80, 80, startAngle, endAngle, PIE);
