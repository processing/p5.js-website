  <p>Deshabilita el dibujo de puntos, líneas, y los trazos o contornos de las formas.</p>
  <p>Llamar a <code> noStroke()</code> es lo mismo que hacer que el trazo sea completamente transparente, por ejemplo: <code>stroke(0, 0)</code>. Si se llama a  <code>noStroke()</code>, al igual que a <a href="/reference/p5/noFill">noFill()</a>, no se dibujará nada en la pantalla.</p>

    function draw() {
      background(0);

      // Style the box.
      noStroke();
      fill(240, 150, 150);
