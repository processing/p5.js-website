description: >
  <p>Un <code>Array</code> de todos los puntos táctiles actuales en un dispositivo táctil.</p>

  <p>La matriz <code>touches</code> está vacía de forma predeterminada. Cuando el usuario toca su
  pantalla, se crea y agrega un nuevo punto táctil a la matriz. Los puntos táctiles son <code>Objets</code> con las siguientes propiedades:</p>

  <code>

  // Iterate over the touches array.

  for (let touch of touches) {
    // x-coordinate relative to the top-left
    // corner of the canvas.
    console.log(touch.x);

  <p>  // y-coordinate relative to the top-left
    // corner of the canvas.
    console.log(touch.y);</p>
  <p>  // x-coordinate relative to the top-left
    // corner of the browser.
    console.log(touch.winX);</p>
  <p>  // y-coordinate relative to the top-left
    // corner of the browser.
    console.log(touch.winY);</p>
  <p>  // ID number
    console.log(touch.id);
  }

  </p></code>
