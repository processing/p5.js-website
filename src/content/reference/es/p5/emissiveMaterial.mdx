description: >
  <p>Establece el color de emisión del material de la superficie de las formas.</p>

  <p>El color <code>emissiveMaterial()</code> establece un color que las formas muestran a
  máxima intensidad, independientemente de la iluminación. Esto puede dar la apariencia de que una forma
  está brillando. Sin embargo, los materiales de emisión no emiten luz que
  pueda afectar a los objetos circundantes.</p>

  <p><code>emissiveMaterial()</code> se puede llamar de tres formas diferentes con diferentes
  parámetros para establecer el color del material.</p>

  <p>La primera forma de llamar a <code>emissiveMaterial()</code> tiene un parámetro,
  <code>gray</code>.

  Se pueden pasar valores de escala de grises entre 0 y 255, como en <code>emissiveMaterial(50)</code>,
  para establecer el color del material. Valores de escala de grises más altos hacen que las formas
  parezcan más brillantes.</p>

  <p>La segunda forma de llamar a <code>emissiveMaterial()</code> tiene un parámetro,
  <code>color</code>. Un
  objeto <a href="/reference/p5/p5.Color">p5.Color</a>, un arreglo de valores de color o una
  cadena de color CSS, como en <code>emissiveMaterial('magenta')</code>, se pueden pasar para establecer
  el color del material.</p>

  <p>La tercera forma de llamar a <code>emissiveMaterial()</code> tiene cuatro parámetros,
  <code>v1</code>, <code>v2</code>,
  <code>v3</code> y <code>alpha</code>. <code>alpha</code> es opcional. Se pueden pasar valores RGBA,
  HSBA o HSLA para establecer los colores del material, como en <code>emissiveMaterial(255, 0,
  0)</code> o
  <code>emissiveMaterial(255, 0, 0, 30)</code>. Los valores de color se interpretarán
  utilizando el
  <a href="/reference/p5/colorMode">colorMode()</a> actual.</p>

  <p>Nota: <code>emissiveMaterial()</code> solo se puede usar en el modo WebGL.</p>
