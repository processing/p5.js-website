description: >
  <p>Una función que se ejecuta una vez cuando el botón del mouse se hace doble
  click rápidamente.</p>

  <p>Declarar la función <code>doubleClicked()</code> establece un bloque de
  código para ejecutarse

  automáticamente cuando el usuario presiona y suelta el botón del mouse dos

  veces rápidamente:</p>

  <code>

  function doubleClicked() {
    // Code to run.
  }

  </code>


  <p>Las variables de sistema del ratón, como <a
  href="/reference/p5/mouseX">mouseX</a> y

  <a href="/reference/p5/mouseY">mouseY</a>, se actualizarán con su valor más

  reciente cuando <code>doubleClicked()</code> se ejecute automáticamente por
  p5.js:</p>

  <code>

  function doubleClicked() {
    if (mouseX < 50) {
      // Code to run if the mouse is on the left.
    }

  <p>  if (mouseY > 50) {
      // Code to run if the mouse is near the bottom.
    }
  }

  </p></code>

  <p>El parámetro, <code>event</code>, es opcional. <code>doubleClicked()</code>
  se le pasa siempre un

  <a href="https://developer.mozilla.org/es/docs/Web/API/MouseEvent"
  target="_blank">MouseEvent</a>

  objeto con propiedades que describen el evento de doble click:</p>

  <code>

  function doubleClicked(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>En dispositivos táctiles, el código colocado en <code>doubleClicked()</code>
  se ejecutará después de dos

  pulsaciones táctiles que ocurran en un breve lapso de tiempo.</p>

  <p>Los navegadores pueden tener comportamientos predeterminados asociados a
  eventos de ratón. Por ejemplo, algunos navegadores resaltan texto cuando el
  usuario se mueve el ratón mientras presiona un botón del ratón. Para evitar cualquier
  comportamiento predeterminado para este evento, agrega <code>return false;</code>
  al final de la función.</p>
