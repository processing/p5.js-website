description: >
  <p>Obtiene el valor de luminosidad de un color.</p>

  <p><code>lightness()</code> extrae el valor de luminosidad HSL de un

  objeto <a href="/reference/p5/p5.Color">p5.Color</a>, un arreglo de componentes de 
  color, o

  una cadena de color CSS.</p>

  <p>De forma predeterminada, <code>lightness()</code> devuelve la luminosidad HSL de un color en el 
  rango 0

  a 100. Si el <a href="/reference/colorMode">colorMode()</a> está configurado en HSL, 
  devuelve

  el valor de luminosidad en el rango dado.</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(50);

      // Use HSL color.
      colorMode(HSL);

      // Create a p5.Color object using HSL values.
      let c = color(0, 100, 75);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // Set 'lightValue' to 75.
      let lightValue = lightness(c);

      // Draw the right rectangle.
      fill(lightValue);
      rect(50, 15, 35, 70);

      describe('Two rectangles. The left one is salmon pink and the right one is gray.');
    }
