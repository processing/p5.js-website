description: >
  <p>Una función que se llama cuando el mouse se mueve mientras se mantiene pulsado
  un botón.</p>

  <p>Declarar la función <code>mouseDragged()</code> establece un bloque de código
  para ejecutarse

  automáticamente cuando el usuario hace clic y arrastra el mouse:</p>

  <code>

  function mouseDragged() {
    // Code to run.
  }

  </code>


  <p>Las variables del sistema del mouse, como <a
  href="/reference/p5/mouseX">mouseX</a> y

  <a href="/reference/p5/mouseY">mouseY</a>, se actualizarán con su valor más
  reciente

  cuando <code>mouseDragged()</code> sea llamado por p5.js:</p>

  <code>

  function mouseDragged() {
    if (mouseX < 50) {
      // Code to run if the mouse is on the left.
    }

  <p>  if (mouseY > 50) {
      // Code to run if the mouse is near the bottom.
    }
  }

  </p></code>

  <p>El parámetro <code>event</code> es opcional. <code>mouseDragged()</code> siempre
  se le pasa un

  <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent"
  target="_blank">MouseEvent</a>

  objeto con propiedades que describen el evento de arrastre del mouse:</p>

  <code>

  function mouseDragged(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>En dispositivos táctiles, <code>mouseDragged()</code> se ejecutará cuando el
  usuario mueve un punto de

  tacto si <a href="/reference/p5/touchMoved">touchMoved()</a> no está declarado.
  Si

  <a href="/reference/p5/touchMoved">touchMoved()</a> está declarado, entonces

  <a href="/reference/p5/touchMoved">touchMoved()</a> se ejecutará cuando el usuario
  mueve un punto de
  tacto y <code>mouseDragged()</code> no se ejecutará.</p>

  <p>Los navegadores pueden tener comportamientos predeterminados asociados a
  eventos del mouse. Por ejemplo, algunos navegadores resaltan texto cuando el
  usuario mueve el mouse mientras mantiene pulsado un botón. Para evitar cualquier
  comportamiento predeterminado para este evento, agregue <code>return false;</code>
  al final de la función.</p>
