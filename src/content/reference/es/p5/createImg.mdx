description: >
  <p>Crea un elemento <code><img></code> que puede aparecer fuera del 
  lienzo.</p>

  <p>El primer parámetro, <code>src</code>, es una cadena con la ruta al 
  archivo de imagen.

  <code>src</code> debe ser una ruta relativa, como en 
  <code>'/assets/image.png'</code>, o una URL, como 
  
  en <code>'https://example.com/image.png'</code>.</p>

  <p>El segundo parámetro, <code>alt</code>, es una cadena con el 
  
   <a
  href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLImageElement/alt#usage_notes"
  target="_blank">texto alternativo</a> 
  
  para la imagen. Se puede usar una cadena vacía <code>''</code> para imágenes que 
  no se muestran.</p>

  <p>El tercer parámetro, <code>crossOrigin</code>, es opcional. Es una cadena 
  que establece la 
  
  <a
  href="https://developer.mozilla.org/en-US/docs/Web/HTML/CORS_settings_attributes"
  target="_blank">propiedad crossOrigin</a> 
  
  de la imagen. Usa <code>'anonymous'</code> o <code>'use-credentials'</code> 
  para recuperar la imagen 
  
  con acceso de origen cruzado.</p>

  <p>El cuarto parámetro, <code>callback</code>, también es opcional. Establece una 
  función para 
  
  llamar después de que se cargue la imagen. La nueva imagen se pasa a la función de devolución de llamada

  como un objeto <a href="/reference/p5/p5.Element">p5.Element</a>.</p>
