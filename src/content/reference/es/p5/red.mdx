description: >
  <p>Obtiene el valor rojo de un color.</p>

  <p><code>red()</code> extrae el valor rojo de un

  objeto <a href="/reference/p5/p5.Color">p5.Color</a>, un arreglo de componentes de 
  color, o

  una cadena de color CSS.</p>

  <p>De forma predeterminada, <code>red()</code> devuelve el valor rojo de un color en el rango 0

  a 255. Si el <a href="/reference/colorMode">colorMode()</a> está configurado en RGB, 
  devuelve

  el valor rojo en el rango dado.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Create a p5.Color object.
      let c = color(175, 100, 220);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // Set 'redValue' to 175.
      let redValue = red(c);

      // Draw the right rectangle.
      fill(redValue, 0, 0);
      rect(50, 15, 35, 70);

      describe('Two rectangles. The left one is light purple and the right one is red.');
