description: >
  <p>Comienza a crear un agujero dentro de una forma plana.</p>

  <p>Las funciones <code>beginContour()</code> y <a href="/reference/p5/endContour">endContour()</a> permiten crear espacio negativo dentro de formas personalizadas que son planas. <code>beginContour()</code> comienza a agregar vértices a un espacio negativo y <a href="/reference/p5/endContour">endContour()</a> detiene el proceso. <code>beginContour()</code> y <a href="/reference/p5/endContour">endContour()</a> deben llamarse entre <a href="/reference/p5/beginShape">beginShape()</a> y <a href="/reference/p5/endShape">endShape()</a>.</p>

  <p>Las transformaciones como <a href="/reference/p5/translate">translate()</a>, <a href="/reference/p5/rotate">rotate()</a> y <a href="/reference/p5/scale">scale()</a> no funcionan entre <code>beginContour()</code> y <a href="/reference/p5/endContour">endContour()</a>. Tampoco es posible usar otras formas, como <a href="/reference/p5/ellipse">ellipse()</a> o <a href="/reference/p5/rect">rect()</a>, entre <code>beginContour()</code> y <a href="/reference/p5/endContour">endContour()</a>.</p>

  <p>Nota: Los vértices que definen un espacio negativo deben "girar" en la dirección opuesta a la forma exterior. Primero, dibuja los vértices para la forma exterior en orden horario. Luego, dibuja los vértices para el espacio negativo en orden antihorario.</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Start drawing the shape.
      beginShape();

      // Exterior vertices, clockwise winding.
      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // Interior vertices, counter-clockwise winding.
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // Stop drawing the shape.
      endShape(CLOSE);

      describe('A white square with a square hole in its center drawn on a gray background.');
    }
    </code>
