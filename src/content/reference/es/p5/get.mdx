description: |-
  <p>Obtiene un píxel o una región de píxeles del lienzo.</p>

  <p><code>get()</code> es fácil de usar pero no es tan rápido como <a href="/reference/p5/pixels">pixels</a>. Usa <a href="/reference/p5/pixels">pixels</a> para leer muchos valores de píxeles.</p>

  <p>La versión de <code>get()</code> sin parámetros devuelve todo el lienzo.</p>

  <p>La versión de <code>get()</code> con dos parámetros se interpreta como coordenadas. Devuelve un array con los valores <code>[R, G, B, A]</code> del píxel en el punto dado.</p>

  <p>La versión de <code>get()</code> con cuatro parámetros se interpreta como coordenadas y dimensiones. Devuelve una subsección del lienzo como un objeto <a href="/reference/p5/p5.Image">p5.Image</a>. Los dos primeros parámetros son las coordenadas de la esquina superior izquierda de la subsección. Los dos últimos parámetros son el ancho y el alto de la subsección.</p>

  <p>Usa <a href="/reference/p5.Image/get">p5.Image.get()</a> para trabajar directamente con objetos <a href="/reference/p5/p5.Image">p5.Image</a>.</p>
