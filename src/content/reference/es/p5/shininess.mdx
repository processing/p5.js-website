description: >
  <p>Establece la cantidad de brillo ("brillantez") de un
  <a href="/reference/p5/specularMaterial">specularMaterial()</a>.</p>

  <p>Los materiales brillantes enfocan la luz reflejada más que los materiales opacos.
  <code>shininess()</code> afecta la forma en que los materiales reflejan las fuentes de luz,
  incluyendo <a href="/reference/p5/directionalLight">directionalLight()</a>,
  <a href="/reference/p5/pointLight">pointLight()</a> y <a href="/reference/p5/spotLight">spotLight()</a>.</p>

  <p>El parámetro <code>shine</code> es un número que establece la cantidad de brillo.
  <code>shine</code> debe ser mayor que 1, que es su valor predeterminado.</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100, WEBGL);

      describe(
        'Two red spheres drawn on a gray background. White light reflects from their surfaces as the mouse moves. The right sphere is shinier than the left sphere.'
      );
    }

    function draw() {
      background(200);

      // Turn on a red ambient light.
      ambientLight(255, 0, 0);

      // Get the mouse's coordinates.
      let mx = mouseX - 50;
      let my = mouseY - 50;

      // Turn on a white point light that follows the mouse.
      pointLight(255, 255, 255, mx, my, 50);

      // Style the sphere.
      noStroke();

      // Add a specular material with a grayscale value.
      specularMaterial(255);

      // Draw the left sphere with low shininess.
      translate(-25, 0, 0);
      shininess(10);
      sphere(20);

      // Draw the right sphere with high shininess.
      translate(50, 0, 0);
      shininess(100);
      sphere(20);
    }
    </code>
