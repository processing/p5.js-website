description: >
  <p>Establece el número de segmentos utilizados para dibujar curvas spline en modo WebGL.</p>

  <p>En modo WebGL, las formas suaves se dibujan usando muchos segmentos planos. 
  Agregar más segmentos planos hace que las formas se vean más suaves.</p>

  <p>El parámetro, <code>detail</code>, es el número de segmentos que se utilizarán 
  para dibujar una curva spline. Por ejemplo, llamar a <code>curveDetail(5)</code> 
  usará 5 segmentos para dibujar curvas con la función 
  <a href="/reference/p5/curve">curve()</a>. Por defecto, <code>detail</code> es 20.</p>

  <p>Nota: <code>curveDetail()</code> no tiene efecto en modo 2D.</p>

    function setup() {
      createCanvas(100, 100, WEBGL);

      curveDetail(3);

      noFill();
      strokeWeight(1);
      stroke(0);
      curve(-45, -24, 0, 23, -26, 0, 23, 11, 0, -35, 15, 0);

      stroke(255, 0, 0);

      curve(23, -26, 0, 23, 11, 0, -35, 15, 0, -35, 15, 0);

      // Draw the anchor points in black.

      stroke(0);
      point(23, -26);
      point(23, 11);

      // Draw the control points in red.
      stroke(255, 0, 0);
      point(-45, -24);
      point(-35, 15);
