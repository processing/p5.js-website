  <p>Una variable de tipo String del sistema que contiene el valor del último botón

  del mouse presionado.</p>

  <p>La variable <code>mouseButton</code> es <code>LEFT</code>,
  <code>RIGHT</code> o <code>CENTER</code>,

  dependiendo del botón que fue presionado por última vez.</p>

  <p>Tenga en cuenta que los diferentes navegadores pueden rastrear
  <code>mouseButton</code> de manera diferente. Consulte

  <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent/buttons"
  target="_blank">MDN</a>

  para más información.</p>
