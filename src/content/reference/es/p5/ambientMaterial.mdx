description: >
  <p>Establece el color ambiental del material de la superficie de las formas.</p>

  <p>El color <code>ambientMaterial()</code> establece los componentes del color de la
  <a href="/reference/p5/ambientLight">luz ambiental</a> que las formas reflejarán.
  Por ejemplo, llamar a <code>ambientMaterial(255, 255, 0)</code> haría que una forma
  refleje luz roja y verde, pero no luz azul.</p>

  <p><code>ambientMaterial()</code> se puede llamar de tres formas diferentes con
  diferentes parámetros para establecer el color del material.</p>

  <p>La primera forma de llamar a <code>ambientMaterial()</code> tiene un parámetro,
  <code>gray</code>. Se pueden pasar valores de escala de grises entre 0 y 255, como
  en <code>ambientMaterial(50)</code>, para establecer el color del material. Valores
  de escala de grises más altos hacen que las formas parezcan más brillantes.</p>

  <p>La segunda forma de llamar a <code>ambientMaterial()</code> tiene un parámetro,
  <code>color</code>. Se puede pasar un objeto <a href="/reference/p5/p5.Color">p5.Color</a>,
  un arreglo de valores de color o una cadena de color CSS, como en
  <code>ambientMaterial('magenta')</code>, para establecer el color del material.</p>

  <p>La tercera forma de llamar a <code>ambientMaterial()</code> tiene tres parámetros,
  <code>v1</code>, <code>v2</code> y <code>v3</code>. Se pueden pasar valores RGB, HSB o HSL,
  como en <code>ambientMaterial(255, 0, 0)</code>, para establecer los colores del material.
  Los valores de color se interpretarán utilizando el <a href="/reference/p5/colorMode">colorMode()</a>
  actual.</p>

  <p>Nota: <code>ambientMaterial()</code> solo se puede usar en el modo WebGL.</p>
