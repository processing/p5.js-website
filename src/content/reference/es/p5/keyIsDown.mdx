description: >
  <p>Devuelve <code>true</code> si la tecla que se está comprobando está presionada y
  <code>false</code> si no lo está.</p>

  <p><code>keyIsDown()</code> es útil para comprobar si se están presionando varias
  teclas diferentes.

  Por ejemplo, <code>keyIsDown()</code> se puede usar para comprobar si se presionan
  tanto la tecla <code>LEFT_ARROW</code> como la tecla
  <code>UP_ARROW</code>:</p>

  <code>

  if (keyIsDown(LEFT_ARROW) && keyIsDown(UP_ARROW)) {
    // Move diagonally.
  }

  </code>


  <p><code>keyIsDown()</code> también puede comprobar la presión de teclas mediante

  los valores <a href="/reference/p5/keyCode">keyCode</a>. Por ejemplo,
  <code>keyIsDown(37)</code> o <code>keyIsDown(LEFT_ARROW)</code>. Los valores
  de keyCode se pueden encontrar en sitios web como

  <a href="https://keycode.info" target="_blank">keycode.info</a>.</p>
