  <p>Crea un elemento <code><audio></code> oculto para reproducción de audio 
  simple.</p>

  <p><code>createAudio()</code> devuelve un nuevo objeto

  <a href="/reference/p5/p5.MediaElement">p5.MediaElement</a>.</p>

  <p>El primer parámetro, <code>src</code>, es la ruta del video. Si se pasa una sola 
  cadena, como 
  
  en <code>'/assets/video.mp4'</code>, se carga un solo video. Se puede usar un 
  arreglo

  de cadenas para cargar el mismo video en diferentes formatos. Por 
  
  ejemplo, <code>['/assets/video.mp4', '/assets/video.ogv', 
  '/assets/video.webm']</code>.

  Esto es útil para asegurar que el video pueda reproducirse en diferentes
  
  navegadores con diferentes capacidades. Consulta

  <a
  href="https://developer.mozilla.org/en-US/docs/Web/HTML/Supported_media_formats"
  target="_blank">MDN</a>

  para obtener más información sobre los formatos admitidos.</p>

  <p>El segundo parámetro, <code>callback</code>, es opcional. Es una función 
  para llamar una vez que

  el audio está listo para reproducirse.</p>
