description: >
  <p>Establece el color de un píxel o dibuja una imagen en el lienzo.</p>

  <p><code>set()</code> es fácil de usar pero no es tan rápido como

  <a href="/reference/p5/pixels">pixels</a>. Usa <a
  href="/reference/p5/pixels">pixels</a>

  para establecer muchos valores de píxeles.</p>

  <p><code>set()</code> interpreta los dos primeros parámetros como coordenadas x, y. 

  Interpreta el último parámetro como un valor de escala de grises, un arreglo de píxeles <code>[R, G, B,
  A]</code>, un objeto <a href="/reference/p5/p5.Color">p5.Color</a>, o un

  objeto <a href="/reference/p5/p5.Image">p5.Image</a>. Si se pasa una imagen,
  los dos primeros

  parámetros establecen las coordenadas para la esquina superior izquierda de la imagen,

  independientemente del <a
  href="/reference/p5/imageMode">imageMode()</a> actual.</p>

  <p>Se debe llamar a <a href="/reference/p5/updatePixels">updatePixels()</a>
  después de usar

  <code>set()</code> para que los cambios aparezcan.</p>

    function setup() {
      createCanvas(100, 100);

      background(255);

      // Draw a horizontal color gradient.
      for (let x = 0; x < 100; x += 1) {
        for (let y = 0; y < 100; y += 1) {
          // Calculate the grayscale value.
          let c = map(x, 0, 100, 0, 255);

          // Set the pixel using the grayscale value.
          set(x, y, c);
        }
      }

      // Update the canvas.
      updatePixels();

      describe('A horiztonal color gradient from black to white.');
    }
    </code>
