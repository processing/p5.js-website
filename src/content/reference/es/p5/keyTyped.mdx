description: >
  <p>Una función que se ejecuta una vez cuando se presiona una tecla con un
  caracter imprimible.</p>

  <p>Declarar la función <code>keyTyped()</code> establece un bloque de código a
  ejecutarse

  automáticamente cuando el usuario presiona cualquier tecla con un caracter

  imprimible como <code>a</code> o 1. Las teclas de modificador como <code>SHIFT</code>,

  <code>CONTROL</code> y las flechas

  serán ignoradas:</p>

  <code>

  function keyTyped() {
    // Code to run.
  }

  </code>


  <p>La variable <a href="/reference/p5/key">key</a> y <a
  href="/reference/p5/keyCode">keyCode</a>

  se actualizarán con el valor más reciente que se ha liberado cuando

  <code>keyTyped()</code> es llamada por p5.js:</p>

  <code>

  function keyTyped() {
    // Check for the "c" character using key.
    if (key === 'c') {
      // Code to run.
    }

  <p>  // Check for "c" using keyCode.
    if (keyCode === 67) {
      // Code to run.
    }
  }

  </p></code>

  <p>El parámetro, <code>event</code>, es opcional. <code>keyTyped()</code> siempre
  se le paso un objeto

  <a href="https://developer.mozilla.org/es/docs/Web/API/KeyboardEvent"
  target="_blank">KeyboardEvent</a>

  con propiedades que describen el evento de presión de tecla:</p>

  <code>

  function keyReleased(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>Nota: Use la función <a href="/reference/p5/keyPressed">keyPressed()</a> y

  <a href="/reference/p5/keyCode">keyCode</a> para responder a las teclas de

  modificador como <code>ALT</code>.</p>

  <p>Los navegadores pueden tener comportamientos predeterminados asociados a

  ciertos eventos de tecla. Para evitar cualquier comportamiento predeterminado

  para este evento, agregue <code>return false;</code> al final de la función.</p>
