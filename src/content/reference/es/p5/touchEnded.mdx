description: >
  <p>Una función que se ejecuta una vez por cada toque en la pantalla que termina.</p>

  <p>Declarar una función llamada <code>touchEnded()</code> establece un bloque de código para ejecutarse automáticamente cuando el usuario deja de tocar una pantalla táctil:</p>

  <code>

  function touchEnded() {
    // Code to run.
  }

  </code>


  <p>La matriz <a href="/reference/p5/touches">touches</a> se actualizará con los puntos táctiles más recientes cuando <code>touchEnded()</code> es llamada por p5.js:</p>

  <code>

  function touchEnded() {
    // Paint over the background.
    background(200);

  <p>  // Mark each remaining touch point when the user stops
    // a touch.
    for (let touch of touches) {
      circle(touch.x, touch.y, 40);
    }
  }

  </p></code>

  <p>El parámetro, event, es opcional. <code>touchEnded()</code> se le pasará un objeto <a href="https://developer.mozilla.org/en-US/docs/Web/API/TouchEvent" target="_blank">TouchEvent</a> con propiedades que describen el evento táctil:</p>

  <code>

  function touchEnded(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>En dispositivos táctiles, <a
  href="/reference/p5/mouseReleased">mouseReleased()</a>

  se ejecutará cuando el usuario deja de tocar su pantalla si <code>touchEnded()</code>
  no está declarado. Si

  <code>touchEnded()</code> está declarado, entonces <code>touchEnded()</code> se ejecutará
  cuando el usuario deja un toque y <a href="/reference/p5/mouseReleased">mouseReleased()</a>
  no se ejecutará.</p>

  <p>Nota: <a href="/reference/p5/touchStarted">touchStarted()</a>,

  <code>touchEnded()</code>, y <a
  href="/reference/p5/touchMoved">touchMoved()</a> estan todos

  relacionados. <a href="/reference/p5/touchStarted">touchStarted()</a> se ejecuta
  inmediatamente cuando el usuario toca una pantalla táctil. <code>touchEnded()</code>
  se ejecuta inmediatamente cuando el usuario deja un toque. <a href="/reference/p5/touchMoved">touchMoved()</a>
  se ejecuta repetidamente mientras el usuario mueve cualquier punto táctil.</p>
