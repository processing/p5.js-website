description: >
  <p>Al llamar a <code>noErase()</code>, se detiene la acción de borrar que se había iniciado previamente con <a href="/reference/p5/erase">erase()</a>. En el ejemplo <code>erase()</code> comienza la acción de borrar del lienzo la forma de un círculo, y <code>noErase()</code> detiene el proceso de borrado. </p>
  <p>Además, las configuraciones de <a href="/reference/p5/fill">fill()</a>, <a href="/reference/p5/stroke">stroke()</a>,  y <a href="/reference/p5/blendMode">blendMode()</a> volverán a lo que eran antes de que se llamara a <a href="/reference/p5/erase">erase()</a>.</p>

    function setup() {
      createCanvas(100, 100);

      background(235, 145, 15);

      // Draw the left rectangle.
      noStroke();
      fill(30, 45, 220);
      rect(30, 10, 10, 80);

      // Erase a circle.
      erase();
      circle(50, 50, 60);
      noErase();

      // Draw the right rectangle.
      rect(70, 10, 10, 80);

      describe('An orange canvas with two tall blue rectangles. A circular hole in the center erases the rectangle on the left but not the one on the right.');
