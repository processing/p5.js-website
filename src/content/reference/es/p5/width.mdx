description: |-
  <p>La variable <code>width</code> almacena el ancho del lienzo en píxeles. </p>
  <p>Por defecto, <code>width</code> tiene un valor de 100, el cual puede ser modificado mediante la llamada a <a href="./createCanvas">createCanvas()</a> o <a href="./resizeCanvas">resizeCanvas()</a>. Por ejemplo: createCanvas(50, 100) define el valor de <code>width</code> como 50. </p>
  <p>Al llamar  <a href="./noCanvas">noCanvas()</a> se definirá el valor de <code>width</code> como 0.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Display the canvas' width.
      text(width, 42, 54);

      describe('The number 100 written in black on a gray square. When the mouse is pressed, the square becomes a rectangle and the number becomes 50.');
    }

    // If the mouse is pressed, reisze
    // the canvas and display its new
    // width.
    function mousePressed() {
      if (mouseX > 0 && mouseX < width && mouseY > 0 && mouseY < height) {
        resizeCanvas(50, 100);
        background(200);
        text(width, 21, 54);
      }
    }
    </code>
