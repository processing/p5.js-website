description: |2-

  <p>Draws an ellipse (oval).</p>


  <p>An ellipse is a round shape defined by the <code>x</code>, <code>y</code>,
  <code>w</code>, and


  <code>h</code> parameters. <code>x</code> and <code>y</code> set the location
  of its center. <code>w</code> and


  <code>h</code> set its width and height. See


  <a href="/reference/p5/ellipseMode">ellipseMode()</a> for other ways to set


  its position.</p>


  <p>If no height is set, the value of width is used for both the width and


  height. If a negative height or width is specified, the absolute value is


  taken.</p>


  <p>The fifth parameter, <code>detail</code>, is also optional. It determines
  how many


  vertices are used to draw the ellipse in WebGL mode. The default value is


  25.</p>
