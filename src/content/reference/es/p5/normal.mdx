description: >
  <p>Establece el vector normal para vértices en una forma 3D personalizada.</p>

  <p>Las formas 3D creadas con <a href="/reference/p5/beginShape">beginShape()</a> y <a href="/reference/p5/endShape">endShape()</a> se hacen conectando conjuntos de puntos llamados vértices. Cada vértice agregado con <a href="/reference/p5/vertex">vertex()</a> tiene un vector normal que apunta hacia afuera de él. El vector normal controla cómo se refleja la luz en la forma.</p>

  <p><code>normal()</code> se puede llamar de dos maneras con diferentes parámetros para definir los componentes del vector normal.</p>

  <p>La primera forma de llamar a <code>normal()</code> tiene tres parámetros: <code>x</code>, <code>y</code> y <code>z</code>. Si se pasan <code>Number</code>s, como en <code>normal(1, 2, 3)</code>, se establecen las componentes x, y, y z del vector normal.</p>

  <p>La segunda forma de llamar a <code>normal()</code> tiene un parámetro, <code>vector</code>. Si se pasa un objeto <a href="/reference/p5/p5.Vector">p5.Vector</a>, como en <code>normal(myVector)</code>, sus componentes se usarán para establecer el vector normal.</p>

  <p><code>normal()</code> cambia el vector normal de los vértices agregados a una forma personalizada con <a href="/reference/p5/vertex">vertex()</a>. <code>normal()</code> debe llamarse entre las funciones <a href="/reference/p5/beginShape">beginShape()</a> y <a href="/reference/p5/endShape">endShape()</a>, al igual que <a href="/reference/p5/vertex">vertex()</a>. El vector normal configurado al llamar a <code>normal()</code> afectará a todos los vértices siguientes hasta que se llame a <code>normal()</code> nuevamente:</p>

  <code>

  beginShape();


  <p>// Set the vertex normal.

  normal(-0.4, -0.4, 0.8);</p>

  <p>// Add a vertex.

  vertex(-30, -30, 0);</p>

  <p>// Set the vertex normal.

  normal(0, 0, 1);</p>

  <p>// Add vertices.

  vertex(30, -30, 0);

  vertex(30, 30, 0);</p>

  <p>// Set the vertex normal.

  normal(0.4, -0.4, 0.8);</p>

  <p>// Add a vertex.

  vertex(-30, 30, 0);</p>

  <p>endShape();

  </p></code>
