description: >
  <p>Declara una nueva variable.</p>

  <p>Una variable es un contenedor para un valor. Por ejemplo, una variable podría contener la coordenada x de una criatura como un <code>Number</code> o su nombre como un <code>String</code>. Las variables pueden cambiar de valor reasignándolas de la siguiente manera:</p>
  <code>
    // Declare the variable x and assign it the value 10.
    let x = 10;
    <p>// Reassign x to 50
    x = 50;</p>
  </code>
  <p>Las variables tienen alcance de bloque. Cuando se declara una variable entre llaves <code>{}</code>, esta solo existe dentro del bloque definido por esas llaves. Por ejemplo, el siguiente código arrojaría un <code>ReferenceError</code> porque <code>x</code> está declarado dentro del bloque de la función <code>setup()</code>:</p>

  <code>
    function setup() {
        createCanvas(100, 100);
        <p>let x = 50;</p>
    }

    <p>function draw() {
        background(200);
        <p>// x was declared in setup(), so it can't be referenced here.
        circle(x, 50, 20);
        </p>
    }
  </code>
  <p>Las variables declaradas fuera de todas las llaves <code>{}</code> están en el alcance global. Una variable que está en el alcance global se puede usar y cambiar en cualquier lugar de un boceto:</p>
  <code>
    let x = 50;
    <p>function setup() {
        createCanvas(100, 100);
    }</p>

    <p>function draw() {
        background(200);
        <p>// Change the value of x.
        x += 10;</p>
        <p>circle(x, 50, 20);
        </p>
    }
  </code>
