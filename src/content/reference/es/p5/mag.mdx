description: |-
  <p><code>mag()</code> calcula la magnitud, o distancia de un vector. </p>
  <p>Un vector puede ser pensado de diferente maneras. Una de ellas, es pensando en vectores como puntos en un espacio. Los componentes de un vector, <code>x</code> y <code>y</code>, son las coordenadas que definen su posición <code>(x,y)</code>. La magnitud de un vector es entonces la distancia desde el punto origen <code>(0,0)</code> hasta el punto donde se encuentra el vector <code>(x,y)</code>. Utilizar <code>mag(x,y)</code> es una opción rápida equivalente a llamar a <code>dist(0,0,x,y)</code>.</p>
  <p>Un vector también puede ser pensado como una flecha que puede apuntar a distintas direcciones en un espacio. Esta perspectiva es útil para programar movimiento.</p>
  <p>Consulta <a href="/reference/p5/p5.Vector">p5.Vector</a> para más detalles.</p>
  <p>Usa <a href="/reference/p5.Vector/mag">p5.Vector.mag()</a> para calcular la magnitud de un objeto <a href="/reference/p5/p5.Vector">p5.Vector</a> .</p>
