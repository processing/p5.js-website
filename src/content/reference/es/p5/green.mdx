description: >
  <p>Obtiene el valor verde de un color.</p>

  <p><code>green()</code> extrae el valor verde de un objeto 

  <a href="/reference/p5/p5.Color">p5.Color</a>, un arreglo de componentes de 
  color, o

  una cadena de color CSS.</p>

  <p>Por defecto, <code>green()</code> devuelve el valor verde de un color en el rango
  0

  a 255. Si el <a href="/reference/colorMode">colorMode()</a> está establecido en RGB,
  devuelve

  el valor verde en el rango dado.</p>

    function setup() {

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // Set 'greenValue' to 100.
      let greenValue = green(c);

      // Draw the right rectangle.
      fill(0, greenValue, 0);
      rect(50, 15, 35, 70);
