description: >
  <p>Una función que se ejecuta cada vez que el usuario toca la pantalla y se mueve.</p>

  <p>Declarar una función llamada <code>touchMoved()</code> establece un bloque de código para ejecutarse automáticamente cuando el usuario toca un dispositivo táctil y se mueve:</p>

  <code>

  function touchMoved() {
    // Code to run.
  }

  </code>


  <p>La matriz <a href="/reference/p5/touches">touches</a> se actualizará con los puntos táctiles más recientes cuando <code>touchMoved()</code> es llamado por p5.js:</p>

  <code>

  function touchMoved() {
    // Paint over the background.
    background(200);

  <p>  // Mark each touch point while the user moves.
    for (let touch of touches) {
      circle(touch.x, touch.y, 40);
    }
  }

  </p></code>

  <p>El parámetro, event, es opcional. <code>touchMoved()</code> se le pasará un objeto <a href="https://developer.mozilla.org/en-US/docs/Web/API/TouchEvent" target="_blank">TouchEvent</a> con propiedades que describen el evento táctil:</p>

  <code>

  function touchMoved(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>En dispositivos táctiles, <a href="/reference/p5/mouseDragged">mouseDragged()</a> se ejecutará cuando el usuario mueve su puntero sobre la pantalla si <code>touchMoved()</code> no está declarado. Si <code>touchMoved()</code> está declarado, entonces <code>touchMoved()</code> se ejecutará cuando el usuario mueve sus puntos táctiles y <a href="/reference/p5/mouseDragged">mouseDragged()</a> no se ejecutará.</p>

  <p>Tenga en cuenta que <a href="/reference/p5/touchStarted">touchStarted()</a>, <a href="/reference/p5/touchEnded">touchEnded()</a> y <code>touchMoved()</code> estan todos relacionados. <a href="/reference/p5/touchStarted">touchStarted()</a> se ejecuta cuando el usuario toca un dispositivo táctil, <a href="/reference/p5/touchEnded">touchEnded()</a> se ejecuta cuando el usuario termina de tocar y <code>touchMoved()</code> se ejecuta repetidamente mientras el usuario mueve cualquier punto táctil.</p>
