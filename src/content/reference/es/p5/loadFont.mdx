description: >
  <p>Carga una fuente y crea un objeto <a href="/reference/p5/p5.Font">p5.Font</a>.</p>
  <p><code>loadFont()</code> puede cargar fuentes en formato .otf o .ttf. Las fuentes cargadas pueden ser utilizadas para dar estilo al texto en el lienzo y en elementos HTML.</p>
  <p>El primer parámetro, <code>path</code>, es la ruta a un archivo de fuente. Las rutas a archivos locales deben ser relativas. Por ejemplo, <code>'/assets/inconsolata.otf'</code>. La fuente Inconsolata utilizada en los siguientes ejemplos se puede descargar de forma gratuita <a href="https://www.fontsquirrel.com/fonts/inconsolata" target="_blank">aquí</a>. Las rutas a archivos remotos deben ser URL. Por ejemplo, <code>'https://example.com/inconsolata.otf'</code>. Las URL pueden estar bloqueadas debido a la seguridad del navegador.</p>
  <p>El segundo parámetro, <code>successCallback</code>, es opcional. Si se pasa una función, se llamará una vez que se haya cargado la fuente. La función de devolución de llamada puede utilizar el nuevo objeto <a href="/reference/p5/p5.Font">p5.Font</a> si es necesario.</p>
  <p>El tercer parámetro, <code>failureCallback</code>, también es opcional. Si se pasa una función, se llamará si la fuente no se carga correctamente. La función de devolución de llamada puede utilizar el objeto de error <a href="https://developer.mozilla.org/en-US/docs/Web/API/Event" target="_blank">Event</a> si es necesario.</p>
  <p>Las fuentes pueden tardar en cargarse. Llamar a <code>loadFont()</code> en <a href="/reference/p5/preload">preload()</a> asegura que las fuentes se carguen antes de ser utilizadas en <a href="/reference/p5/setup">setup()</a> o <a href="/reference/p5/draw">draw()</a>.</p>

      text('p5*js', 10, 50);
