description: >
  <p>Establece la textura que se utilizará en las formas.</p>

  <p>Una textura es como una piel que envuelve una forma. <code>texture()</code>
  funciona con formas incorporadas, como <a href="/reference/p5/square">square()</a> y
  <a href="/reference/p5/sphere">sphere()</a>, y formas personalizadas creadas con
  funciones como <a href="/reference/p5/buildGeometry">buildGeometry()</a>.
  Para texturizar una geometría creada con <a href="/reference/p5/beginShape">beginShape()</a>,
  se deben pasar coordenadas UV a cada llamada de <a href="/reference/p5/vertex">vertex()</a>.</p>

  <p>El parámetro <code>tex</code> es la textura a aplicar.
  <code>texture()</code> puede usar una variedad de fuentes, incluyendo imágenes, videos y renderizadores externos como
  objetos <a href="/reference/p5/p5.Graphics">p5.Graphics</a> y
  <a href="/reference/p5/p5.Framebuffer">p5.Framebuffer</a>.</p>

  <p>Para texturizar una geometría creada con <a href="/reference/p5/beginShape">beginShape()</a>,
  deberás especificar coordenadas UV en <a href="/reference/p5/vertex">vertex()</a>.</p>

  <p>Nota: <code>texture()</code> solo se puede usar en modo WebGL.</p>
