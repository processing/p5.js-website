description: >
  <p>Crea un elemento de botón de radio.</p>

  <p>El parámetro es opcional. Si se pasa una cadena, como en

  <code>let myRadio = createSelect('food')</code>, entonces cada opción de radio

  tendrá <code>"food"</code> como su parámetro <code>name</code>: <code><input
  name="food"></input></code>.

  Si se pasa un elemento existente <code><div></div></code> o <code><span></span></code>,

  como en <code>let myRadio = createSelect(container)</code>, 
  este se

  convertirá en el elemento padre del botón de radio.</p>

  <p>Los botones de radio extienden la clase <a href="/reference/p5/p5.Element">p5.Element</a>
  con algunos

  métodos útiles para gestionar las opciones:</p>

  <ul>

  <li><code>myRadio.option(value, [label])</code> agrega una opción al menú.
  El primer parámetro, <code>value</code>, es una cadena que establece el 
  valor y la etiqueta de la opción. El segundo parámetro, <code>label</code>, es opcional. Si 
  se proporciona, establece la etiqueta que se muestra para el <code>value</code>. Si una opción 
  con <code>value</code> ya existe, su etiqueta se cambia y se devuelve su 
  valor.</li>

  <li><code>myRadio.value()</code> devuelve el valor de la opción actualmente 
  seleccionada.</li>

  <li><code>myRadio.selected()</code> devuelve la opción actualmente 
  seleccionada.</li>

  <li><code>myRadio.selected(valor)</code> selecciona la opción dada y la devuelve 
  como un <a
  href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement"
  target="_blank"><code>HTMLInputElement</code></a>.</li>

  <li><code>myRadio.disable(shouldDisable)</code> habilita el botón de radio 
  completo si se pasa <code>true</code> y lo deshabilita si se pasa <code>false</code>
  .</li>

  </ul>
