description: >
  <p>Una función que se ejecuta una vez cuando se libera un botón del mouse.</p>

  <p>Declarar la función <code>mouseReleased()</code> establece un bloque de
  código para ejecutarse automáticamente cuando el usuario libera un botón del
  mouse después de haberlo presionado:</p>

  <code>

  function mouseReleased() {
    // Code to run.
  }

  </code>


  <p>Las variables del sistema del mouse, como <a
  href="/reference/p5/mouseX">mouseX</a> y

  <a href="/reference/p5/mouseY">mouseY</a>, se actualizarán con su valor más

  reciente cuando <code>mouseReleased()</code> sea llamado por p5.js:</p>

  <code>

  function mouseReleased() {
    if (mouseX < 50) {
      // Code to run if the mouse is on the left.
    }

  <p>  if (mouseY > 50) {
      // Código a ejecutar si el mouse está cerca del final.
    }
  }

  </p></code>

  <p>El parámetro, <code>event</code>, es opcional. <code>mouseReleased()</code>
  siempre

  se le pasa un objeto <a
  href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent"
  target="_blank">MouseEvent</a> con propiedades que describen el evento de
  liberación del mouse:</p>

  <code>

  function mouseReleased(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>En dispositivos táctiles, <code>mouseReleased()</code> se ejecutará cuando
  el usuario termina de tocar la pantalla si <a
  href="/reference/p5/touchEnded">touchEnded()</a>
  no está declarado. Si <a
  href="/reference/p5/touchEnded">touchEnded()</a> está declarado, entonces

  <a href="/reference/p5/touchEnded">touchEnded()</a> se ejecutará cuando el
  usuario termina de tocar la pantalla y <code>mouseReleased()</code> no se
  ejecutará.</p>

  <p>Los navegadores pueden tener comportamientos predeterminados asociados a
  eventos del mouse. Por ejemplo, algunos navegadores resaltan texto cuando el
  usuario mueve el mouse mientras presiona un botón del mouse. Para evitar cualquier
  comportamiento predeterminado para este evento, agregue <code>return
  false;</code> al final de la función.</p>

  <p>Nota: <a href="/reference/p5/mousePressed">mousePressed()</a>,
  <code>mouseReleased()</code>,

  y <a href="/reference/p5/mouseClicked">mouseClicked()</a> estan todos relacionados.

  <a href="/reference/p5/mousePressed">mousePressed()</a> se ejecuta justo
  después de que el usuario presiona el mouse.
  <code>mouseReleased()</code> se ejecuta justo después de que el usuario
  libera el botón del mouse. <a href="/reference/p5/mouseClicked">mouseClicked()</a>

  se ejecuta justo después de que <code>mouseReleased()</code> se ejecute.</p>
