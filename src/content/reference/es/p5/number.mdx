 <p>Un número que puede ser positivo, negativo o cero.</p>

  <p>El tipo de datos <code>Number</code> es útil para describir valores como posición, tamaño y color. Un número puede ser un número entero como 20 o un número decimal como 12.34. Por ejemplo, la posición y el tamaño de un círculo pueden describirse con tres números:</p>

  <code>
    circle(50, 50, 20);
  </code>
  <code>
    circle(50, 50, 12.34);
  </code>

  <p>Los números admiten operaciones aritméticas básicas y siguen el orden estándar de operaciones: Paréntesis, Exponentes, Multiplicación, División, Suma y Resta. Por ejemplo, es común usar operadores aritméticos con variables del sistema de p5.js que son números:</p>

  <code>
    // Draw a circle at the center. 
    circle(width / 2, height / 2, 20);
  </code>
  <code>
    // Draw a circle that moves from left to right.
    circle(frameCount * 0.01, 50, 20);
  </code>

  <p>Aquí tienes una descripción rápida de los operadores aritméticos:</p>

  <code>
    1 + 2 // Add
    1 - 2 // Substract
    1 * 2 // Multiply
    1 / 2 // Divide
    1 % 2 // Remainder
    1 ** 2 // Exponentiate
  </code>

  <p>Es común actualizar una variable numérica usando aritmética. Por ejemplo, la ubicación de un objeto se puede actualizar así:</p>

  <code>
    x = x + 1;
  </code>

  <p>La declaración anterior suma 1 a la variable <code>x</code> usando el operador <code>+</code>. El operador de asignación de suma <code>+=</code> expresa la misma idea:</p>

  <code>
    x += 1;
  </code>

  <p>Aquí tienes una descripción rápida de los operadores de asignación:</p>

  <code>
    x += 2 // Addition assignment
    x -= 2 // Subtraction assignment 
    x *= 2 // Multiplication assignment
    x /= 2 // Division assignment
    x %= 2 // Remainder assignment
  </code>

  <p>Los números se pueden comparar usando los <a href="https://developer.mozilla.org/es/docs/Web/JavaScript/Reference/Operators/Comparison_Operators" target="_blank">operadores relacionales</a> <code>></code>, <code><</code>, <code>>=</code>, <code><=</code>, <code>===</code> y <code>!==</code>. Por ejemplo, el <a href="/reference/p5/frameCount">frameCount</a> de un boceto se puede usar como temporizador:</p>

  <code>
    if (frameCount > 1000) {
        text('Game over!', 50, 50);
    }
  </code>

  <p>Una expresión como <code>frameCount > 1000</code> evalúa a un valor <code>Boolean</code> que es <code>true</code> o <code>false</code>. Todos los operadores relacionales producen valores <code>Boolean</code>:</p>

  <code>
    2 > 1 // true
    2 < 1 // false
    2 >= 2 // true
    2 <= 2 // true
    2 === 2 // true
    2 !== 2 // false
  </code>

  <p>Consulta <a href="/reference/p5/Boolean">Boolean</a> para obtener más información sobre comparaciones y condiciones.</p>

  <p>Nota: También existen los operadores <code>==</code> y <code>!=</code> con un solo <code>=</code>. No los uses.</p>

  <p>Las expresiones con números también pueden producir valores especiales cuando algo sale mal:</p>

  <code>
    sqrt(-1) // NaN
    1 / 0 // Infinity
  </code>

  <p>El valor <code>NaN</code> significa <a href="https://developer.mozilla.org/es/docs/Web/JavaScript/Reference/Global_Objects/NaN" target="_blank">Not-A-Number</a>. <code>NaN</code> aparece cuando los cálculos o conversiones no funcionan. <code>Infinity</code> es un valor que es mayor que cualquier número. Aparece durante ciertos cálculos.</p>
