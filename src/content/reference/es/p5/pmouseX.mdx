description: >
  <p>Una variable del sistema <code>Number</code> que sigue la posición horizontal
  anterior

  del mouse.</p>

  <p>En modo 2D, <code>pmouseX</code> sigue la posición del mouse
  relativa a la

  esquina superior izquierda del lienzo. Su valor es

  <a href="/es/reference/p5/mouseX">mouseX</a> del último fotograma. Por ejemplo, si

  el mouse estaba a 50 pixeles de la parte izquierda del lienzo en el último
  fotograma, entonces <code>pmouseX</code> será 50.</p>

  <p>En modo WebGL, <code>pmouseX</code> sigue la posición del mouse
  relativa al centro

  del lienzo. Por ejemplo, si el mouse estaba a 50 pixeles a la derecha del
  centro del lienzo en el último fotograma, entonces <code>pmouseX</code> será
  50.</p>

  <p>Si se usa el touch en lugar del mouse, entonces <code>pmouseX</code>
  contendrá la coordenada x del punto de contacto más reciente.</p>

  <p>Nota: <code>pmouseX</code> se resetea a la
  <a href="/reference/es/p5/mouseX">mouseX</a> actual
  al inicio de cada evento de touch.</p>

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A line follows the mouse as it moves. The line grows longer with faster movements.');
    }

    function draw() {
      background(200);

      // Adjust coordinates for WebGL mode.
      // The origin (0, 0) is at the center of the canvas.
      let pmx = pmouseX - 50;
      let pmy = pmouseY - 50;
      let mx = mouseX - 50;
      let my = mouseY - 50;

      // Draw the line.
      line(pmx, pmy, mx, my);
    }
