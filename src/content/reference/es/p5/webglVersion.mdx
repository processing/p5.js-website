description: |-
  <p>La variable <code>webglVersion</code> devuelve la versión  de WebGL que está en uso.</p>
  <p>El valor que <code>webglVersion</code> devuelve es igual a una las siguientes cadenas de caracteres constantes:</p>
  <ul>
  "<li><code>WEBGL2</code> cuyo valor es <code>'webgl2'</code>,</li>"
  "<li><code>WEBGL</code> cuyo valor es <code>'webgl'</code>, o</li>"
  "<li><code>P2D</code> cuyo valor es <code>'p2d'</code>. Este es el valor predeterminado"
  para bocetos (sketches) 2D.</li>
  </ul>
  <p>Consulta <a href="./setAttributes">setAttributes()</a> para conocer formas de configurar la
  versión de WebGL.</p>

    function setup() {
      background(200);

      // Display the current WebGL version.
      text(webglVersion, 42, 54);

    function setup() {
      // Create a canvas using WEBGL mode.
      createCanvas(100, 50, WEBGL);
      background(200);

      // Display the current WebGL version.
      fill(0);
      textFont(font);
      text(webglVersion, -15, 5);

      describe('The text "webgl2" written in black on a gray background.');
    }
    </code>

    function preload() {

      font = loadFont('/assets/inconsolata.otf');
    }

    function setup() {
      // Create a canvas using WEBGL mode.
      createCanvas(100, 50, WEBGL);

      // Set WebGL to version 1.
      setAttributes({ version: 1 });

      background(200);

      // Display the current WebGL version.
      fill(0);
      textFont(font);
      text(webglVersion, -14, 5);

      describe('The text "webgl" written in black on a gray background.');
    }
    </code>
