description: >
  <p>Comienza a agregar vértices a una forma personalizada.</p>

  <p>Las funciones <code>beginShape()</code> y <a href="/reference/p5/endShape">endShape()</a> permiten crear formas personalizadas en 2D o 3D. <code>beginShape()</code> comienza a agregar vértices a una forma personalizada y <a href="/reference/p5/endShape">endShape()</a> detiene el proceso.</p>

  <p>El parámetro <code>kind</code> define el tipo de forma a crear. Por defecto, se puede dibujar cualquier polígono irregular. Los modos disponibles para el tipo de forma son:</p>

  <ul>
  <li><code>POINTS</code> para dibujar una serie de puntos.</li>
  <li><code>LINES</code> para dibujar una serie de segmentos de línea no conectados.</li>
  <li><code>TRIANGLES</code> para dibujar una serie de triángulos separados.</li>
  <li><code>TRIANGLE_FAN</code> para dibujar una serie de triángulos conectados que comparten el primer vértice en forma de abanico.</li>
  <li><code>TRIANGLE_STRIP</code> para dibujar una serie de triángulos conectados en forma de tira.</li>
  <li><code>QUADS</code> para dibujar una serie de cuadriláteros separados.</li>
  <li><code>QUAD_STRIP</code> para dibujar una tira de cuadriláteros utilizando bordes adyacentes para formar el siguiente cuadrilátero.</li>
  <li><code>TESS</code> para crear una curva de relleno mediante teselación explícita (solo para WebGL).</li>
  </ul>

  <p>Después de llamar a <code>beginShape()</code>, se pueden construir formas llamando a <a href="/reference/p5/vertex">vertex()</a>, <a href="/reference/p5/bezierVertex">bezierVertex()</a>, <a href="/reference/p5/quadraticVertex">quadraticVertex()</a>, y/o <a href="/reference/p5/curveVertex">curveVertex()</a>. Llamar a <a href="/reference/p5/endShape">endShape()</a> detendrá la adición de vértices a la forma. Cada forma se delineará con el color de trazo actual y se rellenará con el color de relleno actual.</p>

  <p>Las transformaciones como <a href="/reference/p5/translate">translate()</a>, <a href="/reference/p5/rotate">rotate()</a>, y <a href="/reference/p5/scale">scale()</a> no funcionan entre <code>beginShape()</code> y <a href="/reference/p5/endShape">endShape()</a>. Tampoco es posible usar otras formas, como <a href="/reference/p5/ellipse">ellipse()</a> o <a href="/reference/p5/rect">rect()</a>, entre <code>beginShape()</code> y <a href="/reference/p5/endShape">endShape()</a>.</p>
