description: |-
  <p>Guarda el lienzo actual como una imagen.</p>

  <p>Por defecto, <code>saveCanvas()</code> guarda el lienzo como una imagen PNG llamada <code>untitled.png</code>.</p>

  "<p>El primer parámetro, <code>filename</code>, es opcional. Es una cadena que establece el nombre del archivo. Si se incluye una extensión de archivo, como en <code>saveCanvas('drawing.png')</code>, entonces la imagen se guardará utilizando ese formato.</p>"

  "<p>El segundo parámetro, <code>extension</code>, también es opcional. Establece el formato de los archivos. Se pueden usar <code>'png'</code> o <code>'jpg'</code>. Por ejemplo, <code>saveCanvas('drawing', 'jpg')</code> guarda el lienzo en un archivo llamado <code>drawing.jpg</code>.</p>"

  <p>Nota: El navegador guardará el archivo inmediatamente o pedirá al usuario que lo haga a través de una ventana de diálogo.</p>
