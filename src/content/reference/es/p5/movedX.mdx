description: >
  <p>Una variable de sistema <code>Number</code> que sigue el movimiento horizontal del ratón.</p>

  <p><code>movedX</code> rastrea el número de píxeles que el ratón se mueve hacia la derecha o hacia la izquierda entre fotogramas. <code>movedX</code> tendrá un valor negativo si el ratón se mueve hacia la izquierda entre fotogramas y un valor positivo si se mueve hacia la derecha.</p>

  <p><code>movedX</code> puede ser calculado como <code>mouseX - pmouseX</code>.</p>

  <p>Nota: <code>movedX</code> sigue actualizándose incluso cuando <a href="/reference/p5/requestPointerLock">requestPointerLock()</a> está activo.</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square. The text ">>" appears when the user moves the mouse to the right. The text "<<" appears when the user moves the mouse to the left.'
      );
    }

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display >> when movedX is positive and
      // << when it's negative.
      if (movedX > 0) {
        text('>>', 50, 50);
      } else if (movedX < 0) {
        text('<<', 50, 50);
      }
    }
