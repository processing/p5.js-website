description: >
  <p>Bloquea el puntero del ratón en su posición actual y lo hace invisible.</p>

  <p><code>requestPointerLock()</code> permite que el ratón se mueva para siempre
  sin salir de la

  pantalla. Llamar a <code>requestPointerLock()</code> bloquea los valores de

  <a href="/reference/p5/mouseX">mouseX</a>, <a
  href="/reference/p5/mouseY">mouseY</a>,

  <a href="/reference/p5/pmouseX">pmouseX</a>, y <a
  href="/reference/p5/pmouseY">pmouseY</a>.

  <a href="/reference/p5/movedX">movedX</a> y <a
  href="/reference/p5/movedY">movedY</a>

  se continúan actualizando y pueden ser utilizados para obtener la distancia que el ratón
  se movió desde la última vez que se dibujó un fotograma. Llamar a
  <a href="/reference/p5/exitPointerLock">exitPointerLock()</a> retoma la
  actualización de las

  variables de sistema del ratón.</p>

  <p>Nota: La mayoría de los navegadores requieren una entrada, como un clic, antes de
  llamar a <code>requestPointerLock()</code>. Se recomienda llamar a
  <code>requestPointerLock()</code> en una función de evento como <a
  href="/reference/p5/doubleClicked">doubleClicked()</a>.</p>

    let score = 0;

    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square with the text "Score: X" at its center. The score increases when the user moves the mouse upward. It decreases when the user moves the mouse downward.'
      );
    }

    function draw() {
      background(200);

      // Update the score.
      score -= movedY;

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display the score.
      text(`Score: ${score}`, 50, 50);
    }

    // Lock the pointer when the user double-clicks.
    function doubleClicked() {
      requestPointerLock();
    }
