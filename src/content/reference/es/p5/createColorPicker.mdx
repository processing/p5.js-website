description: >
  <p>Crea un elemento de selector de color.</p>

  <p>El parámetro, <code>value</code>, es opcional. Si se pasa una cadena de color o

  un objeto <a href="/reference/p5/p5.Color">p5.Color</a>, se establecerá el
  color 
  
  predeterminado.</p>

  <p>Los selectores de color extienden la clase <a href="/reference/p5/p5.Element">p5.Element</a>
  con un par de 
  
  métodos útiles para gestionar colores:</p>

  <ul>

  <li><code>mySelector.value()</code> devuelve el color actual como una cadena hexadecimal en
  el formato <code>'#rrggbb'</code>.</li>

  <li><code>mySelector.color()</code> devuelve el color actual como un objeto <a
  href="/reference/p5/p5.Color">p5.Color</a>.</li>

  </ul>
