description: >
  <p>Una variable de sistema <code>Number</code> que sigue la posición
  vertical del

  ratón.</p>

  <p>En modo 2D, <code>pmouseY</code> mantiene un registro de la posición del
  ratón relativa a la

  esquina superior izquierda del lienzo. Su valor es

  <a href="/reference/p5/mouseY">mouseY</a> en el fotograma anterior. Por
  ejemplo, si

  el ratón estaba a 50 píxeles del borde superior del lienzo durante el
  fotograma anterior, entonces <code>pmouseY</code> será 50.</p>

  <p>En modo WebGL, <code>pmouseY</code> mantiene un registro de la posición del
  ratón relativa al

  centro del lienzo. Por ejemplo, si el ratón estaba a 50 píxeles debajo del
  centro del lienzo durante el

  fotograma anterior, entonces <code>pmouseY</code> será 50.</p>

  <p>Si se utiliza en lugar del ratón el touch, entonces <code>pmouseY</code>
  mantendrá la

  coordenada y del último punto de contacto.</p>

  <p>Nota: <code>pmouseY</code> se restablece al valor actual de
  <a href="/reference/p5/mouseY">mouseY</a>

  al inicio de cada evento de touch.</p>

    function setup() {

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A line follows the mouse as it moves. The line grows longer with faster movements.');
    }

    function draw() {
      background(200);

      // Adjust coordinates for WebGL mode.
      // The origin (0, 0) is at the center of the canvas.
      let pmx = pmouseX - 50;
      let pmy = pmouseY - 50;
      let mx = mouseX - 50;
      let my = mouseY - 50;

      // Draw the line.
      line(pmx, pmy, mx, my);
    }
    </code>
