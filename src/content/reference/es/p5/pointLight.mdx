description: >
  <p>Crea una luz que brilla desde un punto en todas las direcciones.</p>

  <p>Las luces de punto son como bombillas que brillan en todas las direcciones. Se pueden colocar en diferentes posiciones para lograr diferentes efectos de iluminación. Se pueden activar un máximo de 5 luces de punto a la vez.</p>

  <p>Hay cuatro formas de llamar a <code>pointLight()</code> con parámetros para establecer el color y la posición de la luz.</p>

  <p>La primera forma de llamar a <code>pointLight()</code> tiene seis parámetros. Los primeros tres parámetros, <code>v1</code>, <code>v2</code> y <code>v3</code>, establecen el color de la luz utilizando el <a href="/reference/p5/colorMode">colorMode()</a> actual. Los últimos tres parámetros, <code>x</code>, <code>y</code> y <code>z</code>, establecen la posición de la luz. Por ejemplo, <code>pointLight(255, 0, 0, 50, 0, 0)</code> crea una luz roja <code>(255, 0, 0)</code> que brilla desde las coordenadas <code>(50, 0, 0)</code>.</p>

  <p>La segunda forma de llamar a <code>pointLight()</code> tiene cuatro parámetros. Los primeros tres parámetros, <code>v1</code>, <code>v2</code> y <code>v3</code>, establecen el color de la luz utilizando el <a href="/reference/p5/colorMode">colorMode()</a> actual. El último parámetro, <code>position</code>, establece la posición de la luz utilizando un objeto <a href="/reference/p5/p5.Vector">p5.Vector</a>. Por ejemplo, <code>pointLight(255, 0, 0, lightPos)</code> crea una luz roja <code>(255, 0, 0)</code> que brilla desde la posición establecida por el vector <code>lightPos</code>.</p>

  <p>La tercera forma de llamar a <code>pointLight()</code> tiene cuatro parámetros. El primer parámetro, <code>color</code>, establece el color de la luz utilizando un objeto <a href="/reference/p5/p5.Color">p5.Color</a> o un arreglo de valores de color. Los últimos tres parámetros, <code>x</code>, <code>y</code> y <code>z</code>, establecen la posición de la luz. Por ejemplo, <code>directionalLight(myColor, 50, 0, 0)</code> crea una luz que brilla desde las coordenadas <code>(50, 0, 0)</code> con el valor de color de <code>myColor</code>.</p>

  <p>La cuarta forma de llamar a <code>pointLight()</code> tiene dos parámetros. El primer parámetro, <code>color</code>, establece el color de la luz utilizando un objeto <a href="/reference/p5/p5.Color">p5.Color</a> o un arreglo de valores de color. El segundo parámetro, <code>position</code>, establece la posición de la luz utilizando un objeto <a href="/reference/p5/p5.Vector">p5.Vector</a>. Por ejemplo, <code>directionalLight(myColor, lightPos)</code> crea una luz que brilla desde la posición establecida por el vector <code>lightPos</code> con el valor de color de <code>myColor</code>.</p>

    // Click and drag the mouse to view the scene from different angles.
    // Double-click to turn on the point light.

    let isLit = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A sphere drawn on a gray background. A red light starts shining from above when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Control the light.
      if (isLit === true) {
        // Add a red point light from above.
        // Use RGB values and XYZ coordinates.
        pointLight(255, 0, 0, 0, -150, 0);
      }

      // Style the sphere.
      noStroke();

      // Draw the sphere.
      sphere(30);
    }

    // Turn on the point light when the user double-clicks.
    function doubleClicked() {
      isLit = true;
    }
