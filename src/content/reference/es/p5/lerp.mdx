description: "<p><code>lerp()</code> calcula un número que se encuentra entre dos números, con un incremento específico.</p>\n<p>Esta función toma tres parámetros. El parámetro <code>start</code> define el primer número, mientras que el parámetro <code>stop</code> define el segundo número.  El tercer parámetro <code>amt</code> define la cantidad con la cual se va a interpolar  (encontrar un valor) entre los dos números <code>start</code> y <code>stop</code>.</p> \n<ul>\n<li>Si el valor de <code>amt</code> es 0.0, el resultado será igual al número <code>start</code>.</li>\n<li>Si el valor de <code>amt</code> es 0.1, el resultado será un número muy cercano al número <code>start</code>.</li>\n<li>Si el valor de <code>amt</code> es 0.5, el resultado será la mitad exacta entre el número <code>start</code> y el número </code>stop</code>.</li>\n<li>Si el valor de <code>amt</code> es 1, el resultado será igual al número <code>stop</code>.</li>\n<li>Si el valor de <code>amt</code> es menor que 0 o mayor que 1, <code>lerp()</code> devolverá un número fuera del rango original. Por ejemplo, si llamamos <code>lerp(0, 10, 1.5)</code>, obtendremos 15 como resultado.</li>\n</ul>\n<p>La función <code>lerp()</code> es conveniente para crear movimiento a lo largo de un camino recto, y para dibujar líneas punteadas.</p>"
