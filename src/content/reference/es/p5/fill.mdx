description: >
  <p>Establece el color que se va a utilizar para rellenar las formas.</p>
  <p>Llamar a <code>fill(255, 165, 0)</code> o <code>fill('orange')</code> significa que todas las formas dibujadas después del comando de relleno estarán rellenas con el color naranja.</p>
  <p>La versión de <code>fill()</code> con un parámetro interpreta el valor de tres maneras. Si el parámetro es un <code>Number</code> (número), se interpreta como un valor de escala de grises. Si el parámetro es un <code>String</code> (cadena de caracteres), se interpreta como una cadena de color CSS (colores predefinidos de CSS). También se puede proporcionar un objeto <a href="/reference/p5/p5.Color">p5.Color</a> para establecer el color de relleno.</p>
  <p>La versión de <code>fill()</code> con tres parámetros los interpreta como colores RGB, HSB o HSL, dependiendo del modo de color actual (<a href="/reference/p5/colorMode">colorMode()</a>). El espacio de color predeterminado es RGB, con cada valor en el rango de 0 a 255.</p>
