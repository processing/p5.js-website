description: >
  <p>Crea un elemento <code><a></a></code> que enlaza a otra página web.</p>

  <p>El primer parámetro, <code>href</code>, es una cadena que establece la URL de la 
  página 
  
  enlazada.</p>

  <p>El segundo parámetro, <code>html</code>, es una cadena que establece el HTML 
  interno del 
  
  enlace. Es común usar texto, imágenes o botones como enlaces.</p>

  <p>El tercer parámetro, <code>target</code>, es opcional. Es una cadena que 
  indica al 
  
  navegador web dónde abrir el enlace. Por defecto, los enlaces se abren en la pestaña 
  
  actual del navegador. Pasar <code>'_blank'</code> hará que el enlace se abra en una nueva 
  pestaña 
  
  del navegador. MDN describe algunas 
  
  <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a#target"
  target="_blank">otras opciones</a>.</p>
