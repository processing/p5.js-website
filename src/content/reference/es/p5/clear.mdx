description: >
  <p>El método <code>clear()</code> hace que cada píxel sea 100% transparente. Llamar a <code>clear()</code> no elimina los objetos creados por las funciones <code>createX()</code> como por ejemplo <a href="/reference/p5/createGraphics">createGraphics()</a> , <a href="/reference/p5/createVideo">createVideo()</a> , y <a href="/reference/p5/createImg">createImg()</a>. Estos objetos permanecerán sin cambios después de llamar a <code>clear()</code> y pueden ser dibujados nuevamente.</p>
  <p>En modo WebGL, esta función puede borrar la pantalla a un color específico. El método <code>clear()</code> puede tomar cuatro parámetros numéricos que interpreta como valores de color RGBA normalizados. También borra el búfer de profundidad.</p>
  <p> Si no estás utilizando el renderizador WebGL, estos parámetros no tendrán efecto.</p>
