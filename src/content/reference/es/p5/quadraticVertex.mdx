description: >
  <p>Agrega un segmento de curva Bézier cuadrática a una forma personalizada.</p>

  <p><code>quadraticVertex()</code> agrega un segmento curvado a formas personalizadas. Los segmentos de curvas Bézier que crea son similares a los que se hacen con la función <a href="/reference/p5/bezierVertex">bezierVertex()</a>. <code>quadraticVertex()</code> debe llamarse entre las funciones <a href="/reference/p5/beginShape">beginShape()</a> y <a href="/reference/p5/endShape">endShape()</a>. El segmento curvado utiliza el vértice anterior como primer punto de anclaje, por lo que debe haber al menos una llamada a <a href="/reference/p5/vertex">vertex()</a> antes de que se pueda usar <code>quadraticVertex()</code>.</p>

  <p>Los dos primeros parámetros, <code>cx</code> y <code>cy</code>, establecen el punto de control de la curva. El punto de control "tira" de la curva hacia él.</p>

  <p>Los dos últimos parámetros, <code>x3</code> y <code>y3</code>, establecen el último punto de anclaje. El último punto de anclaje es donde termina la curva.</p>

  <p>Las curvas Bézier también se pueden dibujar en 3D utilizando el modo WebGL. La versión 3D de <code>bezierVertex()</code> tiene ocho argumentos porque cada punto tiene coordenadas x, y y z.</p>

  <p>Nota: <code>quadraticVertex()</code> no funcionará cuando se pasa un argumento a <a href="/reference/p5/beginShape">beginShape()</a>.</p>

    // Click the mouse near the red dot in the top-right corner
    // and drag to change the curve's shape.

    let x2 = 80;
    let y2 = 20;
    let isChanging = false;

    function setup() {
      createCanvas(100, 100);

      describe('A black curve that starts at the top-left corner and ends at the center. Its anchor and control points are marked with dots. Red lines connect both anchor points to the control point.');
    }

    function draw() {
      background(200);

      // Style the curve.
      noFill();
      strokeWeight(1);
      stroke(0);

      // Draw the curve.
      beginShape();
      vertex(20, 20);
      quadraticVertex(x2, y2, 50, 50);
      endShape();

      // Draw red lines from the anchor points to the control point.
      stroke(255, 0, 0);
      line(20, 20, x2, y2);
      line(50, 50, x2, y2);

      // Draw the anchor points in black.
      strokeWeight(5);
      stroke(0);
      point(20, 20);
      point(50, 50);

      // Draw the control point in red.
      stroke(255, 0, 0);
      point(x2, y2);
    }

    // Start changing the first control point if the user clicks near it.
    function mousePressed() {
      if (dist(mouseX, mouseY, x2, y2) < 20) {
        isChanging = true;
      }
    }

    // Stop changing the first control point when the user releases the mouse.
    function mouseReleased() {
      isChanging = false;
    }

    // Update the first control point while the user drags the mouse.
    function mouseDragged() {
      if (isChanging === true) {
        x2 = mouseX;
        y2 = mouseY;
      }
    }
