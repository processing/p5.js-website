  <p>Un grupo de declaraciones con nombre.</p>
  <p><a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions" target="_blank">Las funciones</a> ayudan a organizar y reutilizar código. Por ejemplo, las funciones hacen que sea fácil expresar la idea "Dibujar una flor.":</p>
  <code> function drawFlower() { // Style the text. textAlign(CENTER, CENTER); textSize(20); <p> // Draw a flower emoji. text('🌸', 50, 50); } </p></code>
  <p>El encabezado de la función comienza con la palabra clave <code>function</code>. El nombre de la función, <code>drawFlower</code>, va seguido de paréntesis <code>()</code> y llaves <code>{}</code>. El código entre las llaves corresponde al cuerpo de la función. El cuerpo de la función se ejecuta cuando la función es llamada de la siguiente manera: </p>
  <code> drawFlower(); </code>
  <p>Las funciones pueden aceptar entradas agregando parámetros a sus encabezados. Los parámetros son marcadores de posición para los valores que se proporcionarán cuando se llame a la función. Por ejemplo, la función <code>drawFlower()</code>, podría incluir un parámetro para definir el tamaño de la flor:</p>
  <code>
  function drawFlower(size) { 
  // Style the text. textAlign(CENTER, CENTER); 
  <p> // Use the size parameter. textSize(size);</p> 
  <p> // Draw a flower emoji. text('🌸', 50, 50); } </p></code>
  <p> Los parámetros son parte de la declaración de la función. Los argumentos son proporcionados por el código que llama a una función. Cuando se llama a una función, los argumentos se asignan a los parámetros:</p>
  <code> // The argument 20 is assigned to the parameter size. drawFlower(20); </code>	
  <p>Las funciones pueden tener varios parámetros separados por comas, y los parámetros pueden ser de cualquier tipo de datos. Por ejemplo, la función <code>drawFlower()</code> podría aceptar parámetros de tipo <code>Number</code> (número) para las coordenadas-x y las coordenadas-y de la flor junto con su tamaño:</p>
  <code> function drawFlower(x, y, size) { // Style the text. textAlign(CENTER, CENTER); <p> // Use the size parameter. textSize(size);</p> <p> // Draw a flower emoji. // Use the x and y parameters. text('🌸', x, y); } </p></code>
  <p>Las funciones también pueden producir salidas agregando una declaración <code>return</code></p>
  <code> function double(x) { let answer = 2 * x; return answer; } </code>
  <p>La expresión que sigue a <code>return</code> puede producir una salida que se utiliza en otro lugar. Por ejemplo, la salida de la función <code>double()</code> se puede asignar a una variable:</p>
  <code> let six = double(3); text(`3 x 2 = ${six}`, 50, 50); </code>
