description: >
  <p>Dibuja texto en el lienzo.</p>
  <p>El primer parámetro, <code>str</code>, es el texto que se va a dibujar. Los segundo y tercer parámetros, <code>x</code> e <code>y</code>, establecen las coordenadas de la esquina inferior izquierda del texto. Consulta <a href="/reference/p5/textAlign">textAlign()</a> para otras formas de alinear el texto.</p>
  <p>Los cuarto y quinto parámetros, <code>maxWidth</code> y <code>maxHeight</code>, son opcionales. Establecen las dimensiones del rectángulo invisible que contiene el texto. Por defecto, establecen su ancho y alto máximos. Consulta <a href="/reference/p5/rectMode">rectMode()</a> para otras formas de definir la caja de texto rectangular. El texto se ajustará para encajar dentro de la caja de texto. El texto fuera de la caja no se dibujará.</p>
  <p>El texto puede tener algunos estilos. Llama a la función <a href="/reference/p5/fill">fill()</a> para establecer el color de relleno del texto. Llama a <a href="/reference/p5/stroke">stroke()</a> y <a href="/reference/p5/strokeWeight">strokeWeight()</a> para establecer el contorno del texto. Llama a <a href="/reference/p5/textSize">textSize()</a> y <a href="/reference/p5/textFont">textFont()</a> para establecer el tamaño y la fuente del texto, respectivamente.</p>
  <p>Nota: El modo <code>WEBGL</code> solo admite fuentes cargadas con <a href="/reference/p5/loadFont">loadFont()</a>. Llamar a <a href="/reference/p5/stroke">stroke()</a> no tiene efecto en el modo <code>WEBGL</code>.</p>
