description: >
  <p>Establece el color a utilizar para el fondo del lienzo.</p>
  <p>Por defecto, el fondo es transparente. <code>background()</code> se utiliza típicamente dentro de <a href="/reference/p5/draw">draw()</a> para limpiar la ventana de visualización al principio de cada cuadro (fotograma). También se puede usar dentro de <a href="/reference/p5/setup">setup()</a> para establecer el fondo en el primer cuadro (fotograma) de la animación.</p>
  <p>La versión de <code>background()</code> con un parámetro interpreta el valor de cuatro maneras diferentes. Si el parámetro es un <code>Number</code> (número), se interpreta como un valor de escala de grises. Si el parámetro es un <code>String</code> (cadena de caracteres), se interpreta como una cadena de caracteres de color de CSS (colores predefinidos de CSS). Se admiten cadenas de colores RGB, RGBA, HSL, HSLA, hexadecimal y colores nombrados. Si el parámetro es un objeto <a href="/reference/p5/p5.Color">p5.Color</a>, se utilizará como el color de fondo. Si el parámetro es un objeto <a href="/reference/p5/p5.Image">p5.Image</a>, se utilizará como la imagen de fondo.</p>
  <p>La versión de <code>background()</code> con dos parámetros interpreta el primero como un valor de escala de grises. El segundo parámetro establece el valor alfa (transparencia).</p>
  <p>La versión de <code>background()</code> con tres parámetros los interpreta como colores RGB, HSB o HSL, dependiendo del modo de color actual (<a href="/reference/p5/colorMode">colorMode()</a>). Por defecto, los colores se especifican en valores RGB. Llamar a <code>background(255, 204, 0)</code> establece el fondo a un color amarillo brillante.</p>
