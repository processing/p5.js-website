description: >
  <p>Crea un objeto <a href="/reference/p5/p5.Color">p5.Color</a>.</p>

  <p>Por defecto, los parámetros se interpretan como valores RGB. Llamar a

  <code>color(255, 204, 0)</code> devolverá un color amarillo brillante. La forma
  en que

  se interpretan estos parámetros puede cambiarse con la función

  <a href="/reference/p5/colorMode">colorMode()</a>.</p>

  <p>La versión de <code>color()</code> con un parámetro interpreta el valor de
  dos maneras.

  Si el parámetro es un número, se interpreta como un valor de escala de grises.

  Si el parámetro es una cadena, se interpreta como una cadena de color CSS.</p>

  <p>La versión de <code>color()</code> con dos parámetros interpreta el
  primero como un

  valor de escala de grises. El segundo parámetro establece el valor alfa (transparencia).</p>

  <p>La versión de <code>color()</code> con tres parámetros los interpreta como
  colores RGB, HSB,

  o HSL, dependiendo del <code>colorMode()</code> actual.</p>

  <p>La versión de <code>color()</code> con cuatro parámetros los interpreta como
  colores RGBA, HSBA,

  o HSLA, dependiendo del <code>colorMode()</code> actual. El último 
  parámetro

  establece el valor alfa (transparencia).</p>
