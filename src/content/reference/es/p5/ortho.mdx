description: >
  <p><code>ortho()</code> establece una proyección ortográfica para la cámara actual en un

  boceto 3D. En una proyección ortográfica, las formas con el mismo tamaño siempre

  aparecen con el mismo tamaño, independientemente de si están cerca o lejanas

  respecto a la cámara.</p>

  <p><code>ortho()</code> cambia la perspectiva de la cámara al cambiar su
  frustum de vision

  de un pirámide truncada a un prisma rectangular. La cámara se coloca delante del

  frustum y muestra todo lo que hay entre el plano cercano del frustum y su plano

  lejano. <code>ortho()</code> tiene seis parámetros opcionales para definir el

  frustum.</p>

  <p>Los primeros cuatro parámetros, <code>left</code>, <code>right</code>,
  <code>bottom</code> y <code>top</code>, establecen las coordenadas de

  los lados del frustum como son la parte superior e inferior. Por ejemplo, llamar a <code>ortho(-100, 100, 200, -200)</code>
  crea un frustum que mide 200 pixeles de ancho y

  400 pixeles de alto. Por defecto, estos parámetros están establecidos

  basados en el ancho y el alto del boceto, como en

  <code>ortho(-ancho / 2, ancho / 2, -alto / 2, alto / 2)</code>.</p>

  <p>Los últimos dos parámetros, <code>near</code> y <code>far</code>, establecen la
  distancia del plano cercano del frustum y del plano lejano respectivamente. Por

  ejemplo, llamar a <code>ortho(-100, 100, 200, 200, 50, 1000)</code> crea un
  frustum que mide 200 pixeles de ancho, 400 pixeles de alto, comienza 50 pixeles
  desde la cámara y termina 1,000 pixeles desde la cámara. Por defecto, <code>near</code>
  y <code>far</code> están establecidos en 0 y

  <code>max(ancho, alto) + 800</code>, respectivamente.</p>

  <p>Nota: <code>ortho()</code> solo puede ser utilizado en modo WebGL.</p>

    function setup() {

      describe('A white cube on a gray background.');

    function draw() {
      background(200);

      // Apply an orthographic projection.
      // Center the frustum.
      // Set its width and height to 20.
      // Place its near plane 300 pixels from the camera.
      // Place its far plane 350 pixels from the camera.
      ortho(-10, 10, -10, 10, 300, 350);

      // Translate the origin toward the camera.
      translate(-10, 10, 600);

      // Rotate the coordinate system.
      rotateY(-0.1);
      rotateX(-0.1);

      // Draw the row of boxes.
