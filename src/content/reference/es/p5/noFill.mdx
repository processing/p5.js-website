  <p>Deshabilita la configuración del color de relleno para las formas.</p>
  <p>Llamar a <code> noFill()</code> es lo mismo que hacer que el relleno sea completamente transparente, por ejemplo: <code>fill(0, 0)</code>. Si se llama a <a href="/reference/p5/noStroke">noStroke()</a> al igual que a <code>noFill()</code>, no se dibujará nada en la pantalla.</p>

    function setup() {

      noFill();
