description: >
  <p>Obtiene el valor de tono de un color.</p>

  <p><code>hue()</code> extrae el valor de tono de un objeto

  <a href="/reference/p5/p5.Color">p5.Color</a>, un arreglo de componentes de 
  color, o

  una cadena de color CSS.</p>

  <p>El tono describe la posición de un color en la rueda de colores. Por defecto,
  <code>hue()</code>

  devuelve el tono HSL de un color en el rango 0 a 360. Si el

  <a href="/reference/colorMode">colorMode()</a> está establecido en HSB o HSL, devuelve
  el valor de tonalidad

  en el modo dado.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Use HSL color.
      colorMode(HSL);

      // Create a p5.Color object.
      let c = color(0, 50, 100);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 20, 35, 60);

      // Set 'hueValue' to 0.
      let hueValue = hue(c);

      // Draw the right rectangle.
      fill(hueValue);
      rect(50, 20, 35, 60);

      describe(
