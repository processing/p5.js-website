description: >
  <p>Una variable de sistema <code>Number</code> que sigue el movimiento vertical del ratón.</p>

  <p><code>movedY</code> rastrea el número de píxeles que el ratón se mueve hacia arriba o hacia abajo entre fotogramas. <code>movedY</code> tendrá un valor negativo si el ratón se mueve hacia arriba entre fotogramas y un valor positivo si se mueve hacia abajo.</p>

  <p><code>movedY</code> puede ser calculado como <code>mouseY - pmouseY</code>.</p>

  <p>Nota: <code>movedY</code> sigue actualizándose incluso cuando <a href="/reference/p5/requestPointerLock">requestPointerLock()</a> está activo.</p>

    function setup() {
      createCanvas(100, 100);

        'A gray square. The text "▲" appears when the user moves the mouse upward. The text "▼" appears when the user moves the mouse downward.'
      );
    }

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display ▼ when movedY is positive and
      // ▲ when it's negative.
      if (movedY > 0) {
        text('▼', 50, 50);
      } else if (movedY < 0) {
        text('▲', 50, 50);
