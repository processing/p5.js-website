  <p>Ajusta la forma en que <a href="/reference/p5/curve">curve()</a> y 
  <a href="/reference/p5/curveVertex">curveVertex()</a> dibujan.</p>

  <p>Las curvas spline son como cables que están sujetos a un conjunto de puntos. 
  <code>curveTightness()</code> ajusta qué tan firmemente el cable está sujeto a los 
  puntos.</p>

  <p>El parámetro <code>tightness</code> determina cómo la curva se ajusta a los 
  puntos vértices. Por defecto, <code>tightness</code> está configurado en 0. 
  Establecer <code>tightness</code> en 1, como en <code>curveTightness(1)</code>, 
  conecta los puntos de la curva usando líneas rectas. Los valores en el rango de 
  –5 a 5 deforman las curvas manteniéndolas reconocibles.</p>

    <div>
    <code>
    // Move the mouse left and right to see the curve change.

    function setup() {
      createCanvas(100, 100);

      describe('A black curve forms a sideways U shape. The curve deforms as the user moves the mouse from left to right');
    }

    function draw() {
      background(200);

      // Set the curve's tightness using the mouse.
      let t = map(mouseX, 0, 100, -5, 5, true);
      curveTightness(t);

      // Draw the curve.
      noFill();
      beginShape();
      curveVertex(10, 26);
      curveVertex(10, 26);
      curveVertex(83, 24);
      curveVertex(83, 61);
      curveVertex(25, 65);
      curveVertex(25, 65);
      endShape();
    }
