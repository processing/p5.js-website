description: >
  <p>Una función que se ejecuta una vez cuando se presiona cualquier tecla.</p>

  <p>Declarar la función <code>keyPressed()</code> establece un bloque de código para

  ejecutarse automáticamente cuando el usuario presiona cualquier tecla:</p>

  <code>

  function keyPressed() {
    // Code to run.
  }

  </code>


  <p>Las variables <a href="/reference/p5/key">key</a> y <a
  href="/reference/p5/keyCode">keyCode</a>

  se actualizarán con el valor más reciente que se ha digitado cuando

  <code>keyPressed()</code> es llamada por p5.js:</p>

  <code>

  function keyPressed() {
    if (key === 'c') {
      // Code to run.
    }

  <p>  if (keyCode === ENTER) {
      // Code to run.
    }
  }

  </p></code>

  <p>El parámetro, <code>event</code>, es opcional. <code>keyPressed()</code> siempre
  se le pasa un objeto

  <a href="https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent"
  target="_blank">KeyboardEvent</a>

  con propiedades que describen el evento de presión de tecla:</p>

  <code>

  function keyPressed(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>Los navegadores pueden tener comportamientos predeterminados asociados a

  diferentes eventos de teclado. Por ejemplo, algunos navegadores pueden saltar
  al final de una página web cuando se presiona la tecla <code>SPACE</code>. Para
  evitar cualquier comportamiento predeterminado para este evento, añada
  <code>return false;</code> al final de la función.</p>
