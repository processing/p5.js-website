description: >
  <p>Una lista que mantiene diferentes tipos de datos en orden</p>
  <p>Los arreglos son útiles para almacenar datos relacionados. Pueden contener datos de cualquier tipo. Por ejemplo, un arreglo podría contener una lista de colores favoritos de alguien como cadenas de caracteres. Los arreglos se crean de la siguiente manera:</p>
  <code> let colors = ['deeppink', 'darkorchid', 'magenta']; </code>
  <p>Cada dato dentro de un arreglo se llama elemento. Cada elemento tiene una dirección, o índice, dentro de su arreglo. La variable <code>colors</code> se refiere a un arreglo con tres elementos de tipo <a href="/reference/p5/String">cadena de caracteres</a>, <code>'deeppink'</code>, <code>'darkorchid'</code>, y <code>'magenta'</code>. Los arreglos empiezan con índice cero, lo que significa que <code>'deeppink'</code> está en el índice 0, <code>'darkorchid'</code> está en el índice 1, y <code>'magenta'</code> está en el índice 2. Los elementos del arreglo se pueden acceder utilizando sus índices de la siguiente manera:</p>
  <code> let zeroth = colors[0]; // 'deeppink' let first = colors[1]; // 'darkorchid' let second = colors[2]; // 'magenta' </code>
  <p>Se pueden agregar elementos al final de un arreglo llamando al método <code>push()</code> de la siguiente manera:</p>
  <code> colors.push('lavender'); <p>let third = colors[3]; // 'lavender' </p></code>
  <p>Consulta <a href="https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/Array" target="_blank">MDN</a> para obtener más información sobre arreglos.</p>
