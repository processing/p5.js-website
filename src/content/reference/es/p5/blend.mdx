description: |-
  <p>Copia una región de píxeles de una imagen a otra.</p>

  <p>El primer parámetro, <code>srcImage</code>, es el objeto <a href="/reference/p5/p5.Image">p5.Image</a> para mezclar.</p>

  <p>Los siguientes cuatro parámetros, <code>sx</code>, <code>sy</code>, <code>sw</code> y <code>sh</code>, determinan la región de la cual mezclar la imagen de origen. <code>(sx, sy)</code> es la esquina superior izquierda de la región. <code>sw</code> y <code>sh</code> son el ancho y la altura de la región.</p>

  <p>Los siguientes cuatro parámetros, <code>dx</code>, <code>dy</code>, <code>dw</code> y <code>dh</code>, determinan la región del lienzo en la cual mezclar. <code>(dx, dy)</code> es la esquina superior izquierda de la región. <code>dw</code> y <code>dh</code> son el ancho y la altura de la región.</p>

  <p>El décimo parámetro, <code>blendMode</code>, establece el efecto utilizado para mezclar los colores de las imágenes. Las opciones son <code>BLEND</code>, <code>DARKEST</code>, <code>LIGHTEST</code>, <code>DIFFERENCE</code>, <code>MULTIPLY</code>, <code>EXCLUSION</code>, <code>SCREEN</code>, <code>REPLACE</code>, <code>OVERLAY</code>, <code>HARD_LIGHT</code>, <code>SOFT_LIGHT</code>, <code>DODGE</code>, <code>BURN</code>, <code>ADD</code> o <code>NORMAL</code>.</p>

    <code>
    let img0;
    let img1;

    // Load the images.
    function preload() {
      img0 = loadImage('/assets/rockies.jpg');
      img1 = loadImage('/assets/bricks_third.jpg');
    }

    function setup() {
      createCanvas(100, 100);

      // Use the mountains as the background.
      background(img0);

      // Display the bricks.
      image(img1, 0, 0);

      // Display the bricks faded into the landscape.
      blend(img1, 0, 0, 33, 100, 67, 0, 33, 100, LIGHTEST);

      describe('A wall of bricks in front of a mountain landscape. The same wall of bricks appears faded on the right of the image.');
    }
