description: >
  <p><code>String</code> constante que se utiliza para establecer el

  <a href="/reference/p5/angleMode">angleMode()</a>.</p>

  <p>Por defecto, funciones como <a href="/reference/p5/rotate">rotate()</a>
  y

  <a href="/reference/p5/sin">sin()</a> esperan ángulos medidos en unidades de
  radianes.

  Al llamar a <code>angleMode(DEGREES)</code> se garantiza que los ángulos se
  miden en unidades de

  grados.</p>

  <p>Nota: Tenga en cuenta que <code>TWO_PI</code> radianes equivale a 360˚.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Draw a red arc from 0 to HALF_PI radians.
      fill(255, 0, 0);
      arc(50, 50, 80, 80, 0, HALF_PI);

      // Use degrees.
      angleMode(DEGREES);

      // Draw a blue arc from 90˚ to 180˚.
      fill(0, 0, 255);
      arc(50, 50, 80, 80, 90, 180);
