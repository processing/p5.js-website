  <p> Una secuencia de caracteres de texto </p> <p>El tipo de datos <code>String</code> es útil para trabajar con texto. Por ejemplo una cadena podría contener un mensaje de bienvenida:</p> 
  <code> // Use a string literal. text('Hello!', 10, 10); </code> 
  <code> // Create a string variable. let message = 'Hello!'; </code>
  <code>// Use the string variable. text(message, 10, 10); </p></code> 
  <p> La forma más común de crear cadenas es usando algún tipo de comillas: 
  <code> text("hi", 50, 50); </code> 
  <code> text('hi', 50, 50); </code> 
  <code> text(`hi`, 50, 50); </code> 
  <p><code>"hi"</code>, <code>'hi'</code>, and <code>hi</code> son todos "literales" de cadena. Un "literal" es un elemento que representa directamente un valor, por ejemplo <code>text('hi', 50, 50)</code>. En cambio, <code>text(message, 50, 50)</code> usa la variable <code>message</code>, por lo que no es un literal de cadena.</p>
  <p>Las comillas simples <code>''</code> y las comillas dobles <code>""</code> significan lo mismo. Es bueno tener la opción para casos en los que una cadena contiene un tipo de comilla:</p>
  <code> text("What's up?", 50, 50); </code> <code> text('Air quotes make you look "cool."', 50, 50); </code>
  <p>Las comillas invertidas ```` crean plantillas de literales . Las plantillas de literales tienen muchos usos. Por ejemplo, pueden contener tanto comillas simples como dobles según sea necesario:</p>
  <code> text(`"Don't you forget about me"`, 10, 10); </code>
  <p>Las plantillas de literales son útiles cuando las cadenas se crean a partir de variables de la siguiente manera:</p>
  <code>
  let size = random(10, 20);
  circle(50, 50, size);
  <p>text(<code>The circle's diameter is ${size} pixels.</code>,  10, 10);
  </p>
  </code>
  <p>El valor de la variable <code>size</code> reemplazará <code>${size}</code> cuando se cree la cadena. <code>${}</code> is a placeholder for any value. es un marcador de posición para cualquier valor. Eso significa que se puede usar una expresión, como <code>${round(PI, 3)}</code>. Todos los siguientes son plantillas de literales válidos:</p>
  <code> text(`π is about {mouseX < width / 2} that I'm on the left half of the canvas.`, 10, 30); </code>
  <p>Las plantillas de literales pueden incluir distintas variables:</p>
  <code> let x = random(0, 100); let y = random(0, 100); let size = random(10, 20); circle(x, y, size); <p>text(<code>The circle at (${x}, ${y}) has a diameter of ${size} pixels.</code>, 10, 10); </p></code>
  <p>Las plantillas de literales también son útiles para crear texto de varias líneas como:</p>
  <code> let poem = `My sketch doesn't run; it waits for me patiently while bugs point the way.`; <p>text(poem, 10, 10); </p></code>
