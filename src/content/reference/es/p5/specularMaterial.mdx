description: >
  <p>Establece el color especular del material de la superficie de las formas.</p>

  <p>El color <code>specularMaterial()</code> establece los componentes del color de la luz que

  los revestimientos brillantes en las formas reflejarán. Por ejemplo, llamar a

  <code>specularMaterial(255, 255, 0)</code> haría que una forma refleje luz roja y

  verde, pero no luz azul.</p>

  <p>A diferencia de <a href="/reference/p5/ambientMaterial">ambientMaterial()</a>,

  <code>specularMaterial()</code> reflejará el color completo de las fuentes de luz, incluyendo

  <a href="/reference/p5/directionalLight">directionalLight()</a>,

  <a href="/reference/p5/pointLight">pointLight()</a>,

  y <a href="/reference/p5/spotLight">spotLight()</a>. Esto es lo que le da a las formas

  su apariencia "brillante". El brillo del material se puede controlar mediante la

  función <a href="/reference/p5/shininess">shininess()</a>.</p>

  <p><code>specularMaterial()</code> se puede llamar de tres formas diferentes con diferentes
  parámetros para

  establecer el color del material.</p>

  <p>La primera forma de llamar a <code>specularMaterial()</code> tiene un parámetro, <code>gray</code>.

  Se pueden pasar valores de escala de grises entre 0 y 255, como en <code>specularMaterial(50)</code>,
  para establecer el color del material. Valores de escala de grises más altos hacen que las formas

  parezcan más brillantes.</p>

  <p>La segunda forma de llamar a <code>specularMaterial()</code> tiene un parámetro, <code>color</code>. Un

  objeto <a href="/reference/p5/p5.Color">p5.Color</a>, un arreglo de valores de color o una cadena de color CSS,
  como en <code>specularMaterial('magenta')</code>, se pueden pasar para establecer el

  color del material.</p>

  <p>La tercera forma de llamar a <code>specularMaterial()</code> tiene cuatro parámetros, <code>v1</code>, <code>v2</code>,

  <code>v3</code> y <code>alpha</code>. <code>alpha</code> es opcional. Se pueden pasar valores RGBA, HSBA o HSLA para establecer

  los colores del material, como en <code>specularMaterial(255, 0, 0)</code> o

  <code>specularMaterial(255, 0, 0, 30)</code>. Los valores de color se interpretarán

  utilizando el <a href="/reference/p5/colorMode">colorMode()</a> actual.</p>

    <div>
    <code>
    // Click and drag the mouse to view the scene from different angles.
    // Double-click the canvas to apply a specular material.

    let isGlossy = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A red torus drawn on a gray background. It becomes glossy when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Turn on a white point light at the top-right.
      pointLight(255, 255, 255, 30, -40, 30);

      // Add a glossy coat if the user has double-clicked.
      if (isGlossy === true) {
        specularMaterial(255);
        shininess(50);
      }

      // Style the torus.
      noStroke();
      fill(255, 0, 0);

      // Draw the torus.
      torus(30);
    }

    // Make the torus glossy when the user double-clicks.
    function doubleClicked() {
      isGlossy = true;
    }
    </code>
    </div>
