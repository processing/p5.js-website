description: >
  <p>Establece el estilo para el ajuste de texto cuando se llama a <a href="/reference/p5/text">text()</a>.</p>
  <p>El parámetro, <code>style</code>, puede ser uno de los siguientes valores:</p>
  <p><code>WORD</code> inicia nuevas líneas de texto en los espacios. Si una cadena de texto no tiene espacios, puede desbordar el cuadro de texto y el lienzo. Este es el estilo predeterminado.</p>
  <p><code>CHAR</code> inicia nuevas líneas según sea necesario para mantenerse dentro del cuadro de texto.</p>
  <p><code>textWrap()</code> solo funciona cuando se establece el ancho máximo para un cuadro de texto. Por ejemplo, llamar a <code>text('Have a wonderful day', 0, 10, 100)</code> establece el ancho máximo en 100 píxeles.</p>
  <p>Llamar a <code>textWrap()</code> sin argumentos devuelve el estilo actual.</p>
