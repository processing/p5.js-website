description: >
  <p>Una función que se ejecuta una vez cuando el rueda del ratón se mueve.</p>

  <p>Declarar la función <code>mouseWheel()</code> establece un bloque de código

  para ejecutarse automáticamente cuando el usuario hace scroll con el ratón:</p>

  <code>

  function mouseWheel() {
    // Code to run.
  }

  </code>


  <p>Las variables del sistema del ratón, como <a
  href="/reference/es/p5/mouseX">mouseX</a> y

  <a href="/reference/es/p5/mouseY">mouseY</a>, se actualizarán con su valor más
  reciente

  cuando <code>mouseWheel()</code> sea llamada por p5.js:</p>

  <code>

  function mouseWheel() {
    if (mouseX < 50) {
      // Code to run if the mouse is on the left.
    }

  <p>  if (mouseY > 50) {
      // Code to run if the mouse is near the bottom.
    }
  }

  </p></code>

  <p>El parámetro, <code>event</code>, es opcional. <code>mouseWheel()</code> 
  siempre se le pasa un

  <a href="https://developer.mozilla.org/es/docs/Web/API/MouseEvent"
  target="_blank">MouseEvent</a>

  objeto con propiedades que describen el evento de desplazamiento del ratón:</p>

  <code>

  function mouseWheel(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>El objeto <code>event</code> tiene muchas propiedades incluyendo
  <code>delta</code>, un <code>Number</code>

  que contiene la distancia que el usuario ha hecho scroll. Por ejemplo,
  <code>event.delta</code>

  podría tener el valor 5 cuando el usuario hace scroll hacia arriba.
  <code>event.delta</code> es positivo

  si el usuario hace scroll hacia arriba y negativo si hace scroll hacia abajo.
  Los signos son

  opuestos en macOS con "natural" scrolling habilitado.</p>

  <p>Los navegadores pueden tener comportamientos predeterminados asociados a

  eventos de ratón. Por ejemplo, algunos navegadores resaltan texto cuando el
  usuario mueve el ratón mientras mantiene un botón presionado. Para evitar cualquier
  comportamiento predeterminado para este evento, agregue <code>return false;</code>

  al final de la función.</p>

  <p>Nota: En Safari, <code>mouseWheel()</code> podría no funcionar de manera

  esperada si no se agrega <code>return false;</code> al final de la función.</p>
