description: >
  <p>Una variable de sistema <code>Number</code> que contiene el código de la
  última tecla presionada.</p>

  <p>Todas las teclas tienen un <code>keyCode</code>. Por ejemplo, la tecla
  <code>a</code> tiene el <code>keyCode</code> 65.

  La variable <code>keyCode</code> es útil para comprobar si una tecla especial
  ha sido presionada. Por ejemplo, el siguiente condicional comprueba si se ha
  pulsado la tecla enter:</p>

  <code>

  if (keyCode === 13) {
    // Code to run if the enter key was pressed.
  }

  </code>


  <p>El mismo código se puede escribir de forma más clara utilizando la
  variable de sistema
  <code>ENTER</code>

  que tiene un valor de 13:</p>

  <code>

  if (keyCode === ENTER) {
    // Code to run if the enter key was pressed.
  }

  </code>


  <p>Las variables de sistema <code>BACKSPACE</code>, <code>DELETE</code>,
  <code>ENTER</code>, <code>RETURN</code>, <code>TAB</code>,

  <code>ESCAPE</code>, <code>SHIFT</code>, <code>CONTROL</code>,
  <code>OPTION</code>, <code>ALT</code>, <code>UP_ARROW</code>,
  <code>DOWN_ARROW</code>,

  <code>LEFT_ARROW</code> y <code>RIGHT_ARROW</code> son alias convenientes
  para los códigos de teclas especiales. Los códigos de teclas se pueden
  encontrar en sitios web como <a href="http://keycode.info/">keycode.info</a>.</p>
