description: |2-

  <p>将图像绘制到画布上。</p>


  <p>第一个参数 <code>img</code> 是要绘制的源图像。
  <code>img</code> 可以是以下任何对象：</p>


  <ul>


  <li><a href="/reference/p5/p5.Image">p5.Image</a></li>


  <li><a href="/reference/p5/p5.Element">p5.Element</a></li>


  <li><a href="/reference/p5/p5.Texture">p5.Texture</a></li>


  <li><a href="/reference/p5/p5.Framebuffer">p5.Framebuffer</a></li>


  <li><a
  href="/reference/p5/p5.FramebufferTexture">p5.FramebufferTexture</a></li>


  </ul>


  <p>第二个和第三个参数，<code>dx</code> 和 <code>dy</code>，设置了
  目标图像左上角的坐标。用于其他方式
  定位图片参考 <a href="/reference/p5/imageMode">imageMode()</a>。</p>


  <p>这是一个图表，解释了在 <code>image()</code> 中，可选参数如何工作：</p>


  <p><img src="assets/drawImage.png"/></p>


  <p>第四个和第五个参数，<code>dw</code>和<code>dh</code> 是可选的。他们设定了绘制目标图像的宽度和高度。默认情况下，<code>image()</code> 按原始尺寸绘制完整的源图像。</p>


  <p>第六和第七个参数，<code>sx</code>和 <code>sy</code> 也是可选的。


  这些坐标定义了从源图像中绘制的分段左上角。</p>


  <p>第八个和第九个参数，<code>sw</code> 和 <code>sh</code> 也是可选的。


  这些坐标定义了从源图像中绘制的分段宽度和高度。默认情况下，<code>image()</code> 绘制从 <code>(sx, sy)</code> 开始并延伸到源图像边缘的完整分段。</p>


  <p>第九个参数，<code>fit</code>，也是可选的。在不影响其纵横比的情况下，它绘制了源图像的分段。如果 <code>CONTAIN</code> 通过了，在目标矩形中会出现完整的分段。如果 <code>COVER</code> 通过了，则该分段将完全覆盖目标矩形。这可能会产生分段放大的效果。</p>


  <p>第十个和第十一个参数，<code>xAlign</code> 和
  <code>yAlign</code>，也是可选的。他们确定如何对齐拟合的分段。
  <code>xAlign</code> 可以设置为 <code>LEFT</code>、<code>RIGHT</code> 或
  <code>CENTER</code>。 <code>yAlign</code> 可以设置为


  <code>TOP</code>、<code>BOTTOM</code> 或 <code>CENTER</code>。默认情况下，<code>xAlign</code> 和 <code>yAlign</code> 设置为 <code>CENTER</code>。</p>
