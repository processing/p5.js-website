description: >
  <p>声明一个新的变量。</p>

  <p>变量是用于存储值的容器。例如，一个变量可能包含类型为数字 <code>Number</code> 的 x 坐标，或者类型为字符串 <code>String</code> 的名称。变量可以通过重新赋值来改变其值，如下所示：

  </p>

  <code>

  // Declare the variable x and assign it the value 10.

  let x = 10;


  <p>// Reassign x to 50.

  x = 50;

  </p></code>

  <p>变量具有块级作用域。当一个变量被声明在花括号 <code>{}</code> 之间时，它只存在于由花括号定义的块级内。例如，以下代码会抛出 <code>ReferenceError</code>，
  因为 <code>x</code> 是在 <code>setup()</code> 函数的块级内声明的：</p>

  <code>

  function setup() {
    createCanvas(100, 100);

  <p>  let x = 50;

  }</p>

  <p>function draw() {
    background(200);</p>
  <p>  // x was declared in setup(), so it can't be referenced here.
    circle(x, 50, 20);
  }

  </p></code>

  <p>在所有花括号 <code>{}</code> 之外声明的变量是全局变量。全局变量可以在绘图的任何地方使用和更改：</p>

  <code>

  let x = 50;


  <p>function setup() {
    createCanvas(100, 100);
  }</p>

  <p>function draw() {
    background(200);</p>
  <p>  // Change the value of x.
    x += 10;</p>
  <p>  circle(x, 50, 20);

  }

  </p></code>
