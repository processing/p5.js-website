description: |2-

  <p>设置连接线段的接头样式。</p>


  <p>接头可以是斜接 (<code>MITRE</code>)、斜角
  (<code>BEVEL</code>)，或圆角


  （<code>ROUND</code>）。 2D 模式下默认接头为 <code>MITRE</code>
  WebGL 中为<code>ROUND</code>


  模式。</p>


  <p>传递给<code>strokeJoin()</code>的参数必须为全大写，因为常量 <code>MITRE</code>、<code>BEVEL</code> 和 <code>ROUND</code>
  都是这样定义的。


  JavaScript 是一种区分大小写的编程语言。</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Style the line.
      noFill();
      strokeWeight(10);
      strokeJoin(MITER);

      // Draw the line.
      beginShape();
      vertex(35, 20);
      vertex(65, 50);
      vertex(35, 80);
      endShape();

      describe('A right-facing arrowhead shape with a pointed tip in center of canvas.');
    }
