description: |2-

  <p>将文本绘制到画布上。</p>


  <p>第一个参数 <code>str</code> 是要绘制的文本。第二和第三个参数 <code>x</code> 和 <code>y</code>，设置文本左下角的坐标。

  其他对齐文本方法请参见 <a href="/reference/p5/textAlign">textAlign()</a>。</p>


  <p>第四个和第五个参数，<code>maxWidth</code>和
  <code>maxHeight</code> 是可选的。


  他们设置包含文本的隐形方框的尺寸。默认情况下，他们设置其最大宽度和高度。参考
  <a href="/reference/p5/rectMode">rectMode()</a> 了解其他定义矩形文本框的方式。文本将自动换行以适应文本框。文本无法在框外绘制。</p>


  <p>我们也可以通过多种方式设置文本样式。调用 <a
  href="/reference/p5/fill">fill()</a> 函数可以设置文本的填充颜色。调用 <a href="/reference/p5/stroke">stroke()</a> 和 <a href="/reference/p5/StrokeWeight">StrokeWeight()</a> 可以设置文本的描边。


  调用 <a href="/reference/p5/textSize">textSize()</a> 和 <a href="/reference/p5/textFont">textFont()</a> 可以分别设置文本的大小和字体。</p>


  <p>注意：<code>WebGL</code> 模式仅支持 <a href="/reference/p5/loadFont">loadFont()</a> 加载的字体。调用


  <a href="/reference/p5/stroke">stroke()</a>  对于 <code>WebGL</code> 模式没有效果。</p>
