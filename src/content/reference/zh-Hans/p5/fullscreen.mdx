  <p>切换全屏模式或返回当前模式。</p>


  <p>调用 <code>fullscreen(true)</code> 将绘图切换为全屏模式，调用 <code>fullscreen(false)</code> 将绘图恢复原始大小。</p>


  <p>调用 <code>fullscreen()</code> 不传参数时，如果绘图是全屏模式，返回 <code>true</code>，否则返回 <code>false</code>。</p>


  <p>注意：由于浏览器的限制， <code>fullscreen()</code> 只能在有用户输入时调用，比如点击鼠标的时候。</p>

    function setup() {

      describe('A gray canvas that switches between default and full-screen display when clicked.');
    }

    // If the mouse is pressed,
    // toggle full-screen mode.
    function mousePressed() {
      if (mouseX > 0 && mouseX < width && mouseY > 0 && mouseY < height) {
        let fs = fullscreen();
        fullscreen(!fs);
      }
    }
    </code>
