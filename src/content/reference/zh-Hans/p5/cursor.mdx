description: |2-

  <p>改变光标样式。</p>


  <p>第一个参数，<code>type</code>，设置要显示的光标类型。内置选项有： <code>ARROW</code>、<code>CROSS</code>、
  <code>HAND</code>、<code>MOVE</code>、<code>TEXT</code>和
  <code>WAIT</code>。


  "<code>cursor()</code> 也可以识别以字符串形式传递的标准 CSS 光标属性： <code>'help'</code>，<code>'wait'</code>，<code>'crosshair'</code>，"
  "<code>'not-allowed'</code>，<code>'zoom-in'</code> 和  <code>'grab'</code>。"

  如果传递的是图像路径，例如：
  "<code>cursor('/assets/target.png')</code>，那么光标将以图像的形式展现出来。"

  图像格式必须是 .cur、.gif、.jpg、.jpeg 或 .png，要以这样的形式传递： <a
  href="https://developer.mozilla.org/en-US/docs/Web/CSS/cursor#icon_size_limits">，最高像素为 32x32。</a></p>


  参数 <p><code>x</code> 和 <code>y</code> 都是可选项。如果用图像作光标，<code>x</code> 和 <code>y</code> 用来设置图像中指向的位置。 它们初始值都是 0，此时光标指向图像左上角。并且 <code>x</code> 和 <code>y</code> 的值必须分别小于图像的宽高。</p>

    function setup() {
      createCanvas(100, 100);

      describe('A gray square divided into quadrants. The cursor image changes when the mouse moves to each quadrant.');
    }

    function draw() {
      background(200);

      // Divide the canvas into quadrants.
      line(50, 0, 50, 100);
      line(0, 50, 100, 50);

      // Change cursor based on mouse position.
      if (mouseX < 50 && mouseY < 50) {
        cursor(CROSS);
      } else if (mouseX > 50 && mouseY < 50) {
        cursor('progress');
      } else if (mouseX > 50 && mouseY > 50) {
        cursor('https://avatars0.githubusercontent.com/u/1617169?s=16');
      } else {
        cursor('grab');
      }
    }
    </code>
    </div>
