description: >
  <p>一个按顺序存储多个数据的列表。</p>

  <p>数组有助于存储相关数据。它们可以存储任何类型的数据。例如，数组可以以字符串的形式存储某人最喜欢的颜色列表。数组的创建方式如下: </p>

  <code>

  let colors = ['deeppink', 'darkorchid', 'magenta'];

  </code>


  <p>数组中的每个数据皆为一个元素。每个元素在其数组中都有一个地址或索引。变量 <code>colors</code> 指的是一个包含三个元素的数组。<a href="/reference/p5/String">String> elements,
  <code>'deeppink'</code>，

  <code>'darkorchid'</code>，和 <code>'magenta'</code>。数组的索引是从 0 开始的，这就意味着 <code>'deeppink'</code> 的索引是 0， <code>'darkorchid'</code> 的索引是 1，

  <code>‘magenta'</code> 的索引是 2。我们可以使用数组的索引来访问数组元素，如下所示：</a></p>

  <code>

  let zeroth = colors[0]; // 'deeppink'

  let first = colors[1]; // 'darkorchid'

  let second = colors[2]; // 'magenta'

  </code>


  <p>通过调用 <code>push()</code> 方法，可以在数组末尾添加元素。方法如下：

  method as follows:</p>

  <code>

  colors.push('lavender');


  <p>let third = colors[3]; // 'lavender'

  </p></code>

  <p>了解更多关于数组的信息，请参阅 <a
  href="https://developer.mozilla.org/en-US/docs/Learn/JavaScript/First_steps/Arrays"
  target="_blank">MDN</a></p>
