description: |2-

  <p>设置渲染线条末端的样式。</p>


  <p>线条尾部结束方式可以是圆形 (<code>ROUND</code>)、方形


  (<code>SQUARE</code>)，或扩展 (<code>PROJECT</code>)。默认为
  <code>圆形</code>。</p>


  <p>传入<code>StrokeCap()</code>的参数必须全部大写
  因为常量 ROUND、SQUARE 和
  <code>PROJECT</code> 是这样定义的。


  JavaScript 是一种区分大小写的编程语言。</p>
line: '387'
isConstructor: 'false'
itemtype: method
example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      strokeWeight(12);

      // Top.
      strokeCap(ROUND);
      line(20, 30, 80, 30);

      // Middle.
      strokeCap(SQUARE);
      line(20, 50, 80, 50);

      // Bottom.
      strokeCap(PROJECT);
      line(20, 70, 80, 70);

      describe(
        'Three horizontal lines. The top line has rounded ends, the middle line has squared ends, and the bottom line has longer, squared ends.'
      );
    }
    </code>
    </div>
