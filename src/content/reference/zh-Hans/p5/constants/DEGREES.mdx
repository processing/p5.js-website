  <p>一个 <code>String</code> 常量，用于设置 <a href="/reference/p5/angleMode">angleMode()</a>。</p>

  <p><a href="/reference/p5/rotate">rotate()</a> 和 <a href="/reference/p5/sin">sin()</a> 等函数默认以弧度为单位测量。
  调用 <code>angleMode(DEGREES)</code> 可以确保以角度为单位测量。</p>

  <p>注意：<code>TWO_PI</code>弧度等于 360˚。</p>

      // Use degrees.

      fill(0, 0, 255);
