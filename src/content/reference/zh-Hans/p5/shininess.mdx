  <p>设置 <a href="/reference/p5/specularMaterial">specularMaterial()</a> 的光泽度（"shininess"）。</p>

  <p>有光泽的材料比无光泽的材料更聚焦反射光线。<code>shininess()</code> 影响材料反射光源的方式，包括 <a href="/reference/p5/directionalLight">directionalLight()</a>、<a href="/reference/p5/pointLight">pointLight()</a> 和 <a href="/reference/p5/spotLight">spotLight()</a>。</p>

  <p>参数 <code>shine</code> 是一个数字，表示光泽度的程度。 <code>shine</code> 必须大于 1，这是它的默认值。</p>

  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100, WEBGL);

      describe(
        'Two red spheres drawn on a gray background. White light reflects from their surfaces as the mouse moves. The right sphere is shinier than the left sphere.'
      );
    }

    function draw() {
      background(200);

      // 打开一个红色环境光。
      ambientLight(255, 0, 0);

      // 获取鼠标的坐标。
      let mx = mouseX - 50;
      let my = mouseY - 50;

      // 打开一个跟随鼠标移动的白色点光源。
      pointLight(255, 255, 255, mx, my, 50);

      // 设定球体样式。
      noStroke();

      // 添加一个具有灰度值的高光材质。
      specularMaterial(255);

      // 绘制左侧光泽度较低的球体。
      translate(-25, 0, 0);
      shininess(10);
      sphere(20);

      // 绘制右侧光泽度较高的球体。
      translate(50, 0, 0);
      shininess(100);
      sphere(20);
    }
    </code>
