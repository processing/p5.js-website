description: >
  <p>当任何键被按下时调用一次的函数。</p>

  <p>声明函数 <code>keyPressed()</code> 会设置一个代码块，当用户按下任何键时自动运行一次：</p>

  <code>

  function keyPressed() {
    // 要运行的代码。
  }

  </code>

  <p>当 <code>keyPressed()</code> 被 p5.js 调用时，<a href="/reference/p5/key">key</a> 和 <a href="/reference/p5/keyCode">keyCode</a> 变量会更新为最近输入的值：</p>

  <code>

  function keyPressed() {
    if (key === 'c') {
      // 要运行的代码。
    }

    if (keyCode === ENTER) {
      // 要运行的代码。
    }
  }

  </p></code>

  <p>参数 <code>event</code> 是可选的。<code>keyPressed()</code> 总是传递一个 <a href="https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent" target="_blank">KeyboardEvent</a> 对象，该对象带有描述按键事件的属性：</p>

  <code>

  function keyPressed(event) {
    // 使用事件运行的代码。
    console.log(event);
  }

  </code>

  <p>浏览器可能会将默认行为附加到各种键事件上。例如，一些浏览器在按下 <code>SPACE</code> 键时可能会跳转到网页底部。要防止此事件的任何默认行为，在函数末尾添加 <code>return false;</code>。</p>
