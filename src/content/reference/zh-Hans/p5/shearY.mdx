description: >
  <p>倾斜 y 轴，使形状呈现出倾斜。</p>

  <p>默认情况下，x 轴和 y 轴是垂直的。<code>shearY()</code> 
  函数转换坐标系，平移 y 坐标，而 x 坐标保持不变。</p>

  <p>第一个参数 <code>angle</code> 是倾斜的量。例如，调用 <code>shearY(1)</code> 使用以下公式转换所有的 y 坐标：<code>y = y + x * tan(angle)</code>。shearY() 函数使用当前的 <a href="/reference/p5/angleMode">angleMode()</a> 来解释角度值。</p>

  <p>默认情况下，变换是累积的。例如，连续两次调用 <code>shearY(1)</code> 的效果与一次调用 <code>shearY(2)</code> 相同。可以使用 <a href="/reference/p5/push">push()</a> 和 <a href="/reference/p5/pop">pop()</a> 函数将变换隔离在特定的绘图组中。</p>

  <p>注意：变换会在绘制循环的开头被重置。在 <a href="/reference/p5/draw">draw()</a> 函数内调用 <code>shearY(1)</code> 不会导致形状连续错切。</p>
