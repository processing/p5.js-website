  <p>一个被命名的语句组</p>
  <p><a href=\"https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions\" target=\"\_blank\">Functions</a>
   函数有助于组织和复用代码。例如，函数使得表达“画一朵花”的想法变得容易：</p>
  <code>
  function drawFlower() {
     // Style the text.
   textAlign(CENTER, CENTER);
   textSize(20);
  
  <p> // Draw a flower emoji.
   text('\U0001F338', 50, 50);
  }
  </p></code>
  <p>函数头以关键字 <code>function</code> 开始。函数名 <code>drawFlower</code> 后面是括号 <code>()</code> 和花括号 <code>{}</code>。
   花括号之间的代码是函数的主体。调用函数时，函数主体中的代码就会运行，如下所示：</p>
  <code>
  drawFlower();
  </code>
  
  <p>函数可以通过在其头部添加参数来接收输入。
   参数是值的占位符，调用函数时需要提供这些值。例如，<code>drawFlower()</code> 函数可以接收一个代表花朵大小的参数：</p>
  <code>
  function drawFlower(size) {
     // Style the text.
   textAlign(CENTER, CENTER);
  
  <p> // Use the size parameter.
   textSize(size);</p>
  <p> // Draw a flower emoji.
   text('\U0001F338', 50, 50);
  }
  </p></code>
  <p>参数是函数声明的一部分，实参由调用函数的代码提供。调用函数时，实参会替代形参：</p>
  <code>
  // The argument 20 is assigned to the parameter size.
  drawFlower(20);
  </code>
  
  <p>一个函数可以有多个参数，参数之间用逗号隔开。参数可以是任何类型，例如，<code>drawFlower()</code> 函数可以接收 
  <code>Number</code> 类型的参数，用于花朵的 x 和 y 坐标以及花朵的大小：</p>
  <code>
  function drawFlower(x, y, size) {
     // Style the text.
   textAlign(CENTER, CENTER);
  
  <p> // Use the size parameter.
   textSize(size);</p>
  <p> // Draw a flower emoji.
   // Use the x and y parameters.
   text('\U0001F338', x, y);
  }
  </p></code>
  <p>函数还可以通过添加 <code>return</code> 语句产生输出:</p>
  <code>
  function double(x) {
     let answer = 2 \* x;
   return answer;
  }
  </code>
  
  <p><code>return</code> 后面的表达式可以产生一个输出，这个输出可以在其他地方使用。例如，<code>double()</code> 函数的输出可以赋值给一个变量：</p>
  <code>
  let six = double(3);
  text(`3 x 2 = ${six}`, 50, 50);
  </code>
