description: >
  <p>一个 <code>String</code> 字符串类型的系统变量，包含最后一次键入的键的值。</p>

  <p>key 变量对于检查是否键入了一个 <a href="https://en.wikipedia.org/wiki/ASCII#Printable_characters" target="_blank">ASCII</a> 键非常有用。例如，表达式 <code>key === "a"</code> 如果键入了 <code>a</code> 键则评估为 <code>true</code>，如果没有则为 <code>false</code>。对于特殊键如 <code>LEFT_ARROW</code> 和 <code>ENTER</code>，<code>key</code> 不会更新。对于特殊键，请使用 keyCode。应使用 <a href="/reference/p5/keyIsDown">keyIsDown()</a> 函数检查多个不同的按键同时按下的状态。</p>

      createCanvas(100, 100);

    function draw() {
      // Update x and y if a key is pressed.
      if (keyIsPressed === true) {
        if (key === 'w') {
          y -= 1;
        } else if (key === 's') {
          y += 1;
        } else if (key === 'a') {
          x -= 1;
        } else if (key === 'd') {
          x += 1;
        }
      }

      // Style the circle.
      fill(0);

      // Draw the circle at (x, y).
      circle(x, y, 5);
    }
