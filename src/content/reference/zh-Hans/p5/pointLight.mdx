description: >
  <p>创建一个从一个点朝所有方向发光的光源。</p>

  <p>点光源就像发光的灯泡，它们向所有方向发光。 它们可以放置在不同的位置以实现不同的光照效果。最多可以同时激活 5 个点光源。</p>

  <p>有四种方法可以调用 <code>pointLight()</code> 来设置光的颜色和位置。</p>

  <p>调用 <code>pointLight()</code> 的第一种方式有六个参数。前三个参数 <code>v1</code>、 <code>v2</code> 和 <code>v3</code> 使用当前的 <a href="/reference/p5/colorMode">colorMode()</a> 来设置光的颜色。最后三个参数 <code>x</code>、 <code>y</code> 和<code>z</code> 设置光的位置。例如， <code>pointLight(255, 0, 0, 50, 0, 0)</code> 创建一个红色 <code>(255, 0, 0)</code> 的光，从坐标 <code>(50, 0, 0)</code> 发光。</p>

  <p>调用 <code>pointLight()</code> 的第二种方式有四个参数。前三个参数 <code>v1</code>、 <code>v2</code> 和 <code>v3</code> 使用当前的 <a href="/reference/p5/colorMode">colorMode()</a> 来设置光的颜色。最后一个参数 position 使用一个 <a href="/reference/p5/p5.Vector">p5.Vector</a> 对象来设置光的位置。例如， <code>pointLight(255, 0, 0, lightPos)</code> 创建一个红色 <code>(255, 0, 0)</code> 的光，从由 <code>lightPos</code> 向量设置的位置发出。</p>

  <p>调用 <code>pointLight()</code> 的第三种方式有四个参数。第一个参数 <code>color</code> 使用一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象或颜色值数组来设置光的颜色。最后三个参数 <code>x</code>、 <code>y</code> 和 <code>z</code> 设置光的位置。例如， <code>pointLight(myColor, 50, 0, 0)</code> 创建一个颜色为 <code>myColor</code> 的光，从坐标 <code>(50, 0, 0)</code> 发光。</p>

  <p>调用 <code>pointLight()</code> 的第四种方式有两个参数。第一个参数 <code>color</code> 使用一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象或颜色值数组来设置光的颜色。第二个参数 <code>position</code> 使用一个 <a href="/reference/p5/p5.Vector">p5.Vector</a> 对象来设置光的位置。例如， <code>pointLight(myColor, lightPos)</code> 创建一个从由 <code>lightPos</code> 向量设置的位置发光，其颜色为 <code>myColor</code>。</p>

      background(200);

      // 启用鼠标轨道控制。
      orbitControl();

      // 控制光线。
      if (isLit === true) {
        // 添加一个上方的红色点光源。

        pointLight(255, 0, 0, 0, -150, 0);
      }

      // 设定球体样式。
      noStroke();

      // 绘制球体。
      sphere(30);
    }

    // 当用户双击时打开点光源。
    function doubleClicked() {
      isLit = true;
    }
