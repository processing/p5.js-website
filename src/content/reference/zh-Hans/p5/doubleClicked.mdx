description: >
  <p>当鼠标按钮快速双击时调用一次的函数。</p>

  <p>声明函数 <code>doubleClicked()</code> 会设置一个代码块，当用户快速按下并释放鼠标按钮两次时自动运行：</p>

  <code>

  function doubleClicked() {
    // Code to run.
  }

  </code>

  <p>鼠标系统变量，如 <a href="/reference/p5/mouseX">mouseX</a> 和 <a href="/reference/p5/mouseY">mouseY</a>，在调用 <code>doubleClicked()</code> 时会更新为它们最新的值：</p>

  <code>

  function doubleClicked() {
    if (mouseX < 50) {
      // Code to run if the mouse is on the left.
    }

  <p>  if (mouseY > 50) {
      // Code to run if the mouse is near the bottom.
    }
  }

  </p></code>

  <p>参数 <code>event</code> 是可选的。<code>doubleClicked()</code> 总是传递一个 <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent" target="_blank">MouseEvent</a> 对象，该对象带有描述双击事件的属性：</p>

  <code>

  function doubleClicked(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>

  <p>在触摸屏设备上，放置在 <code>doubleClicked()</code> 中的代码将在短时间内发生两次触摸后运行。</p>

  <p>浏览器可能会将默认行为附加到各种鼠标事件上。例如，一些浏览器在用户按下鼠标按钮同时移动鼠标时会高亮显示文本。为了防止此事件的任何默认行为，在函数的末尾添加 <code>return false;</code>。</p>

    function doubleClicked() {
