description: |2-

  <p>调用 <a href="/reference/p5/image">image()</a> 更改图像的绘制位置。</p>


  <p>默认情况下，<a href="/reference/p5/image">image()</a> 前两个参数是图像左上角的 x 和 y 坐标。接下来的参数是
  它的宽度和高度。和上面相同，调用
  <code>imageMode(CORNER)</code>。</p>


  <p><code>imageMode(CORNERS)</code>也使用 <a href="/reference/p5/image">image()</a> 前两个参数作为图像左上角的 x 和 y 坐标。第三个和第四个参数是它右下角的坐标。</p>


  <p><code>imageMode(CENTER)</code> 使用前两个参数 <a href="/reference/p5/image">image()</a> 作为图像中心的 x 和 y 坐标。接下来的参数是它的宽度和高度。</p>

      imageMode(CORNER);
