description: >
  <p>特殊类型对象的创建模板</p>
  <p>类可以让使用对象编程变得更容易。例如，一个 <code>Frog</code>
   类可以创建行为像青蛙一样的对象。通过类创建的每个对象都是该类的一个实例。同一个类的所有实例都是相同类型。以下是创建 <code>Frog</code>
   类实例的示例：</p>
  <code>
  let fifi = new Frog(50, 50, 20);
  </code>
  
  <p>变量 <code>fifi</code> 代表 <code>Frog</code> 的一个实例。关键字 
  <code>new</code> 用于在语句 
  <code>new Frog()</code> 中调用 <code>Frog</code>。总之，一个新的 <code>Frog</code> 对象被创建并赋值给了变量 <code>fifi</code>。类是模板，因此我们可以用它创建多个实例：</p>
  <code>
  // First Frog instance.
  let frog1 = new Frog(25, 50, 10);
  
  <p>// Second Frog instance.
  let frog2 = new Frog(75, 50, 10);
  </p></code>
  <p>一个简单的 <code>Frog</code> 类可以声明如下：</p>
  <code>
  class Frog {
     constructor(x, y, size) {
     // This code runs once when an instance is created.
   this.x = x;
   this.y = y;
   this.size = size;
   }
  
  <p> show() {
     // This code runs once when myFrog.show() is called.
   textAlign(CENTER, CENTER);
   textSize(this.size);
   text('\U0001F438', this.x, this.y);
   }</p>
  <p> hop() {
     // This code runs once when myFrog.hop() is called.
   this.x += random(-10, 10);
   this.y += random(-10, 10);
   }
  }
  </p></code>
  <p>类的声明以关键字 <code>class</code> 开头，后跟类名。例如 <code>Frog</code>，加上 <code>{}</code>。类名应该使用
  <code>PascalCase</code>，并且不能包含空格。例如：命名为含有空格的 <code>Kermit The Frog</code>，就会引发 
  <code>SyntaxError</code> 错误。花括号 <code>{}</code> 之间的代码定义类。</p>
  <p>属于类的函数被称为方法。在 <code>Frog</code> 类中，<code>constructor()</code>，
  <code>show()</code>，和 <code>hop()</code> 都是方法。方法定义对象的行为。方法可以接受参数并返回值，就像函数一样。注意：方法不使用 <code>function</code> 关键字。</p>
  <p><code>constructor()</code> 是一个特殊的方法，在创建类的实例时调用一次。语句 <code>new Frog()</code> 调用 <code>Frog</code> 类的
  <code>constructor()</code> 方法。</p>
  <p>类定义是实例的模板。关键字 <code>this</code> 指代实例的数据和方法。例如，每个 <code>Frog</code> 实例都有独特的坐标存储在 <code>this.x</code> 和 <code>this.y</code> 中。<code>show()</code> 方法使用这些坐标来绘制青蛙。当调用 <code>hop()</code> 方法时，坐标会更新。一旦创建了一个 <code>Frog</code> 实例，可以使用点运算符 <code>.</code> 来访问它的数据和方法，如下所示：</p>
  <code>
  // Draw a lily pad.
  fill('green');
  stroke('green');
  circle(fifi.x, fifi.y, 2 \* fifi.size);
  
  <p>// Show the Frog.
  fifi.show();</p>
  <p>// Hop.
  fifi.hop();
  </p></code>
