description: >
  <p>开始向自定义形状添加顶点。</p>

  <p><code>beginShape()</code> 和 <a
  href="/reference/p5/endShape">endShape()</a> 函数

  允许在2D或3D中创建自定义形状。 <code>beginShape()</code> 开始添加顶点到自定义形状，而
  
  <a href="/reference/p5/endShape">endShape()</a> 则停止添加。</p>

  <p>参数 <code>kind</code> 设置要创建的形状的种类。默认情况下，可以绘制任何

  不规则多边形。kind可用的模式有：</p>

  <ul>

  <li><code>POINTS</code> 绘制一系列点。</li>

  <li><code>LINES</code> 绘制一系列不相连的线段。</li>

  <li><code>TRIANGLES</code> 绘制一系列单独的三角形。</li>

  <li><code>TRIANGLE_FAN</code> 以扇形方式绘制一系列连接的三角形，共享第一个顶点。</li>

  <li><code>TRIANGLE_STRIP</code> 以条带方式绘制一系列连接的三角形。</li>

  <li><code>QUADS</code> 绘制一系列单独的四边形（四边形）。</li>

  <li><code>QUAD_STRIP</code> 使用相邻边绘制四边形条带形式。</li>

  <li><code>TESS</code> 通过显式细分创建填充曲线（仅限WebGL）。</li>

  </ul>

  <p>调用 <code>beginShape()</code> 后，可以通过用

  <a href="/reference/p5/vertex">vertex()</a>，

  <a href="/reference/p5/bezierVertex">bezierVertex()</a>，

  <a href="/reference/p5/quadraticVertex">quadraticVertex()</a>， 和/或

  <a href="/reference/p5/curveVertex">curveVertex()</a> 来构建形状。调用

  <a href="/reference/p5/endShape">endShape()</a> 将停止向

  形状添加顶点。每个形状将以当前描边颜色勾勒出，并以当前填充颜色填充。

  </p>

  <p><a href="/reference/p5/translate">translate()</a>，

  <a href="/reference/p5/rotate">rotate()</a>，和

  <a href="/reference/p5/scale">scale()</a> 等变换不适用于
  <code>beginShape()</code> 和

  <a href="/reference/p5/endShape">endShape()</a> 之间。也不能在

  <code>beginShape()</code> 和

  <a href="/reference/p5/endShape">endShape()</a> 之间使用其他形状，比如

  <a href="/reference/p5/ellipse">ellipse()</a> 或

  <a href="/reference/p5/rect">rect()</a>。</p>
