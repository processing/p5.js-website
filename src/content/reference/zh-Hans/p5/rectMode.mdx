description: >
  <p>改变矩形和正方形绘制的方式。</p>

  <p>默认情况下，<a href="/reference/p5/rect">rect()</a> 和 <a href="/reference/p5/square">square()</a> 的前两个参数是形状中心的 x 和 y 坐标。接下来的参数设置形状的宽度和高度。这相当于调用 <code>rectMode(CORNER)</code>。</p>

  <p>调用 <code>rectMode(CORNERS)</code> 之后，函数使用前两个参数确定其中一个角的位置，接下来的参数是第一个角的对角的位置。这种模式仅适用于 <a href="/reference/p5/rect">rect()</a>。</p>

  <p><code>rectMode(CENTER)</code> 使用前两个参数作为形状的中心 x 和 y 坐标。接下来的参数是它的宽度和高度。</p>

  <p><code>rectMode(RADIUS)</code> 也使用前两个参数作为形状的中心的 x 和 y 坐标。接下来的参数是形状宽度和高度的一半。</p>

  <p>传递给 <code>rectMode()</code> 的参数必须使用全大写，因为常量 <code>CENTER</code>、<code>RADIUS</code>、<code>CORNER</code> 和 <code>CORNERS</code> 是这样定义的。JavaScript 是区分大小写的编程语言。</p>
