description: >
  <p>一个 <code>Number</code> 数字类型的系统变量，用于跟踪鼠标的水平移动。</p>

  <p><code>movedX</code> 跟踪鼠标在帧之间左右移动了多少像素。如果鼠标在帧之间向左移动，<code>movedX</code> 将会是负值；如果向右移动，则为正值。<code>movedX</code> 可以计算为 <code>mouseX - pmouseX</code>。</p>

  <p>注意：<code>movedX</code> 即使在 <a href="/reference/p5/requestPointerLock">requestPointerLock()</a> 激活时也会继续更新。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square. The text ">>" appears when the user moves the mouse to the right. The text "<<" appears when the user moves the mouse to the left.'
      );
    }

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display >> when movedX is positive and
      // << when it's negative.
      if (movedX > 0) {
        text('>>', 50, 50);
      } else if (movedX < 0) {
        text('<<', 50, 50);
      }
    }
