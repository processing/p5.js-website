description: >
  <p>每当用户触摸屏幕时调用的函数。</p>

  <p>每当用户开始触摸触屏设备时，声明一个名为 <code>touchStarted()</code> 的函数会设置一个代码块，自动运行：</p>

  <code>

  function touchStarted() {
    // Code to run.
  }

  </code>


  <p>当 p5.js 调用 <code>touchStarted()</code> 时，<a href="/reference/p5/touches">touches</a> 数组将被更新为最新的触摸点：</p>

  <code>

  function touchStarted() {
    // Paint over the background.
    background(200);

  <p>  // Mark each touch point once with a circle.
    for (let touch of touches) {
      circle(touch.x, touch.y, 40);
    }
  }

  </p></code>

  <p>参数 event 是可选的。<code>touchStarted()</code> 会被传递为一个描述触摸事件属性的 <a href="https://developer.mozilla.org/en-US/docs/Web/API/TouchEvent" target="_blank">TouchEvent</a> 对象：</p>

  <code>

  function touchStarted(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>


  <p>在触屏设备上，如果未声明 <code>touchStarted()</code>，当用户触摸开始时，<a href="/reference/p5/mousePressed">mousePressed()</a> 会运行。如果声明了 <code>touchStarted()</code>，那么当用户触摸开始时将运行 <code>touchStarted()</code>，而 <a href="/reference/p5/mousePressed">mousePressed()</a> 不会运行。</p>

  <p>注意：<code>touchStarted()</code>、<a href="/reference/p5/touchEnded">touchEnded()</a> 和 <a href="/reference/p5/touchMoved">touchMoved()</a> 都是相关的。<code>touchStarted()</code> 会在用户触摸触屏设备时立即运行。
  <a href="/reference/p5/touchEnded">touchEnded()</a> 会在用户结束触摸时立即运行。<a href="/reference/p5/touchMoved">touchMoved()</a> 会在用户移动任何触摸点时重复运行。</p>

    // On a touchscreen device, touch the canvas using one or more fingers
    // at the same time.

    let bgColor = 50;
    let fillColor = 255;
    let borderWidth = 0.5;

    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square with the number 0 at the top-center. The number tracks the number of places the user is touching the screen. Circles appear at each touch point and change style in response to events.'

    function draw() {
      background(bgColor);

      // Style the text.
      textAlign(CENTER);
      textSize(16);
      fill(0);
      noStroke();

      // Display the number of touch points.
      text(touches.length, 50, 20);

      // Style the touch points.
      fill(fillColor);
      stroke(0);
      strokeWeight(borderWidth);

      // Display the touch points as circles.
      for (let touch of touches) {
        circle(touch.x, touch.y, 40);
      }
    }

    // Set the background color to a random grayscale value.
    function touchStarted() {
      bgColor = random(80, 255);
    }

    // Set the fill color to a random grayscale value.
    function touchEnded() {
      fillColor = random(0, 255);
    }

    // Set the stroke weight.
    function touchMoved() {
      // Increment the border width.
      borderWidth += 0.1;

      // Reset the border width once it's too thick.
      if (borderWidth > 20) {
        borderWidth = 0.5;
      }
    }
