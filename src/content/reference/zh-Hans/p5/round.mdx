  <p>计算最接近一个数字的整数。</p>

  <p>例如，<code>round(133.8)</code> 返回值为 134。</p>

  <p>第二个参数 <code>decimals</code> 是可选的。它设置舍入时要使用的小数位数。例如，<code>round(12.34, 1)</code> 返回值为 12.3。默认情况下，<code>decimals</code> 为 0。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Round a number.
      let x = round(4.2);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display the rounded number.
      text(x, 50, 50);

      describe('The number 4 written in middle of the canvas.');
    }
    </code>
    </div>

    function setup() {

      textSize(16);

      // Display the rounded number.
