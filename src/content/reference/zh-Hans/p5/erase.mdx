description: >
  <p>让程序开始使用形状擦除画布的部分区域。</p>

  <p>在调用 <code>erase()</code> 后的所有绘制的图形都被从画布中减去，显示出网页底部。擦除的区域将变为透明，使得画布后面的内容可以显示出来。一旦调用了 <code>erase()</code>，<a href="/reference/p5/fill">fill()</a>、<a href="/reference/p5/stroke">stroke()</a> 和 <a href="/reference/p5/blendMode">blendMode()</a> 将不起作用。</p>

  <p><code>erase()</code> 函数有两个可选参数。第一个参数设置形状内部擦除的强度。值为 0 表示不会发生擦除。值为 255 表示形状的内部将完全擦除其下的内容。默认值为 255（最大强度）。</p>

  <p>第二个参数设置形状边缘擦除的强度。值为 0 表示不会发生擦除。值为 255 表示形状的边缘将完全擦除其下的内容。默认值为 255（最大强度）。</p>

  <p>要取消擦除效果，请使用 <a href="/reference/p5/noErase">noErase()</a> 函数。</p>

  <p><code>erase()</code> 不会影响调用 <a href="/reference/p5/image">image()</a> 和 <a href="/reference/p5/background">background()</a> 函数进行的绘制。</p>
