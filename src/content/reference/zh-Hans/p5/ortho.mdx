description: >
  <p>为相机设置正交投影。</p>

  <p>在正交投影中，相同大小的形状始终呈现相同的大小，无论它们距离相机远近如何。</p>

  <p><code>ortho()</code> 通过将视锥体从截断的金字塔形状更改为矩形棱柱来改变相机的透视。相机放置在视锥体的前方，并查看视锥体内的所有内容。 <code>ortho()</code> 有六个可选参数来定义视锥体。</p>

  <p>前四个参数 <code>left</code>, <code>right</code>,
  <code>bottom</code>, 和 <code>top</code> 设置视锥体的边、底部和顶部的坐标。例如，调用 创建一个宽度为 200 像素、高度为 400 像素的视锥体。默认情况下，这些尺寸根据绘图的宽度和高度设置，如 <code>ortho(-width / 2, width / 2, -height / 2, height / 2)</code>。</p>

  <p>最后两个参数 <code>near</code> 和 <code>far</code> 设置视锥体近面和远面与相机的距离。例如，调用 <code>ortho(-100, 100, 200, 200, 50, 1000)</code> 创建一个宽度为 200 像素、高度为 400 像素的视锥体，距相机 50 像素开始，距相机 1,000 像素结束。默认情况下，<code>near</code> 和 <code>far</code> 分别设置为 0 和 <code>max(width, height) + 800</code>。</p>

  <p>注意：<code>ortho()</code> 只能在 WebGL 模式下使用。</p>
