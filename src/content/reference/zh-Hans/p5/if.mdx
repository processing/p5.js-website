description: >
  <p>一种选择是否运行一段代码的方法。</p>

  <p><code>if</code> 语句对于根据条件运行一段代码非常有帮助。例如 <code>if</code> 语句使得表达“如果鼠标被按下，则画一个圆”的想法变得简单：</p>

  <code>

  if (mouseIsPressed === true) {
    circle(mouseX, mouseY, 20);
  }

  </code>


  <p>语句头部以关键字 <code>if</code> 开头。括号中的表达式 <code>mouseIsPressed === true</code> 是一个 <code>Boolean</code> 表达式，
  其值为 <code>true</code> 或 <code>false</code>。花括号 <code>{}</code> 之间的代码是 if 语句的主体。
  只有当 <code>Boolean</code> 表达式的值是 <code>true</code> 时，主体中的代码才会运行。</p>

  <p><a href="/reference/p5/mouseIsPressed">mouseIsPressed</a> 系统变量始终为 <code>true</code> 或 <code>false</code>，所以上面的代码片段也可以写成：</p>

  <code>

  if (mouseIsPressed) {
    circle(mouseX, mouseY, 20);
  }

  </code>


  <p><code>if</code>-<code>else</code> 语句中的代码，只有当 <code>Boolean</code> 表达式的值为 <code>false</code> 的时候才会运行。
  例如：下面是一个 <code>if</code>-<code>else</code> 语句，表示“如果鼠标被按下，则画一个圆。否则，显示一条消息”：</p>

  <code>

  if (mouseIsPressed === true) {
    // When true.
    circle(mouseX, mouseY, 20);
  } else {
    // When false.
    text('Click me!', 50, 50);
  }

  </code>


  <p>这段代码片段有两条可能的路径，或者说分支。程序必须遵循其中一条分支。</p>

  <p><code>else</code>-<code>if</code> 语句允许添加多个分支。

  <code>else</code>-<code>if</code> 语句在不同的条件下运行不同的代码块。
  例如，<code>else</code>-<code>if</code> 语句使得表达“如果鼠标在左边，将画布涂白。
  如果鼠标在中间，将画布涂灰。否则，将画布涂黑。”变得简单：</p>

  <code>

  if (mouseX < 33) {
    background(255);
  } else if (mouseX < 67) {
    background(200);
  } else {
    background(0);
  }

  </code>


  <p><code>if</code> 语句可以根据需要添加多个 <code>else</code>-<code>if</code> 语句。但是，只能有一个 <code>else</code> 语句，并且且必须放在末尾。</p>

  <p><code>if</code> 语句还可以一次性判断多个条件。例如，<code>Boolean</code> 运算符 <code>&&</code>（且）判断两个表达式是否同时为 <code>true</code>。</p>

  <code>

  if (keyIsPressed === true && key === 'p') {
    text('You pressed the "p" key!', 50, 50);
  }

  </code>


  <p>如果用户按下键盘中的 <code>'p'</code> 键，则会显示一条信息。</p>

  <p><code>Boolean</code> 运算符 <code>||</code> (或) 判断两个表达式中是否至少有一个为 <code>true</code>。</p>

  <code>

  if (keyIsPressed === true || mouseIsPressed === true) {
    text('You did something!', 50, 50);
  }

  </code>


  <p>如果用户按下一个键，或者按下一个鼠标按钮，或者两者都按下，则会显示一条消息。</p>

  <p><code>if</code> 语句的主体可以包含另一个 <code>if</code> 语句。这被称为嵌套 <code>if</code> 语句。
  例如，嵌套 <code>if</code> 语句使得表达“如果按下一个键，判断这个键是否是'r'键。如果是，那么将填充颜色设置为红色。”变得简单：</p>

  <code>

  if (keyIsPressed === true) {
    if (key === 'r') {
      fill('red');
    }
  }

  </code>


  <p>学习更多关于这些数据类型及其支持的操作，请参阅 <a href="/reference/p5/Boolean">Boolean</a> 和 <a
  href="/reference/p5/Number">Number</a>。</p>
