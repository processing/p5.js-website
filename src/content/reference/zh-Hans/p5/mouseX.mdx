description: >
  <p>一个 <code>Number</code> 数字类型的系统变量，用于跟踪鼠标的水平位置。</p>

  <p>在 2D 模式下，<code>mouseX</code> 跟踪鼠标相对于画布左上角的位置。例如，如果鼠标距离画布的左边缘 50 像素，那么 <code>mouseX</code> 将为 50。</p>

  <p>在 WebGL 模式下，<code>mouseX</code> 跟踪鼠标相对于画布中心的位置。例如，如果鼠标位于画布中心右侧 50 像素处，那么 <code>mouseX</code> 将为 50。</p>

  <p>如果使用触摸代替鼠标，则 <code>mouseX</code> 将保持最近触摸点的 x 坐标。</p>

      // Adjust coordinates for WebGL mode.

      let mx = mouseX - 50;
