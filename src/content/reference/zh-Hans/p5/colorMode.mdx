description: >
  <p>改变颜色值的解读方式。</p>

  <p>默认情况下，<a href="/reference/p5/fill">fill()</a>、<a href="/reference/p5/stroke">stroke()</a>、<a href="/reference/p5/background">background()</a> 和 <a href="/reference/p5/color">color()</a> 的 <code>Number</code> 参数由使用 RGB 颜色模型下的 0 到 255 之间的值定义。这相当于调用了 <code>colorMode(RGB, 255)</code>。在这种模式中，纯红色是 <code>color(255, 0, 0)</code>。</p>

  <p>调用 <code>colorMode(RGB, 100)</code> 将颜色设置为使用介于 0 和 100 之间的 RGB 颜色值。在此模式中，纯红色是 <code>color(100, 0, 0)</code>。</p>

  <p>调用 <code>colorMode(HSB)</code> 或 <code>colorMode(HSL)</code> 将颜色模式改变为 HSB 或 HSL，而不是 RGB。在 HSB 模式中，纯红色是 <code>color(0, 100, 100)</code>，在 HSL 中是 <code>color(0, 100, 50)</code>。</p>

  <p><a href="/reference/p5/p5.Color">p5.Color</a> 对象则会记住它们创建时的颜色模式，它们的外观不会因为此函数更改颜色模式而改变。</p>

      createCanvas(100, 100);

      // Create a p5.Color object.
      let myColor = color(180, 175, 230);
      background(myColor);

      // Use RGB color with values in the range 0-1.
      colorMode(RGB, 1);

      // Get the red, green, and blue color components.
      let redValue = red(myColor);
      let greenValue = green(myColor);
      let blueValue = blue(myColor);

      // Round the color components for display.
      redValue = round(redValue, 2);
      greenValue = round(greenValue, 2);
      blueValue = round(blueValue, 2);

      // Display the color components.
      text(`Red: ${redValue}`, 10, 10, 80, 80);
      text(`Green: ${greenValue}`, 10, 40, 80, 80);
      text(`Blue: ${blueValue}`, 10, 70, 80, 80);

      describe('A purple canvas with the red, green, and blue decimal values of the color written on it.');
    }
