  <p>一个可以是正数、负数或零的数字。</p>

  <p><code>Number</code> 数据类型用于描述位置、大小和颜色等值。数字可以是整数，如 20，也可以是十进制数，如 12.34。例如，圆的位置和大小可以用三个数字来描述：</p>

  <code>

  circle(50, 50, 20);

  </code>

  <code>

  circle(50, 50, 12.34);

  </code>

  <p>数字支持基本的算术运算，并遵循标准的运算顺序：括号、指数、乘法、除法、加法和减法（PEMDAS）。例如，我们通常使用算术运算符与 p5.js 的系统变量，这些变量都是数字：</p>

  <code>

  // Draw a circle at the center.

  circle(width / 2, height / 2, 20);

  </code>

  <code>

  // Draw a circle that moves from left to right.

  circle(frameCount \* 0.01, 50, 20);

  </code>

  <p>以下是算术运算符的简要概述：</p>

  <code>

  1 + 2 // Add

  1 - 2 // Subtract

  1 * 2 // Multiply

  1 / 2 // Divide

  1 % 2 // Remainder

  1 ** 2 // Exponentiate

  </code>

  <p>常见的是使用算术运算更新数字变量。例如，可以像这样更新对象的位置：</p>

  <code>

  x = x + 1;

  </code>

  <p>上面的语句使用 <code>+</code> 运算符将 <code>x</code> 的值变为 <code>x</code> + 1。加法赋值运算符 <code>+=</code> 表达了相同的意思：</p>

  <code>

  x += 1;

  </code>

  <p>以下是赋值运算符的简要概述：</p>

  <code>

  x += 2 // Addition assignment

  x -= 2 // Subtraction assignment

  x *= 2 // Multiplication assignment

  x /= 2 // Division assignment

  x %= 2 // Remainder assignment

  </code>
