description: >
  <p>计算一个向量的大小或长度。</p>

  <p>向量可以用不同的方式来理解。在一种观点中，向量是空间中的一个点。向量的组成部分，<code>x</code>和<code>y</code>，是点的坐标<code>(x, y)</code>。向量的大小是从原点<code>(0, 0)</code>到<code>(x, y)</code>的距离。<code>mag(x, y)</code>是调用<code>dist(0, 0, x, y)</code>的快捷方式。</p>

  <p>向量也可以被认为是空间中指向某方向的箭头。这种视角对编程运动很有帮助。更多细节请参见<a href="/reference/p5/p5.Vector">p5.Vector</a>。</p>

  <p>可以使用<a href="/reference/p5.Vector/mag">p5.Vector.mag()</a>来计算<a href="/reference/p5/p5.Vector">p5.Vector</a>对象的大小。</p>

    function setup() {

      // Set the vector's components.
      let x = 30;

      // Calculate the magnitude.
      let m = mag(x, y);

      // Style the text.
      textSize(16);

      // Display the vector and its magnitude.
      line(0, 0, x, y);
      text(m, x, y);
