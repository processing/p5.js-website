description: |2-


  <p>在特定增量下计算两个数字之间的数值。</p>
  <p><code>amt</code> 参数是在两个数字之间插值的量。0.0 等于第一个数字，0.1 非常接近第一个数字，0.5 是中间值，1.0 等于第二个数字。<code>lerp()</code> 函数方便用于沿直线路径创建运动和绘制虚线。</p>
  <p>如果 <code>amt</code> 的值小于 0 或大于 1，<code>lerp()</code> 将返回原始区间之外的数字。例如，调用 <code>lerp(0, 10, 1.5)</code> 将返回 15。</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Declare variables for coordinates.
      let a = 20;
      let b = 80;
      let c = lerp(a, b, 0.2);
      let d = lerp(a, b, 0.5);
      let e = lerp(a, b, 0.8);

      strokeWeight(5);

      // Draw the original points in black.
      stroke(0);
      point(a, 50);
      point(b, 50);

      // Draw the lerped points in gray.
      stroke(100);
      point(c, 50);
      point(d, 50);
      point(e, 50);

      describe('Five points in a horizontal line. The outer points are black and the inner points are gray.');
    }
    </code>

      background(220);

      // Move x and y toward the target.
      x = lerp(x, targetX, 0.05);
      y = lerp(y, targetY, 0.05);

      // Draw the circle.
      circle(x, y, 20);
    }

    // Set x and y when the user clicks the mouse.
    function mouseClicked() {
      x = mouseX;
      y = mouseY;
    }
