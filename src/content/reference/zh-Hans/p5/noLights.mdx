  <p>从绘图中移除所有光源。</p>

  <p>调用 <code>noLights()</code> 会移除通过 <a href="/reference/p5/lights">lights()</a>, <a href="/reference/p5/ambientLight">ambientLight()</a>, <a href="/reference/p5/directionalLight">directionalLight()</a>, <a href="/reference/p5/pointLight">pointLight()</a>, 或 <a href="/reference/p5/spotLight">spotLight()</a> 创建的任何光源。在调用<code>nolights()</code>后，可以调用这些函数来创建新的光照方案。</p>

    <div>
    <code>
    // 单击并拖动鼠标以从不同角度查看场景。

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('Two spheres drawn against a gray background. The top sphere is white and the bottom sphere is red.');
    }

    function draw() {
      background(50);

      // 使用鼠标进行轨道控制。
      orbitControl();

      // 打开灯光。
      lights();

      // 设定球体样式。
      noStroke();

      // 绘制顶部球体。
      push();
      translate(0, -25, 0);
      sphere(20);
      pop();

      // 关闭灯光。
      noLights();

      // 添加一个指向屏幕内部的红色定向光。
      directionalLight(255, 0, 0, 0, 0, -1);

      // 绘制底部球体。
      push();
      translate(0, 25, 0);
      sphere(20);
      pop();
    }
    </code>
