description: >
  <p>当用户触摸屏幕并移动时调用的函数。</p>
  

  <p>声明函数 <code>touchMoved()</code> 会设置一个代码块，当用户触摸屏幕并移动时调用的函数。</p>

  <code>

  function touchMoved() {
    // Code to run.
  }

  </code>


  <p>当 p5.js 调用 <code>touchMoved()</code> 时，<a href="/reference/p5/touches">touches</a> 数组将被更新为最新的触摸点：</p>

  <code>

  function touchMoved() {
    // Paint over the background.
    background(200);

  <p>  // Mark each touch point while the user moves.
    for (let touch of touches) {
      circle(touch.x, touch.y, 40);
    }
  }

  </p></code>

  <p>参数 event 是可选的。<code>touchStarted()</code> 会被传递为一个描述触摸事件属性的 <a href="https://developer.mozilla.org/en-US/docs/Web/API/TouchEvent" target="_blank">TouchEvent</a> 对象：</p>

  <code>

  function touchMoved(event) {
    // Code to run that uses the event.
    console.log(event);
  }

  </code>

  <p>在屏幕设备上，如果未声明 <code>touchMoved()</code>，当用户的触摸点移动时，<a href="/reference/p5/mouseDragged">mouseDragged()</a> 将运行。如果声明了 <code>touchMoved()</code>，那么当用户的触摸点移动时，将运行 <code>touchMoved()</code>，而 <a href="/reference/p5/mouseDragged">mouseDragged()</a> 不会运行。</p>

  <p>注意：<a href="/reference/p5/touchStarted">touchStarted()</a>、<a href="/reference/p5/touchEnded">touchEnded()</a> 和 <code>touchMoved()</code> 是相关的。当用户触摸触屏设备时，<a href="/reference/p5/touchStarted">touchStarted()</a> 会立即运行。当用户结束触摸时，<a href="/reference/p5/touchEnded">touchEnded()</a> 会立即运行。当用户移动任何触摸点时，<code>touchMoved()</code> 会重复运行。</p>
