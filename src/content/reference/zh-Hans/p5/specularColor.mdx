description: >
  <p>设置灯光的高光颜色。</p>

  <p><code>specularColor()</code> 影响那些在表面上以首选方向反射的光线。这些光线包括 <a href="/reference/p5/directionalLight">directionalLight()</a>， <a href="/reference/p5/pointLight">pointLight()</a>， 和 <a href="/reference/p5/spotLight">spotLight()</a>。 该函数有助于在使用 <a href="/reference/p5/specularMaterial">specularMaterial()</a> 样式化的 <a href="/reference/p5/p5.Geometry">p5.Geometry</a> 对象上创建高光。如果几何体不使用 <a href="/reference/p5/specularMaterial">specularMaterial()</a>，那么 <code>specularColor()</code> 将不起作用。</p>

  <p>注意： <code>specularColor()</code> 不影响所有方向反射的光线，包括 <a href="/reference/p5/ambientLight">ambientLight()</a>
  和 <a href="/reference/p5/imageLight">imageLight()</a>。</p>

  <p>有三种调用 <code>specularColor()</code> 的方式，可以通过可选参数设置高光颜色。</p>

  <p>第一种调用 <code>specularColor()</code> 有两个可选参数， <code>gray</code> 和 <code>alpha</code>。 介于 0 和 255 之间的灰度和 alpha 值， 例如 <code>specularColor(50)</code> 或 <code>specularColor(50, 80)</code>， 可以传递来设置高光颜色。</p>

  <p>第二种调用 <code>specularColor()</code> 有一个可选参数，<code>color</code>。可以传递一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象，一个颜色值数组，或一个 CSS 颜色字符串来设置高光颜色。</p>

  <p>调用 <code>specularColor()</code> 的第三种方式具有四个可选参数， <code>v1</code>, <code>v2</code>, <code>v3</code>， 和 <code>alpha</code>。 RGBA、 HSBA、 或 HSLA 值，例如 <code>specularColor(255, 0, 0, 80)</code>， 可用于设置高光颜色。颜色值将使用当前的 <a href="/reference/p5/colorMode">colorMode()</a> 进行解释。</p>

        // 从右上方添加一个白色点光源。
        pointLight(255, 255, 255, 30, -20, 40);
      }

      // 绘制球体。
      sphere(30);
    }

    function doubleClicked() {
      isLit = true;
    }
