description: >
  <p>一个存储键值对数据容器。</p>
  <p>对象有助于组织任何类型的相关数据，包括其他对象。
   存储在对象中的值可以通过名称访问，称为其键。每个键值对称为“属性”。对象类似于 Python 中的字典，Java、Ruby 中的映射。</p>
  <p>例如，一个对象可以包含 dog 的位置、大小和外观：</p>
  <code>
  // Declare the dog variable and assign it an object.
  let dog = { x: 50, y: 50, size: 20, emoji: '\U0001F436' };
  
  <p>// Style the text.
  textAlign(CENTER, CENTER);
  textSize(dog.size);</p>
  <p>// Draw the dog.
  text(dog.emoji, dog.x, dog.y);
  </p></code>
  <p>变量 <code>dog</code> 被赋予一个具有四个属性的对象。对象声明在花括号 <code>{}</code> 之中。使用点运算符可以访问对象的值，如 <code>dog.size</code>。在上面的例子中，键 <code>size</code> 对应的值是 <code>20</code>。对象的初始值也可以为空：</p>
  <code>
  // Declare a cat variable and assign it an empty object.
  let cat = {};
  
  <p>// Add properties to the object.
  cat.x = 50;
  cat.y = 50;
  cat.size = 20;
  cat.emoji = '\U0001F431';</p>
  <p>// Style the text.
  textAlign(CENTER, CENTER);
  textSize(cat.size);</p>
  <p>// Draw the cat.
  text(cat.emoji, cat.x, cat.y);
  </p></code>
  <p>对象的数据可以在绘图进行时被更新。例如，可以更新变量 <code>cat</code> 的值，使其远离 <code>dog</code>：</p>
  <code>
  // Run to the right.
  cat.x += 5;
  </code>
  
  <p>如果需要，可以使用方括号 <code>[]</code> 和字符串，而不用点符号来访问对象的值：</p>
  <code>
  // Run to the right.
  cat[\"x\"] += 5;
  </code>
  
  <p>当键名有空格时，这种语法会很有帮助，例如，<code>cat['height (m)']</code>。</p>
