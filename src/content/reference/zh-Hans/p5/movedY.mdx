description: >
  <p>一个 <code>Number</code> 数字类型的系统变量，用于跟踪鼠标的垂直移动。</p>

  <p><code>movedY</code> 跟踪鼠标在帧之间上升或下降的像素数。<code>movedY</code> 如果鼠标在帧之间上移会有一个负值，如果下移则有一个正值。<code>movedY</code> 可以计算为 <code>mouseY - pmouseY</code>。</p>

  <p>注意：<code>movedY</code> 即使在 <a href="/reference/p5/requestPointerLock">requestPointerLock()</a> 激活时也会继续更新。</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square. The text "▲" appears when the user moves the mouse upward. The text "▼" appears when the user moves the mouse downward.'
      );
    }

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display ▼ when movedY is positive and
      // ▲ when it's negative.
      if (movedY > 0) {
        text('▼', 50, 50);
      } else if (movedY < 0) {
        text('▲', 50, 50);
      }
    }
