description: >
  <p>创建一个用于简单音频/视频播放的 <code>&lt;video&gt;</code> 元素。</p>

  <p><code>createVideo()</code> 返回一个新的 <a href="#/p5.MediaElement">p5.MediaElement</a> 对象。默认情况下，视频是可见的。通过调用 <code>video.hide()</code> 可以将其隐藏，并使用 <a href="./image">image()</a> 绘制到画布上。</p>

  <p>第一个参数 <code>src</code> 是视频的路径。如果传递了单个字符串，如 <code>'/assets/topsecret.mp4'</code>，则加载单个视频。字符串数组可用于以不同格式加载同一视频。例如，<code>['/assets/topsecret.mp4', '/assets/topsecret.ogv', '/assets/topsecret.webm']</code>。这对于确保视频能够在不同的浏览器中播放很有用，因为不同的浏览器具有不同的功能。有关支持的格式的更多信息，请参见<a href='https://developer.mozilla.org/en-US/docs/Web/HTML/Supported_media_formats'>MDN</a>。</p>

  <p>第二个参数 <code>callback</code> 是可选的。它是一旦视频准备好播放就会调用的函数。</p>

      noCanvas();

      // Load a video and add it to the page.
      // Provide an array options for different file formats.
      // Call mute() once the video loads.
      video = createVideo(
        ['/assets/small.mp4', '/assets/small.ogv', '/assets/small.webm'],
        muteVideo
      );

      // Show the default video controls.
      video.showControls();

      describe('A video of a toy robot with playback controls beneath it.');
    }

    // Mute the video once it loads.
    function muteVideo() {
      video.volume(0);
    }
