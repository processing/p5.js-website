description: |2-

  <p>从画布中获取一个像素或一个像素区域。</p>


  <p><code>get()</code> 很容易使用，但速度不如


  <a href="/reference/p5/pixels">pixels</a>。读取许多像素值时请使用 <a
  href="/reference/p5/pixels">pixels</a>。</p>


  <p>不带参数的 <code>get()</code> 版本返回整个画布。</p>


  <p>带有两个参数的 <code>get()</code> 版本将参数解释为坐标。它返回一个带有定点像素 <code>[R, G, B, A]</code> 值的数组。</p>


  <p>带有四个参数的 <code>get()</code> 版本将参数解释为坐标和尺寸。它返回画布的一分段作为 <a href="/reference/p5/p5.Image">p5.Image</a> 对象。前两个参数是分段左上角的坐标。最后两个参数是分段的宽度和高度。</p>


  <p>使用 <a href="/reference/p5.Image/get">p5.Image.get()</a> 可以直接得到 <a href="/reference/p5/p5.Image">p5.Image</a> 对象进行处理。</p>
