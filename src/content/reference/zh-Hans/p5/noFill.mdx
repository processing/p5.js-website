  <p>禁用形状的填充颜色。</p>


  <p>调用 <code>noFill()</code> 相当于填充一个完全透明的颜色，如同调用 <code>fill(0, 0)</code>。如果 <code>noStroke()</code> 和 <a href="/reference/p5/noFill">noFill()</a> 同时被调用，则画布上不会绘制任何内容。</p>
line: '1219'
isConstructor: 'false'
itemtype: method
example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Draw the top square.
      square(32, 10, 35);

      // Draw the bottom square.
      noFill();
      square(32, 55, 35);

      describe('A white square on above an empty square. Both squares have black outlines.');
    }
    </code>
    </div>

    <div modernizr='webgl'>
    <code>
    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A purple cube wireframe spinning on a black canvas.');
    }

    function draw() {
      background(0);

      // Style the box.
      noFill();
      stroke(100, 100, 240);

      // Rotate the coordinates.
      rotateX(frameCount * 0.01);
      rotateY(frameCount * 0.01);

      // Draw the box.
      box(45);
    }
    </code>
    </div>
