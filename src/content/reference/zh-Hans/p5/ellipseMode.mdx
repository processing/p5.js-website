description: >
  
  <p>改变椭圆、圆和圆弧绘制的方式。</p>

  <p>默认情况下，<a href="/reference/p5/ellipse">ellipse()</a>、<a href="/reference/p5/circle">circle()</a> 和 <a href="/reference/p5/arc">arc()</a> 的前两个参数是形状中心的 x 和 y 坐标。接下来的参数设置形状的宽度和高度。这相当于调用 <code>ellipseMode(CENTER)</code>。</p>

  <p><code>ellipseMode(RADIUS)</code> 也使用前两个参数设置形状中心的 x 和 y 坐标。接下来的参数是形状宽度和高度的一半。调用 <code>ellipse(0, 0, 10, 15)</code> 会绘制一个宽度为 20、高度为 30 的形状。</p>

  <p><code>ellipseMode(CORNER)</code> 使用前两个参数作为形状的左上角。接下来的参数是它的宽度和高度。</p>

  <p><code>ellipseMode(CORNERS)</code> 使用前两个参数作为椭圆外接矩形的一个角的位置。接下来的参数是另一个角的位置。</p>

  <p>传递给 <code>ellipseMode()</code> 的参数必须使用全大写，因为常量 <code>CENTER</code>、<code>RADIUS</code>、<code>CORNER</code> 和 <code>CORNERS</code> 是这样定义的。JavaScript 是区分大小写的编程语言。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // White ellipse.
      ellipseMode(RADIUS);
      fill(255);
      ellipse(50, 50, 30, 30);

      // Gray ellipse.
      ellipseMode(CENTER);
      fill(100);
      ellipse(50, 50, 30, 30);

      describe('A white circle with a gray circle at its center. Both circles have black outlines.');
    }
    </code>
    </div>

      ellipseMode(CORNER);
      fill(255);
      ellipse(25, 25, 50, 50);

      // Gray ellipse.
      ellipseMode(CORNERS);
      fill(100);
      ellipse(25, 25, 50, 50);

      describe('A white circle with a gray circle at its top-left corner. Both circles have black outlines.');
    }
    </code>
