description: >
  <p>一个 <code>Number</code> 系统变量，包含最后输入键的代码。</p>

  <p>所有键都有一个 <code>keyCode</code>。例如，<code>a</code> 键的 <code>keyCode</code> 为 65。

  <code>keyCode</code> 变量对于检查是否输入了特殊键非常有用。例如，以下条件检查是否输入了回车键：</p>

  <code>

  if (keyCode === 13) {
    // 如果按下了回车键，运行的代码。
  }

  </code>

  <p>相同的代码可以使用系统变量 <code>ENTER</code> 更清晰地写出，其值为 13：</p>

  <code>

  if (keyCode === ENTER) {
    // 如果按下了回车键，运行的代码。
  }

  </code>

  <p>系统变量 <code>BACKSPACE</code>、<code>DELETE</code>、<code>ENTER</code>、<code>RETURN</code>、<code>TAB</code>、

  <code>ESCAPE</code>、<code>SHIFT</code>、<code>CONTROL</code>、<code>OPTION</code>、<code>ALT</code>、<code>UP_ARROW</code>、
  <code>DOWN_ARROW</code>、

  <code>LEFT_ARROW</code> 和 <code>RIGHT_ARROW</code> 都是特殊键的键码的有用简写。键码可以在如

  <a href="http://keycode.info/">keycode.info</a> 等网站上找到。</p>
