description: >
  <p>一个 <code>Number</code> 数字类型的系统变量，用于追踪鼠标的前一个垂直位置。</p>

  <p>在 2D 模式下，<code>pmouseY</code> 跟踪鼠标相对于画布左上角的位置。其值是上一帧的 <a href="/reference/p5/mouseY">mouseY</a>。例如，如果上一帧鼠标距离画布顶边 50 像素，则 <code>pmouseY</code> 将为 50。</p>

  <p>在 WebGL 模式下，<code>pmouseY</code> 跟踪鼠标相对于画布中心的位置。例如，如果上一帧鼠标在画布中心下方 50 像素，则 <code>pmouseY</code> 将为 50。</p>

  <p>如果使用触摸代替鼠标，则 <code>pmouseY</code> 将保持最后一个触摸点的 y 坐标。</p>

  <p>注意：<code>pmouseY</code> 在每个触摸事件开始时重置为当前的 <a href="/reference/p5/mouseY">mouseY</a> 值。</p>

    function setup() {

      createCanvas(100, 100, WEBGL);

      describe('A line follows the mouse as it moves. The line grows longer with faster movements.');
    }

    function draw() {
      background(200);

      // Adjust coordinates for WebGL mode.
      // The origin (0, 0) is at the center of the canvas.
      let pmx = pmouseX - 50;
      let pmy = pmouseY - 50;
      let mx = mouseX - 50;
      let my = mouseY - 50;

      // Draw the line.
      line(pmx, pmy, mx, my);
    }
