description: >
  <p>创建一个向所有方向发光的光源。</p>

  <p>环境光不是来自一个方向。相反，3D 形状从所有方向均匀地受到照亮。环境光几乎总是与其他类型的光一起使用。</p>

  <p>有三种调用 <code>ambientLight()</code> 的方式，可以通过可选参数设置光的颜色。</p>

  <p>第一种调用 <code>ambientLight()</code> 的方式有两个参数，<code>gray</code> 和 <code>alpha</code>。 <code>alpha</code> 是可选的。可以传递介于 0 和 255 之间的灰度和 alpha 值来设置环境光的颜色， 例如 <code>ambientLight(50)</code> 或 <code>ambientLight(50, 30)</code>。</p>

  <p>第二种调用 <code>ambientLight()</code> 的方式有一个参数，就是颜色。可以传递一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象、一个颜色值数组或一个 CSS 颜色字符串，例如 <code>ambientLight('magenta')</code>，来设置环境光的颜色。</p>

  <p>第三种调用 <code>ambientLight()</code> 的方式有四个参数，<code>v1</code>、 <code>v2</code>、 <code>v3</code> 和 <code>alpha</code>。 <code>alpha</code> 是可选的。可以传递 RGBA、 HSBA 或 HSLA 值来设置环境光的颜色，例如 <code>ambientLight(255, 0, 0)</code> 或 <code>ambientLight(255, 0, 0, 30)</code>。颜色值将根据当前的 <a href="/reference/p5/colorMode">colorMode()</a> 进行诠释。</p>

      sphere(30);
