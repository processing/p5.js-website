description: >
  <p>停止在平面形状内创建孔。</p>

  <p><a href="/reference/p5/beginContour">beginContour()</a> 和
  <code>endContour()</code> 函数允许在平面自定义形状内创建负空间。
  
  <a href="/reference/p5/beginContour">beginContour()</a> 开始添加顶点到负空间，而

  <code>endContour()</code> 则停止添加。

  <a href="/reference/p5/beginContour">beginContour()</a> 和
  <code>endContour()</code> 必须在

  <a href="/reference/p5/beginShape">beginShape()</a> 和

  <a href="/reference/p5/endShape">endShape()</a> 之间被调用。</p>

  <p>变换如 <a href="/reference/p5/translate">translate()</a>，

  <a href="/reference/p5/rotate">rotate()</a>，和 <a
  href="/reference/p5/scale">scale()</a>

  在 <a href="/reference/p5/beginContour">beginContour()</a> 和 <code>endContour()</code> 之间不起作用。
  
  在 <a href="/reference/p5/beginContour">beginContour()</a> 和
  <code>endContour()</code> 之间之间也不可能使用其他形状，比如

  <a href="/reference/p5/ellipse">ellipse()</a> 或 <a
  href="/reference/p5/rect">rect()</a>。</p>

  <p>注意：定义负空间的顶点必须以与外部形状相反的方向“绕”。首先，以顺时针顺序绘制外部形状的顶点。然后，以逆时针顺序绘制负空间的顶点。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Start drawing the shape.
      beginShape();

      // Exterior vertices, clockwise winding.
      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // Interior vertices, counter-clockwise winding.
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // Stop drawing the shape.
      endShape(CLOSE);

      describe('A white square with a square hole in its center drawn on a gray background.');
    }
    </code>
