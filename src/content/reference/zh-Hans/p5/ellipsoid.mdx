description: >
  <p>绘制一个椭球体。</p>

  <p>椭球体是一个具有连接以形成圆形表面的三角形面的 3D 形状。面数较少的椭球体看起来像晶体。面数较多的椭球体具有光滑的表面，看起来像鸡蛋。

  <code>ellipsoid()</code> 通过其半径来定义形状。这与

  <a href="/reference/p5/ellipse">ellipse()</a> 使用直径（宽度和高度）的方式不同。</p>

  <p>第一个参数 <code>radiusX</code> 是可选的。如果传递一个<code>Number</code>，如 <code>ellipsoid(20)</code>，
  它设置椭球体沿 x 轴的半径。默认情况下，<code>radiusX</code> 是 50。</p>

  <p>第二个参数 <code>radiusY</code> 也是可选的。如果传递一个<code>Number</code>，如 <code>ellipsoid(20, 30)</code>，
  它设置椭球体沿 y 轴的半径。默认情况下，<code>radiusY</code> 设置为椭球体的 <code>radiusX</code>。</p>

  <p>第三个参数 <code>radiusZ</code> 也是可选的。如果传递一个<code>Number</code>，如 <code>ellipsoid(20, 30, 40)</code>，
  它设置椭球体沿 z 轴的半径。默认情况下，<code>radiusZ</code> 设置为椭球体的 <code>radiusY</code>。</p>

  <p>第四个参数 <code>detailX</code> 也是可选的。如果传递一个<code>Number</code>，如 <code>ellipsoid(20, 30, 40, 5)</code>，
  它设置沿 x 轴使用的三角形细分数。所有 3D 形状都是通过连接三角形来形成其表面的。默认情况下，<code>detailX</code> 是 24。</p>

  <p>第五个参数 <code>detailY</code> 也是可选的。如果传递一个<code>Number</code>，如 <code>ellipsoid(20, 30, 40, 5, 7)</code>，
  它设置沿 y 轴使用的三角形细分数。所有 3D 形状都是通过连接三角形来形成其表面的。默认情况下，<code>detailY</code> 是 16。</p>

  <p>注意：<code>ellipsoid()</code> 只能在 WebGL 模式下使用。</p>
