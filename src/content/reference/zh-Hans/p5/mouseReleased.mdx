description: >
  <p>当鼠标按钮释放时调用一次的函数。</p>

  <p>声明函数 <code>mouseReleased()</code> 将设置一个代码块，当用户在按下鼠标按钮后释放时自动运行：</p>

  <code>

  function mouseReleased() {
    // 要运行的代码。
  }

  </code>

  <p>鼠标系统变量，如 <a href="/reference/p5/mouseX">mouseX</a> 和 <a href="/reference/p5/mouseY">mouseY</a>，将在 p5.js 调用 <code>mouseReleased()</code> 时更新为最新的值：</p>

  <code>

  function mouseReleased() {
    if (mouseX < 50) {
      // 如果鼠标在左边，运行的代码。
    }

    if (mouseY > 50) {
      // 如果鼠标在底部附近，运行的代码。
    }
  }

  </code>

  <p>参数 <code>event</code> 是可选的。 <code>mouseReleased()</code> 总是传递一个 <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent" target="_blank">MouseEvent</a> 对象，其属性描述了鼠标释放事件：</p>

  <code>

  function mouseReleased(event) {
    // 使用事件运行的代码。
    console.log(event);
  }

  </code>

  <p>在触摸屏设备上，如果没有声明 <a href="/reference/p5/touchEnded">touchEnded()</a>，当用户的触摸结束时，<code>mouseReleased()</code> 会运行。如果声明了 <a href="/reference/p5/touchEnded">touchEnded()</a>，那么当用户的触摸结束时，<a href="/reference/p5/touchEnded">touchEnded()</a> 会运行，而 <code>mouseReleased()</code> 不会。</p>

  <p>浏览器可能会将默认行为附加到各种鼠标事件上。例如，一些浏览器在用户按下鼠标按钮移动鼠标时会高亮显示文本。要防止此事件的任何默认行为，在函数的末尾添加 <code>return false;</code>。</p>

  <p>注意：<a href="/reference/p5/mousePressed">mousePressed()</a>、<code>mouseReleased()</code> 和 <a href="/reference/p5/mouseClicked">mouseClicked()</a> 都是相关的。<a href="/reference/p5/mousePressed">mousePressed()</a> 在用户点击鼠标时运行。<code>mouseReleased()</code> 在用户释放鼠标点击时运行。<a href="/reference/p5/mouseClicked">mouseClicked()</a> 在 <code>mouseReleased()</code> 之后立即运行。</p>
