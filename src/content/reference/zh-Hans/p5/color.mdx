description: |2-

  <p>创建一个<a href="/reference/p5/p5.Color">p5.Color</a>对象。</p>


  <p>默认情况下，传入的参数将被解读为 RGB 值。调用 <code>color(255, 204, 0)</code> 将返回亮黄色。参数的解读方式可能被 <a href="/reference/p5/colorMode">colorMode()</a> 改变。</p>


  <p>如果 <code>color()</code> 只有一个参数，该参数有两种解读方式。如果参数是数字，则将其解读为灰度值。如果参数是字符串，则将其解读为 CSS 颜色字符串。</p>


  <p>如果 <code>color()</code> 带有两个参数，第一个参数解读为灰度值。第二个参数设置 alpha（透明度）值。</p>


  <p>带有三个参数的 <code>color()</code> 会将参数作为 RGB、HSB 或 HSL 颜色的色值进行解读，具体取决于当前的 <code>colorMode()</code> 设置。</p>


  <p>带有四个参数的 <code>color()</code> 会将参数解读为
  RGBA、HSBA 或 HSLA 颜色，具体取决于当前的 <code>colorMode()</code> 设置。最后一个参数设置 alpha（透明度）值。</p>
