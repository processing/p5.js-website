description: |2-

  <p>从绘图生成 GIF 并将其保存到文件中。</p>


  <p>可以在 <a href="/reference/p5/setup">setup()</a> 或任何绘图运行时的点调用 <code>saveGif()</code> 。</p>


  <p>第一个参数，<code>fileName</code>，设置 GIF 的文件名。</p>


  <p>第二个参数，<code>duration</code>，设置 GIF 的持续秒时间。</p>


  <p>第三个参数<code>options</code>是可选的。如果一个对象通过了， <code>saveGif()</code> 将使用其属性来自定义 GIF。
  <code>saveGif()</code> 识别属性<code>delay</code>、<code>units</code>、
  <code>silent</code>，
  <code>notificationDuration</code> 和 <code>notificationID</code>。</p>

params:
  - name: filename
    description: |2-
  
      <p>GIF 文件名。</p>
    type: 字符串
  - name: duration
    description: |2-
  
      <p>从绘图中捕获的持续时间以秒为单位。</p>
    type: 数字
  - name: options
    description: |2-
  
      <p>一个对象可以包含另外五个属性：
  
                       <code>delay</code>，一个数字，指定录制前等待的时间；
  
                       <code>units</code>，一个字符串，可以是“秒”或“帧”。默认为“秒”；
  
                       <code>silent</code>，一个布尔定义进度通知的存在。默认为 <code>false</code>；
  
                       <code>notificationDuration</code>，一个数字，定义最终通知的时长以秒为单位会活下去。默认情况下它是<code>0</code>，这意味着通知永远不会被删除；
  
      "<code>notificationID</code>，一个指定通知 DOM 的元素 id 的字符串。默认情况下它是<code>'progressBar'</code>。</p>"
