description: >
  <p>设置形状表面材质的自发光颜色。</p>

  <p><code>emissiveMaterial()</code> 颜色设置形状以全强度显示的颜色，不受光照影响。这可以使形状看起来发光。但是，自发光材质实际上并不会发出能够影响周围物体的光。</p>

  <p><code>emissiveMaterial()</code> 可以通过不同的参数以三种方式调用来设置材质的颜色。</p>

  <p>第一种调用 <code>emissiveMaterial()</code> 的方式有一个参数 <code>gray</code>。可以传递介于 0 和 255 之间的灰度值，如 <code>emissiveMaterial(50)</code>，来设置材质的颜色。较高的灰度值会使形状看起来更亮。</p>

  <p>第二种调用 <code>emissiveMaterial()</code> 的方式有一个参数 <code>color</code>。可以传递一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象、一个颜色值数组或一个 CSS 颜色字符串，如 <code>emissiveMaterial('magenta')</code>，来设置材质的颜色。</p>

  <p>第三种调用 <code>emissiveMaterial()</code> 的方式有四个参数，<code>v1</code>、<code>v2</code>、<code>v3</code> 和 <code>alpha</code>。 <code>alpha</code> 是可选的。RGBA、HSBA 或 HSLA值可以传递来设置材质的颜色，例如 <code>emissiveMaterial(255, 0, 0)</code> 或 <code>emissiveMaterial(255, 0, 0, 30)</code>。颜色值将根据当前的 <a href="/reference/p5/colorMode">colorMode()</a>进行解释。</p>

  <p>注意：<code>emissiveMaterial()</code> 只能在 WebGL 模式下使用。</p>
