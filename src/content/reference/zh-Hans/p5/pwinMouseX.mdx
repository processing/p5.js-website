description: >
  <p>一个 <code>Number</code> 数字类型的系统变量，用于跟踪鼠标在浏览器内的前一个水平位置。</p>

  <p><code>pwinMouseX</code> 记录鼠标相对于浏览器窗口左上角的位置。其值是上一帧的 <a href="/reference/p5/winMouseX">winMouseX</a>。例如，如果上一帧鼠标距离浏览器左边缘 50 像素，那么 <code>pwinMouseX</code> 将为 50。</p>

  <p>在触摸屏设备上，<code>pwinMouseX</code> 将保持最近触摸点的 x 坐标。<code>pwinMouseX</code> 在每个触摸事件开始时重置为当前的 <a href="/reference/p5/winMouseX">winMouseX</a> 值。</p>

  <p>注意：使用 <a href="/reference/p5/pmouseX">pmouseX</a> 跟踪鼠标在画布内的前一个 x 坐标。</p>
