description: |2-

  <p>在空间中绘制一个点。</p>


  <p>点的默认宽度是一个像素。要为点着色，使用 <a href="/reference/p5/stroke">stroke()</a> 函数。要更改其宽度，使用
   <a href="/reference/p5/strokeWeight">strokeWeight()</a>函数。一个点无法填充颜色，因此 <a href="/reference/p5/fill">fill()</a> 函数不会影响点的颜色。</p>


  <p>带有两个参数的 <code>point()</code> 通过 x 和 y 坐标确定位置，比如 <code>point(10, 20)</code>。</p>


  <p>带有三个参数的<code>point()</code> 可以将点绘制在具有 x、y 和 z 坐标的 3D 空间中，如 <code>point(10, 20, 30</code>。这样做时需要在调用 <a href="/reference/p5/createCanvas">createCanvas()</a> 时将 <code>WEBGL</code> 参数传入。</p>


  <p>带有一个参数的 <code>point()</code> 其位置通过一个 <a href="/reference/p5/p5.Vector">p5.Vector</a> 对象进行设置。</p>

      createCanvas(100, 100);

      background(200);

      // Top-left.
      point(30, 20);

      // Top-right.
      point(70, 20);

      // Style the next points.
      stroke('purple');
      strokeWeight(10);

      // Bottom-right.
      point(70, 80);

      // Bottom-left.
      point(30, 80);

    }

      point(b);

      point(c);

      describe(

    }
