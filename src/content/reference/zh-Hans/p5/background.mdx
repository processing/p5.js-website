description: >
  <p>设置画布背景的颜色</p>

  <p>默认情况下，画布的背景是透明的。<code>background()</code> 通常和 <a href="/reference/p5/draw">draw()</a> 一起使用在每一帧绘制之初清空显示窗口里的内容。它也可以在 <a href="/reference/p5/setup">setup()</a> 函数内调用，给动画的第一帧设置背景。</p>

  <p>当 <code>background()</code> 带有一个参数的时候，参数有四种解读方式。如果参数是一个 <code>Number</code>, 该参数将作为灰度值解读。如果参数是 <code>String</code>, 该参数将作为 CSS 颜色字符串解读，并支持 RGB, RGBA, HSL, HSLA, hex, 以及 CSS 颜色名。 如果该参数是一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象，它将作为背景颜色使用。如果该参数是一个 <a href="/reference/p5/p5.Image">p5.Image</a> 它将作为背景图片使用。</p>

  <p>当 <code>background()</code> 带有两个参数的时候，第一个参数将作为灰度值解读。第二个参数设置 alpha（透明度）。</p>

  <p>当 <code>background()</code> 带有三个参数的时候，这三个参数将作为 RGB, HSB, 或 HSL 模式下的颜色值解读，具体的模式取决于 <a href="/reference/p5/colorMode">colorMode()</a>。</p>
  <p>默认情况下，颜色数值将以 RGB 模式解读，调用 <code>background(255, 204, 0)</code> 会将背景设为一个亮黄色。</p>
