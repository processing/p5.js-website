description: |2-

  <p>获取颜色的饱和度值。</p>


  <p><code>saturation()</code> 从一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象、一个由颜色组件组成的数组或一个 CSS 颜色字符串中提取饱和度值。</p>


  <p>HSB 和 HSL 中的饱和度缩放比例不同。默认情况下，<code>satruation()</code> 返回数值为 0 到 100 范围内颜色的 HSL 饱和度。如果 <a href="/reference/colorMode">colorMode()</a> 设置为 HSB 或 HSL，则返回给定模式下的饱和度值。</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(50);

      // Use HSB color.
      colorMode(HSB);

      // Create a p5.Color object.
      let c = color(0, 50, 100);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // Set 'satValue' to 50.
      let satValue = saturation(c);

      // Draw the right rectangle.
      fill(satValue);
      rect(50, 15, 35, 70);

      describe('Two rectangles. The left one is salmon pink and the right one is dark gray.');
    }
