description: |2-

  <p>将一个数字从一个范围映射到 0 到 1 之间的值。</p>
  <p>例如，<code>norm(2, 0, 10)</code> 返回 0.2。数字 2 在原始范围 [0, 10] 中的位置与 0.2 在范围 [0, 1] 中的位置成比例。这与调用 <code>map(2, 0, 10, 0, 1)</code> 是相同的。</p>
  <p>超出原始范围的数字不会被限制在 0 和 1 之间。超出范围的值通常是有意的并且很有用。</p>
