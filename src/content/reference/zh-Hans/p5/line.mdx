description: |2-

  <p>在两个点之间绘制一条线段。</p>


  <p>线段的默认粗细为 1 个像素。当向 <code>line()</code> 传入 4 个参数时将在 2D 模式下绘制一条线段。使用 <a href="/reference/p5/stroke">stroke()</a> 函数可以给一条线段上色。使用 <a href="/reference/p5/strokeWeight">strokeWeight()</a> 函数可以调整线段的粗细。线段不能被填充颜色，因此 <a href="/reference/p5/fill">fill()</a> 函数不会影响其颜色。</p>


  <p>当向 <code>line()</code> 传入 6 个参数时将在 3D 空间中绘制。要这样做时，需要在调用 <a href="/reference/p5/createCanvas">createCanvas()</a> 传入 <code>WEBGL</code> 参数。</p>

      background(200);

      // Rotate around the y-axis.
      rotateY(frameCount * 0.01);

      // Draw a line.
      line(0, 0, 0, 30, 20, -10);

      // Draw the center sphere.
      sphere(10);

      translate(30, 20, -10);

      // Draw the bottom-right sphere.
