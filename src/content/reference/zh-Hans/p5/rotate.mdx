description: >
  <p>旋转坐标系。</p>

  <p>默认情况下，正 x 轴指向右侧，正 y 轴指向下方。<code>rotate()</code> 函数通过围绕原点旋转坐标系来改变这种方向。在调用 <code>rotate()</code> 之后绘制的所有内容都将呈现为旋转状态。</p>

  <p>第一个参数 <code>angle</code> 是旋转的角度。例如，调用 <code>rotate(1)</code> 将坐标系顺时针旋转 1 弧度，约为 57 度。<code>rotate()</code> 根据当前的 <a href="/reference/p5/angleMode">angleMode()</a> 解释角度值。</p>

  <p>第二个参数 <code>axis</code> 是可选的。它用于定位 WebGL 模式中的 3D 旋转。如果传递了一个 <a href="/reference/p5/p5.Vector">p5.Vector</a>，如 <code>rotate(QUARTER_PI, myVector)</code>，那么坐标系将围绕 <code>myVector</code> 旋转 <code>QUARTER_PI</code> 弧度。如果传递了一个向量分量数组，如 <code>rotate(QUARTER_PI, [1, 0, 0])</code>，那么坐标系将围绕具有分量 <code>[1, 0, 0]</code> 的向量旋转 <code>QUARTER_PI</code> 弧度。</p>

  <p>默认情况下，转换会累积。例如，连续调用两次 <code>rotate(1)</code> 与调用一次 <code>rotate(2)</code> 效果相同。可以使用 <a href="/reference/p5/push">push()</a> 和 <a href="/reference/p5/pop">pop()</a> 函数将转换隔离到不同的绘图组中。</p>

  <p>注意：转换会在绘制循环的开始处重置。在 <a href="/reference/p5/draw">draw()</a> 函数中调用 <code>rotate(1)</code> 不会导致形状旋转。</p>
