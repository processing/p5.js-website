  <p>使用 <a href="./text">text()</a> 函数设置字体。</p>


  <p>第一个参数<code>font</code>设置字体。
  "<code>textFont()</code> 识别一个 <a href="./p5.Font">p5.Font</a> 对象或一个带有系统字体名称的字符串。例如，<code>'Courier New'</code>。</p>"


  <p>第二个参数<code>size</code>是可选的。它以以像素为单位来设置字体大小。

  这与调用 <a href="./textSize">textSize()</a> 具有相同的效果。</p>


  <p>注意：<code>WebGL</code> 模式仅支持加载 <a href="./loadFont">loadFont()</a> 的字体。</p>

    function setup() {
      background(200);
      textFont(fontRegular);
      text('I am Normal', 10, 30);
      textFont(fontItalic);
      text('I am Italic', 10, 50);
      textFont(fontBold);
      text('I am Bold', 10, 70);
