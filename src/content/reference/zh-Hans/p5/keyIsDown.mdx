description: >
  <p>如果正在检查的键被按下，则返回 <code>true</code>，否则返回 <code>false</code>。</p>

  <p><code>keyIsDown()</code> 在检查多个不同键按下时很有帮助。例如，<code>keyIsDown()</code> 可用于检查是否同时按下了 <code>LEFT_ARROW</code> 和 <code>UP_ARROW</code>：</p>

  <code>
  if (keyIsDown(LEFT_ARROW) && keyIsDown(UP_ARROW)) {
    // 对角线移动。
  }
  </code>

  <p><code>keyIsDown()</code> 可以使用 <a href="/reference/p5/keyCode">keyCode</a> 值检查按键，如 <code>keyIsDown(37)</code> 或 <code>keyIsDown(LEFT_ARROW)</code>。键码可以在如 <a href="https://keycode.info" target="_blank">keycode.info</a> 等网站上找到。</p>

    // Click on the canvas to begin detecting key presses.

    let x = 50;

    function draw() {
      // Update x and y if an arrow key is pressed.
      if (keyIsDown(LEFT_ARROW) === true) {
        x -= 1;
      }

      if (keyIsDown(RIGHT_ARROW) === true) {
        x += 1;
      }

      if (keyIsDown(UP_ARROW) === true) {
        y -= 1;
      }

      if (keyIsDown(DOWN_ARROW) === true) {
        y += 1;
      }

      // Style the circle.
      fill(0);

      // Draw the circle.
      circle(x, y, 5);
    }
