description: |2-

  <p>使用颜色对图像进行着色。</p>


  <p>带有一个参数的 <code>tint()</code> 版本可以用以下四种方式之一来解释。


  如果参数是数字，则将其解释为灰度值。如果
  参数是一个字符串，则将其解释为 CSS 颜色字符串。一个 <code>[R, G, B, A]</code> 数组或 <a href="./p5.Color">p5.Color</a> 对象
  也可用于设置色调颜色。</p>


  <p>带有两个参数的 <code>tint()</code> 版本使用第一个参数作为一个灰度值，第二个作为 alpha 值。例如，调用 <code>tint(255, 128)</code> 可以使图像 50% 透明。</p>

  <p>取决于当前的 <a href="./colorMode"> colorMode()</a> 具有三个参数的 <code>tint()</code> 版本可以将它们解释为
  RGB 或 HSB 值。可选的第四个参数设置 alpha 值。例如， <code>tint(255, 0, 0, 100)</code> 将图片设置成红色调并使它们透明。</p>
