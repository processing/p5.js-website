  <p>获取颜色的绿色值。</p>

  <p><code>green()</code> 从一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象、一个由颜色组件组成的数组或一个 CSS 颜色字符串中提取绿色值。</p>

  <p>默认情况下，<code>green()</code> 返回一个 0 到 255 范围内的绿色值。如果 <a href="/reference/colorMode">colorMode()</a> 设置为 RGB， 它返回给定范围内的绿色值。</p>

  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Create a p5.Color object.
      let c = color(175, 100, 220);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // Set 'greenValue' to 100.
      let greenValue = green(c);

      // Draw the right rectangle.
      fill(0, greenValue, 0);
      rect(50, 15, 35, 70);

      describe('Two rectangles. The left one is light purple and the right one is dark green.');
    }
