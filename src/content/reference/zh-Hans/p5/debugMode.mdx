description: >
  <p>添加网格和轴标志以帮助明确3D绘图中的方向。</p>

  <p><code>debugMode()</code> 函数添加一个网格， 显示绘图中“地面”的位置。默认情况下，网格将通过绘图的原点 <code>(0, 0, 0)</code> 沿着　XZ 平面运行。 <code>debugMode()</code> 还添加了一个沿着正 x、y 和 z 轴指向的坐标轴图标。调用 <code>debugMode()</code> 会显示网格和坐标轴图标， 以及它们的默认大小和位置。</p>

  <p>>有四种调用 <code>debugMode()</code> 的方式，可选参数用于自定义调试环境。</p>

  <p>第一种调用 <code>debugMode()</code> 的方式有一个参数，<code>mode</code>。 如果传递了系统常量 <code>GRID</code>， 例如<code>debugMode(GRID)</code>, 那么将显示网格并隐藏坐标轴图标。 如果传递了常量<code>AXES</code>，例如 <code>debugMode(AXES)</code>， 则会显示坐标轴图标并隐藏网格。</p>

  <p>第二种调用 <code>debugMode()</code> 的方式有六个参数。第一个参数　<code>mode</code>,  选择要显示的 <code>GRID</code> 或　<code>AXES</code>。 接下来的五个参数, <code>gridSize</code>, <code>gridDivisions</code>,　<code>xOff</code>, <code>yOff</code>, 和　<code>zOff</code> 是可选的。 它们是用于设置网格外观　(<code>gridSize</code> 和 <code>gridDivisions</code>) 以及放置坐标轴图标 (<code>xOff</code>, <code>yOff</code>, 和 <code>zOff</code>) 的数字。 例如，调用<code>debugMode(20, 5, 10, 10, 10)</code> 将 <code>gridSize</code> 设置为 20 个像素, 并将 <code>gridDivisions</code> 设置为 5, 并且沿着 x、y 和 z 轴偏移坐标轴图标 1 0像素。</p>

  <p>第三种调用 <code>debugMode()</code> 的方式有五个参数。 第一个参数 <code>mode</code> 选择要显示的 <code>GRID</code> 或 <code>AXES</code>。接下来的四个参数, <code>axesSize</code>, <code>xOff</code>, <code>yOff</code>, 和 <code>zOff</code> 是可选的。它们是用于设置坐标轴图标大小 （ <code>axesSize</code>） 以及其放置位置 （ <code>xOff</code>、 <code>yOff</code> 和 <code>zOff</code>）的数字。</p>

  <p>第四种调用 <code>debugMode()</code> 的方式有九个可选参数。前五个参数 <code>gridSize</code>、 <code>gridDivisions</code>、 <code>gridXOff</code>、 <code>gridYOff</code> 和 <code>gridZOff</code> 是用于设置网格外观的数字。例如，调用 <code>debugMode(100, 5, 0, 0, 0)</code> 将 <code>gridSize</code> 设置为 100，将 <code>gridDivisions</code> 设置为 5，并将所有偏移量设置为 0，以便网格居中于原点。接下来的四个参数 <code>axesSize</code>、 <code>xOff</code>、 <code>yOff</code> 和 <code>zOff</code> 是用于设置坐标轴图标大小的数字（ <code>axesSize</code>）以及其放置位置（ <code>axesXOff</code>、 <code>axesYOff</code> 和 <code>axesZOff</code>）。例如，调用 <code>debugMode(100, 5, 0, 0, 0, 50, 10, 10, 10)</code> 将 <code>gridSize</code> 设置为 100，将 <code>gridDivisions</code> 设置为 5，并将所有偏移量设置为 0，以便网格居中于原点。然后将 <code>axesSize</code> 设置为 50，并将图标沿每个轴偏移 10 像素。</p>
