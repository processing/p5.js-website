description: >
  <p>启用或禁用 3D 草图中线条的透视功能。</p>

  <p>在 WebGL 模式中，当线条距离摄像头较远时，可以使用较细的笔触来绘制线条。这样可以让线条看起来更真实。</p>

  <p>默认情况下，线条会根据所使用的透视类型以不同的方式绘制：</p>

  <ul>

  <li><code>perspective()</code> 和 <code>frustum()</code> 用于模拟逼真的透视效果。在这两种模式中，笔触的重量受线条与相机间距离的影响。这样做可以使线条外观更自然。
  <code>perspective()</code> 是 3D 草图中的默认模式。</li>

  <li><code>ortho()</code> 不会模拟真实的视角。在该模式下，无论线条与摄像机的距离如何，笔划权重都是一致的。这样可以获得更可预测且更一致的外观。</li>

  </ul>

  <p><code>linePerspective()</code> 可以覆盖默认的线条绘制模式。</p>

  <p>参数 <code>enable</code> 是个可选参数。它是一个 <code>Boolean</code> 值，用于设置线条的绘制方式。如果传入的值为 <code>true</code> ，比如 <code>linePerspective(true)</code> ，那么当线条离摄像机较远时，就会显得较细。如果传入值为 <code>false</code> ，比如 <code>linePerspective(false)</code> ，则无论线条与摄像机的距离如何，其笔画权重都将保持一致。默认情况下，启用 <code>linePerspective()</code> 。</p>

  <p>如果启用了 <code>linePerspective()</code> 则调用它而不传递参数将返回 <code>true</code> 。如果未启用，则返回 <code>false</code> 。</p>

  <p>注意：<code>linePerspective()</code> 只能在 WebGL 模式下使用。</p>

    // Double-click the canvas to toggle the line perspective.

    function setup() {
      createCanvas(100, 100, WEBGL);

        'A white cube with black edges on a gray background. Its edges toggle between thick and thin when the user double-clicks.'
      );
    }

    function draw() {
      background(200);

      // Translate the origin toward the camera.
      translate(-10, 10, 600);

      // Rotate the coordinate system.
      rotateY(-0.1);
      rotateX(-0.1);

      // Draw the row of boxes.
      for (let i = 0; i < 6; i += 1) {
        translate(0, 0, -40);
        box(10);
      }
    }

    // Toggle the line perspective when the user double-clicks.
    function doubleClicked() {
      let isEnabled = linePerspective();
      linePerspective(!isEnabled);
    }
