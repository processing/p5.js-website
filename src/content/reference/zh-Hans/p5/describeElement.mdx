description: |2-

  <p>在画布中创建屏幕阅读器可访问的元素描述。</p>


  <p>元素是有实际意义的图形或图形组合。

  例如，几个重叠的圆可以构成“眼睛”这个元素。</p>


  <p>第一个参数，<code>name</code>，是元素的名称。</p>


  <p>第二个参数，<code>text</code>，是对元素的描述。</p>


  "<p>第三个参数，<code>display</code>，是可选项。它决定了描述的展示方式。如果它的值是 <code>LABEL</code>，例如：<code>describe('A description.', LABEL)</code>，描述将会展示在画布旁边的 div 元素中。使用 <code>LABEL</code> 并没有实际用处，它只是屏幕阅读器中的重复项。仅在开发环境中使用 <code>LABEL</code>。如果是 <code>FALLBACK</code>，例如：<code>describe('A description.',"
  FALLBACK）</code>，描述仅对屏幕阅读器可见，这是默认模式。</p>


  <p>了解更多关于制作无障碍绘图的信息，请参阅 <a href="/learn/accessible-labels.html">编写无障碍的画布描述。</a>
  </p>

    function setup() {
      background('pink');

      // Describe the first element
      // and draw it.
      describeElement('Circle', 'A yellow circle in the top-left corner.');
      noStroke();
      fill('yellow');
      circle(25, 25, 40);

      // Describe the second element
      // and draw it.
      describeElement('Heart', 'A red heart in the bottom-right corner.');

      circle(83.2, 66.6, 20);
      triangle(91.2, 72.6, 75, 95, 58.6, 72.6);

      // Add a general description of the canvas.
      describe('A red heart and yellow circle over a pink background.');
    }
