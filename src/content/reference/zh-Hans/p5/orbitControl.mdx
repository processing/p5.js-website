description: >
  <p> 允许用户使用鼠标、触控板或

  触摸屏围绕 3D 绘图进行旋转。</p>

  <p> 通过一个虚拟的摄像机可以查看 3D 绘图。在 

  <a href="/reference/p5/draw">draw()</a> 函数中调用 <code>orbitControl()</code> 
  允许用户

  改变摄像机的位置:</p>

  <code>

  function draw() {
    background(200);

  <p>  // 启用鼠标旋转。
    orbitControl();</p>
  <p>  // 绘图的其余部分。

  }

  </p></code>

  <p> 左键点击并拖动或滑动将使摄像机围绕绘图中心旋转。

  右键点击并拖动或多点滑动可以在不旋转摄像机的情况下平移其位置。

  使用鼠标滚轮（滚动）或进行捏合动作可以使摄像机远离或靠近绘图中心。 </p>

  <p>前三个参数，

  <code>sensitivityX</code>、

  <code>sensitivityY</code> 和

  <code>sensitivityZ</code> 都是可选的。它们是设置绘图对每个轴向移动的灵敏度的数字。例如，调用

  <code>orbitControl(1, 2, -1)</code> 会保持 x 轴的移动在默认值，并使得绘图对 y 轴的移动灵敏度加倍，

  以及使 z 轴反向移动。在默认情况下，所有灵敏度的值均为 1。</p>

  <p>第四个参数，
  <code>options</code>，也是可选的。这是一个

  改变旋转行为的对象。例如，调用

  <code>orbitControl(1, 1, 1, options)</code> 可以在保持默认灵敏度值的同时，

  改变由 <code>options</code> 设置的行为。该对象可以具有以下属性：</p>

  <code>

let options = {
 // 将此设置为 false 可以防止在旋转时意外与页面交互,从而让移动交互更加顺畅。
 // 默认情况下，此值为 true。
    disableTouchActions: true,

  <p>  // 将此设置为 true 会使摄像机总是按照鼠标/触摸的移动方向旋转。
  // 默认情况下，此值为 false。
    freeRotation: false
  };</p>

  <p>orbitControl(1, 1, 1, options);

  </p></code>

    function draw() {
      background(200);

      // 创建一个 options 对象。
      let options = {
        disableTouchActions: false,
        freeRotation: true
      };

      // 启用鼠标控制轨道。
      // 防止触摸屏设备上的意外触控动作
      // 并启用自由旋转。
      orbitControl(1, 1, 1, options);

      // 设定盒子样式。
      normalMaterial();

      // 绘制该盒子。
      box(30, 50);
    }
