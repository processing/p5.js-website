description: >
  <p>设置形状表面材质的环境颜色。</p>

  <p><code>ambientMaterial()</code> 颜色设置形状将反射的 <a href="/reference/p5/ambientLight">ambientLight()</a> 颜色的组件。例如，调用 <code>ambientMaterial(255, 255, 0)</code> 将使形状反射红色和绿色光，但不反射蓝色光。</p>

  <p><code>ambientMaterial()</code> 可以通过不同的参数以三种方式调用来设置材质的颜色。</p>

  <p>第一种调用 <code>ambientMaterial()</code> 的方式有一个参数 <code>gray</code>。可以传递介于 0 和 255 之间的灰度值，如 <code>ambientMaterial(50)</code>，来设置材质的颜色。较高的灰度值会使形状看起来更亮。</p>

  <p>第二种调用 <code>ambientMaterial()</code> 的方式有一个参数 <code>color</code>。可以传递一个 <a href="/reference/p5/p5.Color">p5.Color</a> 对象、一个颜色值数组或一个 CSS 颜色字符串，如 <code>ambientMaterial('magenta')</code>，来设置材质的颜色。</p>

  <p>第三种调用 <code>ambientMaterial()</code> 的方式有三个参数 <code>v1</code>、<code>v2</code> 和 <code>v3</code>。可以传递 RGB、HSB 或 HSL 值，如 <code>ambientMaterial(255, 0, 0)</code>，来设置材质的颜色。颜色值将根据当前的 <a href="/reference/p5/colorMode">colorMode()</a> 进行解释。</p>

  <p>注意：<code>ambientMaterial()</code> 只能在 WebGL 模式下使用。</p>
