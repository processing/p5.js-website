description: >
  <p>创建一个新的 <a href="/reference/p5/p5.Vector">p5.Vector</a> 对象。</p>

  <p>向量可以用不同的方式来理解。在一个视角中，向量就像是空间中指向某方向的箭头。向量既有大小（长度）也有方向。这种视角对于编程运动非常有帮助。</p>

  <p>向量的组成决定了它的大小和方向。例如，调用 <code>createVector(3, 4)</code> 会创建一个新的 <a href="/reference/p5/p5.Vector">p5.Vector</a> 对象，其 x 部分为 3，y 部分为 4。从原点出发，这个向量的尖端向右 3 个单位，向下 4 个单位。</p>

  <p><a href="/reference/p5/p5.Vector">p5.Vector</a> 对象常用于编程运动，因为它们简化了数学计算。例如，一个移动的球有一个位置和一个速度。位置描述了球在空间中的位置。球的位置向量从原点延伸到球的中心。速度描述了球的速度和移动的方向。如果球直线向上移动，它的速度向量就直线向上。向球的位置向量添加球的速度向量，就可以移动它，如 <code>pos.add(vel)</code>。向量数学依赖于 <a href="/reference/p5/p5.Vector">p5.Vector</a> 类内的方法。</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Create p5.Vector objects.
      let p1 = createVector(25, 25);
      let p2 = createVector(50, 50);
      let p3 = createVector(75, 75);

      // Draw the dots.
      strokeWeight(5);
      point(p1);
      point(p2);
      point(p3);

    }

    <div>
    <code>
    let pos;
    let vel;

    function setup() {
      createCanvas(100, 100);

      // Create p5.Vector objects.
      pos = createVector(50, 100);
      vel = createVector(0, -1);

      describe('A black dot moves from bottom to top on a gray square. The dot reappears at the bottom when it reaches the top.');
    }

    function draw() {
      background(200);

      // Add velocity to position.
      pos.add(vel);

      // If the dot reaches the top of the canvas,
      // restart from the bottom.
      if (pos.y < 0) {
        pos.y = 100;
      }

      // Draw the dot.
      strokeWeight(5);
      point(pos);
    }
    </code>
