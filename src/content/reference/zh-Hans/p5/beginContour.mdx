description: >
  <p>开始在一个平面形状中创建一个孔洞。</p>

  <p><code>beginContour()</code> 和 <a
  href="/reference/p5/endContour">endContour()</a>

  函数可以用来在平面自定义形状内创建负空间。<code>beginContour()</code> 开始添加负空间的顶点，而

  <a href="/reference/p5/endContour">endContour()</a> 则停止添加。

  <code>beginContour()</code> 和 <a
  href="/reference/p5/endContour">endContour()</a> 必须在 <a href="/reference/p5/beginShape">beginShape()</a> 和

  <a href="/reference/p5/endShape">endShape()</a> 之间调用。</p>

  <p>在 <code>beginContour()</code> 和

  <a href="/reference/p5/endContour">endContour()</a> 之间，一些变换图形的函数（如 <a href="/reference/p5/translate">translate()</a>，

  <a href="/reference/p5/rotate">rotate()</a>，和 <a
  href="/reference/p5/scale">scale()</a>）不起作用。

  也不能在 <code>beginContour()</code> 和 <a href="/reference/p5/endContour">endContour()</a>

  之间使用其他形状，比如 <a href="/reference/p5/ellipse">ellipse()</a> 或

  <a href="/reference/p5/rect">rect()</a>。</p>

  <p>注意：定义负空间的顶点必须与外部形状的顶点方向相反。首先，以顺时针顺序绘制外部形状的顶点。然后，以逆时针顺序绘制负空间的顶点。</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Start drawing the shape.
      beginShape();

      // Exterior vertices, clockwise winding.
      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // Interior vertices, counter-clockwise winding.
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // Stop drawing the shape.
      endShape(CLOSE);

      describe('A white square with a square hole in its center drawn on a gray background.');
    }
    </code>
