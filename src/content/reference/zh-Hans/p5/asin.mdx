  <p>计算一个数字的反正弦值。</p>

  <p><code>asin()</code> 是 <a href="/reference/p5/sin">sin()</a> 的逆函数。它期望输入值在 -1 到 1 的范围内。默认情况下，<code>asin()</code> 返回的值范围是 -π ÷ 2（约-1.57）到 π ÷ 2（约1.57）。如果 <a href="/reference/p5/angleMode">angleMode()</a> 设置为 <code>DEGREES</code>，则返回的值范围是 -90 到 90。</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Calculate sin() and asin() values.
      let a = PI / 3;
      let s = sin(a);
      let as = asin(s);

      // Display the values.
      text(`${round(a, 3)}`, 35, 25);
      text(`${round(s, 3)}`, 35, 50);
      text(`${round(as, 3)}`, 35, 75);

      describe('The numbers 1.047, 0.866, and 1.047 written on separate rows.');
    }
    </code>

    }
