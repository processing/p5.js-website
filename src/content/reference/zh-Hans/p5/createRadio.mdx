description: >
  <p>创建一个单选按钮元素。</p>

  <p>参数是可选的。如果传递了一个字符串，如

  <code>let myRadio = createSelect('food')</code>，那么每个单选选项将

  有 <code>"food"</code> 作为其 <code>name</code> 参数：<code>&lt;input
  name="food"&gt;&lt;/input&gt;</code>。

  如果传递了一个现有的 <code>&lt;div&gt;&lt;/div&gt;</code> 或
  <code>&lt;span&gt;&lt;/span&gt;</code>

  元素，如 <code>let myRadio = createSelect(container)</code>，
  它将

  成为单选按钮的父元素。</p>

  <p>单选按钮扩展了 <a href="#/p5.Element">p5.Element</a> 类，增加了一些

  有用的方法来管理选项：</p>

  <ul>

  <li><code>myRadio.option(value, [label])</code> 向菜单添加一个选项。
  第一个参数，<code>value</code>，是一个字符串，设置选项的
  值和标签。第二个参数，<code>label</code>，是可选的。如果提供，它设置显示的标签为 <code>value</code>。如果一个选项
  的 <code>value</code> 已经存在，其标签将被更改��返回其值。</li>

  <li><code>myRadio.value()</code> 返回当前选中选项的
  值。</li>

  <li><code>myRadio.selected()</code> 返回当前选中的
  选项。</li>

  <li><code>myRadio.selected(value)</code> 选择给定的选项并将
  它作为一个 <a
  href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement"
  target="_blank"><code>HTMLInputElement</code></a> 返回。</li>

  <li><code>myRadio.disable(shouldDisable)</code> 如果传递 <code>true</code> 则启用整个单选按钮，如果传递 <code>false</code> 则禁用它。</li>

  </ul>
