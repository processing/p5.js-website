  <p>将相机左右旋转。</p>
  <p>平移旋转相机而不改变其位置。旋转发生在相机的“本地”空间中。</p>
  <p>参数 <code>angle</code> 是相机应该旋转的角度。传入正角度，如 <code>myCamera.pan(0.001)</code>，将相机向右旋转。传入负角度，如 <code>myCamera.pan(-0.001)</code>，将相机向左旋转。</p>
  <p>注意：角度根据当前的 <a href="/reference/p5/angleMode">angleMode()</a> 进行解读。</p>

    <div>
    <code>
    let cam;
    let delta = 0.001;

    function setup() {
      createCanvas(100, 100, WEBGL);

      // Create a p5.Camera object.
      cam = createCamera();

      // Place the camera at the top-center.
      cam.setPosition(0, -400, 800);

      // Point the camera at the origin.
      cam.lookAt(0, 0, 0);

      describe(
        'A white cube on a gray background. The cube goes in and out of view as the camera pans left and right.'
      );
    }

    function draw() {
      background(200);

      // Pan with the camera.
      cam.pan(delta);

      // Switch directions every 120 frames.
      if (frameCount % 120 === 0) {
        delta *= -1;
      }

      // Draw the box.
      box();
    }
