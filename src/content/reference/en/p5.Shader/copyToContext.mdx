  <p>Shaders belong to the main canvas or a p5.Graphics. Once they are compiled,
  they can only be used in the context they were compiled on.</p>
  <p>Use this method to make a new copy of a shader that gets compiled on a
  different context.</p>

    let graphic = createGraphics(200, 200, WEBGL);

    graphic.shader(graphicShader); // Use graphicShader on the graphic

    let mainShader = graphicShader.copyToContext(window);
    shader(mainShader); // Use `mainShader` on the main canvas
    </code>
