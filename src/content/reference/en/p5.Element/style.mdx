description: >
  <p>Applies a style to the element by adding a

  <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/Syntax"
  target="_blank">CSS declaration</a>.</p>

  <p>The first parameter, <code>property</code>, is a string. If the name of a
  style

  property is passed, as in <code>myElement.style('color')</code>, the method
  returns

  the current value as a string or <code>null</code> if it hasn't been set. If a

  <code>property:style</code> string is passed, as in

  <code>myElement.style('color:deeppink')</code>, the method sets the style
  <code>property</code>

  to <code>value</code>.</p>

  <p>The second parameter, <code>value</code>, is optional. It sets the
  property's value.

  <code>value</code> can be a string, as in

  <code>myElement.style('color', 'deeppink')</code>, or a

  <a href="/reference/p5/p5.Color">p5.Color</a> object, as in

  <code>myElement.style('color', myColor)</code>.</p>
