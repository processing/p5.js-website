  <p>Returns the color formatted as a <code>String</code>.</p>

  <p>Calling <code>myColor.toString()</code> can be useful for debugging, as in

  <code>print(myColor.toString())</code>. It's also helpful for using p5.js with
  other

  libraries.</p>

  <p>The parameter, <code>format</code>, is optional. If a format string is
  passed, as in

  <code>myColor.toString('#rrggbb')</code>, it will determine how the color
  string is

  formatted. By default, color strings are formatted as <code>'rgba(r, g, b,
  a)'</code>.</p>

  - name: format
    description: >
      <p>how the color string will be formatted.

      Leaving this empty formats the string as rgba(r, g, b, a).

      '#rgb' '#rgba' '#rrggbb' and '#rrggbbaa' format as hexadecimal color
      codes.

      'rgb' 'hsb' and 'hsl' return the color formatted in the specified color
      mode.

      'rgba' 'hsba' and 'hsla' are the same as above but with alpha channels.

      'rgb%' 'hsb%' 'hsl%' 'rgba%' 'hsba%' and 'hsla%' format as
      percentages.</p>
