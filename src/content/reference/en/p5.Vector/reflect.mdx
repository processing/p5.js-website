  <p>Reflects a vector about a line in 2D or a plane in 3D.</p>

  <p>The orientation of the line or plane is described by a normal vector that

  points away from the shape.</p>

  <p>The static version of <code>reflect()</code>, as in
  <code>p5.Vector.reflect(v, n)</code>,

  returns a new <a href="#/p5.Vector">p5.Vector</a> object and doesn't change

  the original.</p>

    function drawArrow(base, vec, myColor) {

      stroke(myColor);
      strokeWeight(3);
      fill(myColor);
      translate(base.x, base.y);
      line(0, 0, vec.x, vec.y);
      rotate(vec.heading());
      let arrowSize = 7;
      translate(vec.mag() - arrowSize, 0);
      triangle(0, arrowSize / 2, 0, -arrowSize / 2, arrowSize, 0);
