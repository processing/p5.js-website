description: >
  <p>Performs modulo (remainder) division with a vector's <code>x</code>,
  <code>y</code>, and <code>z</code>

  components.</p>

  <p><code>rem()</code> can use separate numbers, as in <code>v.rem(1, 2,
  3)</code>,

  another <a href="/reference/p5/p5.Vector">p5.Vector</a> object, as in
  <code>v.rem(v2)</code>, or

  an array of numbers, as in <code>v.rem([1, 2, 3])</code>.</p>

  <p>If only one value is provided, as in <code>v.rem(2)</code>, then all the
  components

  will be set to their values modulo 2. If two values are provided, as in

  <code>v.rem(2, 3)</code>, then <code>v.z</code> won't change. Calling
  <code>rem()</code> with no

  arguments, as in <code>v.rem()</code>, has no effect.</p>

  <p>The static version of <code>rem()</code>, as in <code>p5.Vector.rem(v2,
  v1)</code>, returns a

  new <a href="/reference/p5/p5.Vector">p5.Vector</a> object and doesn't change
  the

  originals.</p>
