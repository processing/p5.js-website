description: >
  <p>Sets the specular color for lights.</p>

  <p><code>specularColor()</code> affects lights that bounce off a surface in a
  preferred

  direction. These lights include

  <a href="/reference/p5/directionalLight/">directionalLight()</a>,

  <a href="/reference/p5/pointLight/">pointLight()</a>, and

  <a href="/reference/p5/spotLight/">spotLight()</a>. The function helps to
  create

  highlights on <a href="/reference/p5/p5.Geometry">p5.Geometry</a> objects that
  are

  styled with <a href="/reference/p5/specularMaterial/">specularMaterial()</a>.
  If a

  geometry does not use

  <a href="/reference/p5/specularMaterial/">specularMaterial()</a>, then

  <code>specularColor()</code> will have no effect.</p>

  <p>Note: <code>specularColor()</code> doesnâ€™t affect lights that bounce in all

  directions, including <a href="/reference/p5/ambientLight/">ambientLight()</a>
  and

  <a href="/reference/p5/imageLight/">imageLight()</a>.</p>

  <p>There are three ways to call <code>specularColor()</code> with optional
  parameters to

  set the specular highlight color.</p>

  <p>The first way to call <code>specularColor()</code> has two optional
  parameters, <code>gray</code>

  and <code>alpha</code>. Grayscale and alpha values between 0 and 255, as in

  <code>specularColor(50)</code> or <code>specularColor(50, 80)</code>, can be
  passed to set the

  specular highlight color.</p>

  <p>The second way to call <code>specularColor()</code> has one optional
  parameter,

  <code>color</code>. A <a href="/reference/p5/p5.Color">p5.Color</a> object, an
  array of color

  values, or a CSS color string can be passed to set the specular highlight

  color.</p>

  <p>The third way to call <code>specularColor()</code> has four optional
  parameters, <code>v1</code>,

  <code>v2</code>, <code>v3</code>, and <code>alpha</code>. RGBA, HSBA, or HSLA
  values, as in

  <code>specularColor(255, 0, 0, 80)</code>, can be passed to set the specular
  highlight

  color. Color values will be interpreted using the current

  <a href="/reference/p5/colorMode/">colorMode()</a>.</p>

      createCanvas(100, 100, WEBGL);

      describe('A sphere drawn on a gray background. A spotlight starts shining when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Style the sphere.
      noStroke();
      specularColor(100);
      specularMaterial(255, 255, 255);

      // Control the light.
      if (isLit === true) {
        // Add a white point light from the top-right.
        pointLight(255, 255, 255, 30, -20, 40);
      }

      // Draw the sphere.
      sphere(30);
    }

    // Turn on the point light when the user double-clicks.
    function doubleClicked() {
      isLit = true;
    }
