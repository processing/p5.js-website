description: >
  <p>An <code>Array</code> of all the current touch points on a touchscreen
  device.</p>

  <p>The <code>touches</code> array is empty by default. When the user touches
  their

  screen, a new touch point is tracked and added to the array. Touch points

  are <code>Objects</code> with the following properties:</p>

  <pre><code class="language-js">// Iterate over the touches array.

  for (let touch of touches) {
    // x-coordinate relative to the top-left
    // corner of the canvas.
    console.log(touch.x);

    // y-coordinate relative to the top-left
    // corner of the canvas.
    console.log(touch.y);

    // x-coordinate relative to the top-left
    // corner of the browser.
    console.log(touch.winX);

    // y-coordinate relative to the top-left
    // corner of the browser.
    console.log(touch.winY);

    // ID number
    console.log(touch.id);
  }

  </code></pre>
