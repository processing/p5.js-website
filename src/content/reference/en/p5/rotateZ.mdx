description: >
  <p>Rotates the coordinate system about the z-axis in WebGL mode.</p>

  <p>The parameter, <code>angle</code>, is the amount to rotate. For example,
  calling

  <code>rotateZ(1)</code> rotates the coordinate system about the z-axis by 1
  radian.

  <code>rotateZ()</code> interprets angle values using the current

  <a href="/reference/p5/angleMode/">angleMode()</a>.</p>

  <p>By default, transformations accumulate. For example, calling
  <code>rotateZ(1)</code>

  twice has the same effect as calling <code>rotateZ(2)</code> once. The

  <a href="/reference/p5/push/">push()</a> and <a
  href="/reference/p5/pop/">pop()</a> functions

  can be used to isolate transformations within distinct drawing groups.</p>

  <p>Note: Transformations are reset at the beginning of the draw loop. Calling

  <code>rotateZ(1)</code> inside the <a href="/reference/p5/draw/">draw()</a>
  function won't cause

  shapes to spin.</p>
