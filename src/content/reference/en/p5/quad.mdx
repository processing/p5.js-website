description: >
  <p>Draws a quadrilateral (four-sided shape).</p>

  <p>Quadrilaterals include rectangles, squares, rhombuses, and trapezoids. The

  first pair of parameters <code>(x1, y1)</code> sets the quad's first point.
  The next

  three pairs of parameters set the coordinates for its next three points

  <code>(x2, y2)</code>, <code>(x3, y3)</code>, and <code>(x4, y4)</code>.
  Points should be added in either

  clockwise or counter-clockwise order.</p>

  <p>The version of <code>quad()</code> with twelve parameters allows the quad
  to be drawn

  in 3D space. Doing so requires adding the <code>WEBGL</code> argument to

  <a href="#/p5/createCanvas">createCanvas()</a>.</p>

  <p>The thirteenth and fourteenth parameters are optional. In WebGL mode, they

  set the number of segments used to draw the quadrilateral in the x- and

  y-directions. They're both 25 by default.</p>
