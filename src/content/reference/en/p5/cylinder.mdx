description: >
  <p>Draws a cylinder.</p>

  <p>A cylinder is a 3D shape with triangular faces that connect a flat bottom

  to a flat top. Cylinders with few faces look like boxes. Cylinders with

  many faces have smooth surfaces.</p>

  <p>The first parameter, <code>radius</code>, is optional. If a
  <code>Number</code> is passed, as in

  <code>cylinder(20)</code>, it sets the radius of the cylinder’s base. By
  default,

  <code>radius</code> is 50.</p>

  <p>The second parameter, <code>height</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>cylinder(20, 30)</code>, it sets the cylinder’s height. By
  default,

  <code>height</code> is set to the cylinder’s <code>radius</code>.</p>

  <p>The third parameter, <code>detailX</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>cylinder(20, 30, 5)</code>, it sets the number of edges used to
  form the

  cylinder's top and bottom. Using more edges makes the top and bottom look

  more like circles. By default, <code>detailX</code> is 24.</p>

  <p>The fourth parameter, <code>detailY</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>cylinder(20, 30, 5, 2)</code>, it sets the number of triangle
  subdivisions

  to use along the y-axis, between cylinder's the top and bottom. All 3D

  shapes are made by connecting triangles to form their surfaces. By default,

  <code>detailY</code> is 1.</p>

  <p>The fifth parameter, <code>bottomCap</code>, is also optional. If a
  <code>false</code> is passed,

  as in <code>cylinder(20, 30, 5, 2, false)</code> the cylinder’s bottom won’t
  be drawn.

  By default, <code>bottomCap</code> is <code>true</code>.</p>

  <p>The sixth parameter, <code>topCap</code>, is also optional. If a
  <code>false</code> is passed, as

  in <code>cylinder(20, 30, 5, 2, false, false)</code> the cylinder’s top won’t
  be

  drawn. By default, <code>topCap</code> is <code>true</code>.</p>

  <p>Note: <code>cylinder()</code> can only be used in WebGL mode.</p>
