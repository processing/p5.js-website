description: >
  <p>Draws certain features with smooth (antialiased) edges.</p>

  <p><code>smooth()</code> is active by default. In 2D mode,

  <a href="/reference/p5/noSmooth/">noSmooth()</a> is helpful for scaling up
  images

  without blurring. The functions don't affect shapes or fonts.</p>

  <p>In WebGL mode, <a href="/reference/p5/noSmooth/">noSmooth()</a> causes all
  shapes to

  be drawn with jagged (aliased) edges. The functions don't affect images or

  fonts.</p>
