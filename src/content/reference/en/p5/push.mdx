description: >
  <p>The <a href="/reference/p5/push">push()</a> function saves the current
  drawing style

  settings and transformations, while <a href="/reference/p5/pop">pop()</a>
  restores these

  settings. Note that these functions are always used together. They allow you
  to

  change the style and transformation settings and later return to what you had.

  When a new state is started with <a href="/reference/p5/push">push()</a>, it
  builds on

  the current style and transform information. The <a
  href="/reference/p5/push">push()</a>

  and <a href="/reference/p5/pop">pop()</a> functions can be embedded to provide
  more

  control. (See the second example for a demonstration.)</p>

  <p><a href="/reference/p5/push">push()</a> stores information related to the
  current transformation state

  and style settings controlled by the following functions:

  <a href="/reference/p5/fill">fill()</a>,

  <a href="/reference/p5/noFill">noFill()</a>,

  <a href="/reference/p5/noStroke">noStroke()</a>,

  <a href="/reference/p5/stroke">stroke()</a>,

  <a href="/reference/p5/tint">tint()</a>,

  <a href="/reference/p5/noTint">noTint()</a>,

  <a href="/reference/p5/strokeWeight">strokeWeight()</a>,

  <a href="/reference/p5/strokeCap">strokeCap()</a>,

  <a href="/reference/p5/strokeJoin">strokeJoin()</a>,

  <a href="/reference/p5/imageMode">imageMode()</a>,

  <a href="/reference/p5/rectMode">rectMode()</a>,

  <a href="/reference/p5/ellipseMode">ellipseMode()</a>,

  <a href="/reference/p5/colorMode">colorMode()</a>,

  <a href="/reference/p5/textAlign">textAlign()</a>,

  <a href="/reference/p5/textFont">textFont()</a>,

  <a href="/reference/p5/textSize">textSize()</a>,

  <a href="/reference/p5/textLeading">textLeading()</a>,

  <a href="/reference/p5/applyMatrix">applyMatrix()</a>,

  <a href="/reference/p5/resetMatrix">resetMatrix()</a>,

  <a href="/reference/p5/rotate">rotate()</a>,

  <a href="/reference/p5/scale">scale()</a>,

  <a href="/reference/p5/shearX">shearX()</a>,

  <a href="/reference/p5/shearY">shearY()</a>,

  <a href="/reference/p5/translate">translate()</a>,

  <a href="/reference/p5/noiseSeed">noiseSeed()</a>.</p>
