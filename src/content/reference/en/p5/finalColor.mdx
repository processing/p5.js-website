description: >-
  <p>A shader hook block that modifies the final color of each pixel after all
  lighting is applied. This hook can be used inside <a
  href="/reference/p5/buildMaterialShader/"><code>buildMaterialShader()</code></a>
  and similar shader <a
  href="/reference/p5.Shader/modify/"><code>modify()</code></a> calls to adjust
  the color before it appears on the screen. Modifications happen between the
  <code>.begin()</code> and <code>.end()</code> methods of the hook.</p>

  <p><code>finalColor</code> has the following properties:</p>

  <ul><li><code>color</code>: a four-component vector representing the pixel
  color (red, green, blue, alpha).</li></ul><p>Call <code>.set()</code> on the
  hook with a vector with four components (red, green, blue, alpha) to update
  the final color.</p>

  <p>This hook is available in:</p>

  <ul><li><a
  href="/reference/p5/buildColorShader/"><code>buildColorShader()</code></a></li><li><a
  href="/reference/p5/buildMaterialShader/"><code>buildMaterialShader()</code></a></li><li><a
  href="/reference/p5/buildNormalShader/"><code>buildNormalShader()</code></a></li><li><a
  href="/reference/p5/buildStrokeShader/"><code>buildStrokeShader()</code></a></li></ul>

example:
  - |-
    let myShader;
    function setup() {
      createCanvas(200, 200, WEBGL);
      myShader = buildMaterialShader(material);
    }

    function material() {
      finalColor.begin();
      let color = finalColor.color;
      // Add a blue tint to the output color
      color.b += 0.4;
      finalColor.set(color);
      finalColor.end();
    }

    function draw() {
      background(230);
      shader(myShader);
      noStroke();
      fill('green');
      circle(0, 0, 100);
    }
