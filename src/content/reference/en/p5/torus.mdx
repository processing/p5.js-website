description: >
  <p>Draws a torus.</p>

  <p>A torus is a 3D shape with triangular faces that connect to form a ring.

  Toruses with few faces look flattened. Toruses with many faces have smooth

  surfaces.</p>

  <p>The first parameter, <code>radius</code>, is optional. If a
  <code>Number</code> is passed, as in

  <code>torus(30)</code>, it sets the radius of the ring. By default,
  <code>radius</code> is 50.</p>

  <p>The second parameter, <code>tubeRadius</code>, is also optional. If a
  <code>Number</code> is

  passed, as in <code>torus(30, 15)</code>, it sets the radius of the tube. By
  default,

  <code>tubeRadius</code> is 10.</p>

  <p>The third parameter, <code>detailX</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>torus(30, 15, 5)</code>, it sets the number of edges used to draw
  the hole

  of the torus. Using more edges makes the hole look more like a circle. By

  default, <code>detailX</code> is 24.</p>

  <p>The fourth parameter, <code>detailY</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>torus(30, 15, 5, 7)</code>, it sets the number of triangle
  subdivisions to

  use while filling in the torusâ€™ height. By default, <code>detailY</code> is
  16.</p>

  <p>Note: <code>torus()</code> can only be used in WebGL mode.</p>
