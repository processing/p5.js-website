description: >-
  <p>A shader hook block that modifies how color components are combined in the
  fragment shader. This hook can be used inside <a
  href="/reference/p5/buildMaterialShader/"><code>buildMaterialShader()</code></a>
  and similar shader <a
  href="/reference/p5.Shader/modify/"><code>modify()</code></a> calls to control
  the final color output of a material. Modifications happen between the
  <code>.begin()</code> and <code>.end()</code> methods of the hook.</p>

  <p><code>combineColors</code> has the following properties:</p>

  <ul><li><code>baseColor</code>: a three-component vector representing the base
  color (red, green, blue).</li><li><code>diffuse</code>: a single number
  representing the diffuse reflection.</li><li><code>ambientColor</code>: a
  three-component vector representing the ambient
  color.</li><li><code>ambient</code>: a single number representing the ambient
  reflection.</li><li><code>specularColor</code>: a three-component vector
  representing the specular color.</li><li><code>specular</code>: a single
  number representing the specular reflection.</li><li><code>emissive</code>: a
  three-component vector representing the emissive
  color.</li><li><code>opacity</code>: a single number representing the
  opacity.</li></ul><p>Call <code>.set()</code> on the hook with a vector with
  four components (red, green, blue, alpha) for the final color.</p>

  <p>This hook is available in:</p>

  <ul><li><a
  href="/reference/p5/buildMaterialShader/"><code>buildMaterialShader()</code></a></li></ul>

example:
  - |-
    let myShader;
    function setup() {
      createCanvas(200, 200, WEBGL);
      myShader = buildMaterialShader(material);
    }

    function material() {
      combineColors.begin();
      // Custom color combination: add a green tint using vector properties
      combineColors.set([
        combineColors.baseColor * combineColors.diffuse +
          combineColors.ambientColor * combineColors.ambient +
          combineColors.specularColor * combineColors.specular +
          combineColors.emissive +
          [0, 0.2, 0], // Green tint
        combineColors.opacity
      ]);
      combineColors.end();
    }

    function draw() {
      background(255);
      shader(myShader);
      lights();
      noStroke();
      fill('white');
      sphere(50);
    }
