description: >
  <p>Converts a positive <code>Number</code> into a <code>String</code> with an
  extra space in front.</p>

  <p><code>nfs()</code> converts positive numbers such as 123.45 into strings
  formatted

  with an extra space in front, as in ' 123.45'. Doing so can be helpful for

  aligning positive and negative numbers.</p>

  <p>The first parameter, <code>num</code>, is the number to convert to a
  string. For

  example, calling <code>nfs(123.45)</code> returns the string <code>'
  123.45'</code>.</p>

  <p>The second parameter, <code>left</code>, is optional. If a number is
  passed, as in

  <code>nfs(123.45, 4)</code>, it sets the minimum number of digits to include
  to the

  left of the decimal place. If <code>left</code> is larger than the number of
  digits in

  <code>num</code>, then unused digits will be set to 0. For example, calling

  <code>nfs(123.45, 4)</code> returns the string <code>' 0123.45'</code>.</p>

  <p>The third parameter, <code>right</code>, is also optional. If a number is
  passed, as

  in <code>nfs(123.45, 4, 1)</code>, it sets the minimum number of digits to
  include to

  the right of the decimal place. If <code>right</code> is smaller than the
  number of

  decimal places in <code>num</code>, then <code>num</code> will be rounded to
  the given number of

  decimal places.  For example, calling <code>nfs(123.45, 4, 1)</code> returns
  the

  string <code>' 0123.5'</code>. If <code>right</code> is larger than the number
  of decimal places

  in <code>num</code>, then unused decimal places will be set to 0.  For
  example,

  calling <code>nfs(123.45, 4, 3)</code> returns the string <code>'
  0123.450'</code>.</p>
