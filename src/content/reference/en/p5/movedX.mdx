description: >
  <p>A <code>Number</code> system variable that tracks the mouse's horizontal
  movement.</p>

  <p><code>movedX</code> tracks how many pixels the mouse moves left or right
  between

  frames. <code>movedX</code> will have a negative value if the mouse moves left
  between

  frames and a positive value if it moves right. <code>movedX</code> can be
  calculated

  as <code>mouseX - pmouseX</code>.</p>

  <p>Note: <code>movedX</code> continues updating even when

  <a href="/reference/p5/requestPointerLock/">requestPointerLock()</a> is
  active.</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square. The text ">>" appears when the user moves the mouse to the right. The text "<<" appears when the user moves the mouse to the left.'
      );
    }

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display >> when movedX is positive and
      // << when it's negative.
      if (movedX > 0) {
        text('>>', 50, 50);
      } else if (movedX < 0) {
        text('<<', 50, 50);
      }
    }
    </code>
