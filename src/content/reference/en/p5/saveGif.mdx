description: >
  <p>Generates a gif from a sketch and saves it to a file.</p>

  <p><code>saveGif()</code> may be called in <a
  href="/reference/p5/setup">setup()</a> or at any

  point while a sketch is running.</p>

  <p>The first parameter, <code>fileName</code>, sets the gif's file name.</p>

  <p>The second parameter, <code>duration</code>, sets the gif's duration in
  seconds.</p>

  <p>The third parameter, <code>options</code>, is optional. If an object is
  passed,

  <code>saveGif()</code> will use its properties to customize the gif.
  <code>saveGif()</code>

  recognizes the properties <code>delay</code>, <code>units</code>,
  <code>silent</code>,

  <code>notificationDuration</code>, and <code>notificationID</code>.</p>
