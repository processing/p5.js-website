description: >
  <p>Sets the color of a pixel or draws an image to the canvas.</p>

  <p><code>set()</code> is easy to use but it's not as fast as

  <a href="#/p5/pixels">pixels</a>. Use <a href="#/p5/pixels">pixels</a>

  to set many pixel values.</p>

  <p><code>set()</code> interprets the first two parameters as x- and
  y-coordinates. It

  interprets the last parameter as a grayscale value, a <code>[R, G, B,
  A]</code> pixel

  array, a <a href="#/p5.Color">p5.Color</a> object, or a

  <a href="#/p5.Image">p5.Image</a> object. If an image is passed, the first

  two parameters set the coordinates for the image's upper-left corner,

  regardless of the current <a href="#/p5/imageMode">imageMode()</a>.</p>

  <p><a href="#/p5/updatePixels">updatePixels()</a> must be called after using

  <code>set()</code> for changes to appear.</p>
