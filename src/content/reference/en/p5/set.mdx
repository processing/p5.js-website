description: >
  <p>Sets the color of a pixel or draws an image to the canvas.</p>

  <p><code>set()</code> is easy to use but it's not as fast as

  <a href="/reference/p5/pixels/">pixels</a>. Use <a
  href="/reference/p5/pixels/">pixels</a>

  to set many pixel values.</p>

  <p><code>set()</code> interprets the first two parameters as x- and
  y-coordinates. It

  interprets the last parameter as a grayscale value, a <code>[R, G, B,
  A]</code> pixel

  array, a <a href="/reference/p5/p5.Color">p5.Color</a> object, or a

  <a href="/reference/p5/p5.Image">p5.Image</a> object. If an image is passed,
  the first

  two parameters set the coordinates for the image's upper-left corner,

  regardless of the current <a
  href="/reference/p5/imageMode/">imageMode()</a>.</p>

  <p><a href="/reference/p5/updatePixels/">updatePixels()</a> must be called
  after using

  <code>set()</code> for changes to appear.</p>

    function setup() {
      createCanvas(100, 100);

      background(255);

      // Draw a horizontal color gradient.
      for (let x = 0; x < 100; x += 1) {
        for (let y = 0; y < 100; y += 1) {
          // Calculate the grayscale value.
          let c = map(x, 0, 100, 0, 255);

          // Set the pixel using the grayscale value.
          set(x, y, c);
        }
      }

      // Update the canvas.
      updatePixels();

      describe('A horiztonal color gradient from black to white.');
    }
    </code>
