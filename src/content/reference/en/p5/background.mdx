description: >
  <p>Sets the color used for the background of the canvas.</p>

  <p>By default, the background is transparent. <code>background()</code> is
  typically used

  within <a href="#/p5/draw">draw()</a> to clear the display window at the

  beginning of each frame. It can also be used inside

  <a href="#/p5/setup">setup()</a> to set the background on the first frame

  of animation.</p>

  <p>The version of <code>background()</code> with one parameter interprets the
  value one

  of four ways. If the parameter is a <code>Number</code>, it's interpreted as a
  grayscale

  value. If the parameter is a <code>String</code>, it's interpreted as a CSS
  color string.

  RGB, RGBA, HSL, HSLA, hex, and named color strings are supported. If the

  parameter is a <a href="#/p5.Color">p5.Color</a> object, it will be used as

  the background color. If the parameter is a

  <a href="#/p5.Image">p5.Image</a> object, it will be used as the background

  image.</p>

  <p>The version of <code>background()</code> with two parameters interprets the
  first one

  as a grayscale value. The second parameter sets the alpha (transparency)

  value.</p>

  <p>The version of <code>background()</code> with three parameters interprets
  them as RGB,

  HSB, or HSL colors, depending on the current

  <a href="#/p5/colorMode">colorMode()</a>. By default, colors are specified

  in RGB values. Calling <code>background(255, 204, 0)</code> sets the
  background a bright

  yellow color.</p>

      describe('A canvas with a royal blue background.');
    }
    </code>
