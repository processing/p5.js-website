description: >
  <p>Clears the pixels on the canvas.</p>

  <p><code>clear()</code> makes every pixel 100% transparent. Calling
  <code>clear()</code> doesn't

  clear objects created by <code>createX()</code> functions such as

  <a href="/reference/p5/createGraphics/">createGraphics()</a>,

  <a href="/reference/p5/createVideo/">createVideo()</a>, and

  <a href="/reference/p5/createImg/">createImg()</a>. These objects will remain

  unchanged after calling <code>clear()</code> and can be redrawn.</p>

  <p>In WebGL mode, this function can clear the screen to a specific color. It

  interprets four numeric parameters as normalized RGBA color values. It also

  clears the depth buffer. If you are not using the WebGL renderer, these

  parameters will have no effect.</p>
