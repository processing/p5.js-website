  <p>Sets the pixel density or returns the current density.</p>

  <p>Computer displays are grids of little lights called <em>pixels</em>. A

  display's <em>pixel density</em> describes how many pixels it packs into an

  area. Displays with smaller pixels have a higher pixel density and create

  sharper images.</p>

  <p><code>pixelDensity()</code> sets the pixel scaling for high pixel density
  displays.

  By default, the pixel density is set to match the display's density.

  Calling <code>pixelDensity(1)</code> turn this off.</p>

  <p>Calling <code>pixelDensity()</code> without an argument returns the current
  pixel

  density.</p>
