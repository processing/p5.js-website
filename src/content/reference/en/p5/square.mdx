description: >
  <p>Draws a square.</p>

  <p>A square is a four-sided shape defined by the <code>x</code>,
  <code>y</code>, and <code>s</code>

  parameters. <code>x</code> and <code>y</code> set the location of its top-left
  corner. <code>s</code> sets

  its width and height. Every angle in the square measures 90Ëš and all its

  sides are the same length. See <a href="/reference/p5/rectMode">rectMode()</a>
  for

  other ways to define squares.</p>

  <p>The version of <code>square()</code> with four parameters creates a rounded
  square.

  The fourth parameter sets the radius for all four corners.</p>

  <p>The version of <code>square()</code> with seven parameters also creates a
  rounded

  square. Each of the last four parameters set the radius of a corner. The

  radii start with the top-left corner and move clockwise around the

  square. If any of these parameters are omitted, they are set to the

  value of the last radius that was set.</p>
