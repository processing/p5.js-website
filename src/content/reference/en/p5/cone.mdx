description: >
  <p>Draws a cone.</p>

  <p>A cone is a 3D shape with triangular faces that connect a flat bottom to a

  single point. Cones with few faces look like pyramids. Cones with many

  faces have smooth surfaces.</p>

  <p>The first parameter, <code>radius</code>, is optional. If a
  <code>Number</code> is passed, as in

  <code>cone(20)</code>, it sets the radius of the cone’s base. By default,
  <code>radius</code> is

  50.</p>

  <p>The second parameter, <code>height</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>cone(20, 30)</code>, it sets the cone’s height. By default,
  <code>height</code> is

  set to the cone’s <code>radius</code>.</p>

  <p>The third parameter, <code>detailX</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>cone(20, 30, 5)</code>, it sets the number of edges used to form
  the

  cone's base. Using more edges makes the base look more like a circle. By

  default, <code>detailX</code> is 24.</p>

  <p>The fourth parameter, <code>detailY</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>cone(20, 30, 5, 7)</code>, it sets the number of triangle
  subdivisions to

  use along the y-axis connecting the base to the tip. All 3D shapes are made

  by connecting triangles to form their surfaces. By default,
  <code>detailY</code> is 1.</p>

  <p>The fifth parameter, <code>cap</code>, is also optional. If a
  <code>false</code> is passed, as

  in <code>cone(20, 30, 5, 7, false)</code> the cone’s base won’t be drawn. By
  default,

  <code>cap</code> is <code>true</code>.</p>

  <p>Note: <code>cone()</code> can only be used in WebGL mode.</p>
