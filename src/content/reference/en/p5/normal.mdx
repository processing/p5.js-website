description: >
  <p>Sets the normal vector for vertices in a custom 3D shape.</p>

  <p>3D shapes created with <a href="/reference/p5/beginShape/">beginShape()</a>
  and

  <a href="/reference/p5/endShape/">endShape()</a> are made by connecting sets
  of

  points called vertices. Each vertex added with

  <a href="/reference/p5/vertex/">vertex()</a> has a normal vector that points
  away

  from it. The normal vector controls how light reflects off the shape.</p>

  <p><code>normal()</code> can be called two ways with different parameters to
  define the

  normal vector's components.</p>

  <p>The first way to call <code>normal()</code> has three parameters,
  <code>x</code>, <code>y</code>, and <code>z</code>.

  If <code>Number</code>s are passed, as in <code>normal(1, 2, 3)</code>, they
  set the x-, y-, and

  z-components of the normal vector.</p>

  <p>The second way to call <code>normal()</code> has one parameter,
  <code>vector</code>. If a

  <a href="/reference/p5/p5.Vector">p5.Vector</a> object is passed, as in

  <code>normal(myVector)</code>, its components will be used to set the normal
  vector.</p>

  <p><code>normal()</code> changes the normal vector of vertices added to a
  custom shape

  with <a href="/reference/p5/vertex/">vertex()</a>. <code>normal()</code> must
  be called between

  the <a href="/reference/p5/beginShape/">beginShape()</a> and

  <a href="/reference/p5/endShape/">endShape()</a> functions, just like

  <a href="/reference/p5/vertex/">vertex()</a>. The normal vector set by calling

  <code>normal()</code> will affect all following vertices until
  <code>normal()</code> is called

  again:</p>

  <pre><code class="language-js">beginShape();


  // Set the vertex normal.

  normal(-0.4, -0.4, 0.8);


  // Add a vertex.

  vertex(-30, -30, 0);


  // Set the vertex normal.

  normal(0, 0, 1);


  // Add vertices.

  vertex(30, -30, 0);

  vertex(30, 30, 0);


  // Set the vertex normal.

  normal(0.4, -0.4, 0.8);


  // Add a vertex.

  vertex(-30, 30, 0);


  endShape();

  </code></pre>
