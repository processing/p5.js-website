description: >
  <p>Use the shape drawn by a callback function to mask subsequent things drawn
  to the canvas.

  Only opaque regions of the mask shape will allow content to be drawn.</p>

  <p>The mask will apply to anything drawn after this call. To draw without a
  mask, contain

  the code to apply the mask and to draw the masked content between

  <a href="#/p5/push">push()</a> and <a href="#/p5/pop">pop()</a>.</p>

  <p>Alternatively, rather than drawing the mask shape in a function, draw the

  shape between <a href="#/p5/beginClip">beginClip()</a> and <a
  href="#/p5/endClip">endClip()</a>.</p>

  <p>Options can include:</p>

  <ul>

  <li><code>invert</code>: A boolean specifying whether or not to mask the areas
  <em>not</em> filled by the mask shape. Defaults to false.</li>

  </ul>

example:
  - |-

    <div>
    <code>
    noStroke();

    // Mask in some shapes
    push();
    clip(() => {
      triangle(15, 37, 30, 13, 43, 37);
      circle(45, 45, 7);
    });

    fill('red');
    rect(5, 5, 45, 45);
    pop();

    translate(50, 50);

    // Mask out the same shapes
    push();
    clip(() => {
      triangle(15, 37, 30, 13, 43, 37);
      circle(45, 45, 7);
    }, { invert: true });

    fill('red');
    rect(5, 5, 45, 45);
    pop();
    </code>
    </div>

    function setup() {
      createCanvas(100, 100, WEBGL);
    }

    function draw() {
      background(255);
      noStroke();

      clip(() => {
        push();
        rotateX(frameCount * 0.01);
        rotateY(frameCount * 0.01);
        scale(0.5);
        torus(30, 15);
        pop();
      });

      beginShape(QUAD_STRIP);
      fill(0, 255, 255);
      vertex(-width/2, -height/2);
      vertex(width/2, -height/2);
      fill(100, 0, 100);
      vertex(-width/2, height/2);
      vertex(width/2, height/2);
      endShape();
    }
    </code>
    </div>
