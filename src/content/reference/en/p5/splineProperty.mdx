  <p>Gets or sets a given spline property.</p>

  <p>Use <code>splineProperty()</code> to adjust the behavior of splines

  created with <code>splineVertex()</code> or <code>spline()</code>. You can
  control

  two key aspects of a spline: its end behavior (<code>ends</code>) and

  its curvature (<code>tightness</code>).</p>

  <p>By default, the ends property is set to <code>INCLUDE</code>, which means

  the spline passes through every point, including the endpoints.

  You can also set it to <code>EXCLUDE</code> i.e. <code>splineProperty('ends',
  EXCLUDE)</code>,

  which makes the spline pass through all points except the endpoints.</p>

  <p><code>INCLUDE</code> case will have the spline passing through

  all points, like this:</p>

  <pre><code class="language-js">splineProperty('ends', INCLUDE); // no need to
  set this, as it is the default

  spline(25, 46, 93, 44, 93, 81, 35, 85);


  point(25, 46);

  point(93, 44);

  point(93, 81);

  point(35, 85);</code></pre><p><img src="/assets/includeSpline.png"/></p>

  <p>EXCLUDE case will have the spline passing through

  the middle points, like this:</p>

  <pre><code class="language-js">splineProperty('ends', EXCLUDE);

  spline(25, 46, 93, 44, 93, 81, 35, 85);


  point(25, 46);

  point(93, 44);

  point(93, 81);

  point(35, 85);</code></pre><p><img src="/assets/excludeSpline.png"/></p>

  <p>By default, the tightness property is set to <code>0</code>,

  producing a smooth curve that passes evenly through

  the vertices. Negative values make the curve looser,

  while positive values make it tighter. Common values

  range between -1 and 1, though values outside this

  range can also be used for different effects.</p>

  <p>For example, To set tightness, use <code>splineProperty('tightness',
  t)</code>,

  (default: t = 0).</p>

  <p>Here's the example showing negetive value of tightness,

  which creates a rounder bulge:</p>

  <pre><code class="language-js">splineProperty('tightness', -5)

  stroke(0);

  strokeWeight(2);

  spline(25, 46, 93, 44, 93, 81, 35, 85);</code></pre><p><img
  src="/assets/roundBulge.png"/>

  Here's the example showing positive value of tightness,

  which makes the curve tighter and more angular:</p>

  <pre><code class="language-js">splineProperty('tightness', 5)

  stroke(0);

  strokeWeight(2);

  spline(25, 46, 93, 44, 93, 81, 35, 85);</code></pre><p><img
  src="/assets/anglurBulge.png"/></p>

  <p>In all cases, the splines in p5.js are <a
  href="https://en.wikipedia.org/wiki/Cubic_Hermite_spline#Cardinal_spline">cardinal
  splines</a>.

  When tightness is 0, these splines are often known as

  <a
  href="https://en.wikipedia.org/wiki/Cubic_Hermite_spline#Catmull%E2%80%93Rom_spline/">Catmull-Rom
  splines</a></p>

      vertexA = createVector(35, 85);
