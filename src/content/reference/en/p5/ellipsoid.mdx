description: >
  <p>Draws an ellipsoid.</p>

  <p>An ellipsoid is a 3D shape with triangular faces that connect to form a

  round surface. Ellipsoids with few faces look like crystals. Ellipsoids

  with many faces have smooth surfaces and look like eggs.
  <code>ellipsoid()</code>

  defines a shape by its radii. This is different from

  <a href="/reference/p5/ellipse/">ellipse()</a> which uses diameters

  (width and height).</p>

  <p>The first parameter, <code>radiusX</code>, is optional. If a
  <code>Number</code> is passed, as in

  <code>ellipsoid(20)</code>, it sets the radius of the ellipsoid along the
  x-axis. By

  default, <code>radiusX</code> is 50.</p>

  <p>The second parameter, <code>radiusY</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>ellipsoid(20, 30)</code>, it sets the ellipsoid’s radius along the
  y-axis.

  By default, <code>radiusY</code> is set to the ellipsoid’s
  <code>radiusX</code>.</p>

  <p>The third parameter, <code>radiusZ</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>ellipsoid(20, 30, 40)</code>, it sets the ellipsoid’s radius along
  the

  z-axis. By default, <code>radiusZ</code> is set to the ellipsoid’s
  <code>radiusY</code>.</p>

  <p>The fourth parameter, <code>detailX</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>ellipsoid(20, 30, 40, 5)</code>, it sets the number of triangle

  subdivisions to use along the x-axis. All 3D shapes are made by connecting

  triangles to form their surfaces. By default, <code>detailX</code> is 24.</p>

  <p>The fifth parameter, <code>detailY</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>ellipsoid(20, 30, 40, 5, 7)</code>, it sets the number of triangle

  subdivisions to use along the y-axis. All 3D shapes are made by connecting

  triangles to form their surfaces. By default, <code>detailY</code> is 16.</p>

  <p>Note: <code>ellipsoid()</code> can only be used in WebGL mode.</p>
