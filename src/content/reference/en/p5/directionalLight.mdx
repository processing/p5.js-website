description: >
  <p>Creates a light that shines in one direction.</p>

  <p>Directional lights don’t shine from a specific point. They’re like a sun

  that shines from somewhere offscreen. The light’s direction is set using

  three <code>(x, y, z)</code> values between -1 and 1. For example, setting a
  light’s

  direction as <code>(1, 0, 0)</code> will light <a
  href="/reference/p5/p5.Geometry">p5.Geometry</a>

  objects from the left since the light faces directly to the right.</p>

  <p>There are four ways to call <code>directionalLight()</code> with parameters
  to set the

  light’s color and direction.</p>

  <p>The first way to call <code>directionalLight()</code> has six parameters.
  The first

  three parameters, <code>v1</code>, <code>v2</code>, and <code>v3</code>, set
  the light’s color using the

  current <a href="/reference/p5/colorMode/">colorMode()</a>. The last three

  parameters, <code>x</code>, <code>y</code>, and <code>z</code>, set the
  light’s direction. For example,

  <code>directionalLight(255, 0, 0, 1, 0, 0)</code> creates a red <code>(255, 0,
  0)</code> light

  that shines to the right <code>(1, 0, 0)</code>.</p>

  <p>The second way to call <code>directionalLight()</code> has four parameters.
  The first

  three parameters, <code>v1</code>, <code>v2</code>, and <code>v3</code>, set
  the light’s color using the

  current <a href="/reference/p5/colorMode/">colorMode()</a>. The last
  parameter,

  <code>direction</code> sets the light’s direction using a

  <a href="/reference/p5/p5.Geometry">p5.Geometry</a> object. For example,

  <code>directionalLight(255, 0, 0, lightDir)</code> creates a red <code>(255,
  0, 0)</code> light

  that shines in the direction the <code>lightDir</code> vector points.</p>

  <p>The third way to call <code>directionalLight()</code> has four parameters.
  The first

  parameter, <code>color</code>, sets the light’s color using a

  <a href="/reference/p5/p5.Color">p5.Color</a> object or an array of color
  values. The

  last three parameters, <code>x</code>, <code>y</code>, and <code>z</code>, set
  the light’s direction. For

  example, <code>directionalLight(myColor, 1, 0, 0)</code> creates a light that
  shines

  to the right <code>(1, 0, 0)</code> with the color value of
  <code>myColor</code>.</p>

  <p>The fourth way to call <code>directionalLight()</code> has two parameters.
  The first

  parameter, <code>color</code>, sets the light’s color using a

  <a href="/reference/p5/p5.Color">p5.Color</a> object or an array of color
  values. The

  second parameter, <code>direction</code>, sets the light’s direction using a

  <a href="/reference/p5/p5.Color">p5.Color</a> object. For example,

  <code>directionalLight(myColor, lightDir)</code> creates a light that shines
  in the

  direction the <code>lightDir</code> vector points with the color value of
  <code>myColor</code>.</p>

    // Click and drag the mouse to view the scene from different angles.
    // Double-click to turn on the directional light.

    let isLit = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A sphere drawn on a gray background. A red light starts shining from above when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Control the light.
      if (isLit === true) {
        // Add a red directional light from above.
        // Use RGB values and XYZ directions.
        directionalLight(255, 0, 0, 0, 1, 0);
      }

      // Style the sphere.
      noStroke();

      // Draw the sphere.
      sphere(30);
    }
