description: >
  <p>Converts a <code>Boolean</code>, <code>String</code>, or decimal
  <code>Number</code> to an integer.</p>

  <p><code>int()</code> converts values to integers. Integers are positive or
  negative

  numbers without decimals. If the original value has decimals, as in -34.56,

  they're removed to produce an integer such as -34.</p>

  <p>The parameter, <code>n</code>, is the value to convert. If <code>n</code>
  is a Boolean, as in

  <code>int(false)</code> or <code>int(true)</code>, then the number 0
  (<code>false</code>) or 1 (<code>true</code>) will

  be returned. If <code>n</code> is a string or number, as in
  <code>int('45')</code> or

  <code>int(67.89)</code>, then an integer will be returned. If an array is
  passed, as

  in <code>int([12.34, 56.78])</code>, then an array of integers will be
  returned.</p>

  <p>Note: If a value can't be converted to a number, as in
  <code>int('giraffe')</code>,

  then the value <code>NaN</code> (not a number) will be returned.</p>
