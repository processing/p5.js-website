  <p>Transpiles a p5.strands callback into executable JavaScript by applying

  a multi-pass AST transformation pipeline.</p>

  <ul><li>Identifies functions passed into uniform() so they are excluded from
  transformation</li></ul></li><li><p>transformSetCallsInControlFlow</p>

  <ul><li>Rewrites <code>.set()</code> calls inside control flow into
  intermediate variable assignments</li></ul></li><li><p>Non-control-flow
  transformations</p>

  <ul><li>Applies ASTCallbacks to transform expressions, assignments,
  etc.</li><li>Skips IfStatement and ForStatement (handled
  later)</li></ul></li><li><p>transformHelperFunctionEarlyReturns</p>

  <ul><li>Converts early returns in helper functions into a single return value
  pattern</li></ul></li><li><p>Control flow transformation (post-order)</p>

  <ul><li>Transforms IfStatement → __p5.strandsIf</li><li>Transforms
  ForStatement → __p5.strandsFor</li><li>Handles variable propagation across
  branches/loops</li></ul></li></ul><p>This staged approach ensures correct
  ordering and avoids transformation conflicts.</p>
