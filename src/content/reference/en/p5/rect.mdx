description: >
  <p>Draws a rectangle.</p>

  <p>A rectangle is a four-sided shape defined by the <code>x</code>,
  <code>y</code>, <code>w</code>, and <code>h</code>

  parameters. <code>x</code> and <code>y</code> set the location of its top-left
  corner. <code>w</code> sets

  its width and <code>h</code> sets its height. Every angle in the rectangle
  measures

  90Ëš. See <a href="/reference/p5/rectMode/">rectMode()</a> for other ways to
  define

  rectangles.</p>

  <p>The version of <code>rect()</code> with five parameters creates a rounded
  rectangle. The

  fifth parameter sets the radius for all four corners.</p>

  <p>The version of <code>rect()</code> with eight parameters also creates a
  rounded

  rectangle. Each of the last four parameters set the radius of a corner. The

  radii start with the top-left corner and move clockwise around the

  rectangle. If any of these parameters are omitted, they are set to the

  value of the last radius that was set.</p>
