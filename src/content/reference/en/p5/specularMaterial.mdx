description: >
  <p>Sets the specular color of shapes’ surface material.</p>

  <p>The <code>specularMaterial()</code> color sets the components of light
  color that

  glossy coats on shapes will reflect. For example, calling

  <code>specularMaterial(255, 255, 0)</code> would cause a shape to reflect red
  and

  green light, but not blue light.</p>

  <p>Unlike <a href="/reference/p5/ambientMaterial/">ambientMaterial()</a>,

  <code>specularMaterial()</code> will reflect the full color of light sources
  including

  <a href="/reference/p5/directionalLight/">directionalLight()</a>,

  <a href="/reference/p5/pointLight/">pointLight()</a>,

  and <a href="/reference/p5/spotLight/">spotLight()</a>. This is what gives it
  shapes

  their "shiny" appearance. The material’s shininess can be controlled by the

  <a href="/reference/p5/shininess/">shininess()</a> function.</p>

  <p><code>specularMaterial()</code> can be called three ways with different
  parameters to

  set the material’s color.</p>

  <p>The first way to call <code>specularMaterial()</code> has one parameter,
  <code>gray</code>.

  Grayscale values between 0 and 255, as in <code>specularMaterial(50)</code>,
  can be

  passed to set the material’s color. Higher grayscale values make shapes

  appear brighter.</p>

  <p>The second way to call <code>specularMaterial()</code> has one parameter,
  <code>color</code>. A

  <a href="/reference/p5/p5.Color">p5.Color> object, an array of color values,
  or a CSS

  color string, as in <code>specularMaterial('magenta')</code>, can be passed to
  set the

  material’s color.</a></p>

  <p>The third way to call <code>specularMaterial()</code> has four parameters,
  <code>v1</code>, <code>v2</code>,

  <code>v3</code>, and <code>alpha</code>. <code>alpha</code> is optional. RGBA,
  HSBA, or HSLA values can be

  passed to set the material’s colors, as in <code>specularMaterial(255, 0,
  0)</code> or

  <code>specularMaterial(255, 0, 0, 30)</code>. Color values will be interpreted
  using

  the current <a href="/reference/p5/colorMode/">colorMode()</a>.</p>

    <code>
    // Click and drag the mouse to view the scene from different angles.
    // Double-click the canvas to apply a specular material.

    let isGlossy = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A red torus drawn on a gray background. It becomes glossy when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Turn on a white point light at the top-right.
      pointLight(255, 255, 255, 30, -40, 30);

      // Add a glossy coat if the user has double-clicked.
      if (isGlossy === true) {
        specularMaterial(255);
        shininess(50);
      }

      // Style the torus.
      noStroke();
      fill(255, 0, 0);

      // Draw the torus.
      torus(30);
    }

    // Make the torus glossy when the user double-clicks.
    function doubleClicked() {
      isGlossy = true;
    }
    </code>
