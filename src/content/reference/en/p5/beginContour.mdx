description: >
  <p>Begins creating a hole within a flat shape.</p>

  <p>The <code>beginContour()</code> and <a
  href="/reference/p5/endContour/">endContour()</a>

  functions allow for creating negative space within custom shapes that are

  flat. <code>beginContour()</code> begins adding vertices to a negative space
  and

  <a href="/reference/p5/endContour/">endContour()</a> stops adding them.

  <code>beginContour()</code> and <a
  href="/reference/p5/endContour/">endContour()</a> must be

  called between <a href="/reference/p5/beginShape/">beginShape()</a> and

  <a href="/reference/p5/endShape/">endShape()</a>.</p>

  <p>Transformations such as <a href="/reference/p5/translate/">translate()</a>,

  <a href="/reference/p5/rotate/">rotate()</a>, and <a
  href="/reference/p5/scale/">scale()</a>

  don't work between <code>beginContour()</code> and

  <a href="/reference/p5/endContour/">endContour()</a>. It's also not possible
  to use

  other shapes, such as <a href="/reference/p5/ellipse/">ellipse()</a> or

  <a href="/reference/p5/rect/">rect()</a>, between <code>beginContour()</code>
  and

  <a href="/reference/p5/endContour/">endContour()</a>.</p>

  <p>Note: The vertices that define a negative space must "wind" in the opposite

  direction from the outer shape. First, draw vertices for the outer shape

  clockwise order. Then, draw vertices for the negative space in

  counter-clockwise order.</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Start drawing the shape.
      beginShape();

      // Exterior vertices, clockwise winding.
      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // Interior vertices, counter-clockwise winding.
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // Stop drawing the shape.
      endShape(CLOSE);

      describe('A white square with a square hole in its center drawn on a gray background.');
    }
    </code>
    </div>
