  <p>Create a new compute shader using p5.strands.</p>

  <p>A compute shader lets you run many calculations all at once on your GPU.
  They

  are similar to a <a href="/reference/p5/for><code>for</code> loop,</a> but
  each iteration of the

  loop happens in parallel on the GPU rather than running one after the other.

  This makes them ideal for calculations or simulations involving many
  items.</p>

  <p>You create a compute shader by passing a function to
  <code>buildComputeShader</code>.

  The function represents one iteration of a loop.</p>

  <p>The compute shader can be run by calling <a href=/" #="" p5=""
  compute"=""><code>compute()</code></a>

  and passing the shader in, along with the number of iterations in up to three

  dimensions. Use the <a href="/reference/p5/index/"><code>index</code></a>
  vector inside of your

  iteration function to refer to the current iteration of the loop. The
  <code>x</code>, <code>y</code>,

  and <code>z</code> properties will count up from zero to the count in each
  dimension passed

  into <code>compute</code>.</p>

  <p>A compute shader will read from and write to storage, which is often an
  array of

  numbers or objects. Use <a
  href="/reference/p5/createStorage/"><code>createStorage</code></a> to
  construct

  initial data. Connect your iteration function to the storage by passing the
  storage

  into <a
  href="/reference/p5/uniformStorage/"><code>uniformStorage</code></a>.</p>

  <p>Often, compute shaders are paired with <a
  href="/reference/p5/model/"><code>model(myGeometry, count)</code></a>

  to draw one instance per object in the storage, and a shader that uses

  <a href="/reference/p5/instanceID/"><code>instanceID()</code></a> to position
  each instance.</p>

  function makeParticles(x, y) {
    let data = [];
    for (let i = 0; i < numParticles; i++) {
      let angle = (i / numParticles) * TWO_PI;
      let speed = random(0.5, 2);
      data.push({
        position: createVector(x, y),
        velocity: createVector(cos(angle) * speed, sin(angle) * speed),
      });
    }
    return data;
  }


  function drawParticle() {
    sphere(2);
  }


  function simulate() {
    let data = uniformStorage(particles);
    let idx = index.x;
    data[idx].position = data[idx].position + data[idx].velocity;
  }


  function display() {
    let data = uniformStorage(particles);
    worldInputs.begin();
    let pos = data[instanceID()].position;
    worldInputs.position.xy += pos - [width / 2, height / 2];
    worldInputs.end();
  }


  function draw() {
    background(30);
    if (frameCount % 60 === 0) {
      particles.update(makeParticles(random(width), random(height)));
    }
    compute(computeShader, numParticles);
    noStroke();
    fill(255, 200, 50);
    shader(displayShader);
    model(instance, numParticles);
