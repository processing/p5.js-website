  <p>A <code>Number</code> variable that tracks the mouse's previous horizontal
  position

  within the browser.</p>

  <p><code>pwinMouseX</code> keeps track of the mouse's position relative to the
  top-left

  corner of the browser window. Its value is

  <a href="/reference/p5/winMouseX/">winMouseX</a> from the previous frame. For

  example, if the mouse was 50 pixels from

  the left edge of the browser during the last frame, then
  <code>pwinMouseX</code> will

  be 50.</p>

  <p>On a touchscreen device, <code>pwinMouseX</code> will hold the x-coordinate
  of the most

  recent touch point. <code>pwinMouseX</code> is reset to the current

  <a href="/reference/p5/winMouseX/">winMouseX</a> value at the start of each
  touch

  event.</p>

  <p>Note: Use <a href="/reference/p5/pmouseX/">pmouseX</a> to track the mouseâ€™s
  previous

  x-coordinate within the canvas.</p>
