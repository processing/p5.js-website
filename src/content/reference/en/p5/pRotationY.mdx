description: >
  <p>The system variable pRotationY always contains the rotation of the

  device along the y axis in the frame previous to the current frame.

  If the sketch <a href="/reference/p5/angleMode/"> angleMode()</a> is set to
  DEGREES,

  the value will be -90 to 90. If it is set to RADIANS, the value will

  be -PI/2 to PI/2.</p>

  <p>pRotationY can also be used with rotationY to determine the rotate

  direction of the device along the Y-axis.</p>

    // A simple if statement looking at whether
    // rotationY - pRotationY < 0 is true or not will be
    // sufficient for determining the rotate direction
    // in most cases.

    // Some extra logic is needed to account for cases where
    // the angles wrap around.
    let rotateDirection = 'clockwise';

    // Simple range conversion to make things simpler.
    // This is not absolutely necessary but the logic
    // will be different in that case.

    let rY = rotationY + 180;
    let pRY = pRotationY + 180;

    if ((rY - pRY > 0 && rY - pRY < 270) || rY - pRY < -270) {
      rotateDirection = 'clockwise';
    } else if (rY - pRY < 0 || rY - pRY > 270) {
      rotateDirection = 'counter-clockwise';
    }
