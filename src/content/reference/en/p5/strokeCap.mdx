description: >
  <p>Sets the style for rendering the ends of lines.</p>

  <p>The caps for line endings are either rounded (<code>ROUND</code>), squared

  (<code>SQUARE</code>), or extended (<code>PROJECT</code>). The default cap is
  <code>ROUND</code>.</p>

  <p>The argument passed to <code>strokeCap()</code> must be written in ALL CAPS
  because

  the constants <code>ROUND</code>, <code>SQUARE</code>, and
  <code>PROJECT</code> are defined this way.

  JavaScript is a case-sensitive language.</p>
line: 388
isConstructor: false
itemtype: method
example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      strokeWeight(12);

      // Top.
      strokeCap(ROUND);
      line(20, 30, 80, 30);

      // Middle.
      strokeCap(SQUARE);
      line(20, 50, 80, 50);

      // Bottom.
      strokeCap(PROJECT);
      line(20, 70, 80, 70);

      describe(
        'Three horizontal lines. The top line has rounded ends, the middle line has squared ends, and the bottom line has longer, squared ends.'
      );
    }
    </code>
    </div>
class: p5

  - name: cap

      <p>either ROUND, SQUARE, or PROJECT</p>
