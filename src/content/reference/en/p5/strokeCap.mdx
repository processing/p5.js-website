description: >
  <p>Sets the style for rendering line endings. These ends are either rounded

  (<code>ROUND</code>), squared (<code>SQUARE</code>), or extended
  (<code>PROJECT</code>). The default cap is

  <code>ROUND</code>.</p>

  <p>The argument passed to <code>strokeCap()</code> must be written in ALL CAPS
  because

  the constants <code>ROUND</code>, <code>SQUARE</code>, and
  <code>PROJECT</code> are defined this way.

  JavaScript is a case-sensitive language.</p>

  - >-

    <div>

    <code>

    strokeWeight(12.0);

    strokeCap(ROUND);

    line(20, 30, 80, 30);

    strokeCap(SQUARE);

    line(20, 50, 80, 50);

    strokeCap(PROJECT);

    line(20, 70, 80, 70);

    describe('Three horizontal lines. The top line has rounded ends, the middle
    line has squared ends, and the bottom line has longer, squared ends.');

    </code>

    </div>
