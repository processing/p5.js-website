  <p>A function that's called when the browser window is resized.</p>

  <p>Code placed in the body of <code>windowResized()</code> will run when the

  browser window's size changes. It's a good place to call

  <a href="/reference/p5/resizeCanvas">resizeCanvas()</a> or make other

  adjustments to accommodate the new window size.</p>

  <p>The <code>event</code> parameter is optional. If added to the function
  declaration, it

  can be used for debugging or other purposes.</p>
