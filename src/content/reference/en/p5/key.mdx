  evaluates to

  <code>true</code> if the <code>a</code> key was typed and <code>false</code>
  if not. <code>key</code> doesnâ€™t update

  keyCode instead for

  special keys. The <a href="/reference/p5/keyIsDown/">keyIsDown()</a> function
  should

  be used to check for multiple different key presses at the same time.</p>
