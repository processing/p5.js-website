description: >
  <p>Creates an <code><a></a></code> element that links to another web page.</p>

  <p>The first parmeter, <code>href</code>, is a string that sets the URL of the
  linked

  page.</p>

  <p>The second parameter, <code>html</code>, is a string that sets the inner
  HTML of the

  link. It's common to use text, images, or buttons as links.</p>

  <p>The third parameter, <code>target</code>, is optional. It's a string that
  tells the

  web browser where to open the link. By default, links open in the current

  browser tab. Passing <code>'_blank'</code> will cause the link to open in a
  new

  browser tab. MDN describes a few

  <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a#target"
  target="_blank">other options</a>.</p>
