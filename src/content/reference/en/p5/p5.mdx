description: >
  <p>Creates a new sketch in "instance" mode.</p>

  <p>All p5.js sketches are instances of the <code>p5</code> class. Put another
  way, all

  p5.js sketches are objects with methods including <code>pInst.setup()</code>,

  <code>pInst.draw()</code>, <code>pInst.circle()</code>, and
  <code>pInst.fill()</code>. By default, sketches

  run in "global mode" to hide some of this complexity.</p>

  <p>In global mode, a default instance of the <code>p5</code> class is created

  automatically. The default <code>p5</code> instance searches the web page's
  source

  code for declarations of system functions such as <code>setup()</code>,
  <code>draw()</code>,

  and <code>mousePressed()</code>, then attaches those functions to itself as
  methods.

  Calling a function such as <code>circle()</code> in global mode actually calls
  the

  default <code>p5</code> object's <code>pInst.circle()</code> method.</p>

  <p>It's often helpful to isolate the code within sketches from the rest of the

  code on a web page. Two common use cases are web pages that use other

  JavaScript libraries and web pages with multiple sketches. "Instance mode"

  makes it easy to support both of these scenarios.</p>

  <p>Instance mode sketches support the same API as global mode sketches. They

  use a function to bundle, or encapsulate, an entire sketch. The function

  containing the sketch is then passed to the <code>p5()</code> constructor.</p>

  <p>The first parameter, <code>sketch</code>, is a function that contains the
  sketch. For

  example, the statement <code>new p5(mySketch)</code> would create a new
  instance mode

  sketch from a function named <code>mySketch</code>. The function should have
  one

  parameter, <code>p</code>, that's a <code>p5</code> object.</p>

  <p>The second parameter, <code>node</code>, is optional. If a string is
  passed, as in

  <code>new p5(mySketch, 'sketch-one')</code> the new instance mode sketch will
  become a

  child of the HTML element with the id <code>sketch-one</code>. If an HTML
  element is

  passed, as in <code>new p5(mySketch, myElement)</code>, then the new instance
  mode

  sketch will become a child of the <code>Element</code> object called
  <code>myElement</code>.</p>
