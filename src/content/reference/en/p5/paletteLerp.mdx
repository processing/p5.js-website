  <p>Blends multiple colors to find a color between them.</p>

  <p>The <code>amt</code> parameter specifies the amount to interpolate between
  the color

  stops which are colors at each <code>amt</code> value "location" with
  <code>amt</code> values

  that are between 2 color stops interpolating between them based on its
  relative

  distance to both.</p>

  <p>The way that colors are interpolated depends on the current

  <a href="/reference/p5/colorMode/">colorMode()</a>.</p>

        ['red', 0.05],
