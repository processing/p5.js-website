description: >
  <p>Sets the emissive color of shapes’ surface material.</p>

  <p>The <code>emissiveMaterial()</code> color sets a color shapes display at
  full

  strength, regardless of lighting. This can give the appearance that a shape

  is glowing. However, emissive materials don’t actually emit light that

  can affect surrounding objects.</p>

  <p><code>emissiveMaterial()</code> can be called three ways with different
  parameters to

  set the material’s color.</p>

  <p>The first way to call <code>emissiveMaterial()</code> has one parameter,
  <code>gray</code>.

  Grayscale values between 0 and 255, as in <code>emissiveMaterial(50)</code>,
  can be

  passed to set the material’s color. Higher grayscale values make shapes

  appear brighter.</p>

  <p>The second way to call <code>emissiveMaterial()</code> has one parameter,
  <code>color</code>. A

  <a href="/reference/p5/p5.Color">p5.Color</a> object, an array of color
  values, or a

  CSS color string, as in <code>emissiveMaterial('magenta')</code>, can be
  passed to set

  the material’s color.</p>

  <p>The third way to call <code>emissiveMaterial()</code> has four parameters,
  <code>v1</code>, <code>v2</code>,

  <code>v3</code>, and <code>alpha</code>. <code>alpha</code> is optional. RGBA,
  HSBA, or HSLA values can be

  passed to set the material’s colors, as in <code>emissiveMaterial(255, 0,
  0)</code> or

  <code>emissiveMaterial(255, 0, 0, 30)</code>. Color values will be interpreted
  using

  the current <a href="/reference/p5/colorMode/">colorMode()</a>.</p>

  <p>Note: <code>emissiveMaterial()</code> can only be used in WebGL mode.</p>
