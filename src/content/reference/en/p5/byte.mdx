description: >
  <p>Converts a <code>Boolean</code>, <code>String</code>, or
  <code>Number</code> to its byte value.</p>

  <p><code>byte()</code> converts a value to an integer (whole number) between
  -128 and

  127. Values greater than 127 wrap around while negative values are

  unchanged. For example, 128 becomes -128 and -129 remains the same.</p>

  <p>The parameter, <code>n</code>, is the value to convert. If <code>n</code>
  is a Boolean, as in

  <code>byte(false)</code> or <code>byte(true)</code>, the number 0
  (<code>false</code>) or 1 (<code>true</code>) will be

  returned. If <code>n</code> is a string or number, as in
  <code>byte('256')</code> or <code>byte(256)</code>,

  then the byte value will be returned. Decimal values are ignored. If an

  array is passed, as in <code>byte([true, 123, '456'])</code>, then an array of
  byte

  values will be returned.</p>

  <p>Note: If a value can't be converted to a number, as in
  <code>byte('giraffe')</code>,

  then the value <code>NaN</code> (not a number) will be returned.</p>
