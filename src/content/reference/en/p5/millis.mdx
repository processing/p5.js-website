  <p>Returns the number of milliseconds since a sketch started running.</p>

  <p><code>millis()</code> keeps track of how long a sketch has been running in

  milliseconds (thousandths of a second). This information is often

  helpful for timing events and animations.</p>

  <p>If a sketch has a

  <a href="/reference/p5/setup/">setup()</a> function, then
  <code>millis()</code> begins tracking

  time before the code in <a href="/reference/p5/setup/">setup()</a> runs.  If a

  sketch includes asynchronous loading using
  <code>async</code>/<code>await</code>, then

  <code>millis()</code> begins tracking time as soon as the asynchronous code

  starts running.</p>
