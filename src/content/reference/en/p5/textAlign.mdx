description: >
  <p>Sets the way text is aligned when <a href="/reference/p5/text/">text()</a>
  is called.</p>

  <p>By default, calling <code>text('hi', 10, 20)</code> places the bottom-left
  corner of

  the text's bounding box at (10, 20).</p>

  <p>The first parameter, <code>horizAlign</code>, changes the way

  <a href="/reference/p5/text/">text()</a> interprets x-coordinates. By default,
  the

  x-coordinate sets the left edge of the bounding box. <code>textAlign()</code>
  accepts

  the following values for <code>horizAlign</code>: <code>LEFT</code>,
  <code>CENTER</code>, or <code>RIGHT</code>.</p>

  <p>The second parameter, <code>vertAlign</code>, is optional. It changes the
  way

  <a href="/reference/p5/text/">text()</a> interprets y-coordinates. By default,
  the

  y-coordinate sets the bottom edge of the bounding box.
  <code>textAlign()</code>

  accepts the following values for <code>vertAlign</code>: <code>TOP</code>,
  <code>BOTTOM</code>, <code>CENTER</code>,

  or <code>BASELINE</code>.</p>
