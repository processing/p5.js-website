description: >
  <p>Converts a <code>Number</code> into a <code>String</code> with a given
  number of digits.</p>

  <p><code>nf()</code> converts numbers such as <code>123.45</code> into strings
  formatted with a set

  number of digits, as in <code>'123.4500'</code>.</p>

  <p>The first parameter, <code>num</code>, is the number to convert to a
  string. For

  example, calling <code>nf(123.45)</code> returns the string
  <code>'123.45'</code>. If an array of

  numbers is passed, as in <code>nf([123.45, 67.89])</code>, an array of
  formatted

  strings will be returned.</p>

  <p>The second parameter, <code>left</code>, is optional. If a number is
  passed, as in

  <code>nf(123.45, 4)</code>, it sets the minimum number of digits to include to
  the

  left of the decimal place. If <code>left</code> is larger than the number of
  digits in

  <code>num</code>, then unused digits will be set to 0. For example, calling

  <code>nf(123.45, 4)</code> returns the string <code>'0123.45'</code>.</p>

  <p>The third parameter, <code>right</code>, is also optional. If a number is
  passed, as

  in <code>nf(123.45, 4, 1)</code>, it sets the minimum number of digits to
  include to

  the right of the decimal place. If <code>right</code> is smaller than the
  number of

  decimal places in <code>num</code>, then <code>num</code> will be rounded to
  the given number of

  decimal places. For example, calling <code>nf(123.45, 4, 1)</code> returns the
  string

  <code>'0123.5'</code>. If right is larger than the number of decimal places in
  <code>num</code>,

  then unused decimal places will be set to 0. For example, calling

  <code>nf(123.45, 4, 3)</code> returns the string <code>'0123.450'</code>.</p>

  <p>When the number is negative, for example, calling <code>nf(-123.45, 5,
  2)</code>

  returns the string <code>'-00123.45'</code>.</p>
