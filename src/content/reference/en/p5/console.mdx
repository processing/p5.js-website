  common to add a <code>console.log()</code> statement while studying how a
  section of

  code works:</p>

  <pre><code class="language-js">if (isPlaying === true) {

    console.log('Got here!');

    // Game logic.
  }

  </code></pre>

  <p><code>console.error()</code> is helpful for tracking errors because it
  prints

  formatted messages. For example, it's common to encounter errors when

  loading media assets:</p>

  <pre><code class="language-js">// Logs an error message with special
  formatting.

  function handleFailure(error) {
    console.error('Oops!', error);
  }

  </code></pre>
