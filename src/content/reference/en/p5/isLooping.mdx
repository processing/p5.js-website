  <p>Returns <code>true</code> if the draw loop is running and
  <code>false</code> if not.</p>

  <p>By default, <a href="/reference/p5/draw">draw()</a> tries to run 60 times
  per

  second. Calling <a href="/reference/p5/noLoop">noLoop()</a> stops

  <a href="/reference/p5/draw">draw()</a> from repeating. The draw loop can be

  restarted by calling <a href="/reference/p5/loop">loop()</a>.</p>

  <p>The <code>isLooping()</code> function can be used to check whether a sketch
  is

  looping, as in <code>isLooping() === true</code>.</p>
