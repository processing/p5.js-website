description: >
  <p>Shears the x-axis so that shapes appear skewed.</p>

  <p>By default, the x- and y-axes are perpendicular. The <code>shearX()</code>
  function

  transforms the coordinate system so that x-coordinates are translated while

  y-coordinates are fixed.</p>

  <p>The first parameter, <code>angle</code>, is the amount to shear. For
  example, calling

  <code>shearX(1)</code> transforms all x-coordinates using the formula

  <code>x = x + y * tan(angle)</code>. <code>shearX()</code> interprets angle
  values using the

  current <a href="/reference/p5/angleMode/">angleMode()</a>.</p>

  <p>By default, transformations accumulate. For example, calling

  <code>shearX(1)</code> twice has the same effect as calling
  <code>shearX(2)</code> once. The

  <a href="/reference/p5/push/">push()</a> and

  <a href="/reference/p5/pop/">pop()</a> functions can be used to isolate

  transformations within distinct drawing groups.</p>

  <p>Note: Transformations are reset at the beginning of the draw loop. Calling

  <code>shearX(1)</code> inside the <a href="/reference/p5/draw/">draw()</a>
  function won't

  cause shapes to shear continuously.</p>
