description: >
  <p>Converts a <code>Number</code> into a <code>String</code> with commas to
  mark units of 1,000.</p>

  <p><code>nfc()</code> converts numbers such as 12345 into strings formatted
  with commas

  to mark the thousands place, as in <code>'12,345'</code>.</p>

  <p>The first parameter, <code>num</code>, is the number to convert to a
  string. For

  example, calling <code>nfc(12345)</code> returns the string
  <code>'12,345'</code>.</p>

  <p>The second parameter, <code>right</code>, is optional. If a number is
  passed, as in

  <code>nfc(12345, 1)</code>, it sets the minimum number of digits to include to
  the

  right of the decimal place. If <code>right</code> is smaller than the number
  of

  decimal places in <code>num</code>, then <code>num</code> will be rounded to
  the given number of

  decimal places. For example, calling <code>nfc(12345.67, 1)</code> returns the
  string

  <code>'12,345.7'</code>. If <code>right</code> is larger than the number of
  decimal places in

  <code>num</code>, then unused decimal places will be set to 0. For example,
  calling

  <code>nfc(12345.67, 3)</code> returns the string
  <code>'12,345.670'</code>.</p>

        let y = (i + 1) * 33;

        // Display the original and formatted numbers.
        text(`${numbers[i]} : ${formatted[i]}`, 50, y);
