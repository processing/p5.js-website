description: >
  <p>Draws a box (rectangular prism).</p>

  <p>A box is a 3D shape with six faces. Each face makes a 90˚ with four

  neighboring faces.</p>

  <p>The first parameter, <code>width</code>, is optional. If a
  <code>Number</code> is passed, as in

  <code>box(20)</code>, it sets the box’s width and height. By default,
  <code>width</code> is 50.</p>

  <p>The second parameter, <code>height</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>box(20, 30)</code>, it sets the box’s height. By default,
  <code>height</code> is set

  to the box’s <code>width</code>.</p>

  <p>The third parameter, <code>depth</code>, is also optional. If a
  <code>Number</code> is passed, as

  in <code>box(20, 30, 40)</code>, it sets the box’s depth. By default,
  <code>depth</code> is set

  to the box’s <code>height</code>.</p>

  <p>The fourth parameter, <code>detailX</code>, is also optional. If a
  <code>Number</code> is passed,

  as in <code>box(20, 30, 40, 5)</code>, it sets the number of triangle
  subdivisions to

  use along the x-axis. All 3D shapes are made by connecting triangles to

  form their surfaces. By default, <code>detailX</code> is 1.</p>

  <p>The fifth parameter, <code>detailY</code>, is also optional. If a number is
  passed, as

  in <code>box(20, 30, 40, 5, 7)</code>, it sets the number of triangle
  subdivisions to

  use along the y-axis. All 3D shapes are made by connecting triangles to

  form their surfaces. By default, <code>detailY</code> is 1.</p>

  <p>Note: <code>box()</code> can only be used in WebGL mode.</p>
