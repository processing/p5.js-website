description: >
  <p>एक चाप खींचता है।</p>

  <p>एक चाप <code>x</code>, <code>y</code>, <code>w</code>, और <code>h</code> द्वारा परिभाषित दीर्घवृत्त का एक खंड है पैरामीटर. <code>x</code> और <code>y</code> चाप के केंद्र का स्थान निर्धारित करते हैं। <code>w</code> और <code>h</code> चाप की चौड़ाई और ऊंचाई निर्धारित करते हैं। अधिक जानकारी के लिए <a href="/reference/p5/ellipse">ellipse()</a> और <a href="/reference/p5/ellipseMode">ellipseMode()</a> देखें।</p>

  <p>पांचवें और छठे पैरामीटर, <code>START</code> और <code>STOP</code>, उन कोणों को सेट करते हैं जिनके बीच चाप खींचना है। चाप सदैव <code>START</code> से <code>START</code> तक दक्षिणावर्त खींचे जाते हैं। कोण हमेशा रेडियन में दिए जाते हैं।</p>

  <p>सातवाँ पैरामीटर, <code>MODE</code>, वैकल्पिक है। यह चाप की भरण शैली निर्धारित करता है। भरण मोड एक अर्धवृत्त (<code>OPEN</code), एक बंद अर्धवृत्त (<code>CHORD</code>), या एक बंद पाई खंड (<codePIE</code>) हैं। .</p>

  <p>आठवां पैरामीटर, <code>detail</code>, भी वैकल्पिक है। यह निर्धारित करता है कि WebGL मोड में चाप खींचने के लिए कितने शीर्षों का उपयोग किया जाता है। डिफ़ॉल्ट मान 25 है.</p>

      background(0);

      // Style the arc.
      noStroke();
      fill(255, 255, 0);

      // Update start and stop angles.
      let biteSize = PI / 16;
      let startAngle = biteSize * sin(frameCount * 0.1) + biteSize;
      let endAngle = TWO_PI - startAngle;

      // Draw the arc.
      arc(50, 50, 80, 80, startAngle, endAngle, PIE);
