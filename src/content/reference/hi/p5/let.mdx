  <p>एक नया वेरिएबल घोषित करता है।</p>

  <p> एक वेरिएबल एक मान के लिए एक कंटेनर है। उदाहरण के लिए, एक चर हो सकता है किसी प्राणी के x-निर्देशांक को <code>संख्या</code> के रूप में या उसके नाम को <code>स्ट्रिंग</code> के रूप में शामिल करें। वेरिएबल्स को निम्नानुसार पुन: असाइन करके मान बदल सकते हैं:</p>

<code>
// वेरिएबल x घोषित करें और इसे मान 10 निर्दिष्ट करें।
 मान लीजिए x = 10;
 <p>// x को 50 पर पुन: असाइन करें।
 एक्स = 50;
   </p></code>
   <p>वेरिएबल्स में ब्लॉक स्कोप होता है। जब घुंघराले ब्रेसिज़ <code>{}</code> के बीच एक वेरिएबल घोषित किया जाता है, तो यह केवल उन ब्रेसिज़ द्वारा परिभाषित ब्लॉक के भीतर मौजूद होता है। उदाहरण के लिए, निम्नलिखित code एक <code>ReferenceError</code> फेंक देगा क्योंकि <code>x</code> को <code>setup()</code> फ़ंक्शन के ब्लॉक के भीतर घोषित किया गया है:</p>

 <code>  function setup() {
    createCanvas(100, 100);
  <p>  let x = 50;
  }</p>

  <p>function draw() {
    background(200);</p>
  <p>  // x was declared in setup(), so it can't be referenced here.
    circle(x, 50, 20);
  }

  </p></code>

<p>सभी घुंघराले ब्रेसिज़ के बाहर घोषित वेरिएबल <code>{}</code> वैश्विक दायरे में हैं। एक वैरिएबल जो वैश्विक दायरे में है, उसे स्केच में कहीं भी इस्तेमाल और बदला जा सकता है:</p>

<code>
  let x = 50;

    createCanvas(100, 100);

    background(200);</p>
  <p>  // Change the value of x.
    x += 10;</p>

  }

  </p></code>
