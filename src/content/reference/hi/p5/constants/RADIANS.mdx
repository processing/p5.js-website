  <p>एक <code>स्ट्रिंग</code> स्थिरांक जिसका उपयोग <a href="/reference/p5/angelMode">angleMode()</a> सेट करने के लिए किया जाता है।</p>

  <p>डिफ़ॉल्ट रूप से, <a href="/reference/p5/rotate">rotate()</a> और <a href="/reference/p5/sin">sin()</a> जैसे फ़ंक्शन की इकाइयों में मापे गए कोणों की अपेक्षा करें
  रेडियंस. <code>angleMode(RADIANS)</code> को कॉल करने से यह सुनिश्चित होता है कि कोणों को रेडियन की इकाइयों में मापा जाता है। ऐसा करना उपयोगी हो सकता है यदि <a href='/reference/p5/angelMode'>angleMode()</a> को <a href='/reference/p5/DEGREES'>DEGREES</a> पर सेट किया गया है। </p> <p>नोट: <code>TWO_PI</code> रेडियंस 360˚ के बराबर है।</p>

      fill(255, 0, 0);
      arc(50, 50, 80, 80, 0, 90);

      // Use radians.
      angleMode(RADIANS);

      // Draw a blue arc from HALF_PI to PI.
      fill(0, 0, 255);
