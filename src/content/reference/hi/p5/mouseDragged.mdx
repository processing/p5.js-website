  जब उपयोगकर्ता क्लिक करता है और माउस खींचता है तो स्वचालित रूप से:</p>

  <code>

  फ़ंक्शन माउसड्रैग्ड() {
    // चलाने के लिए कोड।
  }

  </code>


  <p>माउस सिस्टम वैरिएबल, जैसे <a
  href='/reference/p5/mouseX'>mouseX</a> और

  <a href="/reference/p5/mouseY">mouseY</a>, को उनके अधिकांश के साथ अपडेट किया जाएगा
  हाल ही का

  मान जब <code>mouseDragged()</code> को p5.js द्वारा कॉल किया जाता है:</p>

  <code>

  फ़ंक्शन माउसड्रैग्ड() {
    यदि (माउसएक्स <50) {
      // यदि माउस बाईं ओर है तो चलाने के लिए कोड।
    }

  <p> यदि (माउसY > 50) {
      // यदि माउस नीचे के पास है तो चलाने के लिए कोड।
    }
  }

  </p></code>

  <p>पैरामीटर, <code>इवेंट</code>, वैकल्पिक है। <code>माउसड्रैग्ड()</code>
  हमेशा पारित किया जाता है a

  <a href='https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent'
  target='_blank'>MouseEvent</a>

  गुणों वाली वस्तु जो माउस ड्रैग इवेंट का वर्णन करती है:</p>

  <code>

  फ़ंक्शन माउसड्रैग्ड(इवेंट) {
    // चलाने के लिए कोड जो ईवेंट का उपयोग करता है।
    कंसोल.लॉग(इवेंट);
  }

  </code>


  <p>टचस्क्रीन डिवाइस पर, <code>mouseDragged()</code> उपयोगकर्ता द्वारा चलाया जाएगा
  एक स्पर्श ले जाता है

  यदि <a href="/reference/p5/touchMomed">touchMomed()</a> घोषित नहीं किया गया है तो इंगित करें।
  अगर

  <a href='/reference/p5/touchMomed'>touchMomed()</a> घोषित किया जाता है, तो

  <a href="/reference/p5/touchMomed">touchMomed()</a> उपयोगकर्ता के हिलने पर चलेगा
  ए

  स्पर्श बिंदु और <code>mouseDragged()</code> नहीं होगा।</p>

  <p>ब्राउज़र में विभिन्न माउस ईवेंट से जुड़े डिफ़ॉल्ट व्यवहार हो सकते हैं। के लिए

  उदाहरण के लिए, जब उपयोगकर्ता माउस घुमाता है तो कुछ ब्राउज़र टेक्स्ट को हाइलाइट करते हैं

  माउस बटन दबाना. इस ईवेंट के लिए किसी भी डिफ़ॉल्ट व्यवहार को रोकने के लिए,

  फ़ंक्शन के अंत में <code> return false;</code> जोड़ें।</p>
