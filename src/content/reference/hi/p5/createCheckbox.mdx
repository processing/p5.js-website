  <p>एक चेकबॉक्स <code><input></input></code> तत्व बनाता है।</p>

  <p>चेकबॉक्स <a href="/reference/p5/p5.Element">p5.Element</a> क्लास को <code>checked()</code> विधि से विस्तारित करते हैं। यदि बॉक्स चेक किया गया है तो <code>myBox.checked()</code> को कॉल करने पर <code>true</code> मिलता है और यदि नहीं तो <code>false</code> मिलता है।</p>

  <p>पहला पैरामीटर, <code>label</code>, वैकल्पिक है। यह एक स्ट्रिंग है जो लेबल को चेकबॉक्स के बगल में प्रदर्शित करने के लिए सेट करती है।</p>

  <p>दूसरा पैरामीटर, <code>value</code>, भी वैकल्पिक है। यह एक बूलियन है जो चेकबॉक्स का मान सेट करता है।</p>

    function setup() {

      // Use the checkbox to set the background color.
