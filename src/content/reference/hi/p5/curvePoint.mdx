description: >
  <p>प्रक्षेप का उपयोग करके तख़्ता वक्र के साथ निर्देशांक की गणना करता है।</p>

  <p><code>curvePoint()</code> एंकर और नियंत्रण बिंदुओं का उपयोग करके एक तख़्ता वक्र के साथ निर्देशांक की गणना करता है।

  यह <a href="/reference/p5/curve">curve()</a> फ़ंक्शन के समान क्रम में बिंदुओं की अपेक्षा करता है। 

  <code>curvePoint()</code> एक समय में एक अक्ष पर काम करता है। एंकर और नियंत्रण बिंदुओं के x-निर्देशांक को पार करने से वक्र पर एक बिंदु के x-निर्देशांक की गणना की जाएगी। एंकर और नियंत्रण बिंदुओं के y-निर्देशांक को पार करने से वक्र पर एक बिंदु के y-निर्देशांक की गणना की जाएगी।</p>

  <p>पहला पैरामीटर, <code>a</code>, पहले नियंत्रण बिंदु का निर्देशांक है।</p>

  <p>दूसरा और तीसरा पैरामीटर, <code>b</code> और <code>c</code>, एंकर बिंदुओं के निर्देशांक हैं।</p>

  <p>चौथा पैरामीटर, <code>d</code>, अंतिम नियंत्रण बिंदु का निर्देशांक है।</p>

  <p>पांचवां पैरामीटर, <code>t</code>, वक्र के साथ प्रक्षेपित होने वाली मात्रा है। 0 पहला एंकर पॉइंट है, 1 दूसरा एंकर पॉइंट है, और 0.5 उनके बीच का आधा रास्ता है।</p>

      createCanvas(100, 100);

      let x1 = 5;

      fill(255);

      // Top.
      let x = curvePoint(x1, x2, x3, x4, 0);
      let y = curvePoint(y1, y2, y3, y4, 0);
      circle(x, y, 5);

      // Center.
      x = curvePoint(x1, x2, x3, x4, 0.5);
      y = curvePoint(y1, y2, y3, y4, 0.5);
      circle(x, y, 5);

      // Bottom.
      x = curvePoint(x1, x2, x3, x4, 1);
      y = curvePoint(y1, y2, y3, y4, 1);
      circle(x, y, 5);

    }

      // Draw the curve.
      noFill();
      curve(x1, y1, x2, y2, x3, y3, x4, y4);

      // Calculate the circle's coordinates.
      let t = 0.5 * sin(frameCount * 0.01) + 0.5;
      let x = curvePoint(x1, x2, x3, x4, t);
      let y = curvePoint(y1, y2, y3, y4, t);

      // Draw the circle.
      fill(255);
      circle(x, y, 5);
    }
