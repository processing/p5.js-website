description: >
  <p>कस्टम 3D आकार में शीर्षों के लिए नार्मल वेक्टर सेट करता है।</p>

  <p> <a href="/reference/p5/beginShape">beginShape()</a> और <a href="/reference/p5/endShape">endShape()</a> के साथ 3D आकृतियाँ बनाई जाती हैं बिंदुओं के सेट को जोड़कर, जिन्हें शीर्ष कहा जाता है।

  <a href="/reference/p5/vertex">vertex()</a> के साथ जोड़े गए प्रत्येक शीर्ष पर एक नार्मल वेक्टर होता है जो इससे दूर इंगित करता है।

  नार्मल वेक्टर यह नियंत्रित करता है कि प्रकाश आकृति से कैसे परावर्तित होता है।</p>

  <p><code>normal()</code> को नार्मल वेक्टर के घटकों को परिभाषित करने के लिए विभिन्न मापदंडों के साथ दो तरीकों से बुलाया जा सकता है।</p>

  <p><code>nirmal()</code> को कॉल करने के पहले तरीके में तीन पैरामीटर हैं, <code>x</code>, <code>y</code>, और <code>z</code>।

  यदि <code>Number</code> पास हो जाते हैं, जैसे <code>normal(1, 2, 3)</code> में, वे नार्मल वेक्टर के x-, y- और z-घटक सेट करते हैं।< /पी>

  <p><code>normal()</code> को कॉल करने का दूसरा तरीका एक पैरामीटर है, <code>vector</code>। यदि एक <a href="/reference/p5/p5.Vector">p5.Vector</a> ऑब्जेक्ट पास किया जाता है, जैसे कि <code>normal(myVector)</code>, तो इसके घटकों का उपयोग सेट करने के लिए किया जाएगा नार्मल वेक्टर.</p>

  <p><code>normal()</code> <a href="/reference/p5/vertex">vertex()</a> के साथ कस्टम आकार में जोड़े गए शीर्षों के नार्मल वेक्टर को बदल देता है। <code>normal()</code> को <a href="/reference/p5/beginShape">beginShape()</a> और <a href="/reference/p5/endShape">endShape()</a> फ़ंक्शन के बीच बुलाया जाना चाहिए, बिल्कुल <a href="/reference/p5/vertex">vertex()</a> की तरह।

  <code>normal()</code> को कॉल करके सेट किया गया नार्मल वेक्टर निम्नलिखित सभी शीर्षों को तब तक प्रभावित करेगा जब तक कि <code>normal()</code> को फिर से कॉल न किया जाए:</p>

  <code>

  beginShape();


  <p>// Set the vertex normal.

  normal(-0.4, -0.4, 0.8);</p>

  <p>// Add a vertex.

  vertex(-30, -30, 0);</p>

  <p>// Set the vertex normal.

  normal(0, 0, 1);</p>

  <p>// Add vertices.

  vertex(30, -30, 0);

  vertex(30, 30, 0);</p>

  <p>// Set the vertex normal.

  normal(0.4, -0.4, 0.8);</p>

  <p>// Add a vertex.

  vertex(-30, 30, 0);</p>

  <p>endShape();

  </p></code>
