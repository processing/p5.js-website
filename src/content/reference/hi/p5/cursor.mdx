  <p>कर्सर की दिखावट को बदलता है।</p>

  <p>पहला पैरामीटर, <code>type</code>, कर्सर के प्रकार को प्रदर्शित करता है। अंदरूनी विकल्प हैं <code>ARROW</code>, <code>CROSS</code>, <code>HAND</code>, <code>MOVE</code>, <code>TEXT</code>, और <code>WAIT</code>। <code>cursor()</code> भी मान्यतानुसार स्टैंडर्ड CSS कर्सर गुणों को स्ट्रिंग के रूप में पारित करता है: <code>'help'</code>, <code>'wait'</code>, <code>'crosshair'</code>, <code>'not-allowed'</code>, <code>'zoom-in'</code>, और <code>'grab'</code>। यदि एक छवि के पथ को पास किया जाता है, जैसे कि <code>cursor('/assets/target.png')</code>, तो छवि को कर्सर के रूप में उपयोग किया जाएगा। छवियों का प्रारूप .cur, .gif, .jpg, .jpeg, या .png होना चाहिए और यह <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/cursor#icon_size_limits">अधिकतम 32x32 पिक्सेल के बड़े</a> होने चाहिए।</p>

  <p>पैरामीटर <code>x</code> और <code>y</code> वैकल्पिक हैं। यदि कर्सर के लिए एक छवि का उपयोग किया जाता है, तो <code>x</code> और <code>y</code> छवि के भीतर इंगित की गई स्थान को सेट करते हैं। ये डिफ़ॉल्ट रूप से दोनों 0 होते हैं, ताकि कर्सर छवि के शीर्ष-बाएं कोने को इंगित करें। <code>x</code> और <code>y</code> को छवि की चौड़ाई और ऊँचाई से कम होना चाहिए।</p>

    function setup() {
      createCanvas(100, 100);

      describe('A gray square divided into quadrants. The cursor image changes when the mouse moves to each quadrant.');
    }

    function draw() {
      background(200);

      // Divide the canvas into quadrants.
      line(50, 0, 50, 100);
      line(0, 50, 100, 50);

      // Change cursor based on mouse position.
      if (mouseX < 50 && mouseY < 50) {
        cursor(CROSS);
      } else if (mouseX > 50 && mouseY < 50) {
        cursor('progress');
      } else if (mouseX > 50 && mouseY > 50) {
        cursor('https://avatars0.githubusercontent.com/u/1617169?s=16');
      } else {
        cursor('grab');
      }
    }
    </code>
