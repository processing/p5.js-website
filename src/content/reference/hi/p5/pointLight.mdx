description: >
  <p>सभी दिशाओं में एक बिंदु से चमकने वाला एक प्रकार का प्रकाश बनाता है।</p>

  <p>पॉइंट लाइट्स उन्हें बल्ब के रूप में समझा जा सकता है जो सभी दिशाओं में चमकते हैं। विभिन्न स्थानों पर रखे जा सकते हैं ताकि विभिन्न प्रकाश प्रभाव प्राप्त किए जा सकें। एक समय में अधिकतम 5 पॉइंट लाइट्स सक्रिय हो सकते हैं।</p>

  <p><code>pointLight()</code> को प्रकाश के रंग और स्थिति सेट करने के लिए पैरामीटर के साथ चार तरीके होते हैं।</p>

  <p><code>pointLight()</code> को बुलाने का पहला तरीका छह पैरामीटर्स होते हैं। पहले तीन पैरामीटर, <code>v1</code>, <code>v2</code>, और <code>v3</code>, मौजूदा <a href="/reference/p5/colorMode">colorMode()</a> का उपयोग करके प्रकाश का रंग सेट करते हैं। अंतिम तीन पैरामीटर, <code>x</code>, <code>y</code>, और <code>z</code>, प्रकाश की स्थिति सेट करते हैं। उदाहरण के लिए, <code>pointLight(255, 0, 0, 50, 0, 0)</code> एक लाल <code>(255, 0, 0)</code> प्रकाश बनाता है जो निर्देशांक <code>(50, 0, 0)</code> से चमकता है।</p>

  <p>कॉल करने के दूसरे तरीके <code>pointLight()</code> में चार पैरामीटर हैं। पहले तीन पैरामीटर, <code>v1</code>, <code>v2</code>, और <code>v3</code>, वर्तमान <a href="/reference/p5/ का उपयोग करके प्रकाश का रंग सेट करते हैं कलरमोड">कलरमोड()</a>. अंतिम पैरामीटर, स्थिति <a href="/reference/p5/p5.Vector">p5.Vector</a> ऑब्जेक्ट का उपयोग करके प्रकाश की स्थिति निर्धारित करती है। उदाहरण के लिए, <code>pointLight(255, 0, 0,lightPos)</code> एक लाल <code>(255, 0, 0)</code> प्रकाश बनाता है जो <code>lightPos</code> वेक्टर द्वारा निर्धारित स्थिति से चमकता है.</p>

  <p><code>pointLight()</code> को बुलाने का तीसरा तरीका चार पैरामीटर्स होते हैं। पहला पैरामीटर, <code>color</code>, <a href="/reference/p5/p5.Color">p5.Color</a> ऑब्जेक्ट या रंग मानों के एक एरे का उपयोग करके प्रकाश का रंग सेट करता है। अंतिम तीन पैरामीटर, <code>x</code>, <code>y</code>, और <code>z</code>, प्रकाश की स्थिति सेट करते हैं। उदाहरण के लिए, <code>directionalLight(myColor, 50, 0, 0)</code> उन निर्देशांकों से चमकता है जो <code>(50, 0, 0)</code> होते हैं और उसके रंग मान का उपयोग करता है <code>myColor</code>।</p>
  
  <p><code>pointLight()</code> को बुलाने का चौथा तरीका दो पैरामीटर्स होते हैं। पहला पैरामीटर, <code>color</code>, <a href="/reference/p5/p5.Color">p5.Color</a> ऑब्जेक्ट या रंग मानों के एक एरे का उपयोग करके प्रकाश का रंग सेट करता है। दूसरा पैरामीटर, <code>position</code>, <a href="/reference/p5/p5.Vector">p5.Vector</a> ऑब्जेक्ट का उपयोग करके प्रकाश की स्थिति सेट करता है। उदाहरण के लिए, <code>directionalLight(myColor, lightPos)</code> उस स्थिति से चमकता है जो <code>lightPos</code> वेक्टर द्वारा सेट की गई होती है और उसके रंग मान का उपयोग करता है <code>myColor</code>।</p>

        pointLight(255, 0, 0, 0, -150, 0);
      }

      // Style the sphere.
      noStroke();

      // Draw the sphere.
      sphere(30);
    }

    function doubleClicked() {
      isLit = true;
    }
