  <p>एक बिंदु, मूल बिंदु और धनात्मक x-अक्ष से बने कोण की गणना करता है।</p>

  <p><code>atan2()</code> का उपयोग अक्सर ज्यामिति को माउस की स्थिति में उन्मुख करने के लिए किया जाता है, जैसे 
  <code>atan2(mouseY, माउसX)</code> में। पहला पैरामीटर बिंदु का y-निर्देशांक है और दूसरा पैरामीटर इसका x-निर्देशांक है।</p>

  <p>डिफ़ॉल्ट रूप से, <code>atan2()</code> -π (लगभग -3.14) से π (3.14) की सीमा में मान लौटाता है। 
  यदि <a href="/reference/p5/angelMode">angleMode()</a> <code>DEGREES</code> है, तो मान -180 से 
  180 की सीमा में लौटाए जाते हैं।</p>

    function setup() {
      createCanvas(100, 100);

      describe('A rectangle at the center of the canvas rotates with mouse movements.');
    }

    function draw() {
      background(200);

      // Translate the origin to the center.
      translate(50, 50);

      // Get the mouse's coordinates relative to the origin.
      let x = mouseX - 50;
      let y = mouseY - 50;

      // Calculate the angle between the mouse and the origin.
      let a = atan2(y, x);

      // Rotate.
      rotate(a);

      // Draw the shape.
      rect(-30, -5, 60, 10);
    }
    </code>
