  <p>एक फ़ंक्शन जिसे माउस बटन दबाने पर एक बार कॉल किया जाता है।</p>

  <p>फ़ंक्शन घोषित करना <code>mousePressed()</code> चलाने के लिए एक कोड ब्लॉक सेट करता है

  जब उपयोगकर्ता माउस बटन दबाता है तो स्वचालित रूप से:</p>

  <code>

  फ़ंक्शन माउसप्रेस्ड() {
    // चलाने के लिए कोड।
  }

  </code>


  <p>माउस सिस्टम वैरिएबल, जैसे <a
  href='/reference/p5/mouseX'>mouseX</a> और

  <a href="/reference/p5/mouseY">mouseY</a>, को उनके अधिकांश के साथ अपडेट किया जाएगा
  हाल ही का

  मान जब <code>mousePressed()</code> को p5.js द्वारा कॉल किया जाता है:</p>

  <code>

  फ़ंक्शन माउसप्रेस्ड() {
    यदि (माउसएक्स <50) {
      // यदि माउस बाईं ओर है तो चलाने के लिए कोड।
    }

  <p> यदि (माउसY > 50) {
      // यदि माउस नीचे के पास है तो चलाने के लिए कोड।
    }
  }

  </p></code>

  <p>पैरामीटर, <code>इवेंट</code>, वैकल्पिक है। <code>माउसप्रेस्ड()</code>
  हमेशा पारित किया जाता है a

  <a href='https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent'
  target='_blank'>MouseEvent</a>

  गुणों वाली वस्तु जो माउस प्रेस घटना का वर्णन करती है:</p>

  <code>

  फ़ंक्शन माउसप्रेस्ड(इवेंट) {
    // चलाने के लिए कोड जो ईवेंट का उपयोग करता है।
    कंसोल.लॉग(इवेंट);
  }

  </code>


  <p>टचस्क्रीन डिवाइस पर, <code>mousePressed()</code> तब चलेगा जब उपयोगकर्ता का
  छूना

  यदि <a href="/reference/p5/touchStarted">touchStarted()</a> नहीं है तो प्रारंभ होता है
  घोषित. अगर

  <a href='/reference/p5/touchStarted'>touchStarted()</a> घोषित किया जाता है, तो

  <a href="/reference/p5/touchStarted">touchStarted()</a> तब चलेगा जब कोई उपयोगकर्ता
  छूना

  शुरू होता है और <code>mousePressed()</code> नहीं होगा।</p>

  <p>ब्राउज़र में विभिन्न माउस ईवेंट से जुड़े डिफ़ॉल्ट व्यवहार हो सकते हैं। के लिए

  उदाहरण के लिए, जब उपयोगकर्ता माउस घुमाता है तो कुछ ब्राउज़र टेक्स्ट को हाइलाइट करते हैं

  माउस बटन दबाना. इस ईवेंट के लिए किसी भी डिफ़ॉल्ट व्यवहार को रोकने के लिए,

  फ़ंक्शन के अंत में <code> return false;</code> जोड़ें।</p>

  <p>नोट: <code>mousePressed()</code>, <a
  href='/reference/p5/mouseReleased'>mouseReleased()</a>,

  और <a href="/reference/p5/mouseClicked">mouseClicked()</a> सभी संबंधित हैं।

  जैसे ही उपयोगकर्ता माउस पर क्लिक करता है, <code>mousePressed()</code> चालू हो जाता है।

  <a href='/reference/p5/mouseReleased'>mouseReleased()</a> जैसे ही चलता है
  उपयोगकर्ता

  माउस क्लिक जारी करता है। <ए
  href='/reference/p5/mouseClicked'>mouseClicked()</a>

  <a के तुरंत बाद चलता है
  href='/reference/p5/mouseReleased'>mouseReleased()</a>.</p>
