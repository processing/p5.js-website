  <p>WebGL मोड में स्पलाइन कर्व्स खींचने के लिए उपयोग किए जाने वाले सेगमेंट की संख्या सेट करता है।</p>

  <p>वेबजीएल मोड में, कई सपाट खंडों का उपयोग करके चिकनी आकृतियाँ बनाई जाती हैं। अधिक सपाट खंड जोड़ने से आकृतियाँ अधिक चिकनी दिखाई देती हैं।</p>

  <p>पैरामीटर, <code>dettail</code>, एक तख़्ता वक्र बनाते समय उपयोग किए जाने वाले खंडों की संख्या है।
  उदाहरण के लिए, <code>curveDetail(5)</code> को कॉल करने पर <a href="/reference/p5/curve">curve()</a> फ़ंक्शन के साथ कर्व बनाने के लिए 5 सेगमेंट का उपयोग किया जाएगा।
  
  डिफ़ॉल्ट रूप से, <code>detail</code> 20 है।</p>

  <p>ध्यान दें: <code>curveDetail()</code> का 2डी मोड में कोई प्रभाव नहीं पड़ता है।</p>

      curveDetail(3);

      stroke(0);
      curve(-45, -24, 0, 23, -26, 0, 23, 11, 0, -35, 15, 0);

      // Draw red spline curves from the anchor points to the control points.
      stroke(255, 0, 0);

      curve(23, -26, 0, 23, 11, 0, -35, 15, 0, -35, 15, 0);

      // Draw the anchor points in black.

      stroke(0);
      point(23, -26);
      point(23, 11);

      // Draw the control points in red.
      stroke(255, 0, 0);
      point(-45, -24);
      point(-35, 15);

      describe(
        'A gray square with a jagged curve drawn in three segments. The curve is a sideways U shape with red segments on top and bottom, and a black segment on the right. The endpoints of all the segments are marked with dots.'
