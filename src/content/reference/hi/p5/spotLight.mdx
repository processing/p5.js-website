description: >
  p>एक बिंदु से एक दिशा में चमकने वाला प्रकाश बनाता है।</p>

  <p>स्पॉट लाइट्स फ्लैशलाइट्स की तरह होते हैं जो एक दिशा में चमकते हैं और प्रकाश का एक शंकु बनाते हैं। शंकु का आकार कोण और एकाग्रता पैरामीटर्स का उपयोग करके नियंत्रित किया जा सकता है। एक साथ अधिकतम 5 स्पॉट लाइट्स सक्रिय हो सकती हैं।</p>

  <p>प्रकाश का रंग, स्थिति और दिशा सेट करने के लिए पैरामीटर्स के साथ <code>spotLight()</code> को कॉल करने के आठ तरीके हैं। उदाहरण के लिए, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0)</code> मूल <code>(0, 0, 0)</code> पर एक लाल <code>(255, 0, 0)</code> प्रकाश बनाता है जो दाईं ओर <code>(1, 0, 0)</code> की ओर इशारा करता है।</p>
  
  <p><code>angle</code> पैरामीटर वैकल्पिक है। यह प्रकाश शंकु की त्रिज्या सेट करता है। उदाहरण के लिए, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0, PI / 16)</code> मूल <code>(0, 0, 0)</code> पर एक लाल <code>(255, 0, 0)</code> प्रकाश बनाता है जो <code>PI / 16</code> रेडियन के कोण के साथ दाईं ओर <code>(1, 0, 0)</code> की ओर इशारा करता है। डिफ़ॉल्ट रूप से, <code>angle</code> <code>PI / 3</code> रेडियन होता है।</p>
  
  <p><code>concentration</code> पैरामीटर भी वैकल्पिक है। यह प्रकाश शंकु के केंद्र की ओर प्रकाश को केंद्रित करता है। उदाहरण के लिए, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0, PI / 16, 50)</code> मूल <code>(0, 0, 0)</code> पर एक लाल <code>(255, 0, 0)</code> प्रकाश बनाता है जो 50 के एकाग्रता पर <code>PI / 16</code> रेडियन के कोण के साथ दाईं ओर <code>(1, 0, 0)</code> की ओर इशारा करता है। डिफ़ॉल्ट रूप से, <code>concentration</code> 100 होता है।</p>
