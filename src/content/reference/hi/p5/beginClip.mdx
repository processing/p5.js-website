  <p>एक आकृति को परिभाषित करना प्रारंभ करता है जो बाद में बनाई गई किसी भी आकृति को छिपा देगा।</p>

  <p> <code>beginClip()</code> और <a href="/reference/p5/endClip">endClip()</a> के बीच खींची गई कोई भी आकृति मास्क के आकार में जुड़ जाएगी। मास्क <a href="/reference/p5/endClip">endClip()</a> के बाद खींची गई किसी भी चीज़ पर लागू होगा।</p>

  <p>पैरामीटर, <code>options</code>, वैकल्पिक है। यदि <code>invert</code> प्रॉपर्टी वाला कोई ऑब्जेक्ट पास किया जाता है, जैसे कि <code>beginClip({ invert: true })</code>, तो इसका उपयोग मास्किंग मोड सेट करने के लिए किया जाएगा। <code>{ invert: true }</code> मुखौटे को उलट देता है, जिससे ढकी हुई आकृतियों में छेद हो जाते हैं। <code>invert</code> डिफ़ॉल्ट रूप से <code>गलत</code> है।</p>

  <p>मास्क को <a href="/reference/p5/push">push()</a> और <a href="/reference/p5/pop">pop()</a> के बीच समाहित किया जा सकता है कार्य। ऐसा करने से नकाबपोश आकृतियों के बाद नकाबपोश आकृतियाँ बनाई जा सकती हैं।</p>

  <p>मास्क को कॉलबैक फ़ंक्शन में भी परिभाषित किया जा सकता है जो <a href="/reference/p5/clip">clip()</a> को पास किया जाता है।</p>

      // Create a mask.

      triangle(15, 37, 30, 13, 43, 37);
      circle(45, 45, 7);
      endClip();

      // Draw a backing shape.
      square(5, 5, 45);
      pop();

      translate(50, 50);

      // Draw an inverted masked shape.
      push();
      // Create an inverted mask.
      beginClip({ invert: true });
      triangle(15, 37, 30, 13, 43, 37);
      circle(45, 45, 7);
      endClip();

      // Draw a backing shape.
      square(5, 5, 45);
      pop();

      describe('In the top left, a white triangle and circle. In the bottom right, a white square with a triangle and circle cut out of it.');
    }
