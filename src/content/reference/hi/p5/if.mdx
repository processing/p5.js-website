description: >
  <p>यह चुनने का एक तरीका कि code का ब्लॉक चलाया जाए या नहीं।</p>

   <p><code>if</code> कथन किसी शर्त के आधार पर code के ब्लॉक को चलाने में सहायक होते हैं। उदाहरण के लिए, एक <code>if</code> कथन इस विचार को व्यक्त करना आसान बनाता है "यदि माउस दबाया जाए तो एक वृत्त बनाएं।":</p>

  <code>
  if (mouseIsPressed === true) {
    circle(mouseX, mouseY, 20);
  }
  </code>

  <p>स्टेटमेंट हेडर कीवर्ड <code>if</code> से शुरू होता है। कोष्ठक में अभिव्यक्ति <code>mouseIsPressed === true</code> एक <code>बूलियन</code> अभिव्यक्ति है जो या तो <code>true</code> या <code>गलत</code> है। घुंघराले ब्रेसिज़ के बीच का code <code>{}</code> if स्टेटमेंट का मुख्य भाग है। बॉडी केवल तभी चलती है जब <code>बूलियन</code> अभिव्यक्ति <code>true</code> हो।</p>

   <p><a href="/reference/p5/mouseIsPressed">mouseIsPressed</a> सिस्टम वैरिएबल हमेशा <code>true</code> या <code>गलत</code> होता है, इसलिए ऊपर दिया गया code स्निपेट हो सकता है इस प्रकार भी लिखा जाए:</p>

   <code>
  if (mouseIsPressed) {
    circle(mouseX, mouseY, 20);
  }
  </code>


   <p>एक <code>if</code>-<code>else</code> स्टेटमेंट code का एक ब्लॉक जोड़ता है जो <code>बूलियन</code> एक्सप्रेशन <code>गलत</code> होने पर चलता है। उदाहरण के लिए, यहां एक <code>if</code>-<code>else</code> स्टेटमेंट है जो इस विचार को व्यक्त करता है कि "यदि माउस दबाया जाए तो एक वृत्त बनाएं। अन्यथा, एक संदेश प्रदर्शित करें।":</p>

  <code>

  if (mouseIsPressed === true) {
    // When true.
    circle(mouseX, mouseY, 20);
  } else {
    // When false.
    text('Click me!', 50, 50);
  }
  </code>

   <p>इस code स्निपेट में दो संभावित पथ या शाखाएँ हैं। कार्यक्रम को किसी न किसी शाखा का अनुसरण करना चाहिए।</p>

   <p>एक <code>else</code>-<code>if</code> स्टेटमेंट अधिक शाखाएं जोड़ना संभव बनाता है। <code>else</code>-<code>if</code> स्टेटमेंट अलग-अलग परिस्थितियों में code के अलग-अलग ब्लॉक चलाते हैं। उदाहरण के लिए, एक <code>else</code>-<code>if</code> स्टेटमेंट बनाता है
   इस विचार को व्यक्त करना आसान है "यदि माउस बाईं ओर है, तो कैनवास को सफेद रंग से रंग दें। यदि माउस बीच में है, तो कैनवास को ग्रे रंग से रंग दें। अन्यथा, कैनवास को काले रंग से रंग दें।":</p>

  <code>

  if (mouseX < 33) {
    background(255);
  } else if (mouseX < 67) {
    background(200);
  } else {
    background(0);
  }

  </code>

   <p><code>if</code> स्टेटमेंट में उतने जोड़े जा सकते हैं
   <code>अन्य</code>-<code>यदि</code> आवश्यकतानुसार कथन। हालाँकि, केवल एक ही <code>अन्य</code> कथन हो सकता है और यह अंतिम होना चाहिए।</p>

   <p><code>if</code> स्टेटमेंट एक साथ कई स्थितियों की जांच भी कर सकते हैं। उदाहरण के लिए, <code>बूलियन</code> ऑपरेटर <code>&&</code> (AND) जाँचता है कि क्या दो अभिव्यक्तियाँ <code>true</code> हैं:</p>

   <code>
   यदि (keyIsPressed === true && key === 'p') {
     पाठ ('आपने "पी" कुंजी दबायी!', 50, 50);
   }
   </code>

   <p>यदि उपयोगकर्ता कोई कुंजी दबा रहा है और वह कुंजी <code>'p'</code> है, तो एक संदेश प्रदर्शित होगा।</p>

   <p><code>बूलियन</code> ऑपरेटर <code>||</code> (OR) जाँचता है कि दो में से कम से कम एक अभिव्यक्ति <code>सत्य</code> है या नहीं:</p>

  <code>
  if (keyIsPressed === true && key === 'p') {
    text('You pressed the "p" key!', 50, 50);
  }
  </code>

   <p>यदि उपयोगकर्ता एक कुंजी दबाता है, या एक माउस बटन दबाता है, या दोनों दबाता है, तो एक संदेश प्रदर्शित होगा।</p>

   <p>एक <code>if</code> स्टेटमेंट के मुख्य भाग में एक और स्टेटमेंट शामिल हो सकता है
   <code>यदि</code> कथन। इसे "नेस्टेड <code>अगर</code> स्टेटमेंट" कहा जाता है। उदाहरण के लिए, नेस्टेड <code>यदि</code> कथन इस विचार को व्यक्त करना आसान बनाते हैं "यदि कोई कुंजी दबाया जाता है, तो जांचें कि कुंजी 'आर' है या नहीं। यदि यह है, तो भरण को लाल पर सेट करें।" </p>

  <code>
  if (keyIsPressed === true || mouseIsPressed === true) {
    text('You did something!', 50, 50);
  }
  </code>

  <p> <a href="/reference/p5/Boolean">बूलियन</a> और <a href='/reference/p5/Number'>नंबर</a>
  देखें इन डेटा प्रकारों और उनके द्वारा समर्थित संचालन के बारे में अधिक जानने के लिए।</p>
