  <p>एक फ़ंक्शन जिसे माउस बटन पर दो बार जल्दी क्लिक करने पर एक बार कॉल किया
  जाता है.</p>

  <p>फ़ंक्शन घोषित करना <code>doubleClicked()</code> एक कोड ब्लॉक सेट करता है
  

  जब उपयोगकर्ता माउस बटन को दो बार दबाता और छोड़ता है तो स्वचालित रूप से

  जल्दी:</p>

  <code>

  फ़ंक्शन डबलक्लिक किया गया() {
    // चलाने के लिए कोड।
  }

  </code>


  <p>माउस सिस्टम वैरिएबल, जैसे <a
  href='/reference/p5/mouseX'>mouseX</a> और

  <a href="/reference/p5/mouseY">mouseY</a>, को उनके अधिकांश के साथ अपडेट किया जाएगा
  हाल ही का

  मान जब <code>doubleClicked()</code> को p5.js द्वारा कॉल किया जाता है:</p>

  <code>

  फ़ंक्शन डबलक्लिक किया गया() {
    यदि (माउसएक्स <50) {
      // यदि माउस बाईं ओर है तो चलाने के लिए कोड।
    }

  <p> यदि (माउसY > 50) {
      // यदि माउस नीचे के पास है तो चलाने के लिए कोड।
    }
  }

  </p></code>

  <p>पैरामीटर, <code>इवेंट</code>, वैकल्पिक है।
  <code>doubleClicked()</code> हमेशा पास किया जाता है

  <a href='https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent'
  target='_blank'>MouseEvent</a>

  गुणों वाली वस्तु जो डबल-क्लिक ईवेंट का वर्णन करती है:</p>

  <code>

  फ़ंक्शन डबलक्लिक किया गया (इवेंट) {
    // चलाने के लिए कोड जो ईवेंट का उपयोग करता है।
    कंसोल.लॉग(इवेंट);
  }

  </code>


  <p>टचस्क्रीन डिवाइस पर, <code>doubleClicked()</code> में रखा गया कोड होगा
  दो के पीछे भागो

  ऐसे स्पर्श जो कम समय में घटित होते हैं।</p>

  <p>ब्राउज़र में विभिन्न माउस ईवेंट से जुड़े डिफ़ॉल्ट व्यवहार हो सकते हैं। के लिए

  उदाहरण के लिए, जब उपयोगकर्ता माउस घुमाता है तो कुछ ब्राउज़र टेक्स्ट को हाइलाइट करते हैं

  माउस बटन दबाना. इस ईवेंट के लिए किसी भी डिफ़ॉल्ट व्यवहार को रोकने के लिए,

  फ़ंक्शन के अंत में <code> return false;</code> जोड़ें।</p>
