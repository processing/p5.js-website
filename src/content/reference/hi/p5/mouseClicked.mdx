  <p>एक फ़ंक्शन जिसे माउस बटन दबाने के बाद एक बार कॉल किया जाता है
  जारी किया गया.</p>

  <p>फ़ंक्शन घोषित करना <code>mouseClicked()</code> चलाने के लिए एक कोड ब्लॉक सेट करता है

  स्वचालित रूप से जब उपयोगकर्ता माउस बटन दबाने के बाद उसे छोड़ देता है

  यह:</p>

  <code>

  फ़ंक्शन माउस क्लिक किया गया() {
    // चलाने के लिए कोड।
  }

  </code>


  <p>माउस सिस्टम वैरिएबल, जैसे <a
  href='/reference/p5/mouseX'>mouseX</a> और

  <a href="/reference/p5/mouseY">mouseY</a>, को उनके अधिकांश के साथ अपडेट किया जाएगा
  हाल ही का

  मान जब <code>mouseClicked()</code> को p5.js द्वारा कॉल किया जाता है:</p>

  <code>

  फ़ंक्शन माउस क्लिक किया गया() {
    यदि (माउसएक्स <50) {
      // यदि माउस बाईं ओर है तो चलाने के लिए कोड।
    }

  <p> यदि (माउसY > 50) {
      // यदि माउस नीचे के पास है तो चलाने के लिए कोड।
    }
  }

  </p></code>

  <p>पैरामीटर, <code>इवेंट</code>, वैकल्पिक है। <code>माउसक्लिक्ड()</code>
  हमेशा पारित किया जाता है a

  <a href='https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent'
  target='_blank'>MouseEvent</a>

  गुणों वाली वस्तु जो माउस क्लिक घटना का वर्णन करती है:</p>

  <code>

  फ़ंक्शन माउस क्लिक किया गया (इवेंट) {
    // चलाने के लिए कोड जो ईवेंट का उपयोग करता है।
    कंसोल.लॉग(इवेंट);
  }

  </code>


  <p>टचस्क्रीन डिवाइस पर, <code>mouseClicked()</code> तब चलेगा जब कोई उपयोगकर्ता होगा
  छूना

  समाप्त हो जाता है यदि <a href="/reference/p5/touchEnded">touchEnded()</a> घोषित नहीं किया जाता है। अगर

  <a href='/reference/p5/touchEnded'>touchEnded()</a> घोषित किया जाता है, तो

  <a href="/reference/p5/touchEnded">touchEnded()</a> तब चलेगा जब कोई उपयोगकर्ता
  छूना

  समाप्त होता है और <code>mouseClicked()</code> नहीं होगा।</p>

  <p>ब्राउज़र में विभिन्न माउस ईवेंट से जुड़े डिफ़ॉल्ट व्यवहार हो सकते हैं। के लिए

  उदाहरण के लिए, जब उपयोगकर्ता माउस घुमाता है तो कुछ ब्राउज़र टेक्स्ट को हाइलाइट करते हैं

  माउस बटन दबाना. इस ईवेंट के लिए किसी भी डिफ़ॉल्ट व्यवहार को रोकने के लिए,

  फ़ंक्शन के अंत में <code> return false;</code> जोड़ें।</p>

  <p>नोट: <a href="/reference/p5/mousePressed">mousePressed()</a>,

  <a href='/reference/p5/mouseReleased'>mouseReleased()</a>,

  और <code>mouseClicked()</code> सभी संबंधित हैं।

  <a href="/reference/p5/mousePressed">mousePressed()</a> जैसे ही चलता है
  उपयोगकर्ता

  माउस क्लिक करता है. <a href='/reference/p5/mouseReleased'>mouseReleased()</a>
  के रूप में चलता है

  जैसे ही उपयोगकर्ता माउस क्लिक छोड़ता है। <code>mouseClicked()</code> चलता है

  <a href='/reference/p5/mouseReleased'>mouseReleased()</a>.
  के तुरंत बाद </p>
