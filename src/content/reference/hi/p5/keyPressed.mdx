  <p>एक फ़ंक्शन जो किसी भी कुंजी को दबाने पर एक बार कॉल किया जाता है।</p>

  <p>फ़ंक्शन घोषित करने से <code>keyPressed()</code> चलने के लिए एक code ब्लॉक सेट हो जाता है
  एक बार

  जब उपयोगकर्ता कोई कुंजी दबाता है तो स्वचालित रूप से:</p>

  <code>

  फ़ंक्शन keyPressed() {
    // चलाने के लिए code।
  }

  </code>


  <p><a href="/reference/p5/key">कुंजी</a> और <a
  href='/reference/p5/keyCode'>keyCode</a>

  वेरिएबल को सबसे हाल ही में टाइप किए गए मान के साथ अपडेट किया जाएगा

  <code>keyPressed()</code> को p5.js द्वारा कॉल किया जाता है:</p>

  <code>

  फ़ंक्शन keyPressed() {
    यदि (कुंजी === 'सी') {
      // चलाने के लिए code।
    }

  <p> यदि (keyCode === ENTER) {
      // चलाने के लिए code।
    }
  }

  </p></code>

  <p>पैरामीटर, <code>इवेंट</code>, वैकल्पिक है। <code>कीप्रेस्ड()</code>
  हमेशा पारित किया जाता है a

  <a href='https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent'
  target='_blank'>कीबोर्डइवेंट</a>

  गुणों वाली वस्तु जो कुंजी प्रेस घटना का वर्णन करती है:</p>

  <code>

  फ़ंक्शन कुंजी दबाया गया (घटना) {
    // चलाने के लिए code जो ईवेंट का उपयोग करता है।
    कंसोल.लॉग(इवेंट);
  }

  </code>


  <p>ब्राउज़र में विभिन्न प्रमुख घटनाओं से जुड़े डिफ़ॉल्ट व्यवहार हो सकते हैं। के लिए

  उदाहरण के लिए, कुछ ब्राउज़र किसी वेब पेज के निचले भाग पर जा सकते हैं जब

  <code>स्पेस</code> कुंजी दबाई गई है। इसके लिए किसी भी डिफ़ॉल्ट व्यवहार को रोकने के लिए
  घटना, जोड़ें

  <code>return false;</code> फ़ंक्शन के अंत तक।</p>
