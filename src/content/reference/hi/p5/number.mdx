  <p>एक संख्या जो धनात्मक, ऋणात्मक या शून्य हो सकती है।</p>

  <p><code>संख्या</code> डेटा प्रकार स्थिति, आकार और रंग जैसे मूल्यों का वर्णन करने के लिए उपयोगी है। कोई संख्या पूर्णांक हो सकती है जैसे 20 या दशमलव संख्या जैसे 12.34। उदाहरण के लिए, एक वृत्त की स्थिति और आकार को तीन संख्याओं द्वारा वर्णित किया जा सकता है:</p>

  <code>
  वृत्त(50, 50, 20);
  </code>

   <code>
   वृत्त(50, 50, 12.34);
   </code>

  संचालन: कोष्ठक, घातांक, गुणा, भाग, जोड़,
  और घटाव (PEMDAS)। उदाहरण के लिए, p5.js के सिस्टम वेरिएबल के साथ अंकगणितीय ऑपरेटरों का उपयोग करना आम बात है जो संख्याएँ हैं:</p><code>

   // केंद्र में एक वृत्त बनाएं।

   </code>

  <p>यहां अंकगणितीय ऑपरेटरों का एक त्वरित अवलोकन दिया गया है:</p>
   <code>

   1 / 2 // विभाजित करें
   1 % 2 // शेष
   1 ** 2 // घातांक
   </code>

  <p>अंकगणित का उपयोग करके किसी संख्या चर को अद्यतन करना आम बात है। उदाहरण के लिए, किसी ऑब्जेक्ट का स्थान इस प्रकार अपडेट किया जा सकता है:</p>
  <code>
  एक्स = एक्स + 1;
  </code>

   <p>उपरोक्त कथन <code>+</code> ऑपरेटर का उपयोग करके एक वेरिएबल <code>x</code> में 1 जोड़ता है। अतिरिक्त असाइनमेंट ऑपरेटर <code>+=</code> एक ही विचार व्यक्त करता है:</p> <code>
   एक्स += 1;
   </code>

   <p>यहां असाइनमेंट ऑपरेटरों का एक त्वरित अवलोकन दिया गया है:</p>
   <code>
   x += 2 // अतिरिक्त असाइनमेंट
   x -= 2 // घटाव असाइनमेंट
