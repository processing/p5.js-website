  <p>एक फ़ंक्शन जिसे कोई भी कुंजी जारी होने पर एक बार कॉल किया जाता है।</p>

  <p>फ़ंक्शन घोषित करना <code>keyReleased()</code> चलाने के लिए एक code ब्लॉक सेट करता है
  एक बार

  जब उपयोगकर्ता कोई कुंजी जारी करता है तो स्वचालित रूप से:</p>

  <code>

  फ़ंक्शन keyReleased() {
    // चलाने के लिए code।
  }

  </code>


  <p><a href="/reference/p5/key">कुंजी</a> और <a
  href='/reference/p5/keyCode'>keyCode</a>

  वेरिएबल को हाल ही में जारी किए गए मान के साथ अद्यतन किया जाएगा

  <code>keyReleased()</code> को p5.js द्वारा कॉल किया जाता है:</p>

  <code>

  फ़ंक्शन कुंजी जारी() {
    यदि (कुंजी === 'सी') {
      // चलाने के लिए code।
    }

  <p> यदि (keyCode === ENTER) {
      // चलाने के लिए code।
    }
  }

  </p></code>

  <p>पैरामीटर, <code>इवेंट</code>, वैकल्पिक है। <code>कुंजी जारी()</code>
  हमेशा पारित किया जाता है a

  <a href='https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent'
  target='_blank'>कीबोर्डइवेंट</a>

  गुणों वाली वस्तु जो कुंजी प्रेस घटना का वर्णन करती है:</p>

  <code>

  फ़ंक्शन कुंजी जारी(घटना) {
    // चलाने के लिए code जो ईवेंट का उपयोग करता है।
    कंसोल.लॉग(इवेंट);
  }

  </code>


  <p>ब्राउज़र में विभिन्न प्रमुख घटनाओं से जुड़े डिफ़ॉल्ट व्यवहार हो सकते हैं। को

  इस ईवेंट के लिए किसी भी डिफ़ॉल्ट व्यवहार को रोकें, इसमें <code> return false;</code> जोड़ें
