description: >
  <p>चित्रों की सतह के पदार्थ के चमकदार रंग को सेट करता है।</p> <p><code>specularMaterial()</code> रंग, प्रकाश के उन घटकों को सेट करता है जो चमकीले कोट वाले आकारों को प्रतिबिंबित करेंगे। उदाहरण के लिए, <code>specularMaterial(255, 255, 0)</code> कॉल करने से एक आकार लाल और हरे प्रकाश को प्रतिबिंबित करेगा, लेकिन नीला प्रकाश नहीं।</p> 
  
  <p><a href="/reference/p5/ambientMaterial">ambientMaterial()</a> से विपरीत, <code>specularMaterial()</code> <a href="/reference/p5/directionalLight">directionalLight()</a>, <a href="/reference/p5/pointLight">pointLight()</a>, और <a href="/reference/p5/spotLight">spotLight()</a> सहित प्रकाश स्रोतों के पूर्ण रंग को प्रतिबिंबित करेगा। यही उन्हें "चमकीला" दिखने देता है। पदार्थ की चमक <a href="/reference/p5/shininess">shininess()</a> कार्य से नियंत्रित की जा सकती है।</p> 
  
  <p><code>specularMaterial()</code> को पदार्थ के रंग को सेट करने के लिए तीन तरीकों से कॉल किया जा सकता है।</p> <p>पहला तरीका <code>specularMaterial()</code> को कॉल करने में एक पैरामीटर, <code>gray</code> होता है। 0 और 255 के बीच के ग्रेस्केल मान, जैसे <code>specularMaterial(50)</code>, पदार्थ के रंग को सेट करने के लिए पास किए जा सकते हैं। उच्च ग्रेस्केल मान आकारों को बहुत चमकदार दिखाते हैं।</p> 
  
  <p>दूसरा तरीका <code>specularMaterial()</code> को कॉल करने में एक पैरामीटर, <code>color</code> होता है। <a href="/reference/p5/p5.Color">p5.Color</a> ऑब्जेक्ट, रंग मानों का एक एरे, या CSS रंग स्ट्रिंग, जैसे <code>specularMaterial('magenta')</code>, पदार्थ के रंग को सेट करने के लिए पास की जा सकती हैं।</p> 
  
  <p>तीसरा तरीका <code>specularMaterial()</code> को कॉल करने में चार पैरामीटर होते हैं, <code>v1</code>, <code>v2</code>, <code>v3</code>, और <code>alpha</code>। <code>alpha</code> वैकल्पिक है। RGBA, HSBA, या HSLA मान, जैसे <code>specularMaterial(255, 0, 0)</code> या <code>specularMaterial(255, 0, 0, 30)</code>, पदार्थ के रंगों को सेट करने के लिए पास किए जा सकते हैं। वर्तमान <a href="/reference/p5/colorMode">colorMode()</a> का उपयोग करके रंग मान व्याख्या किए जाएंगे।</p>

      // Enable orbiting with the mouse.

      pointLight(255, 255, 255, 30, -40, 30);

      // Add a glossy coat if the user has double-clicked.
      if (isGlossy === true) {
        specularMaterial(255);
        shininess(50);
      }

      // Style the torus.
      noStroke();
      fill(255, 0, 0);

      // Draw the torus.
      torus(30);
    }

    // Make the torus glossy when the user double-clicks.
    function doubleClicked() {
      isGlossy = true;
    }
