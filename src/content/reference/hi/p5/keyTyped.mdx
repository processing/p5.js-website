  <p>एक फ़ंक्शन जिसे प्रिंट करने योग्य वर्णों वाली कुंजियाँ होने पर एक बार कॉल किया जाता है
  दबाया.</p>

  <p>फ़ंक्शन घोषित करना <code>keyTyped()</code> चलाने के लिए एक code ब्लॉक सेट करता है
  एक बार

  स्वचालित रूप से जब उपयोगकर्ता मुद्रण योग्य वर्ण वाली कोई कुंजी दबाता है

  जैसे <code>a</code> या 1. संशोधक कुंजियाँ जैसे <code>SHIFT</code>,
  <code>नियंत्रण</code>, और तीर कुंजियाँ

  नजरअंदाज कर दिया जाएगा:</p>

  <code>

  फ़ंक्शन keyTyped() {
    // चलाने के लिए code।
  }

  </code>


  <p><a href="/reference/p5/key">कुंजी</a> और <a
  href='/reference/p5/keyCode'>keyCode</a>

  वेरिएबल को हाल ही में जारी किए गए मान के साथ अद्यतन किया जाएगा

  <code>keyTyped()</code> को p5.js द्वारा कॉल किया जाता है:</p>

  <code>

  फ़ंक्शन कुंजी टाइप किया गया() {
    // कुंजी का उपयोग करके "सी" वर्ण की जांच करें।
    यदि (कुंजी === 'सी') {
      // चलाने के लिए code।
    }

  <p> // keyCode का उपयोग करके "c" की जांच करें।
    if(keycode === 67) {
      // चलाने के लिए code।
    }
  }

  </p></code>

  <p>पैरामीटर, <code>इवेंट</code>, वैकल्पिक है। <code>keyTyped()</code> है
  हमेशा एक पारित किया

  <a href='https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent'
  target='_blank'>कीबोर्डइवेंट</a>

  गुणों वाली वस्तु जो कुंजी प्रेस घटना का वर्णन करती है:</p>

  <code>

  फ़ंक्शन कुंजी जारी(घटना) {
    // चलाने के लिए code जो ईवेंट का उपयोग करता है।
    कंसोल.लॉग(इवेंट);
  }

  </code>


  <p>नोट: <a href="/reference/p5/keyPressed">keyPressed()</a> फ़ंक्शन का उपयोग करें
  और

  <a href="/reference/p5/keyCode">keyCode</a> सिस्टम वेरिएबल का जवाब देना है
  संशोधक

  <code>ALT</code> जैसी कुंजियाँ।</p>

  <p>ब्राउज़र में विभिन्न प्रमुख घटनाओं से जुड़े डिफ़ॉल्ट व्यवहार हो सकते हैं। को

  इस ईवेंट के लिए किसी भी डिफ़ॉल्ट व्यवहार को रोकें, इसमें <code> return false;</code> जोड़ें
  समाप्त

  समारोह का.</p>
