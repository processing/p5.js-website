  <p>किसी रंग का लाल मान प्राप्त होता है।</p>

  <p><code>red()</code> a से लाल मान निकालता है <a href="/reference/p5/p5.Color">p5.Color</a> ऑब्जेक्ट, रंग घटकों की एक सरणी, या एक CSS रंग स्ट्रिंग।</p>

  <p>डिफ़ॉल्ट रूप से, <code>red()</code> 0 से 255 की सीमा में रंग का लाल मान लौटाता है। यदि <a href="/reference/colorMode">colorMode()</a> सेट है आरजीबी के लिए, यह दी गई सीमा में लाल मान लौटाता है।</p>
