  <p>एक <code>संख्या</code> सिस्टम वैरिएबल जिसमें अंतिम का code होता है
  कुंजी टाइप की गई.</p>

  <p>सभी कुंजियों में एक <code>keyCode</code> होता है। उदाहरण के लिए, <code>a</code> कुंजी
  <code>keyCode</code> 65 है।

  <code>keyCode</code> वैरिएबल यह जाँचने में सहायक है कि कोई विशेष है या नहीं
  कुंजी है

  टाइप किया गया. उदाहरण के लिए, निम्नलिखित सशर्त जाँच करता है कि क्या एंटर

  कुंजी टाइप कर दी गई है:</p>

  <code>

  अगर (कीcode === 13) {
    // एंटर कुंजी दबाए जाने पर चलने वाला code।
  }

  </code>


  <p>सिस्टम वेरिएबल का उपयोग करके समान code को अधिक स्पष्ट रूप से लिखा जा सकता है
  <code>प्रविष्ट करें</code>

  जिसका मान 13:</p> है

  <code>

  यदि (कुंजीcode === दर्ज करें) {
    // एंटर कुंजी दबाए जाने पर चलने वाला code।
  }

  </code>


  <p>सिस्टम वेरिएबल <code>बैकस्पेस</code>, <code>हटाएं</code>,
  <code>दर्ज करें</code>, <code>वापसी</code>, <code>टैब</code>,

  <code>बचना</code>, <code>शिफ्ट</code>, <code>नियंत्रण</code>,
  <code>विकल्प</code>, <code>एएलटी</code>, <code>UP_ARROW</code>,
  <code>डाउन_एरो</code>,

  <code>LEFT_ARROW</code>, और <code>RIGHT_ARROW</code> सभी सहायक हैं
  शॉर्टहैंड के प्रमुख code

  विशेष चाबियाँ. कुंजी code जैसी वेबसाइटों पर पाए जा सकते हैं
