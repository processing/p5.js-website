  <p>यदि जाँच की जाने वाली कुंजी दबाई जाती है तो <code>सही</code> लौटाता है

  <p><code>keyIsDown()</code> एकाधिक भिन्न की जांच करते समय सहायक होता है
  कुंजी दबाएँ.

  उदाहरण के लिए, <code>keyIsDown()</code> का उपयोग यह जांचने के लिए किया जा सकता है कि दोनों हैं या नहीं
  <code>LEFT_ARROW</code> और

  <code>UP_ARROW</code> दबाया जाता है:</p>

  <code>

  यदि (keyIsDown(LEFT_ARROW) && keyIsDown(UP_ARROW)) {
    // तिरछे ले जाएँ.
  }

  </code>


  <p><code>keyIsDown()</code> का उपयोग करके कुंजी दबाए जाने की जांच कर सकते हैं

  <a href="/reference/p5/keyCode">keyCode</a> मान, जैसा कि
  <code>keyIsDown(37)</code> या

  <code>keyIsDown(LEFT_ARROW)</code>। कुंजी code जैसी वेबसाइटों पर पाए जा सकते हैं

  <a href='https://keycode.info' target=”_blank'>keycode.info</a>.</p>

      if (keyIsDown(LEFT_ARROW) === true) {
        x -= 1;
      }

      if (keyIsDown(RIGHT_ARROW) === true) {
        x += 1;
      }

      if (keyIsDown(UP_ARROW) === true) {
        y -= 1;
      }

      if (keyIsDown(DOWN_ARROW) === true) {
        y += 1;
      }

      // Style the circle.
      fill(0);

      // Draw the circle.
      circle(x, y, 5);
    }
