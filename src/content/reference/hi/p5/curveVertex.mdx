description: >
  <p>कस्टम आकार में एक तख़्ता वक्र खंड जोड़ता है।</p>

  <p><code>curveVertex()</code> कस्टम आकृतियों में एक घुमावदार खंड जोड़ता है। इसके द्वारा बनाए गए तख़्ता वक्र को <a href="/reference/p5/curve">curve()</a> फ़ंक्शन द्वारा बनाए गए वक्रों की तरह परिभाषित किया गया है।

  <code>curveVertex()</code> को <a href="/reference/p5/beginShape">beginShape()</a> और <a href="/reference/p5/endShape">endShape()</a>कार्य के बीच बुलाया जाना चाहिए।</p>

  <p>स्पलाइन वक्र ऐसे आकार और वक्र बना सकते हैं जो धीरे-धीरे ढलान करते हैं। वे उन केबलों की तरह हैं जो बिंदुओं के एक समूह से जुड़े होते हैं।

  स्प्लिंस को दो एंकर बिंदुओं और दो नियंत्रण बिंदुओं द्वारा परिभाषित किया गया है। <code>curveVertex()</code> को <a href="/reference/p5/beginShape">beginShape()</a> और <a href="/reference/p5/endShape">endShape()</a> के बीच कम से कम चार बार कॉल किया जाना चाहिए। 

  एक वक्र बनाने के लिए:</p>

  <code>

  beginShape();


  <p>// Add the first control point.

  curveVertex(84, 91);</p>

  <p>// Add the anchor points to draw between.

  curveVertex(68, 19);

  curveVertex(21, 17);</p>

  <p>// Add the second control point.

  curveVertex(32, 91);</p>

  <p>endShape();

  </p></code>

  <p>उपरोक्त कोड स्निपेट केवल एंकर बिंदुओं के बीच वक्र खींचेगा, <a href="/reference/p5/curve">curve()</a> फ़ंक्शन के समान।

  नियंत्रण और एंकर बिंदुओं के बीच के खंडों को नियंत्रण बिंदुओं के निर्देशांक के साथ <code>curveVertex()</code> पर कॉल करके खींचा जा सकता है:</p>

  <code>

  beginShape();


  <p>// Add the first control point and draw a segment to it.

  curveVertex(84, 91);

  curveVertex(84, 91);</p>

  <p>// Add the anchor points to draw between.

  curveVertex(68, 19);

  curveVertex(21, 17);</p>

  <p>// Add the second control point.

  curveVertex(32, 91);</p>

  <p>// Uncomment the next line to draw the segment to the second control point.

  // curveVertex(32, 91);</p>

  <p>endShape();

  </p></code>

  <p>पहले दो पैरामीटर, <code>x</code> और <code>y</code>, शीर्ष का स्थान निर्धारित करते हैं।

  उदाहरण के लिए, <code>curveVertex(10, 10)</code> को कॉल करने से <code>(10, 10)</code> पर कर्व में एक बिंदु जुड़ जाता है।</p>

  <p>वेबजीएल मोड का उपयोग करके स्पलाइन कर्व्स को 3D में भी खींचा जा सकता है। <code>curveVertex()</code> के 3D संस्करण में तीन तर्क हैं क्योंकि प्रत्येक बिंदु में x-, y- और z-निर्देशांक हैं।

  डिफ़ॉल्ट रूप से, शीर्ष का z-निर्देशांक 0 पर सेट है।</p>

  <p>नोट: <code>curveVertex()</code> तब काम नहीं करेगा जब कोई तर्क <a href="/reference/p5/beginShape">beginShape()</a> को पास किया जाता है।</p>

    // Click the mouse near the red dot in the bottom-left corner
    // and drag to change the curve's shape.

    let x1 = 32;
    let y1 = 91;
    let isChanging = false;

    function setup() {
      createCanvas(100, 100);

      describe(
        'A black U curve drawn upside down on a gray background. The curve passes from one red dot through two black dots and ends at another red dot.'
      );
    }

    function draw() {
      background(200);

      // Style the shape.
      noFill();
      stroke(0);
      strokeWeight(1);

      // Start drawing the shape.
      beginShape();

      // Add the first control point and draw a segment to it.
      curveVertex(x1, y1);
      curveVertex(x1, y1);

      // Add the anchor points.
      curveVertex(21, 17);
      curveVertex(68, 19);

      // Add the second control point and draw a segment to it.
      curveVertex(84, 91);
      curveVertex(84, 91);

      // Stop drawing the shape.
      endShape();

      // Style the anchor and control points.
      strokeWeight(5);

      // Draw the anchor points in black.
      stroke(0);
      point(21, 17);
      point(68, 19);

      // Draw the control points in red.
      stroke(255, 0, 0);
      point(x1, y1);
      point(84, 91);
    }

    // Start changing the first control point if the user clicks near it.
    function mousePressed() {
      if (dist(mouseX, mouseY, x1, y1) < 20) {
        isChanging = true;
      }
    }

    // Stop changing the first control point when the user releases the mouse.
    function mouseReleased() {
      isChanging = false;
    }

    // Update the first control point while the user drags the mouse.
    function mouseDragged() {
      if (isChanging === true) {
        x1 = mouseX;
        y1 = mouseY;
      }
    }
    </code>
