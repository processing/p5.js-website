  <p>सपाट आकार में छेद बनाना बंद कर देता है।</p>

  <p><a href="/reference/p5/beginContour">beginContour()</a> और <code>endContour()</code> फ़ंक्शन फ्लैट कस्टम आकृतियों के भीतर नकारात्मक स्थान बनाने की अनुमति देते हैं।
  
  <a href="/reference/p5/beginContour">beginContour()</a> नकारात्मक स्थान पर शीर्ष जोड़ना शुरू करता है और <code>endContour()</code> उन्हें जोड़ना बंद कर देता है।

  <a href='/reference/p5/beginContour'>beginContour()</a> और <code>endContour()</code> को <a href='/reference/p5/beginShape'>beginShape()</a> और <a href="/reference/p5/endShape">endShape()</a> के बीच कॉल किया जाना चाहिए ।</p>

  <p>रूपांतरण जैसे <a href='/reference/p5/translate'>translate()</a>, <a href='/reference/p5/rotate'>rotate()</a>, और <a href='/reference/p5/scale'>scale()</a> <a href='/reference/p5/beginContour'>beginContour()</a> और <code>endContour()</code> के बीच काम नहीं करता है।
  
  <a href="/reference/p5/beginContour">beginContour()</a> और <code>endContour()</code> के बीच अन्य आकृतियों का उपयोग करना भी संभव नहीं है, जैसे <a href='/reference/p5/ellipse'>ellipse()</a> या <a href='/reference/p5/rect'>rect()</ a>।</p>

  <p>ध्यान दें: नकारात्मक स्थान को परिभाषित करने वाले शीर्षों को बाहरी आकार से विपरीत दिशा में होना चाहिए। सबसे पहले, बाहरी आकृति के लिए दक्षिणावर्त क्रम में शीर्ष बनाएं। फिर, नकारात्मक स्थान के लिए वामावर्त क्रम में शीर्ष बनाएं।</p>

    function setup() {

      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // Interior vertices, counter-clockwise winding.
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // Stop drawing the shape.
      endShape(CLOSE);
