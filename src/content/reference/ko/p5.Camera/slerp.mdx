description: >
  <p>두 개의 다른 카메라의 값 사이에 있는 카메라의 위치와 방향을 설정합니다.</p>
  
  <p><code>myCamera.slerp()</code>는 구형 선형 보간을 사용하여 다른 두 카메라 사이의 위치와 방향을 계산합니다.
  
  이렇게 함으로써 두 시점 간의 부드러운 전환에 도움이 됩니다.</p>
  
  <p>첫 번째 두 매개변수 <code>cam0</code>과 <code>cam1</code>은 현재의 카메라를 설정하는 데 사용되는 <code>p5.Camera</code> 객체입니다.</p>
  
  <p>세 번째 매개변수 <code>amt</code>는 <code>cam0</code>과 <code>cam1</code> 사이를 보간하는 양입니다.
  
  0.0은 카메라의 위치와 방향을 <code>cam0</code>와 같게 유지하고, 0.5는 <code>cam0</code>와 <code>cam1</code> 사이의 중간값으로 설정하며,
  
  1.0은 <code>cam1</code>과 같게 설정합니다.</p>
  
  <p>예를 들어, <code>myCamera.slerp(cam0, cam1, 0.1)</code>을 호출하면 cam의 위치와 방향이 <code>cam0</code>에 매우 가깝게 설정됩니다.
  
  <code>myCamera.slerp(cam0, cam1, 0.9)</code>를 호출하면 cam의 위치와 방향이 <code>cam1</code>에 매우 가깝게 설정됩니다.</p>
  
  <p>참고: 모든 카메라는 동일한 투영을 사용해야 합니다.</p>

    let cam;
    let cam0;
    let cam1;

    function setup() {
      createCanvas(100, 100, WEBGL);

      // Create the main camera.
      // Keep its default settings.
      cam = createCamera();

      // Create the first camera.
      // Keep its default settings.
      cam0 = createCamera();

      // Create the second camera.
      cam1 = createCamera();

      // Place it at the top-right.
      cam1.setPosition(400, -400, 800);

      // Point it at the origin.
      cam1.lookAt(0, 0, 0);

      // Set the current camera to cam.
      setCamera(cam);

      describe('A white cube drawn against a gray background. The camera slowly oscillates between a frontal view and an aerial view.');
    }

    function draw() {
      background(200);

      // Calculate the amount to interpolate between cam0 and cam1.
      let amt = 0.5 * sin(frameCount * 0.01) + 0.5;

      // Update the main camera's position and orientation.
      cam.slerp(cam0, cam1, amt);

      box();
    }
