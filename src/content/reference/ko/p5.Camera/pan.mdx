  <p>카메라를 좌우로 회전합니다.</p>
  
  <p>패닝(panning)은 카메라의 위치를 변경하지 않고 카메라를 회전합니다.
  
  회전은 카메라의 "로컬" 공간에서 발생합니다.</p>
  
  <p>매개변수인 <code>angle</code>은 카메라가 회전해야 하는 각도입니다.
  
  양수 각도를 전달하면, <code>myCamera.pan(0.001)</code>과 같이, 카메라가 오른쪽으로 회전합니다.
  
  음수 각도를 전달하면, <code>myCamera.pan(-0.001)</code>과 같이, 카메라가 왼쪽으로 회전합니다.</p>
  
  <p>참고: 각도는 현재의 <a href="/reference/p5/angleMode">angleMode()</a>에 따라 해석됩니다.</p>

    <code>
    let cam;
    let delta = 0.001;

    function setup() {
      createCanvas(100, 100, WEBGL);

      // Create a p5.Camera object.
      cam = createCamera();

      // Place the camera at the top-center.
      cam.setPosition(0, -400, 800);

      // Point the camera at the origin.
      cam.lookAt(0, 0, 0);

      describe(
        'A white cube on a gray background. The cube goes in and out of view as the camera pans left and right.'
      );
    }

    function draw() {
      background(200);

      // Pan with the camera.
      cam.pan(delta);

      // Switch directions every 120 frames.
      if (frameCount % 120 === 0) {
        delta *= -1;
      }

      // Draw the box.
      box();
    }
