  <p>숫자의 아크 코사인 값을 계산합니다.</p>

  <p><code>acos()</code> 는 <a
  href="/reference/p5/cos">cos()</a>의 역함수입니다. 이 함수는 -1과 1 사이의 값을 가집니다. 기본적으로, <code>acos()</code> 는 0부터 π(3.14) 사이의 값으로 반환합니다.

  만약, <a href="/reference/p5/angleMode">angleMode()</a>이 <code>DEGREES</code>로 설정되어 있다면, 값은 0부터 180 사이로 반환됩니다.</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Calculate cos() and acos() values.
      let a = PI;
      let c = cos(a);
      let ac = acos(c);

      // Display the values.
      text(`${round(a, 3)}`, 35, 25);
      text(`${round(c, 3)}`, 35, 50);
      text(`${round(ac, 3)}`, 35, 75);

      describe('The numbers 3.142, -1, and 3.142 written on separate rows.');
    }
