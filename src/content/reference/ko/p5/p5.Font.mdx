methods:
  textBounds:
    description: >
      <p>폰트를 사용하여 작성된 텍스트의 경계 상자(bounding box)를 반환합니다.</p>
      
      <p>경계 상자는 텍스트 문자열을 포함할 수 있는 가장 작은 직사각형입니다.
      
      <code>font.textBounds()</code>는 경계 상자의 위치와 크기를 포함하는 객체를 반환합니다.
      
      예를 들어 <code>font.textBounds('p5*js', 5, 20)</code>를 호출하면 다음과 같은 형식의 객체가 반환됩니다.
      
      <code>{ x: 5.7, y: 12.1 , w: 9.9, h: 28.6 }</code>.
      
      <code>x</code>와 <code>y</code> 속성은 경계 상자의 왼쪽 상단 모서리의 좌표입니다.</p>
      
      <p>첫 번째 매개변수 <code>str</code>은 텍스트 문자열입니다.
      
      두 번째와 세 번째 매개변수인 <code>x</code>와 <code>y</code>는 텍스트의 위치입니다.
      
      기본적으로, 이들은 경계 상자 좌하단의 좌표로 설정합니다.
      
      더 많은 텍스트 정렬 방법은 <a href="/reference/p5/textAlign">textAlign()</a>를 참조하십시오.</p>
      
      <p>네 번째 매개변수 <code>fontSize</code>는 선택 사항입니다.
      
      경계 상자를 결정하는 데 사용되는 글꼴 크기를 설정합니다.
      
      기본적으로, <code>font.textBounds()</code>는
      
      현재의 <a href="/reference/p5/textSize">textSize()</a>를 사용합니다.</p>
    path: p5.Font/textBounds
  textToPoints:
    description: >
      <p>글꼴을 사용하여 작성된 텍스트의 윤곽을 나타내는 점 배열을 반환합니다.</p>
      
      <p>배열의 각 점 객체에는 점의 위치와 방향을 나타내는 세 가지 속성이 있습니다.
      
      예시로, <code>{ x: 10, y: 20, alpha: 450 }</code>가 있습니다.</p>
      
      <p>첫 번째 매개변수 <code>str</code>은 텍스트 문자열입니다.
      
      두 번째와 세 번째 매개변수인 <code>x</code>와 <code>y</code>는 텍스트의 위치입니다.
      
      기본적으로, 이들은 경계 상자 좌하단의 좌표를 설정합니다.
      
      더 많은 텍스트 정렬 방법은 <a href="/reference/p5/textAlign">textAlign()</a>를 참조하십시오.</p>
      
      <p>네 번째 매개변수 <code>fontSize</code>는 선택 사항입니다.
      
      텍스트의 폰트 크기를 설정합니다.
      
      기본적으로, <code>font.textToPoints()</code>는
      
      현재의 <a href="/reference/p5/textSize">textSize()</a>를 사용합니다.</p>
      
      <p>다섯 번째 매개변수인 <code>options</code>도 선택 사항입니다.
      
      <code>font.textToPoints()</code>는 다음과 같은 속성을 가진 객체가 들어갈 수 있습니다:</p>
      
      <p><code>sampleFactor</code>는 텍스트의 경로 길이와 샘플 수의 비율입니다.
      
      기본값은 0.1입니다. 값이 높으면 경로를 따라 더 많은 점을 생성하며 더 정밀합니다.</p>
      
      <p><code>simplifyThreshold</code>는 0이 아닌 숫자로 설정되면 공선점(collinear point)을 제거합니다.
      
      이 값은 두 가지 모서리가 공선(collinear)인지를 결정할 때 사용할 임계 각도를 나타냅니다.</p>
    path: p5.Font/textToPoints
