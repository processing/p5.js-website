  <p><a href="/reference/p5/specularMaterial">specularMaterial()</a>의 금속성 양을 설정합니다.</p>
  <p><code>metalness()</code>는 재질을 더 금속처럼 보이게 할 수 있습니다. <a href="/reference/p5/directionalLight">directionalLight()</a>, <a href="/reference/p5/pointLight">pointLight()</a>, <a href="/reference/p5/spotLight">spotLight()</a>, 그리고 <a href="/reference/p5/imageLight">imageLight()</a>를 포함한 광원을 반사하는 방식에 영향을 줍니다.</p>
  <p>매개 변수 <code>metallic</code>은 금속성의 양을 설정하는 숫자입니다. <code>metallic</code>은 기본값인 1보다 커야 합니다. <code>metalness(100)</code>과 같이 값이 높으면 표면 재질이 더 금속처럼 보입니다.</p>

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe(
        'Two blue spheres drawn on a gray background. White light reflects from their surfaces as the mouse moves. The right sphere is more metallic than the left sphere.'
      );
    }

    function draw() {
      background(200);

      // Turn on an ambient light.
      ambientLight(200);

      // Get the mouse's coordinates.
      let mx = mouseX - 50;
      let my = mouseY - 50;

      // Turn on a white point light that follows the mouse.
      pointLight(255, 255, 255, mx, my, 50);

      // Style the spheres.
      noStroke();
      fill(30, 30, 255);
      specularMaterial(255);
      shininess(20);

      // Draw the left sphere with low metalness.
      translate(-25, 0, 0);
      metalness(1);
      sphere(20);

      // Draw the right sphere with high metalness.
      translate(50, 0, 0);
      metalness(50);
      sphere(20);
    }
    </code>
