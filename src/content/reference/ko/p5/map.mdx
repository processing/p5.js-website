description: |-
  <p>한 범위에서 다른 범위로 수를 다시 매핑합니다.</p>

  <p>예를 들어, <code>map(2, 0, 10, 0, 100)</code>를 호출하면 20을 반환합니다. 처음 세 매개변수는 원래 값을 2로, 원래의 범위를 0에서 10으로 설정합니다. 마지막 두 매개변수는 목표 범위를 0에서 100으로 설정합니다. 목표 범위인 [0, 100]에서 20의 위치는 원래의 범위인 [0, 10]에서 2의 위치에 비례합니다.</p>

  <p>여섯번째 매개변수인 <code>withinBounds</code>는 선택 사항입니다. 기본적으로 <code>map()</code>은 목표 범위를 벗어난 값을 반환할 수 있습니다. 예를 들어, <code>map(11, 0, 10, 0, 100)</code>는 110을 반환합니다. <code>true</code>를 여섯번째 매개변수에 전달하면 목표 범위로 다시 매핑된 값을 제한할 수 있습니다. 예를 들어, <code>map(11, 0, 10, 0, 100, true)</code>는 100을 반환합니다.</p>
