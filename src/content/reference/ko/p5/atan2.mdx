description: >
  <p>점, 원점, 그리고 양의 x축에 의해 형성된 각도를 계산합니다.</p>

  <p><code>atan2()</code> 은 <code>atan2(mouseY, mouseX)</code>처럼 주로 도형을 마우스의 위치에 맞추기 위해 사용됩니다. 
  첫 번째 매개변수는 점의 y좌표이고 두 번째 매개변수는 x좌표입니다. </p>

  <p>기본적으로, <code>atan2()</code> 는

  -π (약 -3.14) to π (약 3.14)까지의 값을 반환합니다.

  <a href="/reference/p5/angleMode">angleMode()</a>가 <code>DEGREES</code>로 설정되었다면, 값은 -180부터 180 사이로 반환됩니다.</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      describe('A rectangle at the top-left of the canvas rotates with mouse movements.');
    }

    function draw() {
      background(200);

      // Calculate the angle between the mouse
      // and the origin.
      let a = atan2(mouseY, mouseX);

      // Rotate.
      rotate(a);

      // Draw the shape.
      rect(0, 0, 60, 10);
    }
    </code>

    function setup() {
      createCanvas(100, 100);

      describe('A rectangle at the center of the canvas rotates with mouse movements.');
    }

    function draw() {
      background(200);

      // Translate the origin to the center.
      translate(50, 50);

      // Get the mouse's coordinates relative to the origin.
      let x = mouseX - 50;
      let y = mouseY - 50;

      // Calculate the angle between the mouse and the origin.
      let a = atan2(y, x);

      // Rotate.
      rotate(a);

      // Draw the shape.
      rect(-30, -5, 60, 10);
    }
    </code>
    </div>
