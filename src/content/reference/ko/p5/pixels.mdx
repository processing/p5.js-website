description: >
  <p>캔버스의 각 픽셀의 색상을 포함하는 배열입니다.</p>
  
  <p>색상은 빨강, 초록, 파랑 및 알파(RGBA) 값을 나타내는 숫자로 저장됩니다.
  
  성능상의 이유로 <code>pixels</code>는 1차원 배열입니다.</p>
  
  <p><code>pixels</code> 배열에서 각 픽셀은 각 RGBA 값에 대해 네 개의 요소를 차지합니다.
  
  예를 들어, 좌표 (0, 0)의 픽셀은 각각
  
  <code>pixels[0]</code>, <code>pixels[1]</code>, <code>pixels[2]</code>, <code>pixels[3]</code>에 RGBA 값을 저장합니다.
  
  다음 좌표 (1, 0)의 픽셀은 각각
  
  <code>pixels[4]</code>, <code>pixels[5]</code>, <code>pixels[6]</code>, <code>pixels[7]</code>에 RGBA 값을 저장합니다.
  
  이와 같이, 100×100 캔버스의 <code>pixels</code> 배열에는 100 × 100 × 4 = 40,000개의 요소가 있습니다.</p>
  
  <p>일부 디스플레이는 단일 지점에서 색상을 설정하기 위해 여러 작은 픽셀을 사용합니다.
  
  <a href="/reference/p5/pixelDensity">pixelDensity()</a> 함수는 캔버스의 픽셀 밀도를 반환합니다.
  
  높은 밀도 디스플레이는 종종 <a href="/reference/p5/pixelDensity">pixelDensity()</a>가 2입니다.
  
  이러한 디스플레이에서 100×100 캔버스의 <code>pixels</code> 배열에는 200 × 200 × 4 = 160,000개의 요소가 있습니다.</p>
  
  <p>캔버스의 한 지점의 RGBA 값을 액세스하려면 아래와 같이 약간의 수학이 필요합니다.
  
  <code>pixels</code> 배열에 액세스하기 전에 <a href="/reference/p5/loadPixels">loadPixels()</a> 함수를 호출해야 합니다.
  
  어떤 변경 사항이든 발생한 후에는 <a href="/reference/p5/updatePixels">updatePixels()</a> 함수를 호출해야 합니다.</p>

    function setup() {
      createCanvas(100, 100);

      // Load the pixels array.
      loadPixels();

      // Set the dot's coordinates.
      let x = 50;
      let y = 50;

      // Get the pixel density.
      let d = pixelDensity();

      // Set the pixel(s) at the center of the canvas black.
      for (let i = 0; i < d; i += 1) {
        for (let j = 0; j < d; j += 1) {
          let index = 4 * ((y * d + j) * width * d + (x * d + i));
          // Red.
          pixels[index] = 0;
          // Green.
          pixels[index + 1] = 0;
          // Blue.
          pixels[index + 2] = 0;
          // Alpha.
          pixels[index + 3] = 255;
        }
      }

      // Update the canvas.
      updatePixels();

      describe('A black dot in the middle of a gray rectangle.');
    }
    </code>

      let d = pixelDensity();

      // Calculate the halfway index in the pixels array.
      let halfImage = 4 * (d * width) * (d * height / 2);

      // Make the top half of the canvas red.
      for (let i = 0; i < halfImage; i += 4) {
        // Red.
        pixels[i] = 255;
        // Green.
        pixels[i + 1] = 0;
        // Blue.
        pixels[i + 2] = 0;
        // Alpha.
        pixels[i + 3] = 255;
      }

      // Update the canvas.
      updatePixels();

      describe('A red rectangle drawn above a gray rectangle.');
    }
