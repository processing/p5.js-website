  <p>색상에서 빨강값을 추출합니다.</p>

  <p><code>red()</code> 는 

  <a href="/reference/p5/p5.Color">p5.Color</a>객체, 색상 요소 배열, 또는 CSS 색상 문자열로부터

  빨강값을 추출합니다.</p>

  <p>기본적으로, <code>red()</code> 는 0 ~ 255까지범위의 색상의 빨강값을 반환합니다.

  <a href="/reference/colorMode">colorMode()</a> 가 RGB로 설정되어 있는 경우, 


  그것을 지정된 범위의 빨강값으로 반환합니다.</p>
