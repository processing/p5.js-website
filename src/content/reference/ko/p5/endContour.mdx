description: >
  <p>>평면 도형 내부에 구멍 생성을 중지합니다.</p>

  <p><a href="/reference/p5/beginContour">beginContour()</a> 및
  <code>endContour()</code>

  함수를 사용하면 사용자 정의 도형 내에 음의 공간(구멍)을 만들 수 있습니다.

  <a href="/reference/p5/beginContour">beginContour()</a> 은 공간에 
  꼭지점을 추가하고

  <code>endContour()</code> 는 꼭짓점 추가를 중지합니다. 

  <a href="/reference/p5/beginContour">beginContour()</a> 와
  <code>endContour()</code> 는 
  <a href="/reference/p5/beginShape">beginShape()</a> 과

  <a href="/reference/p5/endShape">endShape()</a> 사이에서 호출되어야 합니다.</p>

  <p><a href="/reference/p5/translate">translate()</a>,

  <a href="/reference/p5/rotate">rotate()</a> 그리고 <a
  href="/reference/p5/scale">scale()</a>과 같은 변환 기능은

  <a href="/reference/p5/beginContour">beginContour()</a> 와

  <code>endContour()</code>사이에서는 작동하지 않습니다. 또한

  <a href="/reference/p5/ellipse">ellipse()</a> 와 <a
  href="/reference/p5/rect">rect()</a> 와 같은 다른 도형도

  <a href="/reference/p5/beginContour">beginContour()</a> 와
  <code>endContour()</code> 사이에서는 사용할 수 없습니다.</p>

  <p>참고: 음의 공간을 정의하는 꼭짓점은 반대 방향으로 '감겨야'합니다.

  바깥의 도형에서 먼저 꼭짓점을 시계방향으로 그리고, 

  음의 공간의 꼭짓점을 

  반시계 방향으로 그리세요.</p>

    function setup() {
      createCanvas(100, 100);

      background(200);

      // Start drawing the shape.
      beginShape();

      // Exterior vertices, clockwise winding.
      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // Interior vertices, counter-clockwise winding.
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // Stop drawing the shape.
      endShape(CLOSE);

      describe('A white square with a square hole in its center drawn on a gray background.');
    }
