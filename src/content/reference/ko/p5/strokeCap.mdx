  <p>선의 양 끝 부분 렌더링 스타일을 설정합니다.</p>

  <p>선 끝 부분의 캡은 둥근 모양 (<code>ROUND</code>), 

  사각 모양(<code>SQUARE</code>), 또는 확장된 모양 (<code>PROJECT</code>) 중 하나입니다. 
  기본 캡은 <code>ROUND</code>입니다.</p>

  <p><code>strokeCap()</code> 에 전달된 인수는 모두 대문자로 작성되어야 합니다.
  이유는

  상수(constants) <code>ROUND</code>, <code>SQUARE</code>, 그리고
  <code>PROJECT</code> 가 이렇게 정의되기 때문입니다.

  JavaScript는 대소문자를 구분하는 언어입니다.</p>

  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      strokeWeight(12);

      // Top.
      strokeCap(ROUND);
      line(20, 30, 80, 30);

      // Middle.
      strokeCap(SQUARE);
      line(20, 50, 80, 50);

      // Bottom.
      strokeCap(PROJECT);
      line(20, 70, 80, 70);

      describe(
        'Three horizontal lines. The top line has rounded ends, the middle line has squared ends, and the bottom line has longer, squared ends.'
      );
    }
