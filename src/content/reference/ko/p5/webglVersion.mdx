description: >
  <p>사용 중인 WebGL 버전의 값을 가지는 <code>String</code> 변수입니다.</p>

  <p><code>webglVersion</code>은 다음 문자열(string) 상수 중 하나의 값을 갖습니다.</p>


  <ul>
  <li><code>'webgl2'</code>의 값을 가진 <code>WEBGL2</code></li>
  <li><code>'webgl'</code>의 값을 가진 <code>WEBGL</code></li>
  <li><code>'p2d'</code>의 값을 가진 <code>P2D</code>   - 2D 스케치에서의 기본값</li>
  </ul>

  <p>WebGL 버전을 설정하는 방법을 알아보려면 <a href="./setAttributes">setAttributes()</a>를 확인하세요.</p>

    function setup() {
      // Create a canvas using WEBGL mode.
      createCanvas(100, 50, WEBGL);

      // Set WebGL to version 1.
      setAttributes({ version: 1 });

      background(200);

      // Display the current WebGL version.
      fill(0);
      textFont(font);
