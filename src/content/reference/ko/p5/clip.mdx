description: >
  <p>다음에 그려지는 모든 모양을 마스크할 모양을 정의합니다.</p>

  <p>첫 번째 매개 변수 <code>callback</code> 은 마스크를 정의하는 
  함수입니다.

  <code>callback</code> 에서 그려진 모든 모양은 마스크 모양에 추가됩니다.
  마스크는

  <code>clip()</code> 이 호출된 후에 그려지는 모든 것에 적용됩니다.</p>

  <p>두 번째 매개 변수, <code>options</code>, 는 선택 사항입니다. 
  <code>invert</code> 속성이 포함된 객체가 

  전달되면, 예를 들어 <code>beginClip({ invert: true })</code>처럼, 
  마스킹 모드를

  설정하는데 사용됩니다. <code>{ invert: true }</code> 는 마스크를 반전시켜, 
  마스킹된 모양에

  구멍을 만듭니다. 기본적으로, <code>invert</code>는 <code>false</code> 
  입니다.</p>

  <p>마스크는

  <a href="/reference/p5/push">push()</a> 와 <a
  href="/reference/p5/pop">pop()</a> 함수 사이에 포함될 수 있습니다.

  이렇게 함으로써 마스크된 모양 이후에 마스크되지 않은 모양을 그릴 수 있습니다.</p>

  <p>마스크는 <a
  href="/reference/p5/beginClip">beginClip()</a>

  과 <a href="/reference/p5/endClip">endClip()</a>로도 정의할 수 있습니다.</p>
