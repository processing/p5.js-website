description: >
  <p>마우스 버튼이 눌린 후 떼어질 때 한 번 호출되는 함수입니다.</p>

  <p><code>mouseReleased()</code> 함수를 선언하면 사용자가 마우스 버튼을 누른 후 떼었을 때 자동으로 실행되는 코드 블록을 설정합니다:</p>

  <code>

  function mouseReleased() {
    // 실행할 코드.
  }

  </code>

  <p><a href="/reference/p5/mouseX">mouseX</a>나 <a href="/reference/p5/mouseY">mouseY</a>와 같은 마우스 시스템 변수는
  p5.js에 의해 <code>mouseReleased()</code>가 호출될 때 가장 최근 값으로 업데이트됩니다:</p>
  
  <code>

  function mouseReleased() {
    if (mouseX < 50) {
      // 마우스가 왼쪽에 있을 때 실행할 코드.
    }

  <p>  if (mouseY > 50) {
      // 마우스가 아래쪽에 있을 때 실행할 코드.
    }
  }

  </p></code>

  <p><code>event</code> 매개변수는 선택적(optional) 매개변수입니다.
  <code>mouseReleased()</code>는 항상
  <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent" target="_blank">MouseEvent</a> 객체를 전달받으며,
  이 객체는 마우스 버튼 떼기 이벤트를 설명하는 속성들을 가지고 있습니다:</p>

  <code>

  function mouseReleased(event) {
    // 이벤트를 사용하여 실행할 코드.
    console.log(event);
  }

  </code>

  <p>터치스크린 장치에서는 <a href="/reference/p5/touchEnded">touchEnded()</a>가 선언되지 않은 경우에만
  사용자가 터치를 떼었을 때 <code>mouseReleased()</code>가 실행됩니다.
  만약 <a href="/reference/p5/touchEnded">touchEnded()</a>가 선언되어 있다면,
  사용자가 터치를 떼었을 때 <a href="/reference/p5/touchEnded">touchEnded()</a>가 실행되고
  <code>mouseReleased()</code>는 실행되지 않습니다.</p>

  <p>브라우저는 다양한 마우스 이벤트에 대해 기본적으로 설정된 동작을 가지고 있을 수 있습니다.
  예를 들어, 일부 브라우저는 마우스 버튼을 누르고 마우스를 움직일 때 텍스트를 강조합니다.
  이 이벤트에 대한 기본 동작을 방지하려면, 함수 끝에 <code>return false;</code>를 추가하세요.</p>

  <p>참고: <a href="/reference/p5/mousePressed">mousePressed()</a>,
  <code>mouseReleased()</code>,
  그리고 <a href="/reference/p5/mouseClicked">mouseClicked()</a>는 모두 관련이 있습니다.
  <a href="/reference/p5/mousePressed">mousePressed()</a>는 사용자가 마우스를 클릭했을 때 바로 실행됩니다.
  <code>mouseReleased()</code>는 사용자가 마우스 클릭을 해제했을 때 바로 실행됩니다.
  <a href="/reference/p5/mouseClicked">mouseClicked()</a>는 <code>mouseReleased()</code>가 실행된 직후에 바로 실행됩니다.</p>
