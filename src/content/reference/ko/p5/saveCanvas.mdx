description: >
  <p>현재 캔버스를 이미지로 저장합니다.</p>

  <p><code>saveCanvas()</code> 함수는 기본적으로 캔버스를 <code>untitled.png</code>라는 이름으로 PNG 이미지로 저장합니다.</p>
  
  <p>첫 번째 매개변수인 <code>filename</code>은 선택 사항입니다. 파일의 이름을 설정하는 문자열입니다.
  
  파일 확장자가 포함된 경우, 예를 들어 <code>saveCanvas('drawing.png')</code>와 같이 지정하면

  해당 형식으로 이미지가 저장됩니다.</p>
  
  <p>두 번째 매개변수인 <code>extension</code>도 선택 사항입니다. 파일 형식을 설정합니다.
  
  <code>'png'</code> 또는 <code>'jpg'</code>를 사용할 수 있습니다.
  
  예를 들어, <code>saveCanvas('drawing', 'jpg')</code>는 캔버스를 <code>drawing.jpg</code>라는 이름의 파일로 저장합니다.</p>
  
  <p>참고: 브라우저는 파일을 즉시 저장하거나 사용자에게 대화 상자(dialogue window)를 표시할 것입니다.</p>
