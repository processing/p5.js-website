description: >
  <p>타원체를 그립니다.</p>

  <p>타원체는 삼각형 면들이 연결되어 둥근 표면을 형성하는 3D 도형입니다. 

  삼각형 면의 수가 적은 타원체는 마치 결정체처럼 보이며, 

  많은 삼각형 면으로 이루어진 타원체는 부드러운 표면을 가지고 달걀처럼 보입니다. 
  <code>ellipsoid()</code>

  는 직경(폭과 높이)을 사용하는

  <a href="/reference/p5/ellipse">ellipse()</a> 와 달리 

  반경으로 모양을 정의합니다.</p>

  <p>첫 번째 매개변수 <code>radiusX</code> 는 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>ellipsoid(20)</code> 와 같다면
  x축을 따라 타원체의 반경이 설정됩니다. 

  기본적으로, <code>radiusX</code> 은 50입니다.</p>

  <p>두 번째 매개변수 <code>radiusY</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>ellipsoid(20, 30)</code> 와 같다면,
  y축을 따라 타원체의 반경이 설정됩니다.

  기본적으로, <code>radiusY</code> 는 타원체의 
  <code>radiusX</code> 에 설정됩니다.</p>

  <p>세 번째 매개변수 <code>radiusZ</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>ellipsoid(20, 30, 40)</code> 와 같다면,
  축을 따라 타원체의 반경이 설정됩니다. 

  기본적으로, <code>radiusZ</code> 는 타원체의 
  <code>radiusY</code>에 설정됩니다.</p>

  <p>네 번째 매개변수 <code>detailX</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>ellipsoid(20, 30, 40, 5)</code> 와 같다면, x축을 따라 삼각형 분할면 수가 설정됩니다. 

  모든 3D 도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로, code>detailX</code>는 24입니다.

  <p>다섯 번째 매개변수 <code>detailY</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>ellipsoid(20, 30, 40, 5, 7)</code> 와 같다면, y축을 따라 삼각형 분할 수가 설정됩니다. 

  모든 3D 도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로, <code>detailY</code> 는 16입니다.</p>

  <p>참고: <code>ellipsoid()</code> 는 WebGL 모드에서만 사용할 수 있습니다.</p>
