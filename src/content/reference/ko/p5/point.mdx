description: >
  <p>공간에 하나의 점(point)을 그립니다.</p>

  <p>점의 기본 너비는 하나의 픽셀입니다. 점에 색상을 입히려면

  <a href="/reference/p5/stroke">stroke()</a> 함수를 사용하세요.
  너비를 변경하려면

  <a href="/reference/p5/strokeWeight">strokeWeight()</a> 함수를 사용하세요. 

  점은 채울 수 없으므로, <a href="/reference/p5/fill">fill()</a> 함수가

  점의 색상에 영향을 주지 않습니다.</p>

  <p><code>point()</code> 함수의 하나의 매개변수를 사용하는
  버전은

  해당 점의 위치를 x- 및 y- 좌표로 설정할 수 있으며, 다음과 같이 사용됩니다. <code>point(10, 20)</code>.</p>

  <p><code>point()</code> 함수의 세 개의 매개변수를 사용하는 버전은 
  버전은 

  x축, y축, 및 z축 좌표를 사용하여 3D 공간에 점을 그릴 수 있습니다. 다음과 같이 사용됩니다. <code>point(10, 20,
  30)</code>.

  이것을 하기 위해 <code>WEBGL</code> 인수를

  <a href="/reference/p5/createCanvas">createCanvas()</a>에 추가해야 합니다.</p>

  <p><code>point()</code> 함수의 하나의 매개변수를 사용하는 버전은
  점의 위치를

  <a href="/reference/p5/p5.Vector">p5.Vector</a> 객체로 설정할 수 있습니다.</p>
