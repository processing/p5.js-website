  <p>색상에서 채도값을 추출합니다..</p>

  <p><code>saturation()</code> a에서 채도값을 추출합니다

  <a href="/reference/p5/p5.Color">p5.Color</a> 개체, 색상 배열
  구성, 또는 

  CSS 색상 문자열.</p>

  <p>채도는 HSB와 HSL에서 서로 다르게 조정됩니다. 기본적으로,
  <code>saturation()</code>

  색상의 HSL 채도를 0부터 100까지의 범위로 반환합니다. 만약

  <a href="/reference/colorMode">colorMode()</a> 가 HSB 또는 HSL로 설정되어 있다면,


  주어진 모드에서의 채도 값을 반환합니다.</p>

    function setup() {
      createCanvas(100, 100);

      background(50);

      // Use HSB color.
      colorMode(HSB);

      // Create a p5.Color object.
      let c = color(0, 50, 100);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // Set 'satValue' to 50.
      let satValue = saturation(c);

      // Draw the right rectangle.
      fill(satValue);
      rect(50, 15, 35, 70);

      describe('Two rectangles. The left one is salmon pink and the right one is dark gray.');
