description: >
  <p>한 지점에서 한 방향으로 빛을 내는 조명을 생성합니다.</p>

  <p>집중 광원(spot light)은 한 방향으로 빛을 내는 손전등과 같으며, 빛의 원뿔을 생성합니다. 원뿔의 모양은 각도와 집중도 매개변수를 사용하여 조절할 수 있습니다. 한 번에 최대 5개의 집중 광원이 활성화될 수 있습니다.</p>

  <p><code>spotLight()</code>을 호출하는 방법은 총 8가지이며, 이를 통해 빛의 색상, 위치, 방향을 설정할 수 있습니다. 예를 들어, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0)</code> 은 원점 <code>(0, 0, 0)</code>에서 오른쪽 <code>(1, 0,0)</code>을 향하는 빨간색 <code>(255, 0, 0)</code> 빛을 생성합니다.</p>

  <p><code>angle</code> 매개변수는 선택적입니다. 이는 빛 원뿔의 반지름을 설정합니다. 예를 들어, 
  <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0, PI / 16)</code>은 원점 <code>(0, 0, 0)</code>에서 오른쪽 <code>(1, 0, 0)</code>을 향하며 <code>PI / 16</code> 라디안인 빨간색 <code>(255, 0, 0)</code> 빛을 생성합니다. 기본적으로 <code>angle</code>은 <code>PI / 3</code> 라디안입니다.</p>

  <p><code>concentration</code> 매개변수는 선택적입니다. 이는 빛을 원뿔의 중심에 집중시킵니다. 예를 들어, <code>spotLight(255, 0, 0, 0, 0, 0, 1, 0, 0, PI / 16, 50)</code>는 원점 <code>(0, 0, 0)</code>에서 오른쪽 <code>(1, 0, 0)</code>을 향하며 각도가 <code>PI / 16</code> 라디안이고 집중도가 50인 빨간색 <code>(255, 0, 0)</code> 빛을 생성합니다. 기본적으로 <code>concentration</code>는 100입니다.</p>
