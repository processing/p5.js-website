description: >
  <p>원뿔을 그립니다.</p>

  <p>원뿔은 삼각형 면들이 평평한 밑면과 하나의 꼭지점을 연결하는 3D 모양입니다. 

  삼각형 면의 수가 적은 원뿔은 피라미드처럼 보이며, 

  많은 삼각형 면으로 이루어진 원뿔은 부드러운 표면을 가집니다.</p>

  <p>첫 번째 매개변수 <code>radius</code> 는 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>cone(20)</code> 와 같다면, 원뿔의 밑면 반경이 20으로 설정됩니다.
  기본적으로, <code>radius</code> 는

  50입니다.</p>

  <p>두 번째 매개변수 <code>height</code> 도 선택 사항입니다.
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>cone(20, 30)</code> 와 같다면, 원뿔의 높이가 설정됩니다. 
  기본적으로, <code>height</code> 는

  원뿔의 <code>radius</code>값과 동일합니다.</p>

  <p>세 번째 매개변수 <code>detailX</code> 도 선택 사항입니다.
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>cone(20, 30, 5)</code> 와 같다면, 원뿔의 밑면을 구성하는 모서리의 수가 설정됩니다. 
  더 많은 모서리를 사용할수록 밑면이 원처럼 보입니다.

  기본적으로,

  <code>detailX</code> 는 24입니다.</p>

  <p>네 번째 매개변수 <code>detailY</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>cone(20, 30, 5, 7)</code> 와 같다면 
  밑면과 꼭지점을 연결하는 y축을 따라 삼각형 분할면 수가 설정됩니다.

  모든 3D 도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로,
  <code>detailY</code> 는 1입니다.</p>

  <p>다섯 번째 매개변수 <code>cap</code> 도 선택 사항입니다.
  <code>false</code>가 전달되면, 

  예를 들어 <code>cone(20, 30, 5, 7, false)</code> 와 같다면, 원뿔의 밑면이 그려지지 않습니다. 
  기본적으로,

  <code>cap</code> 는 <code>true</code> 입니다.</p>

  <p>참고: <code>cone()</code> 는 WebGL 모드에서만 사용할 수 있습니다.</p>
