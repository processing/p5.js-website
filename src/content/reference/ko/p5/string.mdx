description: >
  <p>일련의 텍스트 문자들입니다.</p>

  <p><code>String</code> 데이터 유형은 텍스트 작업에 유용합니다. 예를 들어, 문자열(string)은 다음과 같이 환영 메시지를 포함할 수 있습니다:</p>

  <code>

  // Use a string literal.

  text('Hello!', 10, 10);

  </code>


  <code>

  // Create a string variable.

  let message = 'Hello!';


  <p>// Use the string variable.

  text(message, 10, 10);

  </p></code>

  <p>문자열을 생성하는 가장 일반적인 방법은 따옴표를 사용하는 것입니다:</p>

  <code>

  text("hi", 50, 50);

  </code>


  <code>

  text('hi', 50, 50);

  </code>


  <code>

  text(`hi`, 50, 50);

  </code>


  <p><code>"hi"</code>, <code>'hi'</code>, <code>hi</code> 는 모두 문자열 리터럴(string literal)입니다. 
  
  "리터럴"이라는 것은 <code>text('hi', 50, 50)</code>에서와 같이 실제로 문자 그대로의 값을 작성했다는 것을 뜻합니다. 
  
  반면, <code>text(message, 50, 50)</code>는 변수 <code>message</code>를 사용하므로 문자열 리터럴이 아닙니다. 
  
  단일 따옴표 <code>''</code>와 이중 따옴표 <code>""</code>는 같은 의미를 가집니다. 
  
  문자열 안에 특정 종류의 따옴표가 포함되어 있을 때, 다른 종류의 따옴표를 사용할 수 있어서 편리합니다:</p>

  <code>

  text("What's up?", 50, 50);

  </code>


  <code>

  text('Air quotes make you look "cool."', 50, 50);

  </code>


  <p>역따옴표 ```` 는 템플릿 리터럴을 생성합니다. 리터럴은 여러 용도로 사용됩니다. 예를 들어, 필요에 따라 단일 따옴표와 이중 따옴표를 모두 포함할 수 있습니다:</p>

  <code>

  text(`"Don't you forget about me"`,  10, 10); 

  </code>


  <p>템플릿 리터럴은 변수에서 문자열을 생성할 때 유용합니다:</p>

  <code>

  let size = random(10, 20);

  circle(50, 50, size);


  <p>text(<code>The circle's diameter is ${size} pixels.</code>,  10, 10);

  </p>

  </code>

  <p><code>size</code> 변수의 값은 문자열이 생성될 때 <code>${size}</code>를 대체합니다. <code>${}</code>는 어떤 값이든지 받을 수 있는 자리 표시자입니다. 
  
  그것은 <code>${round(PI, 3)}</code>처럼 표현식을 사용할 수 있다는 것을 의미합니다. 다음은 모두 유효한 템플릿 리터럴입니다:</p>

  <code>

  text(`π is about ${round(PI, 2)} pixels.`,  10, 10);

  text(`It's ${mouseX < width / 2} that I'm on the left half of the canvas.`, 
  10, 30);

  </code>


  <p>템플릿 리터럴은 여러 변수를 포함할 수 있습니다:</p>

  <code>

  let x = random(0, 100);

  let y = random(0, 100);

  let size = random(10, 20);

  circle(x, y, size);


  <p>text(<code>The circle at (${x}, ${y}) has a diameter of ${size}
  pixels.</code>,  10, 10);

  </p></code>

  <p>템플릿 리터럴은 또한 다음과 같이 다중 줄 텍스트를 생성하는 데 도움이 됩니다:</p>

  <code>

  let poem = `My sketch doesn't run;

  it waits for me patiently

  while bugs point the way.`;


  <p>text(poem, 10, 10);

  </p></code>
