description: >
  <p><a href="/reference/p5/text">text()</a>를 호출할 때 텍스트가 정렬되는 방식을 설정합니다.</p>
  
  <p>기본적으로 <code>text('hi', 10, 20)</code>을 호출하면
  
  텍스트의 경계 상자(bounding box)의 왼쪽 하단 모서리가 (10, 20)에 위치합니다.</p>
  
  <p>첫 번째 매개변수인 <code>horizAlign</code>은 <a href="/reference/p5/text">text()</a>가
  
  x 좌표를 해석하는 방식을 변경합니다.
  
  기본적으로 x 좌표는 경계 상자의 왼쪽 모서리로 설정합니다.
  
  <code>textAlign()</code>는 <code>horizAlign</code>에 대해 다음 값을 사용합니다:
  
  <code>LEFT</code>, <code>CENTER</code>, <code>RIGHT</code>.</p>
  
  <p>두 번째 매개변수인 <code>vertAlign</code>은 선택 사항입니다.
  
  이는 <a href="/reference/p5/text">text()</a>가 y 좌표를 해석하는 방식을 변경합니다.
  
  기본적으로 y 좌표는 경계 상자의 아래쪽 모서리로 설정합니다.
  
  <code>textAlign()</code>는 <code>vertAlign</code>에 대해 다음 값을 사용합니다:
  
  <code>TOP</code>, <code>BOTTOM</code>, <code>CENTER</code>, <code>BASELINE</code>.</p>

    function setup() {
      createCanvas(100, 100);

      // Draw a vertical line.

      // Top line.
      textSize(16);
      textAlign(RIGHT);
      text('ABCD', 50, 30);

      textAlign(CENTER);
      text('EFGH', 50, 50);

      // Bottom line.
      textAlign(LEFT);

      textAlign(CENTER, TOP);
      text('TOP', 50, 12);

      textAlign(CENTER, CENTER);

    }
