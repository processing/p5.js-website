description: |2-

  <p> 드롭다운 메뉴 <code><select></select></code> 요소를 생성합니다.</p>


  <p>매개변수는 선택적(optional)입니다. <code>let mySelect = createSelect(true)</code>처럼 <code>true</code>가 전달된다면, 드롭다운은 여러 항목을 선택하는 것이 가능해집니다. 만약 <code>let mySelect = createSelect(otherSelect)</code>과 같이 기존에 있던 <code><select></select></code> 요소가 전달된다면, 기존 요소는 새로운 <a
  href="/reference/p5/p5.Element">p5.Element</a> 객체로 감싸집니다.</p>


  <p>드롭다운은 옵션을 관리하는 몇 가지 메서드를 통해 <a href="/reference/p5/p5.Element">p5.Element</a>
  클래스를 확장합니다:</p>


  <ul>


  <li><code>mySelect.option(name, [value])</code>은 메뉴에 옵션을 추가합니다.
  첫 번째 매개변수 <code>name</code>은 옵션의 이름과 값을 설정하는 문자열입니다. 두 번째 매개변수 <code>value</code>는 선택적(optional)입니다. 이 변수가 전달된다면, <code>name</code> 옵션의 값을 설정합니다. 만약 <code>name</code> 옵션이 이미 존재한다면, 해당 옵션의 값이
   <code>value</code>로 덮어씌워집니다.</li>


  <li><code>mySelect.value()</code>는 현재 선택된 옵션의 값을 반환합니다.</li>


  <li><code>mySelect.selected()</code>는 현재 선탣괸 옵션을 반환합니다.</li>


  <li><code>mySelect.selected(option)</code>은 전달된 옵션을 기본값으로 선택합니다.</li>


  <li><code>mySelect.disable()</code>은 모든 드롭다운 요소를 비활성화 상태로 표시합니다.</li>


  <li><code>mySelect.disable(option)</code>은 전달된 옵션을 비활성화 상태로 표시합니다.</li>


  <li><code>mySelect.enable()</code>은 모든 드롭다운 요소를 활성화 상태로 표시합니다.</li>


  <li><code>mySelect.enable(option)</code>은 전달된 옵션을 활성화 상태로 표시합니다.</li>


  </ul>

    // Hold CTRL to select multiple options on Windows and Linux.
    // Hold CMD to select multiple options on macOS.
    let mySelect;

    function setup() {
      createCanvas(100, 100);

      // Create a dropdown and allow multiple selections.
      // Place it beneath the canvas.
      mySelect = createSelect(true);
      mySelect.position(0, 100);

      // Add color options.
      mySelect.option('red');
      mySelect.option('green');
      mySelect.option('blue');
      mySelect.option('yellow');

      describe('A gray square with a dropdown menu beneath it. Colorful circles appear when their color is selected.');
    }

    function draw() {
      background(200);

      // Use the selected value(s) to draw circles.
      let colors = mySelect.selected();
      for (let i = 0; i < colors.length; i += 1) {
        // Calculate the x-coordinate.
        let x = 10 + i * 20;

        // Access the color.
        let c = colors[i];

        // Draw the circle.
        fill(c);
        circle(x, 50, 20);
      }
    }
