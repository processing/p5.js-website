description: >
  <p>사용자 정의 도형에 꼭짓점을 추가하기 시작합니다.</p>

  <p> <code>beginShape()</code> 와 <a
  href="/reference/p5/endShape">endShape()</a> 함수는

  2D 또는 3D에서 사용자 정의 도형을 만드는 것을 가능하게 합니다.
  <code>beginShape()</code> 는 사용자 정의 도형에 점을 추가하기 시작하며,

  <a href="/reference/p5/endShape">endShape()</a> 
  는 점을 추가하는 것을 

  중단합니다.</p>

  <p>매개변수 <code>kind</code> 는 만들 도형의 유형을 설정합니다.
  기본적으로,

  불규칙적인 다각형을 그릴 수 있습니다. 'kind'에 사용할 수 있는 모드는 다음과 같습니다.:</p>

  <ul>

  <li><code>POINTS</code> 로 일련의 점을 그립니다.</li>

  <li><code>LINES</code> 로 연결되지 않은 선분을 그립니다.</li>

  <li><code>TRIANGLES</code> 로 분리된 삼각형을 그립니다.</li>

  <li><code>TRIANGLE_FAN</code> 은 첫 번째 점을 공유하는 연결된 삼각형 시리즈를 
  부채꼴 모양으로 그립니다.</li>

  <li><code>TRIANGLE_STRIP</code> 은 연결된 삼각형을 
  줄 형태로 그립니다.</li>

  <li><code>QUADS</code> 를 사용하면 별개의 사분면(quads)을
  그립니다.</li>

  <li><code>QUAD_STRIP</code> 를 사용하면 인접한 변을 사용하여 
  그 다음 사각형을 구성하는 사각형 줄을 그립니다.</li>

  <li><code>TESS</code> 를 사용하면 뚜렷한 테셀레이션(tessellation)을 통해 곡선을 채웁니다
  (WebGL에서만 가능).</li>

  </ul>

  <p><code>beginShape()</code> 를 호출한 후에는 

  <a href="/reference/p5/vertex">vertex()</a>,

  <a href="/reference/p5/bezierVertex">bezierVertex()</a>,

  <a href="/reference/p5/quadraticVertex">quadraticVertex()</a> 그리고

  <a href="/reference/p5/curveVertex">curveVertex()</a>를 호출하여 도형을 구성할 수 있습니다. 

  <a href="/reference/p5/endShape">endShape()</a>를 호출하면 
  도형에 점을 더하는 것을 중단합니다.

  각 도형은 현재의 색상으로 윤곽선과 도형 색상이

  채워집니다.</p>

  <code>beginShape()</code> 와 <a href="/reference/p5/endShape">endShape()</a>사이에서는
  
  <p><a href="/reference/p5/translate">translate()</a>,

  <a href="/reference/p5/rotate">rotate()</a>, 

  <a href="/reference/p5/scale">scale()</a> 과 같은 변형기능이 작동하지 않습니다. 

  또한

  <a href="/reference/p5/ellipse">ellipse()</a> 와

  <a href="/reference/p5/rect">rect()</a> 은  <code>beginShape()</code> 

  와 <a href="/reference/p5/endShape">endShape()</a> 사이에서 사용할 수 없습니다.</p>
