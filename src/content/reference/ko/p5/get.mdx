  <p>캔버스에서 픽셀 또는 픽셀 영역을 가져옵니다.</p>
  
  <p><code>get()</code>은 사용하기 쉽지만 <a href="/reference/p5/pixels">pixels</a>보다 속도가 느립니다.
  
  많은 픽셀 값을 읽을 때는 <a href="/reference/p5/pixels">pixels</a>를 사용하세요.</p>
  
  <p>매개변수가 없는 <code>get()</code>은 전체 캔버스를 반환합니다.</p>
  
  <p>두 개의 매개변수가 있는 <code>get()</code>은 그것들을 좌표로 해석합니다.
  
  주어진 지점의 픽셀의 <code>[R, G, B, A]</code> 값을 포함하는 배열을 반환합니다.</p>
  
  <p>네 개의 매개변수가 있는 <code>get()</code>의 경우 그것들을 좌표와 면적으로 해석합니다.
  
  이 때 캔버스의 일부를 <a href="/reference/p5/p5.Image">p5.Image</a> 객체로 반환합니다.
  
  처음 두 매개변수는 캔버스 일부의 좌상단 좌표이고, 마지막 두 매개변수는 너비와 높이입니다.</p>
  
  <p><a href="/reference/p5.Image/get">p5.Image.get()</a>을 사용하여
  
  <a href="/reference/p5/p5.Image">p5.Image</a> 객체에 직접 작업할 수 있습니다.</p>
