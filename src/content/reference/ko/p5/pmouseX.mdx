description: >
  <p>마우스의 이전 수평 위치를 추적하는 <code>Number</code> 시스템 변수입니다.</p>

  <p>2D 모드에서, <code>pmouseX</code>는 캔버스의 왼쪽 상단 모서리를 기준으로 마우스의 이전 위치를 추적합니다.
  그 값은 이전 프레임의 <a href="/reference/p5/mouseX">mouseX</a>입니다.
  예를 들어, 마지막 프레임에서 마우스가 캔버스의 왼쪽 가장자리에서 50픽셀 떨어져 있었다면, <code>pmouseX</code>는 50입니다.</p>

  <p>WebGL 모드에서, <code>pmouseX</code>는 캔버스의 중심을 기준으로 마우스의 이전 위치를 추적합니다.
  예를 들어, 마지막 프레임에서 마우스가 캔버스의 중심에서 오른쪽으로 50픽셀 떨어져 있었다면, <code>pmouseX</code>는 50입니다.</p>

  <p>만약 마우스 대신 터치가 사용된다면, <code>pmouseX</code>는 마지막 터치 지점의 x 좌표를 저장합니다.</p>
  
  <p>참고: <code>pmouseX</code>는 각 터치 이벤트가 시작되는 시점에 현재 <a
  href="/reference/p5/mouseX">mouseX</a> 값으로 재설정됩니다.</p>

      // Adjust coordinates for WebGL mode.

      let pmx = pmouseX - 50;
      let pmy = pmouseY - 50;
      let mx = mouseX - 50;
      let my = mouseY - 50;

      line(pmx, pmy, mx, my);
