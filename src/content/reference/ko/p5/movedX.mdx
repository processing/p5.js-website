description: >
  <p>마우스의 수평 이동을 추적하는 <code>Number</code> 시스템 변수입니다.</p>

  <p><code>movedX</code>는 프레임 사이에 마우스가 왼쪽이나 오른쪽으로 몇 픽셀 이동하는지를 추적합니다.
  프레임 사이에 마우스가 왼쪽으로 이동하면 <code>movedX</code>는 음수 값을 가지고, 오른쪽으로 이동하면 양수 값을 가집니다.
  <code>movedX</code>는 <code>mouseX - pmouseX</code>로 계산할 수 있습니다.</p>
  
  <p>참고: <code>movedX</code>는 <a href="/reference/p5/requestPointerLock">requestPointerLock()</a>이 활성화되어 있을 때에도 계속 업데이트됩니다.</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square. The text ">>" appears when the user moves the mouse to the right. The text "<<" appears when the user moves the mouse to the left.'
      );
    }

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display >> when movedX is positive and
      // << when it's negative.
      if (movedX > 0) {
        text('>>', 50, 50);
      } else if (movedX < 0) {
        text('<<', 50, 50);
      }
    }
