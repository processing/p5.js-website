description: >
  <p>원기둥을 그립니다.</p>

  <p>원기둥은 삼각형 면들이 평평한 바닥과 평평한 꼭대기를 연결하는 3D 도형입니다.

  삼각형 면의 수가 적은 원기둥는 박스처럼 보이며, 

  삼각형 면의 수가 많은 원기둥은 부드러운 표면을 가집니다.</p>

  <p>첫 번째 매개변수 <code>radius</code> 는 선택 사항입니다. 
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>cylinder(20)</code> 와 같이 전달되면 원기둥 밑면의 반경이 설정됩니다.
  기본적으로

  <code>radius</code> 는 50입니다.</p>

  <p>두 번째 매개변수 <code>height</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>cylinder(20, 30)</code> 와 같다면, 원기둥의 높이가 설정됩니다. 
  기본적으로

  <code>height</code> 는 원기둥의 <code>radius</code> 값과 동일합니다.</p>

  <p>세 번째 매개변수 <code>detailX</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>cylinder(20, 30, 5)</code> 와 같다면, 원기둥의 꼭대기와 바닥을 
  구성하는 모서리의 수가 설정됩니다. 

  더 많은 모서리를 사용할수록 꼭대기와 바닥이 원처럼 보입니다. 

  기본적으로 <code>detailX</code> 는 24입니다.</p>

  <p>네 번째 매개변수 <code>detailY</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>cylinder(20, 30, 5, 2)</code> 와 같다면, 
  원기둥의 꼭대기와 바닥 사이의 y축을 따라 삼각형 분할 수가 설정됩니다.

  모든 3D도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로,

  <code>detailY</code> 는 1입니다.</p>

  <p>다섯 번째 매개변수, <code>bottomCap</code> 도 선택 사항입니다. 
  <code>false</code> 가 전달되면,

  예를 들어 (20, 30, 5, 2, false)</code> 와 같다면,
  원기둥의 바닥면은 그려지지 않습니다.

  기본적으로, <code>bottomCap</code> 는 <code>true</code> 입니다.</p>

  <p>여섯 번째 매개변수 <code>topCap</code> 도 선택 사항입니다.
  <code>false</code> 가 전달되면,

  예를 들어 <code>cylinder(20, 30, 5, 2, false, false)</code> 와 같이 전달되면,
  원기둥의 윗면이 그려지지 않습니다. 

  기본적으로, <code>topCap</code> 는 <code>true</code> 입니다.</p>

  <p>참고: <code>cylinder()</code> 는 WebGL 모드에서만 사용할 수 있습니다.</p>
