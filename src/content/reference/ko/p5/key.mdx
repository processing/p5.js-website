description: >
  <p>마지막으로 입력된 키의 값을 가지는 <code>String</code> 시스템 변수입니다.</p>

  <p>key 변수는 
  <a href="https://en.wikipedia.org/wiki/ASCII#Printable_characters"
  target="_blank">ASCII</a> 키가 입력되었는지 확인하는데 유용합니다.

  예를 들어, <code>key === "a"</code> 식은 <code>a</code> 키가 입력되었으면 <code>true</code>로 평가되고,
  그렇지 않으면 <code>false</code>로 평가됩니다.

  <code>key</code>는 <code>LEFT_ARROW</code>나 <code>ENTER</code>와 같은 특수 키에 대해서는 업데이트되지 않습니다.

  특수 키에 대해서는 대신 keyCode를 사용하세요.

  여러 키가 동시에 눌려있는지 확인하려면 <a href="/reference/p5/keyIsDown">keyIsDown()</a> 함수를 사용해야 합니다.</p>
