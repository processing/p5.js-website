description: >
  <p>코드 블록을 반복하는 방법입니다.</p>

  <p><code>while</code> 반복문은 조건이 <code>true</code>인 동안 문장을 반복하는 데 유용합니다. 
  
  이는 반복되는 <code>if</code> 문과 비슷합니다. 예를 들어, <code>while</code> 반복문는 "여러 줄 그리기"라는 아이디어를 다음과 같이 쉽게 표현할 수 있습니다:</p>

  <code>

  // Declare a variable to keep track of iteration.

  let x = 10;


  <p>// Repeat as long as x < 100

  while (x < 100) {
    line(x, 25, x, 75);</p>
  <p>  // Increment by 20.
    x += 20;
  }

  </p></code>

  <p>반복문의 헤더는 키워드 <code>while</code>로 시작합니다. 반복문은 보통 반복하면서 증가하거나 감소합니다.
  
  괄호 안의 문장 <code>x < 100</code>은 반복문이 각 반복 때마다 확인하는 조건입니다.

  조건이 <code>true</code>이면 중괄호 <code>{}</code> 사이의 코드, 즉 반복문의 본문이 실행됩니다.

  조건이 <code>false</code>이면 본문은 건너뛰고 반복문은 중단됩니다.
  
  실수로 무한 반복을 만드는 경우가 흔합니다. 예를 들어, 다음 반복문은 증가하지 않기 때문에 결코 반복을 멈추지 않습니다:</p>

  <code>

  // Declare a variable to keep track of iteration.

  let x = 10;


  <p>// Repeat as long as x < 100

  while (x < 100) {
    line(x, 25, x, 75);
  }</p>

  <p>// This should be in the loop's body!

  x += 20;

  </p></code>

  <p>문장 <code>x += 20</code>은 반복문 본문 이후에 나타나므로, 
  
  <code>x</code>는 항상 <code>10</code>에 머물러 있으며, 이는 항상 <code>100</code>보다 작습니다.</p>

  <p>반복 횟수를 미리 알 수 없을 때 <code>while</code> 반복문이 유용합니다. 
  
  예를 들어, 다음과 같이 무작위 간격으로 동심원을 그릴 수 있습니다:</p>

  <code>

  let d = 100;

  let minSize = 5;


  <p>while (d > minSize) {
    circle(50, 50, d);
    d -= random(10);
  }

  </p></code>
