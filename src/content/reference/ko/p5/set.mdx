description: >
  <p>픽셀의 색상을 설정하거나 이미지를 캔버스에 그립니다.</p>
  
  <p><code>set()</code>을 사용하기 쉽지만 <a href="/reference/p5/pixels">pixels</a>보다는 느립니다.
  
  많은 픽셀 값을 설정하려면 <a href="/reference/p5/pixels">pixels</a>를 사용하세요.</p>
  
  <p><code>set()</code>은 처음 두 매개변수를 x 좌표와 y 좌표로 해석합니다.
  
  마지막 매개변수는 그레이스케일 값, <code>[R, G, B, A]</code> 픽셀 배열,
  
  <a href="/reference/p5/p5.Color">p5.Color</a> 객체 또는 <a href="/reference/p5/p5.Image">p5.Image</a> 객체로 해석합니다.
  
  이미지가 전달되면 처음 두 매개변수는 현재 <a href="/reference/p5/imageMode">imageMode()</a>와 관계없이
  
  이미지 좌상단의 좌표를 설정합니다.</p>
  
  <p>변경 사항을 표시하려면 <code>set()</code>을 사용한 후에
  
  <a href="/reference/p5/updatePixels">updatePixels()</a>를 호출해야 합니다.</p>

    function setup() {
      createCanvas(100, 100);

      background(255);

      // Draw a horizontal color gradient.
      for (let x = 0; x < 100; x += 1) {
        for (let y = 0; y < 100; y += 1) {
          // Calculate the grayscale value.
          let c = map(x, 0, 100, 0, 255);

          // Set the pixel using the grayscale value.
          set(x, y, c);
        }
      }

      // Update the canvas.
      updatePixels();

      describe('A horiztonal color gradient from black to white.');
    }
    </code>
