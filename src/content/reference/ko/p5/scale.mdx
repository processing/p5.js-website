description: >
  <p>좌표계를 크기를 조절합니다.</p>
  
  <p>기본적으로 도형들은 원래의 크기대로 그려집니다.
  
  50픽셀 너비의 직사각형은 100픽셀 폭의 캔버스에서 절반 정도의 너비를 차지하도록 보입니다.
  
  <code>scale()</code> 함수는 좌표계를 확대하거나 축소해서 도형이 서로 다른 크기로 보이도록 할 수 있습니다.
  
  크기 조절 요인을 설정하는 매개변수를 사용하여 <code>scale()</code> 함수를 호출하는 두 가지 방법이 있습니다.</p>
  
  <p><code>scale()</code>을 호출하는 첫 번째 방법은 숫자를 사용하여 각 축의 크기를 설정하는 것입니다.
  
  첫 번째 매개변수 <code>s</code>는 각 축의 크기를 설정합니다.
  
  예를 들어, <code>scale(2)</code>를 호출하면 x-, y-, z-축이 각각 2배로 늘어납니다.
  
  다음 두 매개변수 <code>y</code>와 <code>z</code>는 선택 사항입니다.
  
  이들은 y-축과 z-축의 크기를 설정합니다.</p>
  
  예를 들어, <code>scale(2, 0.5, 1)</code>를 호출하면 x-축을 2배로 늘리고, y-축을 0.5배로 줄이고, z-축은 변경되지 않습니다.</p>
  
  <p><code>scale()</code>을 호출하는 두 번째 방법은 <a href="/reference/p5/p5.Vector">p5.Vector</a> 객체를 사용하여 크기 조절 요인을 설정하는 것입니다.
  
  예를 들어, <code>scale(myVector)</code>를 호출하면 <code>myVector</code>의 x-, y-, z- 구성 요소를 사용하여 x-, y-, z-축을 따라 크기를 조절할 양을 설정합니다.
  
  이렇게 하면 <code>scale(myVector.x, myVector.y, myVector.z)</code>를 호출하는 것과 같습니다.</p>
  
  <p>기본적으로 변환은 누적됩니다.
  
  예를 들어, <code>scale(1)</code>을 두 번 호출하면 <code>scale(2)</code>를 한 번 호출하는 것과 동일한 효과가 있습니다.
  
  <a href="/reference/p5/push">push()</a> 및 <a href="/reference/p5/pop">pop()</a> 함수를 사용하여 서로 다른 그룹에서 변환을 분리할 수 있습니다.</p>
  
  <p>참고: 변환은 draw 루프의 시작부에서 재설정됩니다.
  
  <code>scale(2)</code>를 <a href="/reference/p5/draw">draw()</a> 함수 내에서 호출하면 도형이 계속해서 커지지 않습니다.</p>
