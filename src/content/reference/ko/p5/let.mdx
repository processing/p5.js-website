description: >
  <p>새로운 변수를 선언합니다.</p>

  <p>변수는 값의 컨테이너입니다. 예를 들어, 변수는 생물의 x좌표를 <code>Number</code> 형태로 또는 그 이름을 <code>String</code> 형태로 포함할 수 있습니다. 변수는 다음과 같이 재할당함으로써 값이 변경될 수 있습니다:</p>

  <code>

  // Declare the variable x and assign it the value 10.

  let x = 10;


  <p>// Reassign x to 50.

  x = 50;

  </p></code>

  <p>변수는 블록 범위를 가집니다. 중괄호 <code>{}</code> 사이에 변수가 선언되면, 그 변수는 그 중괄호로 정의된 블록 내에서만 존재합니다. 예를 들어, 다음 코드는 <code>x</code>가 <code>setup()</code> 함수의 블록 내에서 선언되었기 때문에 <code>ReferenceError</code>를 발생시킬 것입니다:</p>

  <code>

  function setup() {
    createCanvas(100, 100);

  <p>  let x = 50;

  }</p>

  <p>function draw() {
    background(200);</p>
  <p>  // x was declared in setup(), so it can't be referenced here.
    circle(x, 50, 20);
  }

  </p></code>

  <p>모든 중괄호 <code>{}</code> 바깥에 선언된 변수는 전역 범위에 있습니다. 전역 범위에 있는 변수는 스케치의 어느 곳에서나 사용되고 변경될 수 있습니다:</p>

  <code>

  let x = 50;


  <p>function setup() {
    createCanvas(100, 100);
  }</p>

  <p>function draw() {
    background(200);</p>
  <p>  // Change the value of x.
    x += 10;</p>
  <p>  circle(x, 50, 20);

  }

  </p></code>
