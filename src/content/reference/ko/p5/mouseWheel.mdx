description: >
  <p>마우스 휠이 움직일 때 한 번 호출되는 함수입니다.</p>

  <p><code>mouseWheel()</code> 함수를 선언하면 사용자가 마우스 휠을 움직일 때 자동으로 실행되는 코드 블록을 설정합니다:</p>
  
  <code>

  function mouseWheel() {
    // 실행할 코드.
  }

  </code>

  <p><a href="/reference/p5/mouseX">mouseX</a>나 <a href="/reference/p5/mouseY">mouseY</a>와 같은 마우스 시스템 변수는
  p5.js에 의해 <code>mouseWheel()</code>가 호출될 때 가장 최근 값으로 업데이트됩니다:</p>

  <code>

  function mouseWheel() {
    if (mouseX < 50) {
      // 마우스가 왼쪽에 있을 때 실행할 코드.
    }

  <p>  if (mouseY > 50) {
      // 마우스가 아래쪽에 있을 때 실행할 코드.
    }
  }

  </p></code>

  <p><code>event</code>는 선택적(optional) 매개변수입니다.
  <code>mouseWheel()</code>는 항상
  <a href="https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent" target="_blank">MouseEvent</a> 객체를 전달받으며,
  이 객체는 마우스 스크롤 이벤트를 설명하는 속성들을 가지고 있습니다:</p>
  
  <code>

  function mouseWheel(event) {
    // 이벤트를 사용하여 실행할 코드.
    console.log(event);
  }

  </code>

  <p><code>event</code> 객체에는 유저가 스크롤한 거리를 나타내는 <code>Number</code>, 즉 <code>delta</code>가 포함되어 있습니다.
  예를 들어, 유저가 위로 스크롤할 때 <code>event.delta</code>가 5일 수 있습니다.
  <code>event.delta</code>는 유저가 위로 스크롤할 때 양수이며, 아래로 스크롤할 때 음수입니다.
  macOS에서는 "자연스러운" 스크롤이 활성화되어 있을 때 이 부호가 반대입니다.</p>

  <p>브라우저는 다양한 마우스 이벤트에 대해 기본적으로 설정된 동작을 가지고 있을 수 있습니다.
  예를 들어, 일부 브라우저는 마우스 버튼을 누르고 마우스를 움직일 때 텍스트를 강조합니다.
  이 이벤트에 대한 기본 동작을 방지하려면, 함수 끝에 <code>return false;</code>를 추가하세요.</p>
  
  <p>참고: Safari에서는 <code>mouseWheel()</code>가 예상대로 작동하려면 함수 끝에 <code>return false;</code>를 추가해야 합니다.</p>
