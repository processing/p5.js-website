description: >
  <p>브라우저 내에서 마우스의 이전 수평 위치를 추적하는 <code>Number</code> 변수입니다.</p>
  
  <p><code>pwinMouseX</code>는 브라우저 창의 왼쪽 상단을 기준으로 마우스의 위치를 추적합니다.
  이 값은 이전 프레임의 <a href="/reference/p5/winMouseX">winMouseX</a> 값입니다.
  예를 들어, 마우스가 이전 프레임에서 브라우저 왼쪽 가장자리에서 50픽셀 떨어져 있다면, <code>pwinMouseX</code>는 50이 됩니다.</p>

  <p>터치스크린 장치에서는, <code>pwinMouseX</code>는 가장 최근 터치 지점의 x 좌표를 저장합니다.
  <code>pwinMouseX</code>는 각 터치 이벤트가 시작되는 시점에 현재 <a href="/reference/p5/winMouseX">winMouseX</a> 값으로 재설정됩니다.</p>
  
  <p>참고: 캔버스 내에서 마우스의 이전 x 좌표를 추적하려면 <a href="/reference/ko/p5/pmouseX">pmouseX</a>를 사용하세요.</p>
