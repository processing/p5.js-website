  <p> vertex와 fragment 셰이더를 로드하여 <a href="/reference/p5/p5.Shader">p5.Shader</a> 객체를 생성합니다.</p>

  <p>셰이더는 그래픽 처리 장치(GPU)에서 실행되는 프로그램입니다. 많은 픽셀을 동시에 처리할 수 있어 많으 그래픽 작업에 빠르게 작동합니다. 셰이더는 <a href="https://developer.mozilla.org/en-US/docs/Games/Techniques/3D_on_the_web/GLSL_Shaders" target="_blank">이라는 언어로 작성되며 스케치의 나머지 코드와 함께 실행됩니다.</p>

  <p><a href="/reference/p5/p5.Shader">p5.Shader</a> 객체가 생성되면, <a href="/reference/p5/shader">shader()</a> 함수와 함께 사용할 수 있습니다. 예를 들어, <code>shader(myShader)</code>를 호출하면 됩니다. 셰이더 프로그램은 두 개의 파일, vertex와 fragment 셰이더로 구성됩니다. vertex 셰이더는 3D 기하학이 화면에 그려지는 위치에 영향을 주고, fragment 셰이더는 색상에 영향을 줍니다.
</p>

  <p><code>loadShader()</code> 는 <code>.vert</code> 파일과 <code>.frag</code> 파일에서 vertex와 fragment 셰이더를 로드합니다. 예를 들어, <code>loadShader('/assets/shader.vert', '/assets/shader.frag')</code>를 호출하면 필요한 두 셰이더를 로드하고 <a href="/reference/p5/p5.Shader">p5.Shader</a> 객체를 반환합니다.

세 번째 매개변수인 <code>successCallback</code>은 선택 사항입니다. 함수가 전달되면 셰이더가 로드된 후에 호출됩니다. 콜백 함수는 새로운 <a href="/reference/p5/p5.Shader">p5.Shader</a> 객체를 매개변수로 사용할 수 있습니다.
</p>

  <p>네 번째 매개변수인 <code>failureCallback</code>은 선택 사항입니다. 함수가 전달되면 셰이더 로드에 실패할 경우 호출됩니다. 콜백 함수는 이벤트 오류를 매개변수로 사용할 수 있습니다. 셰이더는 로드하는 데 시간이 걸릴 수 있습니다. <a href="/reference/p5/preload">preload()</a> 에서 <code>loadShader()</code>를 호출하면 <a href="/reference/p5/setup">setup()</a> 또는 <a href="/reference/p5/draw">draw()</a>에서 사용하기 전에 셰이더가 로드됩니다. </p>

  <p>참고: 셰이더는 WebGL 모드에서만 사용할 수 있습니다.</p>

    <div modernizr='webgl'>
    <code>
    // Note: A "uniform" is a global variable within a shader program.

    let mandelbrot;

    // Load the shader and create a p5.Shader object.
    function preload() {
      mandelbrot = loadShader('/assets/shader.vert', '/assets/shader.frag');
    }

    function setup() {
      createCanvas(100, 100, WEBGL);

      // Compile and apply the p5.Shader object.
      shader(mandelbrot);

      // Set the shader uniform p to an array.
      mandelbrot.setUniform('p', [-0.74364388703, 0.13182590421]);

      // Set the shader uniform r to the value 1.5.
      mandelbrot.setUniform('r', 1.5);

      // Add a quad as a display surface for the shader.
      quad(-1, -1, 1, -1, 1, 1, -1, 1);

      describe('A black fractal image on a magenta background.');
    }
    </code>
    </div>

      describe('A fractal image zooms in and out of focus.');

    function draw() {
      // Set the shader uniform r to a value that oscillates between 0 and 2.
      mandelbrot.setUniform('r', sin(frameCount * 0.01) + 1);

      quad(-1, -1, 1, -1, 1, 1, -1, 1);
    }
