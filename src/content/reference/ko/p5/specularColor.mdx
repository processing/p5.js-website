description: >
  <p>빛의 반사색을 설정합니다.</p>

  <p><code>specularColor()</code> 함수는 표면에서 선호하는 방향으로 반사되는 빛에 영향을 미칩니다. 이러한 빛에는 <a href="/reference/p5/directionalLight">directionalLight()</a>,

  <a href="/reference/p5/pointLight">pointLight()</a>,

  <a href="/reference/p5/spotLight">spotLight()</a>가 포함됩니다.
  
  이 함수는 <a href="/reference/p5/specularMaterial">specularMaterial()</a>로 스타일링된 
  
  <a href="/reference/p5/p5.Geometry">p5.Geometry</a> 객체에 하이라이트를 만드는 데 도움을 줍니다. 
  
  만약 기하학적 객체가 <a href="/reference/p5/specularMaterial">specularMaterial()</a>을 사용하지 않는다면,
  
  <code>specularColor()</code>는 효과가 없습니다.</p>

  <p>참고: <code>specularColor()</code> 는 <a href="/reference/p5/ambientLight">ambientLight()</a>와 <a href="/reference/p5/imageLight">imageLight()</a>처럼 모든 방향으로 반사되는 빛에는 영향을 미치지 않습니다.</p>

  <p><code>specularColor()</code> 를 호출하는 방법은 총 세 가지입니다. 이들 모두 선택적 매개변수를 사용하여 반사 하이라이트 색상을 설정합니다.</p>

  <p>첫 번째 방법은 선택적 매개변수 <code>gray</code>와 <code>alpha</code>를 가집니다. 0에서 255 사이의 회색조(gray) 값 및 투명도(alpha) 값을 <code>specularColor(50)</code> 또는 <code>specularColor(50, 80)</code>과 같이 전달하여 반사 하이라이트 색상을 설정할 수 있습니다.</p>

  <p>두 번째 방법은 <code>specularColor()</code>가 선택적 매개변수 <code>color</code>을 가집니다. <a href="/reference/p5/p5.Color">p5.Color</a> 객체, 색상값 배열, 또는 CSS 색상 문자열을 전달하여 반사 하이라이트 색상을 설정할 수 있습니다.</p>

  <p>세 번째 방법은 <code>specularColor()</code>가 선택적 매개변수 <code>v1</code>, <code>v2</code>, <code>v3</code>, <code>alpha</code>를 가집니다. <code>specularColor(255, 0, 0, 80)</code>과 같이 RGBA, HSBA, 또는 HSLA 값이 전달되어 반사 하이라이트 색상을 설정할 수 있습니다. 색상값은 현재 <a href="/reference/p5/colorMode">colorMode()</a>를 사용하여 해석됩니다.</p>
