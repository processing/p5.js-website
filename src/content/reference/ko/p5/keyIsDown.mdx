description: >
  <p>확인하고자 하는 키가 눌려있으면 <code>true</code>, 그렇지 않으면 <code>false</code>를 반환합니다.</p>
  
  <p><code>keyIsDown()</code> 함수는 여러 개의 다른 키 입력을 확인할 때 유용합니다.
  예를 들어, <code>keyIsDown()</code>은 <code>LEFT_ARROW</code>와 <code>UP_ARROW</code>가 둘 다 눌려 있는지 확인할 때 사용할 수 있습니다:</p>
  
  <code>

  if (keyIsDown(LEFT_ARROW) && keyIsDown(UP_ARROW)) {
    // 대각선으로 이동.
  }

  </code>

  <p><code>keyIsDown()</code>은 <code>keyIsDown(37)</code>나 <code>keyIsDown(LEFT_ARROW)</code>와 같이 
  <a href="/reference/p5/keyCode">keyCode</a> 값을 사용하여 키 입력을 확인할 수 있습니다. 
  키 코드는 <a href="https://keycode.info" target="_blank">keycode.info</a>와 같은 웹사이트에서 찾을 수 있습니다.</p>
