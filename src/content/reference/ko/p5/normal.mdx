description: >
  <p>사용자 지정 3D 도형의 꼭짓점/정점(vertex)에 대한 법선 벡터를 설정합니다.</p>

  <p><a href="/reference/p5/beginShape">beginShape()</a>
  와

  <a href="/reference/p5/endShape">endShape()</a> 로 만들어진

  3D 도형은 꼭짓점이라는 점의 집합으로 구성됩니다.

  <a href="/reference/p5/vertex">vertex()</a> 로 추가된 각 꼭짓점은 
  자신으로부터 멀리 향하는 법선 벡터를 갖습니다. 

  법선 벡터는 도형에서 빛이 반사되는 방식을 제어합니다.</p>

  <p><code>normal()</code> 은 서로 다른 매개변수를 사용하여,
  법선 벡터의 구성 요소를 정의하는 

  두 가지 방식으로 호출할 수 있습니다.</p>

  <p>첫 번째 호출 방법은 <code>normal()</code> has three parameters,
  <code>x</code>, <code>y</code>, <code>z</code> 세 가지 매개변수를 사용하는 것입니다.

  예를 들어, <code>normal(1, 2, 3)</code> 처럼 <code>Number</code> 가 전달되면, 
  법선 벡터의

  x, y, z 구성 요소를 설정합니다.</p>

  <p><code>normal()</code> 을 호출하는 두 번째 방법은 한 가지 매개변수
  <code>vector</code> 를 가지며, 

  <code>normal(myVector)</code>처럼 <a href="/reference/p5/p5.Vector">p5.Vector</a> 

  객체를 전달하면, 구성 요소들이 
  법선 벡터를 설정하는 데 사용됩니다.</p>

  <p><code>normal()</code> 는
  <a href="/reference/p5/vertex">vertex()</a>로 추가된 꼭짓점에 대한 정규 백터를 변경합니다.

  <code>normal()</code> 
  은

  <a href="/reference/p5/beginShape">beginShape()</a> 와

  <a href="/reference/p5/endShape">endShape()</a> 함수 사이에서 호출되어야 하며,

  <a href="/reference/p5/vertex">vertex()</a> 처럼

  <code>normal()</code> 이 다시 호출될 때까지 
  다음의 모든 꼭짓점에

  영향을 미칩니다:</p>

  <code>

  beginShape();


  <p>// Set the vertex normal.

  normal(-0.4, -0.4, 0.8);</p>

  <p>// Add a vertex.

  vertex(-30, -30, 0);</p>

  <p>// Set the vertex normal.

  normal(0, 0, 1);</p>

  <p>// Add vertices.

  vertex(30, -30, 0);

  vertex(30, 30, 0);</p>

  <p>// Set the vertex normal.

  normal(0.4, -0.4, 0.8);</p>

  <p>// Add a vertex.

  vertex(-30, 30, 0);</p>

  <p>endShape();

  </p></code>
