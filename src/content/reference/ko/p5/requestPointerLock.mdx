description: >
  <p>마우스 포인터를 현재 위치에 고정하고 보이지 않게 만듭니다.</p>

  <p><code>requestPointerLock()</code> 함수를 호출하면 마우스가 화면을 벗어나지 않고 계속 움직일 수 있습니다.
  <code>requestPointerLock()</code>를 호출하면
  <a href="/reference/p5/mouseX">mouseX</a>, <a href="/reference/p5/mouseY">mouseY</a>,
  <a href="/reference/p5/pmouseX">pmouseX</a>, <a href="/reference/p5/pmouseY">pmouseY</a>의 값이 고정됩니다.

  <a href="/reference/p5/movedX">movedX</a>와 <a href="/reference/p5/movedY">movedY</a>는 계속 업데이트되며,
  마지막 프레임이 그려진 이후 마우스가 이동한 거리를 얻을 수 있습니다.

  <a href="/reference/p5/exitPointerLock">exitPointerLock()</a> 함수를 호출하면
  마우스 시스템 변수가 다시 업데이트됩니다.</p>

  <p>참고: 대부분의 브라우저는 <code>requestPointerLock()</code>을 호출하기 전에 클릭과 같은 입력을 요구합니다.
  따라서 <a href="/reference/p5/doubleClicked">doubleClicked()</a>와 같은 이벤트 함수에서
  <code>requestPointerLock()</code>를 호출하는 것이 좋습니다.</p>

    let score = 0;

      textAlign(CENTER);
      textSize(16);

      // Display the score.
      text(`Score: ${score}`, 50, 50);
    }

    // Lock the pointer when the user double-clicks.
    function doubleClicked() {
      requestPointerLock();
    }
