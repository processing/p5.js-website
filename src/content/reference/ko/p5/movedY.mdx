  <p>마우스의 수직 이동을 추적하는 <code>Number</code> 시스템 변수입니다.</p>

  <p><code>movedY</code>는 프레임 사이에 마우스가 위나 아래로 몇 픽셀 이동하는지를 추적합니다.
  프레임 사이에 마우스가 위로 이동하면 <code>movedY</code>는 양수 값을 가지고, 아래로 이동하면 음수 값을 가집니다.
  <code>movedY</code>는 <code>mouseY - pmouseY</code>로 계산할 수 있습니다.</p>

  <p>참고: <code>movedY</code>는 <a href="/reference/p5/requestPointerLock">requestPointerLock()</a>이 활성화되어 있을 때에도 계속 업데이트됩니다.</p>

    function setup() {
      createCanvas(100, 100);

        'A gray square. The text "▲" appears when the user moves the mouse upward. The text "▼" appears when the user moves the mouse downward.'

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Display ▼ when movedY is positive and
      // ▲ when it's negative.
      if (movedY > 0) {
        text('▼', 50, 50);
      } else if (movedY < 0) {
        text('▲', 50, 50);
