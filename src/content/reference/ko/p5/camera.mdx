description: >
  <p>3D 스케치에서 현재 카메라의 위치와 방향을 설정합니다.</p>
  <p><code>camera()</code>는 객체를 다른 각도에서 볼 수 있게 해 줍니다. 이 함수는 9개의 매개 변수를 가지고 있으며, 모두 선택 사항입니다.</p>
  <p>처음 세 개의 매개 변수인 <code>x</code>, <code>y</code>, <code>z</code>는 카메라 위치의 좌표입니다. 예를 들어, <code>camera(0, 0, 0)</code>를 호출하면 카메라가 원점인 <code>(0, 0, 0)</code>에 있게 됩니다. 기본적으로, 카메라는 <code>(0, 0, 800)</code>에 위치해 있습니다.</p>
  <p>다음 세 개의 매개 변수 <code>centerX</code>, <code>centerY</code>, <code>centerZ</code>는 카메라가 향하는 지점의 좌표입니다. 예를 들어, <code>camera(0, 0, 0, 10, 20, 30)</code>를 호출하면 카메라가 원점인 <code>(0, 0, 0)</code>에 있고 <code>(10, 20, 30)</code>를 향하고 있게 됩니다. 기본적으로, 카메라는 원점 <code>(0, 0, 0)</code>을 향해 있습니다.</p>
  <p>마지막 세 개의 매개 변수 <code>upX</code>, <code>upY</code>, <code>upZ</code>는 "상향" 벡터의 구성 요소입니다. "상향" 벡터는 카메라의 y축을 향합니다. 예를 들어, <code>camera(0, 0, 0, 10, 20, 30, 0, -1, 0)</code>를 호출하면 카메라가 원점인 <code>(0, 0, 0)</code>에 있고 <code>(10, 20, 30)</code>를 향하고 있으며 "상향" 벡터를 <code>(0, -1, 0)</code>로 설정하는, 다시 말해 거꾸로 있도록 합니다. 기본적으로 "상향" 벡터는 <code>(0, 1, 0)</code>입니다.</p>
  <p>참고: <code>camera()</code>는 WebGL 모드에서만 사용할 수 있습니다.</p>

    }

    function draw() {
      background(200);

      // Get the camera's coordinates from the sliders.
      let x = xSlider.value();
      let y = ySlider.value();
      let z = zSlider.value();

      // Move the camera.
      camera(x, y, z);

      // Draw the box.
      box();
    }
