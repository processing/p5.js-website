  <p>캔버스의 일부를 지우기 위해 모양을 사용하는 것을 시작해보세요.</p>

  <p><code>erase()</code> 이후의 모든 그리기는
  캔버스에서, 지워지며

  아래의 웹 페이지를 드러냅니다. T지워진 영역은 투명해지며, 

  캔버스 뒤의 내용이 표시됩니다.

  <a href="/reference/p5/fill">fill()</a>, <a
  href="/reference/p5/stroke">stroke()</a>, 그리고

  <a href="/reference/p5/blendMode">blendMode()</a> 의 효과는
  <code>erase()</code> 를 호출하면

  사라집니다.</p>

  <p><code>erase()</code> 함수에는 두 개의 선택적 매개변수가 있습니다. 
  첫 번째 매개 변수는

  모양의 내부 지우기의 강도를 설정합니다. 값이 0이면 

  지워지지 않습니다. 값이 255이면

  모양 아래의 내용을 완전히 지웁니다. 전체 기본값은 255

  (full strength)입니다.</p>

  <p>두 번째 매개변수는 모양의 가장자리 지우기의 강도를 설정합니다.

  값이 0이면 지워지지 않습니다. 값이 255이면

  모양 가장자리 아래의 내용을 완전히 지웁니다. 전체 기본값은 255

  (full strength)입니다.</p>

  <p>지우기 효과를 취소하려면, <a
  href="/reference/p5/noErase">noErase()</a>

  함수를 사용하세요.</p>

  <p><code>erase()</code> 는

  <a href="/reference/p5/image">image()</a> 와

  <a href="/reference/p5/background">background()</a> 함수가 적용된 그리기에는 영향을 주지 않습니다.</p>
