description: >
  <p>소스 이미지의 픽셀을 캔버스의 영역으로 복사합니다.</p>
  
  <p>첫 번째 매개변수 <code>srcImage</code>는 혼합할 <a href="/reference/p5/p5.Image">p5.Image</a> 객체입니다.
  
  소스 이미지는 캔버스 자체이거나 <a href="/reference/p5/p5.Image">p5.Image</a> 객체일 수 있습니다.
  
  <code>copy()</code>는 소스 영역이 대상 영역과 크기가 다르면 크기를 조정합니다.</p>
  
  <p>다음 네 개의 매개변수 <code>sx</code>, <code>sy</code>, <code>sw</code>, <code>sh</code>는
  
  소스 이미지에서 복사할 영역을 결정합니다.
  
  <code>(sx, sy)</code>는 영역의 좌상단의 좌표입니다.
  
  <code>sw</code>와 <code>sh</code>는 영역의 너비와 높이입니다.</p>
  
  <p>다음 네 개의 매개변수 <code>dx</code>, <code>dy</code>, <code>dw</code>, <code>dh</code>는
  
  복사할 캔버스의 영역을 결정합니다.
  
  <code>(dx, dy)</code>는 영역의 왼쪽 상단입니다.
  
  <code>dw</code>와 <code>dh</code>는 영역의 너비와 높이입니다.</p>

    let img;

    // Load the image.
    function preload() {
      img = loadImage('/assets/rockies.jpg');
    }

    function setup() {
      createCanvas(100, 100);

      // Use the mountains as the background.
      background(img);

      // Copy a region of pixels to another spot.
      copy(img, 7, 22, 10, 10, 35, 25, 50, 50);

      // Outline the copied region.
      stroke(255);
      noFill();
      square(7, 22, 10);

      describe('An image of a mountain landscape. A square region is outlined in white. A larger square contains a pixelated view of the outlined region.');
    }
