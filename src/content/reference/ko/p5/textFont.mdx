  <p><a href="/reference/p5/text">text()</a> 함수에서 사용할 폰트를 설정합니다.</p>
  
  <p>첫 번째 매개변수 <code>font</code>는 폰트를 설정합니다.
  
  <code>textFont()</code>는 <a href="/reference/p5/p5.Font">p5.Font</a> 객체나
  
  시스템 폰트의 이름을 가진 문자열을 인식합니다.
  
  예시로, <code>'Courier New'</code>가 있습니다.</p>
  
  <p>두 번째 매개변수 <code>size</code>는 선택 사항입니다.
  
  픽셀 단위의 폰트 크기를 설정합니다.
  
  이는 <a href="/reference/p5/textSize">textSize()</a>를 호출하는 것과 동일한 효과입니다.</p>
  
  <p>참고: <code>WEBGL</code> 모드는
  
  <a href="/reference/p5/loadFont">loadFont()</a>로 불러온 글꼴만 지원합니다.</p>
