description: >
  <p>캔버스 배경에 사용되는 색상을 설정합니다.</p>

  <p>기본적으로,  배경(background)은 투명합니다. <code>background()</code> 는
  일반적으로

  <a href="/reference/p5/draw">draw()</a>
  각 프레임의 시작에서 

  디스플레이 창을 지우는 데 사용됩니다. 또한 애니메이션의 첫 번째 프레임에서 배경을 설정하기 위해

  <a href="/reference/p5/setup">setup()</a> 
  안에서도 

  사용될 수 있습니다.</p>

  <p> 매개변수가 하나인 <code>background()</code> 버전은
  네 가지 방식 중

  하나로 해석됩니다. 매개변수가 <code>Number</code>일 경우,
  그레이스케일 

  값으로 해석됩니다. 매개변수가 <code>String</code>일 경우, 
  CSS 색상 문자열로 해석됩니다.

  RGB, RGBA, HSL, HSLA, hex 및 명명된 색상 문자열이 지원됩니다. 

  매개변수가 <a href="/reference/p5/p5.Color">p5.Color</a> 
  객체일 경우, 

  배경색으로 사용됩니다. 매개변수가

  <a href="/reference/p5/p5.Image">p5.Image</a> 
  객체일 경우,

  배경 이미지로 사용됩니다.</p>

  <p>두 개의 매개 변수를 사용하는 <code>background()</code>의 버전은 
  첫 번째 매개 변수를 

  그레이스케일 값으로 해석합니다. 두 번째 매개 변수는 알파(투명도)

  값을 설정합니다.</p>

  <p>세 개의 매개 변수를 사용하는 <code>background()</code> 의 버전은 현재
  tRGB,

  HSB, 또는 HSL 색상에따라

  <a href="/reference/p5/colorMode">colorMode()</a>으로 해석됩니다. 기본적으로, 
  색상은

  RGB값으로 지정됩니다. <code>background(255, 204, 0)</code> 를 호출하면
  밝은 노랑색의 배경으로

  설정됩니다.</p>

example:
  - |

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // A grayscale value.
      background(51);

      describe('A canvas with a dark charcoal gray background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // A grayscale value and an alpha value.
      background(51, 0.4);
      describe('A canvas with a transparent gray background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // R, G & B values.
      background(255, 204, 0);

      describe('A canvas with a yellow background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // Use HSB color.
      colorMode(HSB);

      // H, S & B values.
      background(255, 204, 100);

      describe('A canvas with a royal blue background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // A CSS named color.
      background('red');

      describe('A canvas with a red background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // Three-digit hex RGB notation.
      background('#fae');

      describe('A canvas with a pink background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // Six-digit hex RGB notation.
      background('#222222');

      describe('A canvas with a black background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // Integer RGB notation.
      background('rgb(0, 255, 0)');

      describe('A canvas with a bright green background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // Integer RGBA notation.
      background('rgba(0, 255, 0, 0.25)');

      describe('A canvas with a transparent green background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // Percentage RGB notation.
      background('rgb(100%, 0%, 10%)');

      describe('A canvas with a red background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // Percentage RGBA notation.
      background('rgba(100%, 0%, 100%, 0.5)');

      describe('A canvas with a transparent purple background.');
    }
    </code>
    </div>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      // A p5.Color object.
      let c = color(0, 0, 255);
      background(c);

      describe('A canvas with a blue background.');
    }
    </code>
    </div>
