  <p><a href="/reference/p5/curve">curve()</a> 및

  <a href="/reference/p5/curveVertex">curveVertex()</a> 함수를 통해 그려지는 곡선의 형태를 조정합니다.</p>

  <p>스플라인 곡선은 여러 지점에 연결된 케이블과 같습니다. 

  <code>curveTightness()</code> 함수는 케이블이 지점에 얼마나 
  타이트하게 연결되는지를 조정합니다.</p>

  <p>매개변수, <code>tightness</code>는 곡선이 
  꼭지점에 어떻게 맞는지를 결정합니다.

  기본적으로, <code>tightness</code> 는 0으로 설정되어 있습니다.
  0으로 설정되어 있습니다.

  <code>curveTightness(1)</code>처럼 타이트니스를 1로 설정하면,
  곡선의 점을 직선으로 연결합니다.

  –5에서 5 사이의 값은 곡선을 변형시키면서도 곡선으로 인식할 수 있게 합니다.

  recognizable.</p>

    <code>
    // Move the mouse left and right to see the curve change.

    function setup() {
      createCanvas(100, 100);

      describe('A black curve forms a sideways U shape. The curve deforms as the user moves the mouse from left to right');
    }

    function draw() {
      background(200);

      // Set the curve's tightness using the mouse.
      let t = map(mouseX, 0, 100, -5, 5, true);
      curveTightness(t);

      // Draw the curve.
      noFill();
      beginShape();
      curveVertex(10, 26);
      curveVertex(10, 26);
      curveVertex(83, 24);
      curveVertex(83, 61);
      curveVertex(25, 65);
      curveVertex(25, 65);
      endShape();
    }
