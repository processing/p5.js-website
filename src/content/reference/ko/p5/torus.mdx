description: >
  <p>원환면을 그립니다.</p>

  <p>원환면은 삼각형 면들이 연결되어 고리 모양을 형성하는 3D 도형입니다. 

  삼각형 면의 수가 적은 원환면은 평평하게 보이며,

  많은 삼각형 면으로 이루어진 원환면은 부드러운 표면을 가집니다.</p>

  <p>첫 번째 매개변수 <code>radius</code> 는 선택 사항입니다. 
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>torus(30)</code>와 같다면 고리의 반경이 설정됩니다.
  기본적으로, <code>radius</code> 은 50입니다.</p>

  <p>두 번째 매개변수 <code>tubeRadius</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>torus(30, 15)</code> 와 같다면 튜브의 반경이 설정됩니다. 
  기본적으로, 

  <code>tubeRadius</code> 은 10입니다.</p>

  <p>세 번째 매개변수 <code>detailX</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>torus(30, 15, 5)</code> 와 같다면 원환면의 여백을 구성하는 모서리의 수가 설정됩니다. 
  더 많은 모서리를 사용할수록,

  여백은 원처럼 보입니다. 

  기본적으로, <code>detailX</code> 는 24입니다.</p>

  <p>네 번째 매개변수 <code>detailY</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면, 

  예를 들어 <code>torus(30, 15, 5, 7)</code> 와 같다면 원환면의 높이를 채우는 동안
  사용할 삼각형 분할면 수가 설정됩니다. 

  기본적으로, <code>detailY</code> 는
  16입니다.</p>

  <p>참고: <code>torus()</code> 는 WebGL 모드에서만 사용할 수 있습니다.</p>
