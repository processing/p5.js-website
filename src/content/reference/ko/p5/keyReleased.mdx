description: >
  <p>어떤 키가 눌린 후 떼어질 때 한 번 호출되는 함수입니다.</p>

  <p><code>keyReleased()</code> 함수를 선언하면 사용자가 키를 눌렀다 뗄 때 자동으로 한 번 실행되는 코드 블록을 설정합니다:</p>
  
  <code>

  function keyReleased() {
    // 실행할 코드.
  }

  </code>

  <p><a href="/reference/p5/key">key</a> 변수와
  <a href="/reference/p5/keyCode">keyCode</a> 변수는 
  p5.js에 의해 <code>keyReleased()</code>가 호출될 때 가장 최근에 놓인 값으로 업데이트됩니다:</p>

  <code>

  function keyReleased() {
    if (key === 'c') {
      // 실행할 코드.
    }

  <p>  if (keyCode === ENTER) {
      // 실행할 코드.
    }
  }

  </p></code>

  <p><code>event</code>는 선택적(optional) 매개변수입니다. <code>keyReleased()</code>는 항상
  <a href="https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent" target="_blank">KeyboardEvent</a> 객체를 전달받으며, 
  이 객체는 키 입력 이벤트를 설명하는 속성들을 가지고 있습니다:</p>

  <code>

  function keyReleased(event) {
    // 이벤트를 사용하여 실행할 코드.
    console.log(event);
  }

  </code>

  <p>브라우저는 다양한 키 이벤트에 대해 기본적으로 설정된 동작을 가지고 있을 수 있습니다.
  이 이벤트에 대한 기본 동작을 방지하려면, 함수 끝에 <code>return false;</code>를 추가하세요.</p>
