description: >
  <p>3D 스케치에서 선에 대해 원근감을 활성화하거나 비활성화합니다.</p>
  <p>WebGL 모드에서, 선은 카메라로부터 멀리 떨어져 있을 때 더 얇게 그려질 수 있습니다. 이렇게 하면 선이 더 현실적으로 보여집니다.</p>
  <p>기본적으로 사용하는 원근법 유형에 따라 선이 다르게 그려집니다.</p>
  <ul>
  <li><code>perspective()</code>와 <code>frustum()</code>는 현실적인 원근법을 시뮬레이션합니다. 이 모드에서, 선 두께는 카메라로부터 선의 거리에 영향을 받습니다. 이렇게 하면 더 자연스러운 모습을 볼 수 있습니다. 3D 스케치에서 <code>perspective()</code>가 기본 모드입니다.</li>
  <li><code>ortho()</code>는 현실적인 원근법을 시뮬레이션하지 않습니다. 이 모드에서 선 두께는 카메라로부터 선의 거리와 관계없이 일관성을 갖습니다. 이렇게 하면 더 예측 가능하고 일관된 모습을 볼 수 있습니다.</li>
  </ul>
  <p><code>linePerspective()</code>는 선 그리기 기본값을 재정의할 수 있습니다.</p>
  <p>매개 변수 <code>enable</code>은 선택 사항으로, 선을 그리는 방법을 설정하는 <code>Boolean</code> 값입니다. 만약 <code>linePerspective(true)</code>에서와 같이 <code>true</code>가 전달되면, 카메라에서 멀리 떨어져 있을 때 선이 더 가늘게 나타납니다. <code>linePerspective(false)</code>에서와 같이 <code>false</code>가 전달되면, 선은 카메라와의 거리에 관계없이 일관된 선 두께를 갖게 됩니다. 기본적으로 <code>linePerspective()</code>가 활성화됩니다.</p>
  <p>아무 인수 없이 <code>linePerspective()</code>를 호출하면 활성화된 경우 <code>true</code>를, 그렇지 않은 경우 <code>false</code>를 반환합니다.</p>
  <p>참고: <code>linePerspective()</code>는 WebGL 모드에서만 사용할 수 있습니다.</p>

    // Double-click the canvas to toggle the line perspective.

    function setup() {
      createCanvas(100, 100, WEBGL);

      );

    function draw() {
      background(200);

      // Translate the origin toward the camera.
      translate(-10, 10, 600);

      // Rotate the coordinate system.
      rotateY(-0.1);
      rotateX(-0.1);

      // Draw the row of boxes.
      for (let i = 0; i < 6; i += 1) {
        translate(0, 0, -40);
        box(10);
      }
    }

    // Toggle the line perspective when the user double-clicks.
    function doubleClicked() {
      let isEnabled = linePerspective();
      linePerspective(!isEnabled);
    }
    </code>
