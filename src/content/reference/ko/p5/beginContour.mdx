description: >
  <p>평면 도형 내부에 빈 공간을 만들기 시작합니다.</p>

  <p><code>beginContour()</code> 와 <a
  href="/reference/p5/endContour">endContour()</a>

  함수는 평면인 커스텀 도형 내에 여백을 만들 수 있습니다. 

  <code>beginContour()</code> 가 여백에 꼭짓점 추가하기
  시작하고,

  <a href="/reference/p5/endContour">endContour()</a> 는 꼭짓점을 추가하는 것을 중단합니다.

  <code>beginContour()</code> 와 <a
  href="/reference/p5/endContour">endContour()</a> 는

  <a href="/reference/p5/beginShape">beginShape()</a> 와

  <a href="/reference/p5/endShape">endShape()</a> 사이에서 호출되어야 합니다.</p>

  <p>Transformations such as <a href="/reference/p5/translate">translate()</a>,

  <a href="/reference/p5/rotate">rotate()</a>, <a
  href="/reference/p5/scale">scale()</a>와 같은 변환은

  <code>beginContour()</code> 와 

  <a href="/reference/p5/endContour">endContour()</a> 사이에선 작동하지 않습니다. 
  또한

  <code>beginContour()</code>와  <a href="/reference/p5/endContour">endContour()</a> 사이에서 

  <a href="/reference/p5/ellipse">ellipse()</a> 나 <a href="/reference/p5/rect">rect()</a> 와 
  같은 다른 도형을 

  사용할 수 없습니다. </p>

  <p>참고: 여백을 정의하는 꼭지점은 외부 도형의 반대 방향으로 "회전"해야 합니다. 

  먼저 바깥의 도형 꼭짓점을 시계 방향으로 그리고, 

  그 다음 여백의 꼭짓점을

  반시계 방향으로 그립니다.</p>

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Start drawing the shape.
      beginShape();

      // Exterior vertices, clockwise winding.
      vertex(10, 10);
      vertex(90, 10);
      vertex(90, 90);
      vertex(10, 90);

      // Interior vertices, counter-clockwise winding.
      beginContour();
      vertex(30, 30);
      vertex(30, 70);
      vertex(70, 70);
      vertex(70, 30);
      endContour();

      // Stop drawing the shape.
      endShape(CLOSE);

      describe('A white square with a square hole in its center drawn on a gray background.');
    }
    </code>
