  <p>색상에서 파랑값을 추출합니다.</p>


  <p><code>blue()</code> 는 a에서 파랑값을 추출합니다


  <a href="/reference/p5/p5.Color">p5.Color</a> 개체, 색상 배열
  구성요소, 또는


  CSS 색상 문자열.</p>


  <p>기본적으로, <code>blue()</code> 는 0 ~ 255까지범위의 색상의 파랑값을 반환합니다.


  <a href="/reference/colorMode">colorMode()</a> 가 RGB로 설정되어 있는 경우,
  그것을


  지정된 범위의 파랑값으로 반환합니다.</p>

      noStroke();

      // Draw the right rectangle.
      fill(0, 0, blueValue);
