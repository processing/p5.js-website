  <p>이미지에서 주변광을 생성합니다.</p>

  <p><code>imageLight()</code> 는 모든 방향에서 빛을 내는 것을 시뮬레이션합니다.
  이 효과는 이미지를 텍스처로 사용하는 거대한 구의 중심에 스케치를 배치하는 것과 같습니다. 
  
  이미지의 분산광은 <a href="/reference/p5/fill">fill()</a>에 의해 영향을 받고, 
  
  반사광은 <a href="/reference/p5/specularMaterial">specularMaterial()</a>과 <a href="/reference/p5/shininess">shininess()</a>에 의해 영향을 받습니다.</p>

  <p>매개변수 <code>img</code>는 빛의 원천으로 사용할 <a
  href="/reference/p5/p5.Image">p5.Image</a>의 객체입니다.</p>

    <div class="notest">
    <code>
    // Click and drag the mouse to view the scene from different angles.

    let img;

    // Load an image and create a p5.Image object.
    function preload() {
      img = loadImage('/assets/outdoor_spheremap.jpg');
    }

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A sphere floating above a landscape. The surface of the sphere reflects the landscape.');
    }

    function draw() {
      // Enable orbiting with the mouse.
      orbitControl();

      // Draw the image as a panorama (360˚ background).
      panorama(img);

      // Add a soft ambient light.
      ambientLight(50);

      // Add light from the image.
      imageLight(img);

      // Style the sphere.
      specularMaterial(20);
      shininess(100);
      noStroke();

      // Draw the sphere.
      sphere(30);
    }
    </code>
