description: >
  <p>색상을 사용하여 이미지에 틴트(색조, tint)를 적용합니다.</p>
  
  <p>한 개의 매개변수를 사용하는 <code>tint()</code>는 네 가지 방식으로 해석됩니다.
  
  매개변수가 숫자인 경우, 그레이스케일 값으로 해석됩니다.
  
  매개변수가 문자열인 경우, CSS 색상 문자열로 해석됩니다.
  
  <code>[R, G, B, A]</code> 값의 배열이나 <a href="/reference/p5/p5.Color">p5.Color</a> 객체를 사용하여 틴트 색상을 설정할 수도 있습니다.</p>
  
  <p>두 개의 매개변수를 사용하는 <code>tint()</code>는 첫 번째 매개변수를 그레이스케일 값으로, 두 번째 매개변수를 알파 값으로 사용합니다.
  
  예를 들어, <code>tint(255, 128)</code>를 입력하면 이미지를 50% 투명하게 만듭니다.</p>
  
  <p>세 개의 매개변수를 사용하는 <code>tint()</code>는 현재 <a href="/reference/p5/colorMode">colorMode()</a>에 따라 RGB 또는 HSB 값으로 해석됩니다.
  
  선택적인 네 번째 매개변수는 알파 값을 설정합니다.
  
  예를 들어, <code>tint(255, 0, 0, 100)</code>은 이미지에 빨간색 틴트를 적용하고 투명하게 만듭니다.</p>
