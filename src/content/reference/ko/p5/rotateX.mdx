description: >
  <p>WebGL 모드에서 x축을 기준으로 좌표계를 회전시킵니다.</p>
  
  <p>매개변수 <code>angle</code>은 회전하는 양입니다.
  
  예를 들어, <code>rotateX(1)</code>을 호출하면 x축을 중심으로 좌표 시스템이 1 라디안만큼 회전합니다.
  
  <code>rotateX()</code>는 현재의 <a href="/reference/p5/angleMode">angleMode()</a>를 기준으로 각도 값을 해석합니다.</p>
  
  <p>기본적으로 변환은 누적됩니다.
  
  예를 들어, <code>rotateX(1)</code>을 두 번 호출하는 것은 <code>rotateX(2)</code>를 한 번 호출하는 것과 같은 효과가 있습니다.
  
  <a href="/reference/p5/push">push()</a>와 <a href="/reference/p5/pop">pop()</a> 함수를 사용하여 서로 다른 그룹 내에서 변환을 분리할 수 있습니다.</p>
  
  <p>참고: 변환은 draw 루프의 시작 시점에서 재설정됩니다.
  
  <a href="/reference/p5/draw">draw()</a> 함수 내에서 <code>rotateX(1)</code>를 호출해도 도형이 계속해서 회전하지 않습니다.</p>
