  <p><a href="/reference/p5/requestPointerLock">requestPointerLock</a>으로 시작된 포인터 잠금을 종료합니다.</p>
  
  <p><code>requestPointerLock()</code>를 호출하면
  <a href="/reference/p5/mouseX">mouseX</a>, <a href="/reference/p5/mouseY">mouseY</a>, 
  <a href="/reference/p5/pmouseX">pmouseX</a>, <a href="/reference/p5/pmouseY">pmouseY</a>의 값이 잠깁니다.
  
  <code>exitPointerLock()</code>를 호출하면 마우스 시스템 변수의 업데이트가 다시 시작됩니다.</p>
  
  <p>참고: 대부분의 브라우저는 <code>requestPointerLock()</code>을 호출하기 전에 클릭과 같은 입력을 요구합니다.
  따라서 <a href="/reference/p5/doubleClicked">doubleClicked()</a>와 같은 이벤트 함수에서
  <code>requestPointerLock()</code>를 호출하는 것이 좋습니다.</p>

    <div>
    <code>
    let isLocked = false;

    function setup() {
      createCanvas(100, 100);

      describe(
        'A gray square with a word at its center. The word changes between "Unlocked" and "Locked" when the user double-clicks.'
      );
    }

    function draw() {
      background(200);

      // Style the text.
      textAlign(CENTER);
      textSize(16);

      // Tell the user whether the pointer is locked.
      if (isLocked === true) {
        text('Locked', 50, 50);
      } else {
        text('Unlocked', 50, 50);
      }
    }

    // Toggle the pointer lock when the user double-clicks.
    function doubleClicked() {
      if (isLocked === true) {
        exitPointerLock();
        isLocked = false;
      } else {
        requestPointerLock();
        isLocked = true;
      }
    }
