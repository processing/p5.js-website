  <p>박스(직육면체)를 그립니다.</p>

  <p>박스는 여섯 개의 면을 가진 3D 도형입니다. 

  각 면은 네 개의 마주하는 면들과 90˚를 이룹니다.</p>

  <p>첫 번째 매개변수 <code>width</code> 는 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어, <code>box(20)</code> 와 같이 전달되면 박스의 너비와 높이가 설정됩니다.
  기본적으로, <code>width</code> 는 50입니다.</p>

  <p>두 번째 매개변수 <code>height</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어, <code>box(20, 30)</code> 와 같이 전달되면 박스의 높이가 설정됩니다.
  기본적으로, <code>height</code> 는 

  박스의 <code>width</code> 에 설정됩니다.</p>

  <p>세 번째 매개변수 <code>depth</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어, <code>box(20, 30, 40)</code> 와 같이 전달되면 박스의 깊이가 설정됩니다.
  기본적으로,  <code>depth</code> 는

  박스의 <code>height</code> 에 설정됩니다.</p>

  <p>네 번째 매개변수, <code>detailX</code> 도 선택 사항입니다. 
  <code>Number</code> 가 전달되면, 

  예를 들어, <code>box(20, 30, 40, 5)</code> 와 같이 전달되면
   x축을 따라 사용할 삼각형 분할 수가 설정됩니다.

  모든 3D도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로, <code>detailX</code> 는 1입니다.</p>

  <p>다섯 번째 매개변수 <code>detailY</code> 도 선택 사항입니다.
  선택 사항입니다.

  <code>box(20, 30, 40, 5, 7)</code> 와 같이 전달되면,
  y축을 따라 사용할 삼각형 분할면 수가 설정됩니다. 

  모든 3D 도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로, <code>detailY</code> 는 1입니다.</p>

  <p>참고: <code>box()</code> 는 WebGL 모드에서만 사용할 수 있습니다.</p>
