description: >

  <p>지정된 점에서 모든 방향으로 빛을 발하는 빛을 생성합니다.</p>

  <p>점 광원은 모든 방향으로 빛을 발하는 전구와 같습니다. 다양한 조명 효과를

  얻기 위해 다른 위치에 배치할 수 있습니다. 한 번에 최대 5개의 점 광원이 활성화될 수 있습니다.</p>

  <p><code>pointLight()</code>를 호출하는 방법은 빛의 색상과 위치를 설정하는 매개변수가 4가지 있습니다.</p>

  <p><code>pointLight()</code>를 6개의 매개변수로 호출하는 첫 번째 방법은 첫 번째 3개의

  매개변수인 <code>v1</code>, <code>v2</code>, <code>v3</code>를 사용하여 현재

  <a href="/reference/p5/colorMode">colorMode()</a>를 사용하여 빛의 색상을 설정합니다. 마지막 3개의 매개변수인

  <code>x</code>, <code>y</code>, <code>z</code>는 빛의 위치를 설정합니다. 예를 들어,

  <code>pointLight(255, 0, 0, 50, 0, 0)</code>은 <code>(50, 0, 0)</code> 좌표에서 빛이

  발하는 빨간색 <code>(255, 0, 0)</code> 빛을 생성합니다.</p>

  <p><code>pointLight()</code>를 호출하는 두 번째 방법은 4개의 매개변수를 사용합니다. 첫 번째 3개의

  매개변수인 <code>v1</code>, <code>v2</code>, <code>v3</code>를 사용하여 현재

  <a href="/reference/p5/colorMode">colorMode()</a>를 사용하여 빛의 색상을 설정합니다. 마지막 매개변수인

  position은 <a href="/reference/p5/p5.Vector">p5.Vector</a> 객체를 사용하여 빛의 위치를 설정합니다.


  예를 들어, <code>pointLight(255, 0, 0, lightPos)</code>는 <code>lightPos</code> 벡터로 설정된 위치에서

  빛이 발하는 빨간색 <code>(255, 0, 0)</code> 빛을 생성합니다.</p>

  <p><code>pointLight()</code>를 호출하는 세 번째 방법은 4개의 매개변수를 사용합니다. 첫 번째

  매개변수인 <code>color</code>는 <a href="/reference/p5/p5.Color">p5.Color</a> 객체나 색상

  값 배열을 사용하여 빛의 색상을 설정합니다. 마지막 3개의 매개변수인 <code>x</code>, <code>y</code>, <code>z</code>는

  빛의 위치를 설정합니다. 예를 들어, <code>directionalLight(myColor, 50, 0, 0)</code>은

  <code>(50, 0, 0)</code> 좌표에서 빛이 발하는 <code>myColor</code> 색상의 빛을 생성합니다.</p>

  <p><code>pointLight()</code>를 호출하는 네 번째 방법은 2개의 매개변수를 사용합니다. 첫 번째

  매개변수인 <code>color</code>는 <a href="/reference/p5/p5.Color">p5.Color</a> 객체나 색상

  값 배열을 사용하여 빛의 색상을 설정합니다. 두 번째 매개변수인 <code>position</code>은

  <a href="/reference/p5/p5.Vector">p5.Vector</a> 객체를 사용하여 빛의 위치를 설정합니다. 예를 들어,

  <code>directionalLight(myColor, lightPos)</code>는 <code>lightPos</code> 벡터로 설정된 위치에서

  빛이 발하는 <code>myColor</code> 색상의 빛을 생성합니다.</p>

    // Click and drag the mouse to view the scene from different angles.
    // Double-click to turn on the point light.

    let isLit = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A sphere drawn on a gray background. A red light starts shining from above when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Control the light.
      if (isLit === true) {
        // Add a red point light from above.
        // Use RGB values and XYZ coordinates.
        pointLight(255, 0, 0, 0, -150, 0);
      }

      // Style the sphere.
      noStroke();

      // Draw the sphere.
      sphere(30);
    }

    // Turn on the point light when the user double-clicks.
    function doubleClicked() {
      isLit = true;
    }
