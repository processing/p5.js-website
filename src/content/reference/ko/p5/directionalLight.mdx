description: >
  <p>한 방향으로 빛을 내는 빛을 생성합니다.</p>

  <p>방향성 광원(Directional Lights)는 특정 지점에서 빛나지 않습니다. 그것들을 화면 밖 어딘가에서 빛나는 태양과 같습니다. 빛의 방향은 -1에서 1사이의 세개의 <code>(x, y, z)</code> 값으로 설정됩니다. 예를 들어, 빛의 방향을 <code>(1, 0, 0)</code> 으로 설정하면 빛이 직접 오른쪽을 향하므로 <a href="/reference/p5/p5.Geometry">p5.Geometry</a> 객체는 왼쪽에서 비춰집니다.</p>

  <p>조명의 색상과 방향을 설정하기 위한 매개 변수를 사용하여 <code>directionalLight()</code>를 호출하는 방법은 네 가지가 있습니다.</p>

  <p>첫 번째 방법은 여섯 개의 매개변수를 가집니다. 처음 세 개의 매개변수인 <code>v1</code>, <code>v2</code>, <code>v3</code>는 현재 <a href="/reference/p5/colorMode">colorMode()</a>를 사용하여 빛의 색상을 설정합니다.
  
  마지막 세 개의 매개변수인 <code>x</code>, <code>y</code>, <code>z</code>는 빛의 방향을 설정합니다. 예를 들어, <code>directionalLight(255, 0, 0, 1, 0, 0)</code>은 오른쪽을 향해 빛나는 빨간색 <code>(255, 0,
  0)</code> 빛을 생성합니다.</p>

  <p>두 번째 방법은 네 개의 매개변수를 가집니다. 처음 세 개의 매개변수인 <code>v1</code>, <code>v2</code>, <code>v3</code>는 현재 <a href="/reference/p5/colorMode">colorMode()</a>를 사용하여 빛의 색상을 설정합니다. 마지막 매개변수인 <code>direction</code>은 <a href="/reference/p5/p5.Geometry">p5.Geometry</a> 객체를 사용하여 빛의 방향을 설정합니다. 예를 들어, <code>directionalLight(255, 0, 0, lightDir)</code>은 <code>lightDir</code> 벡터가 가리키는 방향으로 빛나는 빨간색 <code>(255, 0, 0)</code> 빛을 생성합니다.</p>

  <p>세 번째 방법은 네 개의 매개변수를 가집니다. 첫 번째 매개변수인 <code>color</code>는 <a href="/reference/p5/p5.Color">p5.Color</a> 객체나 색상 값 배열을 사용하여 빛의 색상을 설정합니다. 마지막 세개의 매개변수인 <code>x</code>, <code>y</code>, <code>z</code>는 빛의 방향을 설정합니다. 예를 들어, <code>directionalLight(myColor, 1, 0, 0)</code>은 <code>myColor</code>의 색상 값을 가진 오른쪽으로 빛나는 빛을 생성합니다.
  
  네 번째 방법은 두 개의 매개변수를 가집니다. 첫 번째 매개변수인 <code>color</code>는 <a href="/reference/p5/p5.Color">p5.Color</a> 객체나 색상 값 배열을 사용하여 빛의 색상을 설정합니다. 두 번째 매개변수인 <code>direction</code>은 <a href="/reference/p5/p5.Color">p5.Color</a> 객체를 사용하여 빛의 방향을 설정합니다. 
  
  예를 들어, <code>directionalLight(myColor, lightDir)</code>은 <code>lightDir</code> 벡터가 가리키는 방향으로 빛나는 <code>myColor</code>의 색상 값을 가진 빛을 생성합니다.</p>

    // Click and drag the mouse to view the scene from different angles.
    // Double-click to turn on the directional light.

    let isLit = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A sphere drawn on a gray background. A red light starts shining from above when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Control the light.
      if (isLit === true) {
        // Add a red directional light from above.
        // Use RGB values and XYZ directions.
        directionalLight(255, 0, 0, 0, 1, 0);
      }

      // Style the sphere.
      noStroke();

      // Draw the sphere.
      sphere(30);
    }
    </code>
