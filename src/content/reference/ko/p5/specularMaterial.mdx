description: >
  <p>도형의 표면 재질의 반사색을 설정합니다.</p>
  <p><code>specularMaterial()</code> 색상은 도형의 광택 코팅이 반사할 밝은 색상의 구성 요소를 설정합니다. 예를 들어, <code>specularMaterial(255, 255, 0)</code>를 호출하면 도형이 빨간색과 초록색 빛을 반사하지만 파란색 빛은 반사하지 않습니다.</p>
  <p><a href="/reference/p5/ambientMaterial">ambientMaterial()</a>과 다르게,<code>specularMaterial()</code>은 <a href="/reference/p5/directionalLight">directionalLight()</a>, <a href="/reference/p5/pointLight">pointLight()</a>, 그리고 <a href="/reference/p5/spotLight">spotLight()</a></p>을 포함한 광원의 전체 색상을 반사합니다. 이렇게 하면 모양의 "빛나는" 외관을 형성할 수 있습니다. 그 재질의 반짝임은 <a href="/reference/p5/shininess">shininess()</a> 함수로 조절할 수 있습니다.
  <p>재질의 색상을 설정하기 위해 <code>specularMaterial()</code>은 세 가지의 다른 방법과 그에 맞는 인수로 호출될 수 있습니다.</p>
  <p><code>specularMaterial()</code>을 호출하는 첫번째 방법은 하나의 인수를 갖는데, 바로 <code>gray</code>입니다. <code>specularMaterial(50)</code>에서와 같이 0에서 255 사이의 회색조 값을 전달해 재질의 색상을 설정할 수 있습니다. 회색조 값이 높을수록 더 밝게 보입니다.</p>
  <p><code>specularMaterial()</code>을 호출하는 두번째 방법은 하나의 인수를 갖는데, 바로 <code>color</code>입니다. <a href="/reference/p5/p5.Color">p5.Color</a> 객체, 색상 값 배열이나 <code>specularMaterial('magenta')</code>에서와 같이 CSS 색상 문자열을 전달해 재질의 색상을 설정할 수 있습니다.</p>
  <p><code>specularMaterial()</code>을 호출하는 세번째 방법은 네 개의 인수를 갖는데, 바로 <code>v1</code>, <code>v2</code>, <code>v3</code>, 그리고 <code>alpha</code>입니다. <code>specularMaterial(255, 0, 0, 30)</code>에서와 같이 RGBA, HSBA, 또는 HSLA 값을 전달해 재질의 색상을 설정할 수 있습니다. 색상 값은 현재의 <a href="/reference/p5/colorMode">colorMode()</a>를 통해 해석됩니다.</p>

    <code>
    // Click and drag the mouse to view the scene from different angles.
    // Double-click the canvas to apply a specular material.

    let isGlossy = false;

    function setup() {
      createCanvas(100, 100, WEBGL);

      describe('A red torus drawn on a gray background. It becomes glossy when the user double-clicks.');
    }

    function draw() {
      background(200);

      // Enable orbiting with the mouse.
      orbitControl();

      // Turn on a white point light at the top-right.
      pointLight(255, 255, 255, 30, -40, 30);

      // Add a glossy coat if the user has double-clicked.
      if (isGlossy === true) {
        specularMaterial(255);
        shininess(50);
      }

      // Style the torus.
      noStroke();
      fill(255, 0, 0);

      // Draw the torus.
      torus(30);
    }

    // Make the torus glossy when the user double-clicks.
    function doubleClicked() {
      isGlossy = true;
    }
    </code>
    </div>
