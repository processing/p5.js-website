description: >
  <p>도형의 표면 재질의 주변색을 설정합니다.</p>
  <p><code>ambientMaterial()</code> 색상은 도형이 반사할 <a href="/reference/p5/ambientLight">ambientLight()</a> 색상의 구성 요소를 설정합니다. 예를 들어, <code>ambientMaterial(255, 255, 0)</code>를 호출하면 도형이 빨간색과 초록색 빛을 반사하지만 파란색 빛은 반사하지 않습니다.</p>
  <p>재질의 색상을 설정하기 위해 <code>ambientMaterial()</code>은 세 가지의 다른 방법과 그에 맞는 매개 변수로 호출될 수 있습니다.</p>
  <p><code>ambientMaterial()</code>을 호출하는 첫번째 방법은 하나의 매개 변수를 갖는데, 바로 <code>gray</code>입니다. <code>ambientMaterial(50)</code>에서와 같이 0에서 255 사이의 회색조 값을 전달해 재질의 색상을 설정하는 것입니다. 회색조 값이 높을수록 더 밝게 보입니다.</p>
  <p><code>ambientMaterial()</code>을 호출하는 두번째 방법은 하나의 매개 변수를 갖는데, 바로 <code>color</code>입니다. <a href="/reference/p5/p5.Color">p5.Color</a> 객체, 색상 값 배열이나 <code>ambientMaterial('magenta')</code>에서와 같이 CSS 색상 문자열을 전달해 재질의 색상을 설정하는 것입니다.</p>
  <p><code>ambientMaterial()</code>을 호출하는 세번째 방법은 세 개의 매개 변수를 갖는데, 바로 <code>v1</code>, <code>v2</code>, <code>v3</code>입니다. <code>ambientMaterial(255, 0, 0)</code>에서와 같이 RGB, HSB, 또는 HSL 값을 전달해 재질의 색상을 설정하는 것입니다. 색상 값은 현재의 <a href="/reference/p5/colorMode">colorMode()</a>를 통해 해석됩니다.</p>
  <p>참고: <code>ambientMaterial()</code>은 WebGL 모드에서만 사용할 수 있습니다.</p>
