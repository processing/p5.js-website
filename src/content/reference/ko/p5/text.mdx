description: >
  <p>캔버스에 텍스트를 그립니다.</p>
  
  <p>첫 번째 매개변수 <code>str</code>은 그려질 텍스트입니다.
  
  두 번째와 세 번째 매개변수인 <code>x</code>와 <code>y</code>는 텍스트 좌하단의 좌표를 설정합니다.
  
  텍스트를 정렬하는 다른 방법은 <a href="/reference/p5/textAlign">textAlign()</a>을(를) 참조하십시오.</p>
  
  <p>네 번째와 다섯 번째 매개변수 <code>maxWidth</code>와 <code>maxHeight</code>는 선택 사항입니다.
  
  이들은 텍스트를 포함하는 보이지 않는 직사각형의 면적을 설정합니다.
  
  기본적으로 이들은 최대 너비와 높이를 설정합니다.
  
  직사각형 텍스트 상자를 정의하는 다른 방법은 <a href="/reference/p5/rectMode">rectMode()</a>를(을) 참조하십시오.
  
  텍스트는 텍스트 상자 내에 맞추어 줄바꿈됩니다.
  
  텍스트 상자를 벗어나는 텍스트는 그려지지 않습니다.</p>
  
  <p>텍스트는 몇 가지 방법으로 스타일을 지정할 수 있습니다.
  
  <a href="/reference/p5/fill">fill()</a> 함수를 호출하여 텍스트의 채우기 색상을 설정합니다.
  
  <a href="/reference/p5/stroke">stroke()</a>와 <a href="/reference/p5/strokeWeight">strokeWeight()</a>를 호출하여
  
  텍스트의 외곽선을 설정합니다.
  
  <a href="/reference/p5/textSize">textSize()</a>와 <a href="/reference/p5/textFont">textFont()</a>를 호출하여
  
  텍스트의 크기와 글꼴을 설정합니다.</p>
  
  <p>참고: <code>WEBGL</code> 모드는
  
  <a href="/reference/p5/loadFont">loadFont()</a>로 불러온 글꼴만 지원합니다.
  
  <code>WEBGL</code> 모드에서
  
  <a href="/reference/p5/stroke">stroke()</a>를 호출하면 효과가 없습니다.</p>
