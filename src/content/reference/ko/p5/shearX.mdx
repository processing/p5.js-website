description: >
  <p>모양이 비스듬하게 보이도록 x축을 기울입니다.</p>
  
  <p>기본적으로 x축과 y축은 수직으로 만납니다.
  
  <code>shearX()</code> 함수는 x 좌표가 변환되는 동안 y 좌표가 고정되도록 좌표계를 변환합니다.</p>
  
  <p>첫 번째 매개변수 <code>angle</code>은 기울임의 정도입니다.
  
  예를 들어, <code>shearX(1)</code>을 호출하면 모든 x 좌표가 <code>x = x + y * tan(angle)</code> 공식을 사용하여 변환됩니다.
  
  <code>shearX()</code>는 현재의 <a href="/reference/p5/angleMode">angleMode()</a>를 사용하여 각도 값을 해석합니다.</p>
  
  <p>기본적으로 변환은 누적됩니다.
  
  예를 들어, <code>shearX(1)</code>을 두 번 호출하는 것은 <code>shearX(2)</code>를 한 번 호출하는 것과 같은 효과가 있습니다.
  
  <a href="/reference/p5/push">push()</a>와 <a href="/reference/p5/pop">pop()</a> 함수를 사용하여 서로 다른 그룹 내에서 변환을 분리할 수 있습니다.</p>
  
  <p>참고: 변환은 draw 루프의 시작 시점에서 재설정됩니다.
  
  <a href="/reference/p5/draw">draw()</a> 함수 내에서 <code>shearX(1)</code>를 호출해도 도형이 계속해서 기울어지지 않습니다.</p>
