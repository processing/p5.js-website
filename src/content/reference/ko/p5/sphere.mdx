description: >
  <p>구를 그립니다.</p>

  <p>구는 삼각형 면들이 연결되어 둥근 표면을 형성하는 3D 모양입니다. 

  삼각형 면의 수가 적은 구는 마치 결정체처럼 보이며, 

  많은 삼각형 면으로 이루어진 구는 부드러운 표면을 가진 공처럼 보입니다.</p>

  <p>첫 번째 매개변수 <code>radius</code> 는 선택 사항입니다. 
  <code>Number</code> 가 전달되면,

  예를 들어 <code>sphere(20)</code> 와 같이 전달되면, 구의 반경이 설정됩니다.
  기본적으로, <code>radius</code> 는 50입니다.</p>

  <p>두 번째 매개변수 <code>detailX</code> 도 선택 사항입니다. 
  <code>Number</code> 가 

  예를 들어 <code>sphere(20, 5)</code> 와 같이 전달되면,
  x축을 따라 사용할 삼각형 분할면 수가 설정됩니다. 

  모든 3D도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로, <code>detailX</code> 는 24입니다.</p>

  <p>세 번째 매개변수 <code>detailY</code> 도 선택 사항입니다. 
  <code>Number</code> 가

  예를 들어 <code>sphere(20, 5, 2)</code> 와 같이 전달되면,
  y축을 따라 사용할 삼각형 분할면 수가 설정됩니다. 

  모든 3D 도형은 삼각형을 연결하여 표면이 만들어집니다.

  기본적으로, <code>detailY</code> 는 16입니다.</p>

  <p>참고: <code>sphere()</code> 는 WebGL 모드에서만 사용할 수 있습니다.</p>
