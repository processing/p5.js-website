  <p>두 점 사이에 직선을 그립니다.</p>

  <p>한 개의 직선 기본 너비는 하나의 픽셀입니다.  네 개의 매개 변수를 사용하는 <code>line()</code>
  의 버전은

  2D로 선을 그립니다. 선의 색상을 지정하려면,

  <a href="/reference/p5/stroke">stroke()</a> 함수를 사용하세요.
  선의 두께를 변경하려면,

  <a href="/reference/p5/strokeWeight">strokeWeight()</a> 함수를 사용하세요. 

  선은 채울 수 없기에, <a href="/reference/p5/fill">fill()</a> 함수는

  선의 색상에 영향을 주지 않습니다.</p>

  <p>여섯 개의 매개 변수를 사용하는 <code>line()</code> 의 버전은
  3D 공간에서

  선을 그릴 수 있습니다. 이것을 시도해보기 위해선 <code>WEBGL</code> 인수를

  <a href="/reference/p5/createCanvas">createCanvas()</a>에 추가해야 합니다.</p>

      background(200);

      // Rotate around the y-axis.
      rotateY(frameCount * 0.01);

      // Draw a line.
      line(0, 0, 0, 30, 20, -10);

      // Draw the center sphere.
      sphere(10);

      // Translate to the second point.
      translate(30, 20, -10);

      // Draw the bottom-right sphere.
      sphere(10);
