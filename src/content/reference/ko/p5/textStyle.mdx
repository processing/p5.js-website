description: >
  <p>Sets the style for system fonts when

  <a href="/reference/p5/text">text()</a> is called.</p>

  <p>The parameter, <code>style</code>, can be either <code>NORMAL</code>,
  <code>ITALIC</code>, <code>BOLD</code>, or

  <code>BOLDITALIC</code>.</p>

  <p><code>textStyle()</code> may be overridden by CSS styling. This function
  doesn't

  affect fonts loaded with <a href="/reference/p5/loadFont">loadFont()</a>.</p>

example:
  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Style the text.
      textSize(12);
      textAlign(CENTER);

      // First row.
      textStyle(NORMAL);
      text('Normal', 50, 15);

      // Second row.
      textStyle(ITALIC);
      text('Italic', 50, 40);

      // Third row.
      textStyle(BOLD);
      text('Bold', 50, 65);

      // Fourth row.
      textStyle(BOLDITALIC);
      text('Bold Italic', 50, 90);

      describe('The words "Normal" displayed normally, "Italic" in italic, "Bold" in bold, and "Bold Italic" in bold italics.');
    }
    </code>
