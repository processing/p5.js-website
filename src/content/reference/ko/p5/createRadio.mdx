description: |2-

  <p>라디오 버튼 요소를 생성합니다.</p>


  "<p>매개변수는 선택적(optional)입니다. <code>let myRadio = createSelect('food')</code>처럼 문자열이 전달된다면, 각 라디오 옵션은 <code>"food"</code>를 <code>name</code> 속성으로 가지게 됩니다. 다음처럼요": <code><input
  name="food"></input></code>.


  만약 <code>let myRadio = createSelect(container)</code>처럼 기존에 있던 <code><div></div></code> 또는 <code><span></span></code> 요소가 전달된다면, 그 요소는 라디오 버튼의 부모 요소가 됩니다.</p>


  <p>라디오 버튼은 옵션을 관리하는 몇 가지 유용한 메소드를 통해 <a href="/reference/p5/p5.Element">p5.Element</a> 클래스를 확장합니다.</p>


  <ul>


  <li><code>myRadio.option(value, [label])</code>는 메뉴에 옵션을 추가합니다.
  첫 번째 매개변수 <code>value</code>는 옵션의 값과 레이블을 설정하는 문자열입니다. 두 번째 매개변수 <code>label</code>는 선택적(optional)입니다. 만약 이 값이 전달되면 <code>value</code>에 해당하는 레이블을 설정합니다. 만약 <code>value</code>를 가진 옵션이 이미 존재한다면, 해당 옵션의 레이블을 덮어씌우고, 해당 옵션의 값을 반환합니다.</li>


  <li><code>myRadio.value()</code>는 현재 선택된 옵션의 값을 반환합니다.</li>


  <li><code>myRadio.selected()</code>는 현재 선택된 옵션을 반환합니다.</li>


  <li><code>myRadio.selected(value)</code>은 전달된 옵션을 선택하고 해당 옵션의 <a
  href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement"
  target="_blank"><code>HTMLInputElement</code></a>를 반환합니다.</li>


  <li><code>myRadio.disable(shouldDisable)</code>에 <code>true</code>가 전달된다면 전체 라디오 버튼을 활성화하고,  <code>false</code>가 전달된다면 전체 라디오 버튼을 비활성화합니다.
  </li>


  </ul>
