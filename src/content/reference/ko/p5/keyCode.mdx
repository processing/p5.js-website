description: >
  <p>마지막으로 입력된 키의 값을 가지는 <code>Number</code> 시스템 변수입니다.</p>

  <p>모든 키는 <code>keyCode</code>를 가집니다. 예를 들어, <code>a</code> 키는 <code>keyCode</code> 65를 가집니다.
  
  <code>keyCode</code> 변수는 특수 키가 입력되었는지 확인하는데 유용합니다. 예를 들어, 다음 조건문은 엔터 키가 입력되었는지 확인합니다:</p>


  <code>

  if (keyCode === 13) {
    // 엔터 키가 눌렸을 때 실행할 코드.
  }

  </code>


  <p><code>ENTER</code>라는 시스템 변수를 사용하면 같은 코드를 더 명확하게 작성할 수 있습니다. 이 변수의 값은 13입니다:</p>


  <code>

  if (keyCode === ENTER) {
    // 엔터 키가 눌렸을 때 실행할 코드.
  }

  </code>


  <p>시스템 변수 <code>BACKSPACE</code>, <code>DELETE</code>,
  <code>ENTER</code>, <code>RETURN</code>, <code>TAB</code>,

  <code>ESCAPE</code>, <code>SHIFT</code>, <code>CONTROL</code>,
  <code>OPTION</code>, <code>ALT</code>, <code>UP_ARROW</code>,
  <code>DOWN_ARROW</code>,

  <code>LEFT_ARROW</code>, 그리고 <code>RIGHT_ARROW</code>는 모두 특수 키 코드를 알아내는 데 도움이 되는 단축어입니다.
  
  키 코드는 <a href="http://keycode.info/">keycode.info</a>와 같은 웹사이트에서 찾을 수 있습니다.</p>
