  <p>숫자의 아크 탄젠트 값을 계산합니다.</p>

  <p><code>atan()</code>는 <a
  href="/reference/p5/tan">tan()</a>의 역함수입니다. -무한대에서 무한대 사이의 입력값을 기대합니다. 기본적으로, <code>atan()</code>
  은 -π ÷ 2 (약 -1.57)에서 π ÷ 2 (약 1.57) 사이의 값을 가집니다.
  <a href="/reference/p5/angleMode">angleMode()</a>가
  <code>DEGREES</code>일때는, -90부터 90 사이의 값으로 반환됩니다.</p>

    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // Calculate tan() and atan() values.
      let a = PI / 3;
      let t = tan(a);
      let at = atan(t);

      // Display the values.
      text(`${round(a, 3)}`, 35, 25);
      text(`${round(t, 3)}`, 35, 50);
      text(`${round(at, 3)}`, 35, 75);

    }
