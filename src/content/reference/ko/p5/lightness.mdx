description: >
  <p>색상의 밝기 값을 가져옵니다.</p>

  <p><code>lightness()</code> 함수는 

  <a href="/reference/p5/p5.Color">p5.Color</a> 객체, 색상 구성 요소의 배열,
  또는 CSS 색상 문자열에서

   HSL 색상의 밝기 값을 추출합니다.</p>

  <p>기본적으로, <code>lightness()</code> 함수는 색상의 HSL 밝기를 
  0부터

  100까지의 범위로 반환합니다. 만약 <a href="/reference/colorMode">colorMode()</a> 가 HSL로 설정되어 있다면,
  주어진 범위 내의

  밝기 값을 반환합니다.</p>

  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(50);

      // Use HSL color.
      colorMode(HSL);

      // Create a p5.Color object using HSL values.
      let c = color(0, 100, 75);

      // Draw the left rectangle.
      noStroke();
      fill(c);
      rect(15, 15, 35, 70);

      // Set 'lightValue' to 75.
      let lightValue = lightness(c);

      // Draw the right rectangle.
      fill(lightValue);
      rect(50, 15, 35, 70);

      describe('Two rectangles. The left one is salmon pink and the right one is gray.');
    }
