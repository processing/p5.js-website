## Introducción

Para comenzar a escribir código usando [p5.js](/reference/), necesitas configurar el entorno de programación que usarás para escribir y guardar tus programas. En esta guía:

1. Configurarás tu entorno de programación en la web usando el [Editor Web de p5.js](https://editor.p5js.org/) o en tu computadora usando [Visual Studio Code (VS Code).](https://code.visualstudio.com/)
2. Usarás la [referencia de p5.js](/reference/) para escribir tu primer programa sencillo con formas y color.
3. Guardarás y compartirás tu código.


## Lo que necesitarás

1. Acceso a internet y un navegador de escritorio como:
   - [Chrome](https://www.google.com/chrome/dr/download/?brand=SJWC\&geo=US\&gclid=Cj0KCQjw-pyqBhDmARIsAKd9XIM88u9RUlUagd7fpeqWx3AL_8VbhoIFULSME9tiz87t00L64-wLAlMaAm_QEALw_wcB\&gclsrc=aw.ds)
   - [Firefox](https://www.mozilla.org/en-US/firefox/)
   - [Safari](https://www.apple.com/safari/)
   - [Edge](https://www.microsoft.com/en-us/edge?ep=193\&form=MA13L2\&es=40)
2. Una computadora de escritorio, laptop o chromebook


## Paso 0: Elige el tipo de entorno de programación

Puedes comenzar tu recorrido de programación con [p5.js](/reference/) usando el [Editor Web de p5.js](https://editor.p5js.org/) o [VS Code](https://code.visualstudio.com/).

El [Editor Web de p5.js](https://editor.p5js.org/) es un sitio web donde las personas que programan pueden escribir, probar, compartir o remezclar programas de p5.js sin tener que descargar ni configurar un *editor de código* en una computadora. Un *editor de código* facilita escribir y leer código al organizar y colorear el texto de una manera que ayuda a distinguir las distintas partes del código.

[VS Code](https://code.visualstudio.com/) es un editor de código popular que instalas en tu computadora y que ofrece una experiencia de programación más avanzada. Según tus necesidades, elige el entorno de programación que sea mejor para ti.

- [Enlace a la guía del Editor Web de p5.js](#web-editor)
- [Enlace a la guía de VS Code](#vscode)

Si eres principiante en p5.js, comienza con el [Editor Web de p5.js](https://editor.p5js.org/).

### Paso 4: Nombra, guarda y ejecuta tu primer sketch

- Nombra tus proyectos haciendo clic en el ícono de lápiz sobre el editor de texto y escribiendo un nombre para tu proyecto.

![Una flecha apunta a un ícono de lápiz para cambiar el nombre de un sketch.](../images/introduction/p5_editor_naming_a_sketch.png)

- Guarda proyectos haciendo clic en *File* en la barra de herramientas superior y seleccionando *Save*.
  - Asegúrate de haber iniciado sesión en tu cuenta; de lo contrario, no podrás guardar el sketch.
  - Guardar los proyectos con frecuencia ayuda a evitar que se pierda código si algo le sucede a tu computadora, navegador o conexión a internet mientras programas.

![Una persona usuaria en el Editor Web de p5.js nombra un proyecto nuevo y lo guarda.](../images/introduction/p5_editor_saving_a_sketch.png)

![Una flecha apunta a un cuadro de notificación con el texto “Sketch saved.”](../images/introduction/p5_editor_saved_sketch_notification.png)

Para ver la salida de tu código, haz clic en el botón *Play* en la esquina superior izquierda:

![Un lienzo vacío aparece en la ventana de vista previa del Editor Web de p5.js cuando una persona usuaria hace clic en el botón “Play” en la esquina superior derecha.](../images/introduction/p5_editor_running_a_sketch.png)

Un *sketch de p5.js* es un archivo de texto con código escrito en el lenguaje de programación *JavaScript*. *JavaScript* es un lenguaje de programación que se usa para hacer que las páginas web sean interactivas. p5.js es una biblioteca escrita en *JavaScript*, por eso termina en “*.js*”, de *JavaScript*. Con p5.js, puedes crear programas coloridos y animados, con funciones con las que las personas usuarias pueden interactuar. Para aprender más sobre algunas de las cosas que puedes hacer con p5.js, mira el [video de bienvenida de p5.js](https://hello.p5js.org/). Para aprender más sobre JavaScript, puedes visitar [este recurso.](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

El editor comienza con el siguiente código en el archivo *sketch.js*:

```js
function setup() {
  createCanvas(400, 400);
}
function draw() {
  background(220);
}
```

El código anterior crea en la vista previa un elemento de lienzo de 400 píxeles de ancho y 400 píxeles de alto. También establece el fondo en un tono de gris.


#### `createCanvas()`

Las pantallas de computadora están formadas por luces diminutas llamadas *píxeles*, que son los elementos más pequeños que componen cualquier imagen. La línea de código que crea un lienzo en la ventana de vista previa es [`createCanvas(400, 400)`](/reference/p5/createCanvas). Sin esta línea de código, no habrá un lienzo donde dibujar. Los números 400, 400 corresponden al ancho y la altura del lienzo en píxeles. Estos números también se conocen como *argumentos* de la función [`createCanvas()`](/reference/p5/createCanvas).

Los valores que colocas dentro de los paréntesis de una función se llaman *argumentos*: valores que se usan para personalizar funciones. [`createCanvas()`](/reference/p5/createCanvas) aparece dentro de la función [`setup()`](/reference/p5/setup) para crear un elemento de lienzo HTML en el que puedes dibujar.

Para aprender más, visita las páginas de la [referencia de p5.js](/reference) sobre [`setup()`](/reference/p5/setup) y [`createCanvas()`](/reference/p5/createCanvas).


### Paso 5: Cambia el color del lienzo

- Puedes escribir comandos para cualquier sketch en el editor de texto y ver la salida de tu código en la ventana de vista previa haciendo clic en *Play*.
- Cambia el color de fondo de tu lienzo modificando el *argumento* de la función [`background()`](/reference/p5/background).
  - Cambia `background(220);` por `background("aqua");` y presiona *Play*.

```js
function setup() {
  createCanvas(400, 400);
}
function draw() {
  background("aqua");
}
```


#### `background()`

La función `background()` colorea el fondo con un color específico. Puedes usar palabras entre comillas o números para colorear el fondo del lienzo. Para aprender más, visita las páginas de la [referencia de p5.js](/reference) sobre [`background()`](/reference/p5/background) y [color](/reference/p5/color).


### Paso 6: Dibuja formas en el lienzo

- Dibuja un círculo en el lienzo.
  - Agrega el siguiente texto debajo de la función [`background()`](/reference/p5/background):

    ```js
    //círculo en el centro con un ancho de 100
    circle(200,200,100);
    ```

- Haz clic en el botón *Play*.

Tu código debería verse así:

```js
function setup() {
  createCanvas(400, 400);
}
function draw() {
  background(220);

  // Dibuja un círculo en el centro con
  // un ancho de 100.
  circle(200, 200, 100);
}
```

#### `draw()`

Puedes dibujar formas en el lienzo escribiendo comandos específicos para formas dentro de las llaves `{}` después de [`function draw()`](/reference/p5/draw).

El sketch anterior dibuja un círculo en el lienzo al llamar a la función [`circle()`](/reference/p5/circle) dentro de [`draw()`](/reference/p5/draw). Los dos primeros *argumentos*, `200, 200`, colocan el círculo en el centro del lienzo, y el último *argumento*, `100`, indica que el círculo tiene 100 píxeles de ancho. Los comentarios incluidos en el sketch, en las líneas sobre la función [`circle()`](/reference/p5/circle), describen lo que hace el código. Para aprender más, visita las páginas de la [referencia de p5.js](/reference/) sobre [`draw()`](/reference/p5/draw) y [`circle()`](/reference/p5/circle).


### Paso 7: ¡Crea!

p5.js tiene muchas funciones que puedes usar para incorporar elementos estáticos e interactivos en tu lienzo.

- Reemplaza los comandos de fondo y círculo con el siguiente texto:

  ```js
  //cuando se presiona el botón del mouse, los círculos se vuelven negros
  if (mouseIsPressed === true) {
    fill(0);
  } else {
    fill(255);
  }

  //círculos blancos dibujados en la posición del mouse
  circle(mouseX, mouseY, 100);
  ```

Tu código debería verse así:

<EditableSketch code={`
function setup() {
  createCanvas(400, 400);
}
function draw() {
  //cuando se presiona el botón del mouse, los círculos se vuelven negros
  if (mouseIsPressed === true) {
    fill(0);
  } else {
    fill(255);
  }

  //círculos blancos dibujados en la posición del mouse
  circle(mouseX, mouseY, 100);
}
`} />

<Callout>
Mantén presionado el botón del mouse mientras arrastras el puntero sobre el lienzo.
</Callout>


El código anterior dibuja círculos blancos en la misma posición que el puntero del mouse. Cuando se presiona un botón del mouse, el color del círculo cambia a negro.

Visita la [referencia de p5.js](/reference/) para conocer más funciones de p5.js, como [formas primitivas 2D](/reference/#Shape/).


### Errores

Es fácil escribir mal los nombres de las funciones u olvidar una coma. Las reglas de sintaxis ayudan a la computadora a interpretar el código. Cuando se rompe una “regla”, aparece un mensaje de error en la consola (por ejemplo, si [`circle()`](/reference/p5/circle) está escrito incorrectamente). Estos errores se conocen comúnmente como “bugs”. La consola muestra mensajes del editor con detalles sobre cualquier error que hayas cometido. Si tu código no se ejecuta correctamente, podría haber un bug en tu código.

Visita la [Guía de campo para depurar](/tutorials/field-guide-to-debugging) para obtener más información sobre cómo corregir errores en tu código.


#### Nota de accesibilidad:

Si usas un lector de pantalla, debes activar las salidas accesibles en el [Editor Web de p5.js](https://editor.p5js.org/) y agregar la biblioteca de accesibilidad en tu HTML. Para aprender más, visita nuestra guía sobre [cómo usar el Editor Web de p5.js con un lector de pantalla.](/tutorials/p5js-with-screen-reader/)

### Paso 8: Comparte tu proyecto

Una vez que tu proyecto esté guardado, puedes compartirlo.

- Comparte proyectos haciendo clic en *File* en la barra de herramientas superior, seleccionando *Share* y copiando uno de los enlaces proporcionados. Puedes compartir tus proyectos de tres maneras diferentes:
  - Embed: Agrega tu sketch de p5.js a un sitio web o blog que use HTML (no muestra el código).
  - Fullscreen: Comparte tu proyecto con un enlace (no muestra el código).
  - Edit: Comparte el código de tu proyecto en el editor de p5.js con un enlace.

## Uso de VS Code<a id="vscode"></a>

[VS Code](https://code.visualstudio.com/) ofrece una experiencia de edición de código más avanzada en comparación con el Editor Web. Recomendamos esta opción si ya estás familiarizado con VS Code.


### Paso 1: Descarga VS Code

- Descarga VS Code en tu dispositivo usando [este enlace](https://code.visualstudio.com/download).
- Explora las funciones de VS Code y sus distintas opciones de configuración usando [estos recursos](https://code.visualstudio.com/docs/getstarted/introvideos).

- Abre VS Code y ve al administrador de extensiones en la barra de herramientas izquierda.
- Escribe [*p5.js 2.x Project Generator*](https://marketplace.visualstudio.com/items?itemName=Irti.p5js-project-generator) en la barra de búsqueda, selecciona la extensión y haz clic en el botón de instalación.

### Paso 3: Crea un proyecto de p5.js

- Sigue las indicaciones para configurar tu proyecto.
- Selecciona la carpeta de tu computadora donde quieres guardar tu proyecto.
