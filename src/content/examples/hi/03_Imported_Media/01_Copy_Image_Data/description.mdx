title: छवि डेटा कॉपी करें

---

<a href='https://p5js.org/reference/#/p5/copy' target='_blank'>copy()</a> विधि का उपयोग करके, आप किसी छवि को रंगने की प्रक्रिया का अनुकरण कर सकते हैं। यह काले और सफेद छवि के शीर्ष पर जहां भी कर्सर खींचा जाता है, रंगीन छवि की एक छवि की प्रतिलिपि बनाकर किया जाता है।
