---

<a href='https://p5js.org/reference/p5.Element/drop' target='_blank'>drop()</a> एक p5.js तत्व विधि है जो हर बार कॉलबैक फ़ंक्शन को कॉल करती है जब एक फ़ाइल तत्व में लोड करना पूरा करती है। 
अपलोड की गई फ़ाइल एक <a href='https://p5js.org/reference/p5.File' target='_blank'>p5.File</a> क्लास में बनाई गई है।
आप फ़ाइल प्रकार की जांच करने के लिए <a href='https://p5js.org/reference/p5.Element/drop' target='_blank'>drop()</a> कॉलबैक का उपयोग कर सकते हैं, और फिर विभिन्न फ़ाइल प्रकारों पर अलग-अलग प्रतिक्रिया देने के लिए सशर्त विवरण लिख सकते हैं।
