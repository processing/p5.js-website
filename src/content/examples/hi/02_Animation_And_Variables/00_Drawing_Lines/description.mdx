एक रेखा खींचने के लिए माउस को क्लिक करें और खींचें।

यह उदाहरण कई बिल्ट-इन वेरिएबल के उपयोग को प्रदर्शित करता है।
<a href='https://p5js.org/reference/p5/mouseX'>mouseX</a> और <a href='https://p5js.org/reference/p5/mouseY'> mouseY</a> वर्तमान माउस स्थिति को संग्रहीत करता है, जबकि पिछली माउस स्थिति को <a href="https://p5js.org/reference/p5/pmouseX">pmouseX</a> और <a href="https://p5js.org/reference/p5/pmouseY">pmouseY</a> द्वारा दर्शाया जाता है।

 *

यह उदाहरण HSB (रंग-संतृप्ति-चमक) के साथ <a href="https://p5js.org/reference/p5/colorMode">colorMode</a> का उपयोग भी दिखाता है, ताकि पहला वेरिएबल रंग सेट कर पाये।
