इस उदाहरण में, <a href='https://p5js.org/reference/p5/accelerationX' target='_blank'>accelerationX</a>, <a href='https://p5js.org/reference/p5/accelerationY' target='_blank'>accelerationY</a>, और <a href='https://p5js.org/reference/p5/accelerationZ' target='_blank'>accelerationZ </a> मान वृत्त की स्थिति और आकार निर्धारित करते हैं। यह केवल मोबाइल उपकरणों के लिए काम करता है।
