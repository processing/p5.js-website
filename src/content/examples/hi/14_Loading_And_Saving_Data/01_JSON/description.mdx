<a href='https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/JSON' target='_blank'>जावास्क्रिप्ट ऑब्जेक्ट नोटेशन, या JSON</a>,
 एक प्रारूप है किसी फ़ाइल में डेटा लिखना. जबकि सिंटैक्स जावास्क्रिप्ट से आता है, JSON का उपयोग कई अन्य संदर्भों में किया जाता है। 
 यह उदाहरण जावा में लिखे डैनियल शिफमैन के लोडिंग JSON डेटा उदाहरण फॉर प्रोसेसिंग पर आधारित है। यह एक 
 <a href='https://p5js.org/reference/p5/class' target='_blank'>class</a> का उपयोग करता है
एक बुलबुले के लिए डेटा व्यवस्थित करने के लिए। जब स्केच शुरू होता है, तो यह JSON फ़ाइल से दो बबल के लिए डेटा लोड करता है। 
विज़िटर नए बबल जोड़ सकता है, एक अद्यतन JSON फ़ाइल डाउनलोड कर सकता है और JSON फ़ाइल में लोड कर सकता है।
