title: 画布上的视频
oneLineDescription: 在画布上显示并设置视频样式。
---

通过使用 <a href="https://p5js.org/reference/p5/createVideo" target="_blank">createVideo()</a> 和 <a href="https://p5js.org/reference/p5/image" target="_blank">image()</a> 函数，你可以将视频上传到画布中。由于视频捕捉是通过 <a href="https://p5js.org/reference/p5/image" target="_blank">image()</a> 构造函数传递的，你可以使用 <a href="https://p5js.org/reference/p5/filter" target="_blank">filter()</a> 方法给视频捕捉添加滤镜。要在本地运行这个示例，你需要一个正在运行的 <a href="https://github.com/processing/p5.js/wiki/Local-server">本地服务器</a>。如果想要构建一个不在画布内嵌入的视频，请参见 <a href="https://p5js.org/examples/imported-media-video/">视频</a> 示例。
