title: 透明蒙版
oneLineDescription: 使用一张图像来裁除另一张图像中的对应部分。
---

通过使用 
<a href="https://p5js.org/reference/p5/mask" target="_blank">mask()</a> 方法，
你可以为图像创建一个蒙版，
以指定图像不同部分的透明度。若要在本地运行这个示例，
你需要两个图像文件和一个正在运行的
<a href="https://github.com/processing/p5.js/wiki/Local-server">本地服务器</a>。
