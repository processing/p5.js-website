title: 图像透明度
oneLineDescription: 将画布上的一个图像设为半透明。
---

本程序通过使用 
<a href="https://p5js.org/reference/p5/tint" target="_blank">tint()</a> 函数
修改图像的 alpha 值，将一张图像覆盖在另一张图像上。
在画布上左右移动光标可以改变图像的位置。
若要在本地运行这个示例，
你需要一个图像文件和一个正在运行的
<a href="https://github.com/processing/p5.js/wiki/Local-server">本地服务器</a>。
