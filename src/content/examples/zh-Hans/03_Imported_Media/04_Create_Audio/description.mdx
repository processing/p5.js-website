<a href="https://p5js.org/reference/p5/createAudio" target="_blank">createAudio()</a> 
用于创建一个音频播放器。本示例展示了播放器的控制按钮，并可调整播放速度。当鼠标在窗口的左边缘时，播放速度是正常的，而随着鼠标向右移动，播放速度会变快。 有关使用音频播放器等媒体元素的更多信息，请参见 <a href="https://p5js.org/reference/p5.MediaElement" target="_blank">p5.MediaElement</a> 参考页面。此处使用的音频文件是一段 <a href="https://freesound.org/people/josefpres/sounds/711156/" target="_blank">公共版权的 Josef Pres 钢琴曲循环</a>。
