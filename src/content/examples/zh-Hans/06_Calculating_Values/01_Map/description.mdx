<a href="https://p5js.org/reference/#/p5/map" target="_blank">map()</a>
函数将一个范围内的值转换到另一个范围。在这个例子中，map
将光标的水平位置从 0-720 的范围转换到 0-360 。最终结果为圆形的色调。Map 还将光标的
垂直位置从 0-400 的范围转换到 20-300。最终结果为圆形的直径。
