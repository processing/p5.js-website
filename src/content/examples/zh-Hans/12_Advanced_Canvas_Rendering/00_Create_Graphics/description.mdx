<a href="https://p5js.org/reference/p5/createGraphics" target="_blank">createGraphics()</a>
函数可用于创建一个新的 p5.Graphics 对象，它可以作为画布内的一个屏幕外图形缓冲区。屏幕外缓冲区可以具有与当前显示表面不同的尺寸和属性，尽管它们似乎存在于同一空间中。
