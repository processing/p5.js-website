---

Comma-Separated Values (CSV) 是一种在文件中编写数据的格式。p5 可以使用 
<a href="https://p5js.org/reference/p5.Table" target="_blank">p5.Table</a> 处理这种格式。
这个例子基于 Daniel Shiffman 为 Processing 创建的 
<a href="https://processing.org/examples/loadsavetable.html" target="_blank">加载表格数据</a> 
的例子。它使用一个类来组织代表气泡的数据。当绘图启动时，它从一个 CSV 文件中加载了四个气泡的数据。访问者可以添加新的气泡，下载更新后的 CSV 文件，并加载一个 CSV 文件。
