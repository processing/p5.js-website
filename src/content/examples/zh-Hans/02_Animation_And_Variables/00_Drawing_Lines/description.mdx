oneLineDescription: 用鼠标绘制线条

可以通过点击并拖拽鼠标来绘制线条。

本示例展示了几个内置变量的使用方法。<a href="https://p5js.org/reference/#/p5/mouseX">mouseX</a> 和 <a href="https://p5js.org/reference/#/p5/mouseY">mouseY</a>
可以存储当前的鼠标位置，而 <a href="https://p5js.org/reference/#/p5/pmouseX">pmouseX</a> 和 <a href="https://p5js.org/reference/#/p5/pmouseY">pmouseY</a>
代表上一次的鼠标位置。
 *这个示例也展示了 HSB（色调-饱和度-亮度）颜色模式 <a href="https://p5js.org/reference/#/p5/colorMode">colorMode</a> 的使用方法，所以第一个变量用来设置色调。
