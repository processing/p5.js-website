基于 Dan Shiffman 为 Processing 创建的原始 
<a href="https://natureofcode.com/book/chapter-4-particle-systems/">例子</a>，
来演示烟雾粒子系统。

这段代码使用了 
<a href="https://p5js.org/reference/p5.Vector">p5.Vector</a> 
类，包括 
<a href="https://p5js.org/reference/p5/createVector">createVector()</a> 
函数。使用 p5.Vector 方法执行更新粒子位置和速度的各种计算。

粒子系统被创建为一个 
<a href="https://p5js.org/reference/p5/class">类</a>，其中包含一个对象数组（属于 Particle 类）。
