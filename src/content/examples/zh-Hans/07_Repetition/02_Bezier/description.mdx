<a href="https://p5js.org/reference/p5/bezier" target="_blank">bezier()</a> 曲线是使用控制点和锚点创建的。对于 <a href="https://p5js.org/reference/p5/bezier" target="_blank">bezier()</a> 函数，前两个参数指定曲线的第一个点，最后两个参数指定最后一个点。中间的参数设置控制点，这些控制点定义了曲线的形状。
