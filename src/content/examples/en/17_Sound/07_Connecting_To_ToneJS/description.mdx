At some point you might want to connect your p5 sound sketch to another library like Tone.js, for processing. A common way sound libraries connect to each other is by sharing an 'AudioContext.'

Think of an AudioContext like a Canvas (or drawing context). Anything drawn to the same Canvas is viewed in the same window. When working with JavaScript sound libraries a similar paradigm exists. Anything using the same AudioContext can be connected to each other and used in the same browser window.
