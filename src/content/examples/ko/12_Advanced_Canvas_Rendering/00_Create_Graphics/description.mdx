<a href="https://p5js.org/reference/#/p5/createGraphics" target="_blank">createGraphics()</a> 함수는
새로운 p5.Graphics 객체를 생성하는 데 사용됩니다. 이 객체는 캔버스 내에서 오프-스크린 그래픽 버퍼(off-screen graphic buffer)로 사용될 수 있습니다.
오프-스크린 버퍼(off-screen buffer)는 현재 디스플레이 표면과 다른 차원과 속성을 가질 수 있음에도 불구하고 같은 공간에 존재하는 것처럼 보입니다.
