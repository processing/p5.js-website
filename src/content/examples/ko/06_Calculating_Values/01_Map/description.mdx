title: 매핑(Map)
oneLineDescription: 한 범위에서 다른 범위로 숫자를 변환합니다.
---

<a href="https://p5js.org/reference/p5/map" target="_blank">map()</a> 함수는 값을 하나의 범위에서 다른 범위로 변환합니다. 이 예제에서, 매핑은 커서의 수평 위치를 0-720의 범위에서 0-360 범위로 변환합니다. 결과값은 원의 색상이 됩니다. 또한 매핑은 커서의 수직 위치를 0-400의 범위에서 20-300의 범위로 변환합니다. 결과값은 원의 지름이 됩니다.
