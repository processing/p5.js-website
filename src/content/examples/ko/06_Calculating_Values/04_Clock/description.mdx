현재 시각은 <a href="https://p5js.org/reference/p5/second" target="_blank">second()</a>,
<a href="https://p5js.org/reference/p5/minute" target="_blank">minute()</a>,
그리고 <a href="https://p5js.org/reference/p5/minute" target="_blank">hour()</a> 함수를 사용하여 읽을 수 있습니다.
이 예제는 <a href="https://p5js.org/reference/p5/map" target="_blank">map()</a> 함수를 사용하여 시계 바늘의 각도를 계산합니다. 그런 다음 <a href="https://p5js.org/reference/#group-Transform" target="_blank">좌표계 변환</a>을 사용하여 그 위치를 설정합니다.
