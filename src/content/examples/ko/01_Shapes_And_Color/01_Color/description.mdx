색상을 추가하여 기본 조형 예제를 확장해 봅시다.
<a href="https://p5js.org/reference/p5/background" target="_blank">background()</a>는 캔버스를 하나의 색상으로 채우고,
<a href="https://p5js.org/reference/p5/stroke" target="_blank">stroke()</a>는 그려지는 윤곽선의 색상을 정하며,
<a href="https://p5js.org/reference/p5/fill" target="_blank">fill()</a>은 도형 내부의 색상을 정합니다.
<a href="https://p5js.org/reference/p5/noFill" target="_blank">noStroke()</a>와
<a href="https://p5js.org/reference/p5/noFill" target="_blank">noFill()</a>은 각각 윤곽선의 색상과 채우기 색상을 없앨 수 있습니다. 

색상은 다양한 방법으로 표현될 수 있으며, 이 예제는 몇 가지의 옵션을 보여주고 있습니다.
