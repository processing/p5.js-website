oneLineDescription: 입자 시스템(particle system)을 활용하여 연기를 시뮬레이션합니다.
---

다니엘 쉬프만(Dan Shiffman)의 원본 Processing 
<a href="https://natureofcode.com/book/chapter-4-particle-systems/">예제</a>를 기반으로 한 연기 입자 시스템 데모입니다.

이 코드는 <a href="https://p5js.org/reference/#/p5.Vector">p5.Vector</a> 클래스를 활용하며, 
이는 <a href="https://p5js.org/reference/#/p5/createVector">createVector()</a> 함수를 포함합니다. 
입자의 위치와 속도를 업데이트하는 다양한 계산은 p5.Vector 메서드를 사용하여 수행됩니다.

입자 시스템은 <a href="https://p5js.org/reference/#/p5/class">클래스</a>로 구현되어 있으며, 
이는 (Particle 클래스의) 객체 배열을 포함합니다.
