title: 폼 요소

---

<a href="https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model" target="_blank">문서 객체 모델 (DOM)</a>은 웹 페이지의 결과로 나오는 구조를 나타냅니다.
<a href="https://p5js.org/reference/p5/createInput" target="_blank">createInput()</a>,
<a href="https://p5js.org/reference/p5/createSelect" target="_blank">createSelect()</a>,
그리고 <a href="https://p5js.org/reference/p5/createRadio" target="_blank">createRadio()</a>와 같은 p5.js의 폼 요소를 사용하여 <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select" target="_blank">[드롭다운] 메뉴 선택</a>, <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input" target="_blank">입력칸</a> 또는 <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio" target="_blank">라디오 버튼</a>을 통해 제출된 정보를 갖는 다양한 방법을 구축하고, 그 정보를 기반으로 DOM을 업데이트할 수 있습니다.
