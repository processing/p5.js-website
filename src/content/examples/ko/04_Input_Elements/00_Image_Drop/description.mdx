<a href="https://p5js.org/reference/#/p5.Element/drop" target="_blank">drop()</a>은 p5.js 요소 메서드로, 파일이 요소에 로드될 때마다 콜백을 등록합니다. 업로드된 파일은 <a href="https://p5js.org/reference/#/p5.File" target="_blank">p5.File</a>로 만들어집니다. 
<a href="https://p5js.org/reference/#/p5.Element/drop" target="_blank">drop()</a> 콜백을 사용하여 파일 유형을 확인한 다음 각 파일 유형에 대응하는 조건문을 작성할 수 있습니다.
