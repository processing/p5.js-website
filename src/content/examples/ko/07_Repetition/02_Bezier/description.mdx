<a href="https://p5js.org/reference/p5/bezier" target="_blank">bezier()</a> 곡선은 제어점과 고정점을 이용하여 생성됩니다.
<a href="https://p5js.org/reference/p5/bezier" target="_blank">bezier()</a> 함수의 첫 두 매개변수는 곡선의 첫 번째 점을 지정하고, 마지막 두 매개변수는 마지막 점을 지정합니다. 중간 매개변수는 곡선 모양을 정의하는 제어점입니다.
