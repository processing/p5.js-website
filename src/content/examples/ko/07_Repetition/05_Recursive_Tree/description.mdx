이것은 재귀를 통해 간단한 트리같은 구조를 렌더링하는 예제입니다.
분기 각도는 마우스의 수평 위치에 대한 함수로 계산됩니다. 마우스를 왼쪽과 오른쪽으로 움직여 각도를 변경하세요. 다니엘 쉬프먼의 Processing에서 작성된 <a href="https://processing.org/examples/tree.html">재귀 트리 예제</a>를 바탕으로 합니다.
