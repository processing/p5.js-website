<a href="https://en.wikipedia.org/wiki/Perlin_noise" target="_blank">펄린 노이즈</a>
는 켄 펄린(Ken Perlin)이 작성한 알고리즘으로, 무작위성과 유기성을 모두 지닌 시퀀스입니다.
p5의 <a href="https://p5js.org/reference/p5/noise" target="_blank">noise()</a>함수는 펄린 노이즈를 만듭니다. 이 예제에서 점들은 노이즈 값에 기반하여 크기가 결정됩니다. 왼쪽의 슬라이더를 이용해 점들 사이의 거리를 설정합니다. 오른쪽 슬라이더를 이용해 노이즈 계산에서의 오프셋(offset) 값을 설정합니다.
