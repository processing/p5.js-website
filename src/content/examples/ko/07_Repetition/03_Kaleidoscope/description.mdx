만화경은 두 개 이상의 반사면이 서로를 향해 비스듬히 기울어진 광학 기기입니다.
<a href="https://p5js.org/reference/#/p5/translate" target="_blank">translate()</a>,
<a href="https://p5js.org/reference/#/p5/rotate" target="_blank">rotate()</a>,
그리고 <a href="https://p5js.org/reference/#/p5/scale" target="_blank">scale()</a> 함수를 사용하면 캔버스에서 만화경을 통해 만든 비주얼을 복제할 수 있습니다.
