<a href="https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/JSON" target="_blank">JavaScript Object Notation, 또는 JSON</a>은 
파일에 데이터를 작성하는 형식입니다.
구문은 JavaScript에서 가져왔지만, JSON은 다른 맥락에서도 많이 사용됩니다. 
이 예제는 다니엘 쉬프만(Daniel Shiffman)의 Java로 작성된 Processing용 JSON 데이터 로드 예제를 기반으로 합니다. 
이는 버블에 대한 데이터를 구성하기 위해 
<a href="https://p5js.org/reference/p5/class" target="_blank">클래스</a>를 사용합니다.
스케치가 시작될 때, 두 개의 버블에 대한 데이터를 JSON 파일에서 로드합니다. 
방문자는 새로운 버블을 추가할 수 있고, 업데이트된 JSON 파일을 다운로드하고, JSON 파일을 로드할 수 있습니다.
