쉼표로 구분된 값(CSV)은 파일에 데이터를 작성하는 형식입니다.
p5는 <a href="https://p5js.org/reference/p5.Table" target="_blank">p5.Table</a>을 사용하여 이 형식을 작업할 수 있습니다. 
이 예제는 다니엘 쉬프만(Daniel Shiffman)의 
Processing에서 작성된 <a href="https://processing.org/examples/loadsavetable.html" target="_blank">테이블 데이터 가져오기(Loading Tabular Data)</a> 예제를 기반으로 합니다. 
이는 버블을 나타내는 데이터를 구성하기 위해 클래스를 사용합니다. 
스케치가 시작될 때, 네 개의 버블에 대한 데이터를 CSV 파일에서 로드합니다. 
방문자는 새로운 버블을 추가할 수 있고, 업데이트된 CSV 파일을 다운로드하고, CSV 파일을 로드할 수 있습니다.
