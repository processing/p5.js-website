title: 로컬 스토리지(Local Storage)
one-line-description: 브라우저에서 기기에 데이터를 저장합니다.
---

브라우저는 웹사이트가 방문자의 기기에 데이터를 저장할 수 있도록 허용하고 있습니다. 
이를 <a href="https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage" target="_blank">로컬 스토리지</a>라고 합니다.
<a href="https://p5js.org/reference/#/p5/getItem" target="_blank">getItem()</a>, 
<a href="https://p5js.org/reference/#/p5/storeItem" target="_blank">storeItem()</a>, 
<a href="https://p5js.org/reference/#/p5/clearStorage" target="_blank">clearStorage()</a>, 
그리고 <a href="https://p5js.org/reference/#/p5/removeItem" target="_blank">removeItem()</a> 함수들이 이를 제어합니다.

이 예제는 다니엘 쉬프만(Daniel Shiffman)의 Java로 작성된 
로딩 JSON 데이터(Loading JSON Data)와 로딩 테이블 데이터(Loading Tabular Data) 예제를 참고하여 만들어졌습니다.
이는 버블에 대한 데이터를 구성하기 위해 
<a href="https://p5js.org/reference/#/p5/class" target="_blank">클래스</a>를 사용합니다. 
방문자는 새로운 버블을 추가할 수 있으며, 그들의 데이터는 로컬 스토리지에 저장됩니다.
방문자가 스케치를 다시 방문하면, 동일한 버블이 다시 로드됩니다.
