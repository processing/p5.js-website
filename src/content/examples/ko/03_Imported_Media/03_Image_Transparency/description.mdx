title: 이미지 투명도
one-line-description: 캔버스에 이미지를 반투명하게 만듭니다.
---

이 프로그램은 <a href="https://p5js.org/reference/#/p5/tint" target="_blank">tint()</a> 함수로 이미지의 알파 값을 수정하여 한 이미지를 다른 이미지 위에 덮어씌웁니다. 이미지의 위치를 변경하려면 캔버스를 가로질러 커서를 좌우로 이동합니다. 이 예제를 로컬에서 실행하려면 이미지 파일과 실행 중인 <a href="https://github.com/processing/p5.js/wiki/Local-server">로컬 서버</a>가 필요합니다.
