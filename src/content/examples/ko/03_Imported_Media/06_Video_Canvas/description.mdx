title: 캔버스에서의 비디오
oneLineDescription: 캔버스에서 비디오를 표시하고 스타일을 지정합니다.
---

<a href="https://p5js.org/reference/p5/createVideo" target="_blank">createVideo()</a>와
<a href="https://p5js.org/reference/p5/image" target="_blank">image()</a> 함수를 사용하면 비디오를 캔버스에 업로드할 수 있습니다.
비디오 캡처는 <a href="https://p5js.org/reference/p5/image" target="_blank">image()</a> 생성자를 통해 전달될 수 있으므로
<a href="https://p5js.org/reference/p5/filter" target="_blank">filter()</a> 메서드를 사용하여 비디오 캡처에 필터를 추가할 수 있습니다.
이 예제를 로컬에서 실행하려면 실행 중인 <a href="https://github.com/processing/p5.js/wiki/Local-server">로컬 서버</a>가 필요합니다. 
비디오를 캔버스 내에 내장하지 않고 구축하려면 <a href="https://p5js.org/examples/imported-media-video/">Video</a> 예제를 확인해 보세요.
