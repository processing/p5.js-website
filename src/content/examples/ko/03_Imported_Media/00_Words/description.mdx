<a href="https://p5js.org/reference/#/p5/text" target="_blank">text()</a> 함수는 캔버스에 글자를 삽입하는 데 사용됩니다. 
<a href="https://p5js.org/reference/#/p5/loadFont" target="_blank">loadFont()</a> 및 
<a href="https://p5js.org/reference/#/p5/fontSize" target="_blank">fontSize()</a> 함수를 사용하여 글꼴 및 글자 크기를 변경할 수 있습니다.
<a href="https://p5js.org/reference/#/p5/textAlign" target="_blank">textAlign()</a> 함수를 사용하여 왼쪽, 가운데 또는 오른쪽으로 정렬할 수 있으며 도형에서와 같이 <a href="https://p5js.org/reference/#/p5/fill" target="_blank">fill()</a>을 이용하여 글자에 색상을 입힐 수 있습니다.
