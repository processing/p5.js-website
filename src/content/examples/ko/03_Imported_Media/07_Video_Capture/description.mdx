title: 비디오 캡처
one-line-description: 카메라의 라이브 비디오 피드를 표시합니다.
---

<a href="https://p5js.org/reference/#/p5/createCapture" target="_blank">createCapture()</a>와
<a href="https://p5js.org/reference/#/p5/image" target="_blank">image()</a> 함수를 사용하면 장치의 비디오 캡처를 가져와 캔버스에 그릴 수 있습니다.
비디오 캡처는 <a href="https://p5js.org/reference/#/p5/image" target="_blank">image()</a> 생성자를 통해 전달될 수 있으므로
<a href="https://p5js.org/reference/#/p5/filter" target="_blank">filter()</a> 메서드를 사용하여 비디오 캡처에 필터를 추가할 수 있습니다.
비디오를 업로드하거나, 발표하거나, 스타일을 지정하기 위한 다양한 전략을 보려면 
<a href="https://p5js.org/examples/dom-video.html" target="_blank">Video</a>와
<a href="https://p5js.org/examples/dom-video-canvas.html" target="_blank">Video Canvas</a>를 확인해 보세요.
