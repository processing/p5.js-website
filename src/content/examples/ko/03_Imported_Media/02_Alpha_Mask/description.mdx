title: 알파 마스크
oneLineDescription: 한 이미지를 사용해 다른 이미지의 일부를 잘라냅니다.
---

<a href="https://p5js.org/reference/p5/mask" target="_blank">mask()</a> 메서드를 사용하면 이미지의 마스크를 만들어 이미지의 다른 부분에서 투명도를 지정할 수 있습니다.
이 예제를 로컬에서 실행하려면 두 개의 이미지 파일과 실행 중인 <a href="https://github.com/processing/p5.js/wiki/Local-server">로컬 서버</a>가 필요합니다.
