Este expande el ejemplo de Primitivas de Formas añadiendo 
color.
<a href="https://p5js.org/reference/p5/background" target="_blank">background()</a>
llena el lienzo con un color,
<a href="https://p5js.org/reference/p5/stroke" target="_blank">stroke()</a>
establece el color para dibujar líneas, y
<a href="https://p5js.org/reference/p5/fill" target="_blank">fill()</a>
establece el color para el interior de las formas.
<a href="https://p5js.org/reference/p5/noFill" target="_blank">noStroke()</a> y
<a href="https://p5js.org/reference/p5/noFill" target="_blank">noFill()</a>
apagan el color de línea y el color interior, respectivamente.

Los colores pueden ser representados de muchas formas diferentes. Este ejemplo demuestra varias opciones.
