<a href="https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/JSON" target="_blank">JavaScript Object Notation, o JSON</a>,
es un formato para escribir datos en un archivo. Aunque la sintaxis proviene de
JavaScript, JSON se utiliza en muchos otros contextos. Este ejemplo está
basado en el Ejemplo de Carga de Datos JSON de Daniel Shiffman para Processing
escrito en Java. Utiliza una
<a href="https://p5js.org/reference/#/p5/class" target="_blank">clase</a>
para organizar los datos de una burbuja. Cuando el boceto comienza, carga
los datos de dos burbujas desde un archivo JSON. El visitante puede agregar
nuevas burbujas, descargar un archivo JSON actualizado y cargar un archivo JSON.
