Los navegadores permiten a los sitios web almacenar datos en el dispositivo del visitante.
Esto se llama
<a href="https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage" target="_blank">almacenamiento local</a>.
Las funciones
<a href="https://p5js.org/reference/p5/getItem" target="_blank">getItem()</a>,
<a href="https://p5js.org/reference/p5/storeItem" target="_blank">storeItem()</a>,
<a href="https://p5js.org/reference/p5/clearStorage" target="_blank">clearStorage()</a> y
<a href="https://p5js.org/reference/p5/removeItem" target="_blank">removeItem()</a>
controlan esto.

Este ejemplo está inspirado en los ejemplos de Daniel Shiffman de Carga de Datos JSON y Carga de
Datos Tabulares para Processing escritos en Java. Utiliza una
<a href="https://p5js.org/reference/p5/class" target="_blank">clase</a>
para organizar los datos de una burbuja. El visitante puede agregar nuevas burbujas y sus datos
se guardarán en el almacenamiento local. Si el visitante vuelve a visitar el boceto, se cargarán
las mismas burbujas.
