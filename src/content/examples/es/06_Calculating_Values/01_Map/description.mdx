title: Map
oneLineDescription: Convierte un número de un rango a otro rango.
---

La función <a href="https://p5js.org/reference/p5/map" target="_blank">map()</a> 
convierte un valor de un rango a otro. En este ejemplo, map convierte la posición 
horizontal del cursor de un rango de 0 a 720 a un rango de 0 a 360. El valor 
resultante se convierte en el tono del círculo. Map también convierte 
la posición vertical del cursor de un rango de 0 a 400 a un rango de 20 a 300. 
El valor resultante se convierte en el diámetro del círculo.
