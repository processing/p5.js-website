La hora actual se puede obtener con las funciones
<a href="https://p5js.org/reference/p5/second" target="_blank">second()</a>,
<a href="https://p5js.org/reference/p5/minute" target="_blank">minute()</a> y
<a href="https://p5js.org/reference/p5/minute" target="_blank">hour()</a>.
Este ejemplo utiliza
<a href="https://p5js.org/reference/p5/map" target="_blank">map()</a>
para calcular el ángulo de las manecillas. Luego utiliza
<a href="https://p5js.org/reference/#group-Transform" target="_blank">transformaciones</a>
para establecer su posición.
