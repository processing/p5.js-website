---

El modo <a href="https://p5js.org/reference/#/p5/WEBGL" target="_blank">WEBGL</a> 
de p5.js incluye 7 formas primitivas. Estas formas son plano, caja, cilindro, cono, toro, esfera y elipsoide. 
Además, <a href="https://p5js.org/reference/#/p5/model" target="_blank">model()</a> muestra una geometría 
personalizada cargada a través de <a href="https://p5js.org/reference/#/p5/loadModel" target="_blank">loadModel()</a>. 
Este ejemplo incluye cada una de las formas primitivas. También incluye un modelo de la <a href="https://nasa3d.arc.nasa.gov/models" 
target="_blank"> colección de la NASA</a>.
