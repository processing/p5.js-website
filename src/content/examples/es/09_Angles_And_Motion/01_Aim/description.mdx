La función <a href="https://p5js.org/reference/p5/atan2" target="_blank">atan2()</a> 
calcula el ángulo entre dos posiciones. El ángulo que calcula atan2() puede ser utilizado 
para rotar una forma hacia algo. En este ejemplo, los ojos rotan y apuntan para mirar al cursor.
