Este ejemplo demuestra una pelota rebotando en una superficie inclinada,
implementada usando cálculos vectoriales para la reflexión.

El código hace uso extensivo de la
<a href="https://p5js.org/reference/#/p5.Vector">clase p5.Vector</a>,
incluyendo la función
<a href="https://p5js.org/reference/#/p5/createVector">createVector()</a> para crear nuevos vectores,
y los métodos vectoriales
<a href="https://p5js.org/reference/#/p5.Vector/add">add()</a>
y
<a href="https://p5js.org/reference/#/p5.Vector/dot">dot()</a>
para los cálculos vectoriales.
