Demostración de un sistema de partículas de humo, basado en el
<a href="https://natureofcode.com/book/chapter-4-particle-systems/">ejemplo</a>
original de Dan Shiffman para Processing.

El código hace uso de la
<a href="https://p5js.org/reference/p5.Vector">clase p5.Vector</a>,
incluyendo la función
<a href="https://p5js.org/reference/p5/createVector">createVector()</a>.
Los diversos cálculos para actualizar las posiciones y velocidades de las partículas
se realizan con métodos de p5.Vector.

El sistema de partículas se implementa como una
<a href="https://p5js.org/reference/p5/class">clase</a>, que contiene un array de
objetos (de la clase Particle).
