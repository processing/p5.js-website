La funci√≥n
<a href="https://p5js.org/reference/#/p5/rotate">rotate()</a>
rota el sistema de coordenadas actual respecto del origen actual.

Nota que por defecto el origen se ubica en la esquina superior izquierda del lienzo.
Para poder rotar alrededor del centro del lienzo, tenemos primero que trasladar
el sistema de coordenadas y luego rotar respecto del nuevo origen.

Las funciones
<a href="https://p5js.org/reference/#/p5/push">push()</a>
y
<a href="https://p5js.org/reference/#/p5/pop">pop()</a>
salval y restauran el sistema de coordenadas, respectivamente.
