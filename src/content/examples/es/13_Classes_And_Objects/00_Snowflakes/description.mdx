Este ejemplo demuestra el uso de un sistema de partículas
para simular el movimiento de copos de nieve que caen. Este programa define una
clase de copo de nieve
<a href="https://p5js.org/reference/#/p5/class">class</a>
y utiliza un array para almacenar los objetos de copo de nieve.
