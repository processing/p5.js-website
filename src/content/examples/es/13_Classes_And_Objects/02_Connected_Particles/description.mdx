---

Este ejemplo utiliza dos
<a href="https://p5js.org/reference/p5/class" target="_blank">clases</a> personalizadas.
La clase Particle almacena una posición, velocidad y tono. Renderiza
un círculo usando la posición y el tono actuales, y actualiza la
posición usando la velocidad actual. La clase Path almacena una matriz
de objetos creados a partir de la clase Particle. Renderiza líneas
conectando cada una de las partículas. Cuando el usuario hace clic en el ratón, el
boceto crea una nueva instancia de la clase Path. Cuando el usuario arrastra
el ratón, el boceto añade una nueva instancia de la clase Particle al
trayecto actual.
