---

La función
<a href="https://p5js.org/reference/#/p5/createGraphics" target="_blank">createGraphics()</a>
puede ser usada para crear un nuevo objeto p5.Graphics, que puede servir como
un búfer de gráficos fuera de pantalla dentro del lienzo. Los búferes fuera de pantalla pueden tener
diferentes dimensiones y propiedades que su superficie de visualización actual, incluso
aunque parezcan existir en el mismo espacio.
