Haz clic y arrastra el ratón para dibujar una línea.

Este ejemplo demuestra el uso de varias variables 
integradas.
<a href="https://p5js.org/reference/#/p5/mouseX">mouseX</a>
y
<a href="https://p5js.org/reference/#/p5/mouseY">mouseY</a>
almacenan la posición actual del ratón, mientras que la
posición anterior del ratón está representada por
<a href="https://p5js.org/reference/#/p5/pmouseX">pmouseX</a>
y
<a href="https://p5js.org/reference/#/p5/pmouseY">pmouseY</a>.

Este ejemplo también muestra el uso de
<a href="https://p5js.org/reference/#/p5/colorMode">colorMode</a> con HSB
(tono-saturación-brillo), de modo que la primera variable establece el tono.
