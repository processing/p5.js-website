Este ejemplo demuestra el uso de
<a href="https://p5js.org/reference/#/p5/loop">loop()</a>
y
<a href="https://p5js.org/reference/#/p5/noLoop">noLoop()</a>
para pausar y reanudar la animación.

Al hacer clic en el ratón, se alterna el bucle de animación. Si el bucle de
animación está detenida, el usuario puede presionar cualquier tecla para avanzar un fotograma.
Nota: el usuario debe hacer clic para establecer el enfoque en el lienzo para que
las pulsaciones de teclas se registren.

Avanzar un solo fotograma se logra llamando a la función
<a href="https://p5js.org/reference/#/p5/redraw">redraw()</a>,
lo que resulta en una sola llamada a la función draw().
