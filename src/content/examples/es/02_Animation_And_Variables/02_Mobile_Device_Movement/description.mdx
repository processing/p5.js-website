La función <a href="https://p5js.org/reference/p5/deviceMoved" target="_blank">deviceMoved()</a>
se ejecuta cuando el dispositivo móvil que muestra el sketch se mueve.
En este ejemplo, los valores de
<a href="https://p5js.org/reference/p5/accelerationX" target="_blank">accelerationX</a>,
<a href="https://p5js.org/reference/p5/accelerationY" target="_blank">accelerationY</a> 
y <a href="https://p5js.org/reference/p5/accelerationZ" target="_blank">accelerationZ</a>
configuran la posición y el tamaño de un círculo.
Esto solo funciona para dispositivos móviles.
