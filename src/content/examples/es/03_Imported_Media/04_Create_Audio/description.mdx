title: Reproductor de Audio
oneLineDescription: Crea un reproductor para un archivo de audio.
---

<a href="https://p5js.org/reference/#/p5/createAudio" target="_blank">createAudio()</a>
crea un reproductor de audio. Este ejemplo muestra los controles del reproductor
y ajusta su velocidad. La velocidad de reproducción es normal cuando el ratón
está en el borde izquierdo de la ventana, y se vuelve más rápida a medida que el ratón
se mueve hacia la derecha. Más información sobre el uso de elementos multimedia como
reproductores de audio se encuentra en la página de referencia de
<a href="https://p5js.org/reference/#/p5.MediaElement" target="_blank">p5.MediaElement</a>.
El archivo de audio es un
<a href="https://freesound.org/people/josefpres/sounds/711156/" target="_blank">bucle de piano de dominio público de Josef Pres</a>.
