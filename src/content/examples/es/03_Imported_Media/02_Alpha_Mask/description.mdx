title: Máscara Alfa
oneLineDescription: Usa una imagen para recortar una sección de otra imagen.
---

Usando el método
<a href="https://p5js.org/reference/p5/mask" target="_blank">mask()</a>,
puedes crear una máscara para una imagen para especificar la transparencia en
diferentes partes de la imagen. Para ejecutar este ejemplo localmente, necesitarás dos
archivos de imagen y un servidor local en funcionamiento
<a href="https://github.com/processing/p5.js/wiki/Local-server">local server</a>.
