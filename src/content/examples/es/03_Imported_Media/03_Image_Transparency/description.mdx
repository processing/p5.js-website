title: Transparencia de Imagen
oneLineDescription: Hacer una imagen transparente en el lienzo.
---

Este programa superpone una imagen sobre otra modificando el
valor alfa de la imagen con la función
<a href="https://p5js.org/reference/#/p5/tint" target="_blank">tint()</a>.
Mueve el cursor hacia la izquierda y hacia la derecha a través del lienzo para cambiar
la posición de la imagen. Para ejecutar este ejemplo
localmente, necesitarás un archivo de imagen y un servidor local en funcionamiento
<a href="https://github.com/processing/p5.js/wiki/Local-server">local server</a>.
