La función <a href="https://p5js.org/reference/p5/text" target="_blank">text()</a> se utiliza para insertar texto en el lienzo.
Puedes cambiar la fuente y el tamaño del texto usando las funciones <a href="https://p5js.org/reference/p5/loadFont" target="_blank">loadFont()</a>
y <a href="https://p5js.org/reference/p5/fontSize" target="_blank">fontSize()</a>.
El texto puede alinearse a la izquierda, al centro o a la derecha con la función <a href="https://p5js.org/reference/p5/textAlign" target="_blank">textAlign()</a>,
y, al igual que las formas, el texto puede tener color con <a href="https://p5js.org/reference/p5/fill" target="_blank">fill()</a>.
