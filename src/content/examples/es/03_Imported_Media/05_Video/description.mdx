title: Reproductor de Video
oneLineDescription: Crea un reproductor para un archivo de video.
---

Usando las funciones <a href="https://p5js.org/reference/p5/noCanvas" target="_blank">noCanvas()</a>
y <a href="https://p5js.org/reference/p5/createVideo" target="_blank">createVideo()</a>,
puedes cargar un video dentro del 
<a href="https://developer.mozilla.org/en-US/docs/Glossary/DOM" target="_blank">DOM</a>
sin incrustar el video dentro de un lienzo. Para construir un video incrustado dentro del elemento canvas,
visita el ejemplo 
<a href="https://p5js.org/examples/imported-media-video-canvas/" target="_blank">Video Canvas</a>.
