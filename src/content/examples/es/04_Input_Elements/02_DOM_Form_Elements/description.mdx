El <a href="https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model" target="_blank">Document Object Model</a>,
o DOM, representa la estructura resultante de una página web. Usar los elementos de forma de p5.js's,
tales como <a href="https://p5js.org/reference/p5/createInput" target="_blank">createInput()</a>,
<a href="https://p5js.org/reference/p5/createSelect" target="_blank">createSelect()</a>
y <a href="https://p5js.org/reference/p5/createRadio" target="_blank">createRadio()</a>,
te permiten crear distintas maneras de acceder a la información ingresada a través de un botón
<a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select" target="_blank">select</a>,
<a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input" target="_blank">input</a>,
o <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio" target="_blank">radio</a>
y actualizar el DOM en base a la información obtenida.
