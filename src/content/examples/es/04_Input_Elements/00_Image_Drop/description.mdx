<a href="https://p5js.org/reference/p5.Element/drop" target="_blank">drop()</a>
es el método de un elemento en p5.js que registra una callback cada vez que se
carga un archivo en el elemento. El archivo cargado se crea dentro de una clase
<a href="https://p5js.org/reference/p5.File" target="_blank">p5.File</a>.
Puedes usar la callback de
<a href="https://p5js.org/reference/p5.Element/drop" target="_blank">drop()</a>
para revisar el tipo de archivo y usarlo en declaraciones condicionales
que respondan al mismo.
