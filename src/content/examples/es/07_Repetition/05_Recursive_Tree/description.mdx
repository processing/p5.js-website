Este es un ejemplo de cómo renderizar una estructura similar a un árbol mediante recursión.
 El ángulo de ramificación se calcula como una función de la posición horizontal del mouse. 
 Mueve el mouse hacia la izquierda y hacia la derecha para cambiar el ángulo. Basado en el ejemplo
 <a href="https://processing.org/examples/tree.html">Recursive Tree Example</a> de Daniel Shiffman para Processing.
