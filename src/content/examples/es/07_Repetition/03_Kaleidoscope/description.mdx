Un caleidoscopio es un instrumento óptico con dos o más superficies reflectantes inclinadas entre sí a un ángulo. 
Utilizando las funciones <a href="https://p5js.org/reference/p5/translate" target="_blank">translate()</a>,
<a href="https://p5js.org/reference/p5/rotate" target="_blank">rotate()</a>, y 
<a href="https://p5js.org/reference/p5/scale" target="_blank">scale()</a>, puedes replicar 
el efecto visual resultante de un caleidoscopio en un lienzo.
