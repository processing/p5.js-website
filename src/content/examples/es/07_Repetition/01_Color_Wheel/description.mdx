Este ejemplo ilustra la apariencia de diferentes tonos. Utiliza un <a href="https://p5js.org/reference/#/p5/for" target="_blank">bucle for </a> para repetir transformaciones. 
El bucle inicializa una variable llamada ángulo, que cambia la rotación de un círculo así como su tono. 
Cada vez que se repite el bucle, se dibuja un círculo relativo al centro del lienzo. 
Las funciones <a href="https://p5js.org/reference/#/p5/push" target="_blank">push()</a> y <a href="https://p5js.org/reference/#/p5/pop" target="_blank">pop()</a> hacen que estas 
transformaciones afecten solo al círculo individual.
